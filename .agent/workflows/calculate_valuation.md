# バリュエーション算出ワークフロー

起業大全に基づき、バリュエーションを決定する6要素を自己評価し、企業価値向上施策を策定する。

> [!IMPORTANT]
> **起業大全の教え**: バリュエーションは「交渉」で決まるが、根拠のある評価が交渉力を高める

## 入力・出力

| 項目 | 内容 |
|------|------|
| **入力** | `finance_plan.md`, `flywheel.md`, 事業データ |
| **出力** | `documents/3_planning/valuation.md` |
| **実施タイミング** | 資金調達前、年次レビュー |
