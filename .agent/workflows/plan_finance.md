# ファイナンス・資金調達計画ワークフロー

起業大全に基づき、フェーズ別の資金戦略・持分政策・ランウェイ管理を策定する。

> [!IMPORTANT]
> **起業大全の教え**: 最初の資本政策で失敗しない（持分比率の死守）
> Seedでは18ヶ月分のランウェイ確保が推奨

## 入力・出力

| 項目 | 内容 |
|------|------|
| **入力** | 現在の財務状況、`lean_canvas.md` |
| **出力** | `documents/3_planning/finance_plan.md` |
| **実施タイミング** | 資金調達前、四半期ごとのレビュー |
