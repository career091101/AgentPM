# 5つの眼ワークフロー

起業参謀の教科書に基づき、5つの視点からビジネスを多角的に分析する。

## 前提条件
- `lean_canvas.md` が作成済み
- 事業の方向性検討または定期レビュー時に実施

## ステップ

```markdown
|---------|---------|-------------|

| **E**nvironmental（環境） | [トレンド] | [影響] |
| **P**olitical（政治・法規制） | [トレンド] | [影響] |

### メガトレンド
1. [トレンド1]: [事業への影響]
2. [トレンド2]: [事業への影響]

[この視点から見えた機会・脅威]
```
