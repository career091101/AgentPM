# カスタマーサクセス設計ワークフロー

起業大全のDEARモデルに基づき、サブスクリプション時代のカスタマーサクセス戦略を設計する。

> [!IMPORTANT]
> **起業大全の教え**: 「売って終わり」ではなく「売ってからが始まり」
> サブスクリプションモデルではLTVを最大化することが成功の鍵

## 入力・出力

| 項目 | 内容 |
|------|------|
| **入力** | `persona.md`, `lean_canvas.md`, 顧客データ |
| **出力** | `documents/4_executing/customer_success.md` |
| **実施タイミング** | PMF達成後、定期的なCS戦略レビュー時 |
