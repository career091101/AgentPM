# SNSコンテンツ作成ワークフロー

MVP公開後のユーザー獲得に向けて、SNS投稿コンテンツを作成する。

## 前提条件
- `deploy_report.md` （公開URL確定済み）
- `persona.md` が作成済み
- `lean_canvas.md` （UVP定義済み）

## ステップ

### 1. SNS戦略の確認

```markdown
## SNS戦略

### ターゲットプラットフォーム
| プラットフォーム | 優先度 | 理由 |
|----------------|:------:|------|
| X (Twitter) | ⭐⭐⭐ | IT/ビジネス層のリーチ |
| LinkedIn | ⭐⭐ | BtoBの場合有効 |
| Facebook | ⭐ | コミュニティ形成向け |

### 投稿目標
- [ ] 認知獲得（インプレッション）
- [ ] エンゲージメント獲得（いいね・リツイート）
- [ ] LP流入（クリック）
- [ ] コンバージョン（事前登録）

### KPI設定
- 目標インプレッション: [数値]/投稿
- 目標エンゲージメント率: [X]%
- 目標クリック率: [X]%
```

### 2. コンテンツカテゴリ設計

```markdown
## コンテンツカテゴリ

### カテゴリ1: 課題提起（共感獲得）
ペルソナの課題を言語化し、共感を得る

**テンプレート**:
```
[課題の状況描写]

こんな経験ありませんか？

・[課題ポイント1]
・[課題ポイント2]
・[課題ポイント3]

実は[ターゲット]の[X]%が同じ悩みを抱えています。

#[関連ハッシュタグ]
```

### カテゴリ2: ソリューション紹介
サービスの価値を伝える

**テンプレート**:
```
[ベネフィットの一言表現]

[サービス名]を使うと：
✅ [ベネフィット1]
✅ [ベネフィット2]
✅ [ベネフィット3]

詳しくはこちら👇
[LP URL]

#[関連ハッシュタグ]
```

### カテゴリ3: Tips/ノウハウ提供
価値提供による信頼獲得

**テンプレート**:
```
【[テーマ]のコツ】

[ターゲット]が知らないと損する[X]つのポイント：

1️⃣ [ポイント1]
2️⃣ [ポイント2]
3️⃣ [ポイント3]

保存して後で見返してね💡

#[関連ハッシュタグ]
```

### カテゴリ4: リリース告知
サービスの正式告知

**テンプレート**:
```
🎉 [サービス名] リリースしました！

[1行でUVPを説明]

✨ 特徴
・[特徴1]
・[特徴2]
・[特徴3]

今なら[特典/限定オファー]

▼ 詳細・登録はこちら
[LP URL]

#[関連ハッシュタグ]
```
```


