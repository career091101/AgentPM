# 事業計画書生成ワークフロー

AIエージェントがこれまでの成果物を統合し、投資家・ステークホルダー向けの事業計画書を自動生成する。

## 入力・出力

| 項目 | 内容 |
|------|------|
| **入力** | 全成果物（business_idea.md, lean_canvas.md, persona.md, など） |
| **出力** | `documents/6_closing/business_plan.md` |
| **前提** | Phase1完了、できればPMF達成後 |

---

## STEP 1: 成果物収集【自動実行】

### 使用ツール
`view_file`

// turbo
以下のファイルを順次読み込み：

```
必須:
- business_idea.md
- lean_canvas.md
- persona.md
- cpf_diagnosis.md
- psf_diagnosis.md
- deploy_report.md（公開URL）

オプション（存在する場合）:
- pmf_diagnosis.md
- mvv.md
- flywheel.md
- scorecard.md
- engagement_report.md
```
