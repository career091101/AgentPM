# スタートアップ・スコアカードワークフロー

起業大全のバランス・スコアカードに基づき、経営の健康状態を4つの視点から可視化する。

## 前提条件
- PMF達成済みまたは達成間近
- `flywheel.md` が作成済み（推奨）
- 定期的な経営レビュー時に実施

## ステップ
