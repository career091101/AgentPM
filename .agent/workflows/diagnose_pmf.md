# PMF達成判定ワークフロー

プロダクトが市場に適合しているか（PMF）を多角的に判定する。

## 前提条件
- `psf_diagnosis.md` でPSF達成済み
- MVPが公開済みで実ユーザーが存在
- `startup_science_knowledge.md` を参照可能

## ステップ
