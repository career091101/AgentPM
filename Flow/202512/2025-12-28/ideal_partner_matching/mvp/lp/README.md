**MVP類型**: Landing Page型
**目的**: メール登録数の獲得、PMFテストの準備
**バージョン**: 1.0

## LP概要

**タイトル**: 「真剣な出会いを、6ヶ月で。」

**サブタイトル**: ideal_partner_matching - AI婚活コーチ×真剣度マッチングで本物のつながりを

**ターゲット**: 30-40代の真剣交際希望者

**ゴール**: 事前登録メールアドレス1,000件獲得（3ヶ月以内）

## ページ構成（7セクション）

### 1. ヒーローセクション

**見出し（H1）**:
```
「また遊び目的か...」にうんざりしていませんか？

真剣な出会いを、6ヶ月で実現。
ideal_partner_matching
```

**サブ見出し**:
```
AI婚活コーチと真剣度スコアリングで、
無駄なやりとりを排除し、本物のパートナーに出会う。
```

**CTA（Call To Action）**:
```
[無料で事前登録する]
（ボタン - オレンジ色、目立つデザイン）
```

**背景画像**:
- 幸せそうなカップルの写真（30-40代、ナチュラル）

---

### 2. 問題提起セクション

**見出し（H2）**:
```
こんな婚活の悩み、ありませんか？
```

**3つの課題（アイコン付き）**:

1. ❌ **真剣度の低い人との無駄なやりとりに疲弊**
   「マッチングしても、すぐゴーストされる...」

2. ❌ **条件にこだわりすぎて、誰も選べない**
   「年収、身長、学歴...完璧な人なんていない」

3. ❌ **自分の気持ちをどう伝えていいか分からない**
   「真剣交際に進んでも、3組に1組は破局」

---

### 3. ソリューションセクション

**見出し（H2）**:
```
ideal_partner_matchingなら、解決できます
```

**3つの差別化要素（アイコン + 説明）**:

1. ✅ **真剣度スコアリング**
   - 行動パターンからAIが真剣度を自動評価
   - 真剣度80%以上の人同士を優先マッチング
   - 「無駄な時間を20分の1に削減」

2. ✅ **AI婚活コーチ24時間常駐**
   - 「この人とデートすべき？」迷ったらAIに相談
   - Red flag自動検出で、失敗を防ぐ
   - 「気持ちの伝え方」をロールプレイで練習

3. ✅ **価値観マッチング**
   - 条件ではなく、結婚観・家族観で相性分析
   - 「減点法→加点法」でポジティブな出会いを
   - 「6ヶ月以内に真剣交際移行率30%」

---

### 4. How It Worksセクション

**見出し（H2）**:
```
使い方はシンプル、3ステップ
```

**3ステップ（図解付き）**:

**STEP 1**: 1分で登録
- アプリをダウンロード
- プロフィール作成（AIガイド付き）

**STEP 2**: AIが最適な相手を紹介
- 価値観診断（50問）
- 真剣度スコアリング
- 相性の高い人を自動マッチング

**STEP 3**: AI婚活コーチと一緒に進める
- メッセージのコツをAIがアドバイス
- デート前後の相談も24時間対応
- 真剣交際モードで1対1に集中

---

### 5. 料金プランセクション

**見出し（H2）**:
```
シンプルな料金プラン
```

| プラン | 月額料金 | 主な機能 |
|--------|:--------:|---------|
| **ベーシック** | ¥3,980/月 | 真剣度マッチング、価値観診断、月5回AI相談 |
| **プレミアム** | ¥7,980/月（推奨） | ベーシック + AI無制限相談、優先マッチング、詳細分析 |

**比較（小さく表示）**:
- マッチングアプリ: ¥2,000-3,000/月（真剣度低い）
- 結婚相談所: ¥30-50万/年（高額）
- ideal_partner_matching: ¥3,980-7,980/月（**ちょうど良い*）

### 6. 社会的証明セクション

**見出し（H2）**:
```
なぜideal_partner_matchingなのか？
```

**3つの強み**:

1. **データに裏付けられた信頼**
   - 「真剣交際移行率30%（業界平均の3倍）」
   - 「平均6ヶ月で真剣交際に移行」

2. **最新AI技術の活用**
   - 「GPT-4ベースのAI婚活コーチ」
   - 「真剣度スコアリング精度90%以上（Year 3目標）」

3. **創業チームの専門性**
   - データサイエンティスト + 心理学者 + 婚活業界経験者
   - AI × 心理学 × 婚活の3領域を深く理解

**（将来的に追加）成婚カップルの声**:
- 「6ヶ月で理想のパートナーに出会えました！」（佐藤さん、34歳）
- 「AIコーチのおかげで、自信を持ってアプローチできた」（田中さん、36歳）

## 技術スタック（MVP版）

### フロントエンド

- **HTML5**: セマンティックタグ使用
- **CSS3**: Flexbox/Gridでレスポンシブデザイン
- **Vanilla JavaScript**: メール登録フォームのバリデーション

### ホスティング

- **Netlify** or **Vercel**: 無料プラン、CI/CDで自動デプロイ

### メール管理

- **Google Forms**: メールアドレス収集（最初はシンプルに）
- （将来的に）**Mailchimp**: ニュースレター配信、セグメント管理

### アナリティクス

- **Google Analytics 4**: 訪問数、CVR測定
- **Hotjar**: ヒートマップ、ユーザー行動分析

## KPI設定

| 指標 | 目標値（3ヶ月） | 測定方法 |
|------|:---------------:|---------|
| **訪問数** | 10,000人 | Google Analytics |
| **メール登録数** | 1,000件 | Google Forms |
| **CVR（コンバージョン率）** | 10% | 登録数 ÷ 訪問数 |
| **平均滞在時間** | 2分以上 | Google Analytics |
| **直帰率** | 50%以下 | Google Analytics |
