# MVV（Mission / Vision / Values）

**作成日**: 2025-12-28
**プロジェクト**: ideal_partner_matching（理想のパートナーマッチング）
**バージョン**: 1.0

---

## Mission（存在意義）

**真剣な出会いを求める全ての人に、本物のつながりと成長の機会を提供する。**

私たちは、「いい人に出会えない」と嘆く人々が、実は「真剣度の高い人と効率的に出会える仕組み」と「自己成長しながらパートナーシップを育むスキル」の両方を必要としていることに気づきました。

ideal_partner_matchingは、単なるマッチングプラットフォームではなく、ユーザーが**本物の愛を見つけ、持続可能な関係を築く**ための総合的なパートナーシップ・エコシステムです。

**Why we exist（なぜ私たちが存在するのか）:**
- カジュアルな出会いと高額な結婚相談所の間に大きなギャップが存在する
- デートが「chore（雑用）」になってしまっている現代の婚活市場を変革する
- 真剣度の低い人との無駄なやりとりに疲弊している人々を救う
- 「条件」ではなく「価値観」でつながる本物の関係を創出する

---

## Vision（目指す未来 - 3-5年後）

**2030年までに、日本の真剣交際市場におけるNo.1プラットフォームになり、10万組のカップルに本物のつながりを届ける。**

### 定量目標（2030年時点）

| 指標 | 目標 |
|------|:----:|
| アクティブユーザー数 | 50万人 |
| 累計成婚カップル数 | 10万組 |
| 月間マッチング成立数 | 10万件 |
| ユーザー満足度（NPS） | 60以上 |
| 真剣交際移行率 | 30%（業界平均の3倍） |
| 平均成婚期間 | 6ヶ月（業界平均の50%短縮） |

### 定性的ビジョン

1. **業界のスタンダードになる**
   - 「真剣交際ならideal_partner_matching」が常識になる
   - 既存マッチングアプリが「真剣度フィルター」を真似する存在になる

2. **ユーザー中心の文化を創る**
   - 「減点法」から「加点法」へのマインドシフトを促進
   - 「条件」より「価値観」を重視する新しい婚活文化を醸成

3. **社会課題を解決する**
   - 日本の未婚率上昇・出生率低下の一因となっている「良質な出会い不足」を解消
   - 「結婚したいけどできない」人々に希望を提供

4. **技術革新をリードする**
   - AI婚活コーチが当たり前になる未来を作る
   - パーソナライズされた婚活サポートを民主化（誰でもアクセス可能に）

## Values（行動指針）

私たちが大切にする5つの価値観と、日々の行動基準：

### 1. Authenticity（本物であること）

**「本物のつながりは、本物の自分から始まる」**

- ユーザーに対して真摯であり、誇張した宣伝や誤解を招く表現をしない
- データを正直に開示し、透明性を保つ（成婚率、真剣交際率など）
- プロフィールの虚偽記載を厳格に取り締まり、本人確認を徹底する
- AIアルゴリズムの仕組みをブラックボックスにせず、可能な限り説明する

**行動例:**
- 「月間10万マッチング成立！」だけでなく「そのうち真剣交際に進んだのは3万件」まで開示
- ユーザーにAIが「なぜこの人をおすすめしたか」を説明する機能を提供

### 2. Empowerment（成長を支援すること）

**「私たちはマッチングだけでなく、ユーザーの成長を支援する」**

- ユーザーが自己認識を深め、パートナーシップスキルを向上できるツールを提供
- AI婚活コーチは「答えを与える」のではなく「自分で気づく」ことを促す
- 失敗（交際終了）も学びの機会と捉え、振り返り支援を提供
- 「条件にこだわりすぎ」などの気づきを、批判ではなく成長機会として提示

**行動例:**
- 交際終了時に「次に活かせる学び」を一緒に振り返るリフレクション機能
- 「減点法→加点法」トレーニングのゲーミフィケーション

### 3. Respect（尊重すること）

**「すべてのユーザーは尊重されるべき存在である」**

- 性別、年齢、価値観、過去の経験に関わらず、すべてのユーザーを平等に尊重
- ゴーストング、breadcrumbingなど、相手を尊重しない行動に対して警告・ペナルティ
- 「条件に合わない」からといって粗末に扱われない仕組み（例: 真剣度スコアで全員が価値ある存在）
- ユーザーのプライバシーを最大限保護し、データを悪用しない

**行動例:**
- 返信がない相手への催促メッセージを制限（ストーカー行為防止）
- 退会後もデータを速やかに削除し、プライバシーを守る

### 4. Innovation（革新を追求すること）

**「婚活の当たり前を疑い、より良い未来を創造する」**

- 「スワイプ型」「いいね数至上主義」など、既存の常識を疑う姿勢
- AI/機械学習の最新技術を積極的に取り入れる
- ユーザーフィードバックを迅速にプロダクトに反映（週次アップデート）
- 失敗を恐れず、小さく試して学ぶ文化（MVP思考）

**行動例:**
- 月1回の新機能リリース（A/Bテストで効果測定）
- ユーザーインタビューを月20件実施し、課題を発掘

### 5. Commitment（コミットメントを促進すること）

**「真剣な関係は、真剣なコミットメントから始まる」**

- ユーザーが「この人と向き合う」決断をしやすい仕組みを提供
- 真剣交際モード（1対1集中モード）など、コミットメントを後押しする機能
- スタッフ自身が「長期的視点」を持ち、短期的利益に流されない
- ユーザーの成婚後も関係を継続（コミュニティ、イベント、カップル向けコンテンツ）

**行動例:**
- 真剣交際モードに入ると他のマッチングが一時停止され、1人に集中できる
- 成婚カップル向けコミュニティで、結婚後も関係を継続

## MVV整合性検証

### Mission → Vision → Values の流れ

✅ **Mission（真剣な出会いと成長機会の提供）**
  ↓
✅ **Vision（10万組のカップルに本物のつながりを届ける）**
  ↓
✅ **Values（Authenticity, Empowerment, Respect, Innovation, Commitment）**

**整合性: 3/3 ✅**

- Mission が「なぜ存在するか」を明確に定義
- Vision が Mission の具体的な実現形を示している
- Values が Vision 達成のための行動指針になっている

## ベンチマーク企業MVV比較

| 企業 | Mission/Slogan | 差別化ポイント |
|------|---------------|--------------|
| **Hinge** | "Designed to be deleted" | 長続きする関係を見つける支援 |
| **Bumble** | "Designed to be deleted" | 女性主導、エンパワーメント |
| **Match.com** | 真剣な関係を促進 | 互換性ベースのマッチング |
| **ideal_partner_matching** | **真剣な出会いと成長機会を提供** | **真剣度スコアリング + AI婚活コーチの統合** |

**差別化要素:**
1. Hinge/Bumbleは「削除されることが目標」だが、当サービスは「削除後も成長を支援」（成婚カップル向けコミュニティ）
2. 既存サービスは「マッチング」のみだが、当サービスは「マッチング + スキル向上 + AI常駐コーチ」の3本柱
3. 「真剣度」を可視化・フィルタリングする点で独自性
