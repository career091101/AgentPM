# プロジェクト憲章

**プロジェクト名**: Merriman Financial Astrology Analysis

---

## 1. 背景と目的

### 背景

金融市場において、価格変動のタイミングを予測することは投資家にとって重要な課題である。メリマン（Raymond Merriman）が体系化した金融占星術は、天体サイクルと市場サイクルの相関を分析することで、相場の転換点を事前に把握するアプローチを提供する。

これまでの検討を正式なプロジェクトとして体系化し、継続的な分析・検証基盤を構築する必要性が認識された。

### 目的

メリマンの金融占星術を用いて:
- 相場の**転換点**を事前に把握する
- **警戒期間**を明確にする
- **優位な時間帯**を特定する

これにより、実際のトレード判断に耐える**「時間ベースの分析フレーム」**を構築する。

### 重要ポイント

| 焦点 | 説明 |
|------|------|
| **予測対象** | 価格予測ではなく時間（タイミング） |
| **意思決定支援** | いつ注意すべきか／いつ無理をしないか／いつ仕掛けを考えるか |
| **方法論** | 感覚論ではなく検証・記録・振り返りが可能な形 |

## 2. スコープ

### 含む（In Scope）

- メリマン金融占星術理論の学習・体系化
- 天体サイクル（惑星逆行、アスペクト等）のカレンダー化
- 過去データとの相関分析・検証
- 分析記録・振り返りシステムの構築
- AI/自動化導入の基盤設計

### 含まない（Out of Scope）

- 価格予測モデルの構築
- 自動売買システムの開発（将来フェーズ）
- 他の占星術手法（メリマン以外）
