# CEO層インタビューガイド - SNSプロジェクト

**作成日**: 2025-12-27
**プロジェクト**: SNS生産性向上プロジェクト
**調査フェーズ**: 詳細市場調査（オプションB）
**対象**: CEO/代表取締役（従業員50-500名）
**インタビュー時間**: 60分
**ステータス**: Draft

---

## 1. インタビュー概要

### 1.1 調査目的

1. **ペルソナの検証**: 「田中雄一郎CEO」の実在性と行動パターンの確認
2. **ニーズの深掘り**: AI情報収集の課題と理想的な解決策の特定
3. **価格感度の把握**: 有料情報サービスへの支払意思額の把握
4. **購買決定要因**: 何があれば「購読したい」と思うかの特定
5. **競合評価**: 既存インフルエンサーの利用状況と満足度

### 1.2 対象者プロフィール

**必須条件:**
- 役職: CEO/代表取締役
- 従業員規模: 50-500名
- 業種: IT/SaaS/コンサル/製造業のいずれか
- AI関心度: 高（SNSでAI投稿にエンゲージメントしている）

**推奨条件:**
- LinkedInアクティブユーザー
- 過去6ヶ月以内にAI関連の投稿にいいね/コメント
- 東京圏在住（リモート可）

## 2. インタビュー構成（60分）

### フェーズ1: アイスブレイク（5分）

**目的**: リラックスした雰囲気作り、信頼関係の構築

**スクリプト:**
```
本日はお忙しい中、お時間をいただきありがとうございます。
私は〇〇と申しまして、現在AI業界の情報発信サービスの企画を
進めております。

経営者の方々がAI情報をどのように収集され、どのような課題を
お持ちかをお伺いし、より良いサービス設計に活かしたいと考えて
おります。

インタビュー内容は匿名化して利用させていただき、個別の発言が
外部に公開されることはございません。ご安心ください。

それでは、まず簡単に自己紹介をお願いできますでしょうか？
```

**質問:**
- お名前と会社名、事業内容を簡単に教えてください
- 従業員数と創業年を教えてください
- AI/生成AIとの関わりを教えてください（利用状況、導入検討等）

---

### フェーズ2: 現状の情報収集行動（15分）

**目的**: 日常的な情報収集の実態を把握

#### Q1. 日常的な情報収集について

**質問:**
```
まず、経営者としての日常の情報収集についてお伺いします。

1-1. 普段、ビジネスや業界の情報をどのように収集されていますか？
     （媒体、頻度、時間帯など具体的に）

1-2. その中で、AI/生成AIに関する情報はどのように収集されていますか？
```

**深掘りポイント:**
- [ ] 利用しているSNS（LinkedIn、Facebook、X等）
- [ ] 利用時間帯（朝通勤時、昼休み、夜等）
- [ ] 1日あたりの情報収集時間
- [ ] 保存機能の利用状況（「後で読む」を実際に読むか？）

#### Q2. AI情報収集の課題

**質問:**
```
2-1. AI/生成AI情報を収集する上で、現在どのような課題を
     感じていらっしゃいますか？

2-2. 具体的に、どのような場面で「困った」「面倒だ」と
     感じたことがありますか？
```

**深掘りポイント:**
- [ ] 情報過多の問題（多すぎて選べない）
- [ ] 信頼性の問題（何を信じればいいかわからない）
- [ ] 時間不足の問題（読む時間がない）
- [ ] 実務への応用の問題（自社で使えるかわからない）

**ペルソナ検証チェック:**
- [ ] 「情報が多すぎて、何が重要かわからない」 → ペルソナの Pain Point と一致？
- [ ] 「時間がなくて、後で読むつもりが読めていない」 → ペルソナの行動パターンと一致？

#### Q3. 理想的な情報の受け取り方

**質問:**
```
3-1. もし理想的なAI情報サービスがあるとしたら、
     どのような形で情報を受け取りたいですか？

3-2. どのくらいの頻度で、どのくらいのボリュームが適切ですか？
```

**深掘りポイント:**
- [ ] フォーマット（長文、箇条書き、動画、音声等）
- [ ] 頻度（毎日、週1回、隔週等）
- [ ] 長さ（5分で読める、10分で読める等）
- [ ] 配信方法（メール、SNS、専用アプリ等）

### フェーズ3: 既存サービスの利用状況（15分）

**目的**: 競合インフルエンサーの評価と満足度

#### Q4. 現在フォローしているAI系インフルエンサー

**質問:**
```
4-1. 現在、AI/生成AI分野で定期的に情報をチェックしている
     インフルエンサーや情報源はありますか？

4-2. その中で、特に参考にしている方は誰ですか？
     それはなぜですか？
```

**リストを見せながら:**
```
こちらのリストの中で、ご存知の方、フォローされている方は
いらっしゃいますか？

- 木内翔大（SHIFT AI）
- 深津貴之（THE GUILD / note）
- チャエン（デジライズ）
- 池田朋弘（ChatGPT著者）
- AI速報
- その他（　　　　　　　）
```

**深掘りポイント:**
- [ ] フォローしている理由（速報性、わかりやすさ、専門性等）
- [ ] 満足している点
- [ ] 不満な点（ここが改善されたらいいのに）

#### Q5. 有料情報サービスの利用経験

**質問:**
```
5-1. AI/ビジネス情報で、有料サービス（note有料記事、
     オンラインサロン、ニュースレター等）を利用されたことは
     ありますか？

5-2. （利用経験がある場合）どのサービスを、いくらで
     利用されましたか？満足度はいかがでしたか？

5-3. （利用経験がない場合）有料サービスを利用しない理由は
     何ですか？
```

**深掘りポイント:**
- [ ] 過去の支払い経験（月額、単発）
- [ ] 支払った金額のレンジ
- [ ] 継続/解約の理由
- [ ] 無料と有料の使い分け基準

### フェーズ4: 価格感度と購買意向（15分）

**目的**: 支払意思額と購買決定要因の特定

#### Q6. 価格感度テスト（Van Westendorp法）

**質問:**
```
ここからは、仮にAI情報の有料サービスがあった場合の
お考えをお聞かせください。

想定サービス:
- CEO層向けに特化したAI/生成AI情報
- 週1回配信、5分で読める要約形式
- 投資判断・経営判断に役立つ視点を提供
- 著名人の発言、市場データ、事例を含む

6-1. このサービスに対して、「安すぎて品質が心配」と感じる
     最低価格はいくらですか？

6-2. 「お得だと感じる」価格はいくらですか？

6-3. 「少し高いが、払ってもいい」と思う価格はいくらですか？

6-4. 「高すぎて絶対に払わない」価格はいくらですか？
```

**価格帯の選択肢を提示:**
- [ ] 月額500円
- [ ] 月額1,000円
- [ ] 月額2,000円
- [ ] 月額3,000円
- [ ] 月額5,000円
- [ ] 月額10,000円
- [ ] その他（　　　円）

#### Q7. 購買決定要因

**質問:**
```
7-1. このサービスを「購読したい」と思うには、
     どのような要素が必要ですか？（複数回答可）

□ 情報の速報性（誰よりも早く知れる）
□ 情報の独自性（ここでしか得られない視点）
□ 情報の信頼性（データに基づいている）
□ 時間効率（5分で重要情報を把握できる）
□ 実務への応用（自社で使える具体例）
□ 経営判断への貢献（投資判断の材料になる）
□ ネットワーキング（他のCEOとの交流）
□ その他（　　　　　　　　　　　　）

7-2. 上記の中で、最も重視するのはどれですか？（1つ）
```

#### Q8. 競合との比較

**質問:**

## 3. インタビュー実施の注意点

### 3.1 事前準備

**必須準備物:**
- [ ] Zoom URLの準備（録画設定ON）
- [ ] 同意書の準備（録音・録画の許可）
- [ ] 謝礼の準備（Amazonギフト券コード）
- [ ] インタビューガイドのプリントアウト
- [ ] メモ用ノート/PC

### 3.2 インタビュー中の心得

**DO（すべきこと）:**
- [ ] 相槌を適度に打ち、傾聴の姿勢を示す
- [ ] 「なぜ？」を3回深掘りする（Why Why Why分析）
- [ ] 具体例を求める（「例えば、最近そういうことありましたか？」）
- [ ] 沈黙を恐れない（考える時間を与える）
- [ ] 感謝の気持ちを言葉に出す

**DON'T（避けること）:**
- [ ] 誘導質問をしない（「〇〇ですよね？」）
- [ ] 自分の意見を押し付けない
- [ ] 話を遮らない
- [ ] ジャッジしない（「それは間違っています」等）
- [ ] 専門用語を使いすぎない

### 3.3 記録方法

**録画・録音:**
- Zoom録画（クラウド保存）
- 許可を必ず取る
- バックアップとして音声レコーダーも併用

**メモ:**
- キーワードを箇条書き
- 感情の動き（声のトーン、表情）も記録
- 印象的な発言は「 」で引用

## 4. 分析フレームワーク

### 4.1 インタビュー直後（5分）

**すぐにメモすること:**
- [ ] 第一印象（この人はペルソナに近いか？）
- [ ] 最も印象的だった発言
- [ ] 予想外の発見
- [ ] 追加で聞きたかったこと

### 4.2 インタビュー後の分析（30分）

**実施項目:**
1. **文字起こし**: 録画から重要部分を文字起こし
2. **テーマ抽出**: KJ法で共通テーマをグルーピング
3. **ペルソナとの照合**: 仮説との一致/不一致を確認
4. **インサイトの抽出**: 「なるほど!」と思った発見を3つ

### 4.3 全インタビュー終了後の統合分析

**分析観点:**
- [ ] 共通の課題（全員が言及した課題）
- [ ] 価格感度のレンジ（中央値、最頻値）
- [ ] 購買決定要因のランキング（TOP 3）
- [ ] ペルソナの修正点（仮説との差分）
