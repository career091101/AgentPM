**期待成果:**
- ペルソナの検証・更新
- 価格感度の把握（月額1,000円〜5,000円の範囲で最適価格帯特定）

### 2.3 調査③: 市場規模の詳細推定

**目的**: 「CEO向けAI情報発信市場」のTAM/SAM/SOMを算出

#### A. TAM（Total Addressable Market）推定

**アプローチ1: トップダウン**
```
国内生成AI市場（2024年）: 1,016億円
└─ 法人向け導入ソリューション: 503億円
   └─ コンサル・教育・情報サービス: 推定200億円（40%）
      └─ 情報発信・メディア: 推定50億円（25%）
         └─ CEO層向けAI情報発信: 推定10億円（20%）

TAM = 10億円/年
```

**アプローチ2: ボトムアップ**
```
国内のCEO数（従業員50-500名企業）: 約10万社
└─ AI関心層: 30%（3万社）
   └─ 有料情報購読意向層: 10%（3,000社）
      └─ 平均年間支出: 10万円（月8,300円×12ヶ月）

TAM = 3,000社 × 10万円 = 3億円/年
```

**採用TAM: 3-10億円/年**（保守的に3億円で試算）

#### B. SAM（Serviceable Addressable Market）推定

```
TAM: 3億円/年
└─ オンライン情報発信でリーチ可能: 70%（SNS・Web利用層）
   SAM = 3億円 × 70% = 2.1億円/年
```

#### C. SOM（Serviceable Obtainable Market）推定

```
SAM: 2.1億円/年
└─ 競合存在下でのシェア獲得可能性: 5-10%
   SOM = 2.1億円 × 5-10% = 1,050万〜2,100万円/年

月間換算: 87.5万〜175万円/月
```

**調査方法:**
1. **統計データ収集**:
   - 帝国データバンク: 企業規模別CEO数
   - 総務省統計: IT投資額・教育研修費
   - IDC Japan: AI市場レポート

2. **アンケート調査**:
   - LinkedIn投票機能で簡易調査（n=100-200）
   - 「AI情報に月いくら払うか？」の価格感度調査

3. **ベンチマーク**:
   - note有料記事市場全体の規模
   - ビジネス系オンラインサロンの市場規模
   - 海外AI情報発信市場（Substack等）

**期待成果:**
- TAM/SAM/SOMの定量化
- 市場成長率の推定（年20-30%成長と仮定）
- 自社の到達可能市場規模: 月100-200万円

## 6. リスクと対策

### 6.1 リクルーティングリスク

**リスク**: CEO層のインタビュー協力者が集まらない

**対策**:
1. 謝礼額を3万円→5万円に引き上げ
2. インタビュー時間を60分→30分に短縮
3. セカンダリペルソナ（企画担当者）もターゲットに含める
4. ビザスク以外のプラットフォーム（ココナラ、YOUTRUST）も活用

### 6.2 競合データ入手リスク

**リスク**: 競合の収益データが入手できない

**対策**:
1. 公開情報の徹底的な収集（決算資料、プレスリリース、インタビュー記事）
2. 業界平均値での推定（note平均単価、企業研修平均単価）
3. 類似事例からの類推（海外インフルエンサーの収益モデル）

### 6.3 時間超過リスク

**リスク**: 5週間で完了しない

**対策**:
1. Week 3終了時点で進捗確認、遅延項目の優先度調整
2. 必須項目（CEO層インタビュー5名、TAM算出）に集中
3. Nice-to-have項目（競合インタビュー）は後回し
