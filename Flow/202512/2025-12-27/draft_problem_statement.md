## 3. 課題の定義

### 3.1 表層的な課題（What）

> 「月間インプレッション目標の34.7%しか達成できていない」

### 3.2 根本的な課題（Why）

#### 🎯 課題A: 投稿の「再現性」が低い

**現象:**
- トップ10%の投稿が全体の40%のimpを稼いでいる
- 残り90%の投稿は平均以下のパフォーマンス
- 「当たった理由」が明確でなく、次回に活かせていない

**根本原因:**
- 投稿作成時に「勝ちパターン」が型化されていない
- トピック選定が属人的で、データドリブンでない
- 成功投稿の分析はあるが、テンプレート化されていない

**影響:**
- 投稿ごとのパフォーマンスのバラつきが大きい
- 平均impを安定的に向上させられない
- 投稿作成に時間がかかる（試行錯誤）

---

#### 🎯 課題B: プラットフォーム戦略の「最適化不足」

**現象:**
- Xが全体の4.4%しか貢献していない
- Facebook/LinkedInでも投稿内容は同じ
- 各プラットフォームの特性を活かせていない

**根本原因:**
- 「ワンソース・マルチユース」を意識しすぎている
- 各SNSのアルゴリズム特性を理解していない
- プラットフォーム別の最適化が後回しになっている

**影響:**
- Xの潜在impが活かされていない
- 各SNSの最大パフォーマンスを引き出せていない
- 同一投稿をコピペする運用になっている

---

#### 🎯 課題C: ターゲット層への「刺さり方」が不明確

**現象:**
- フォロワー32,056人中、CEO層は4.3%（約1,400人）しかいない
- 投稿内容がCEO層に特化されていない
- 広く一般層を狙い、結果的に誰にも深く刺さっていない

**根本原因:**
- ペルソナ（田中CEO）は定義されているが、投稿との紐付けが弱い
- CEO層の「Jobs to be Done」を満たす投稿設計ができていない
- ターゲット以外の層（エンジニア、一般層）も取り込もうとして焦点がぼやけている

**影響:**
- CEO層のエンゲージメント率が低い
- フォロワーの質と量のバランスが悪い
- 「誰に向けて書いているか」が不明確な投稿が多い

---

#### 🎯 課題D: 投稿頻度と品質の「トレードオフ」

**現象:**
- Facebook: 77投稿/3ヶ月 = 約26投稿/月
- 競合（木内翔大）は1日3-5投稿 = 約90-150投稿/月
- 投稿頻度が競合の**1/4〜1/6**しかない

**根本原因:**
- 1投稿あたりの制作時間が長い（特に動画投稿）
- 「質を下げたくない」という意識が強すぎる
- 投稿テンプレートがないため、毎回ゼロから作成している

**影響:**
- 投稿頻度が低く、アルゴリズムに評価されにくい
- 目標達成には頻度増加が必須だが、現状では限界
- 時間不足で投稿が後回しになる

## 4. なぜこの課題を解決する必要があるのか？

### 4.1 ビジネス上の理由

| 観点 | 理由 |
|------|------|
| **影響力の拡大** | 月間100万impは「業界インフルエンサー」としての信頼性の閾値 |
| **マネタイズの前提** | noteサブスク、企業研修、コンサルへの導線には一定規模のフォロワーが必要 |
| **競合優位性** | AI業界は競争が激化しており、今ポジションを確立しないと埋もれる |
| **時間的制約** | 2025年はAI市場が急成長中（年84%成長）、今が勝負の時期 |

### 4.2 個人的な理由

- **キャリア戦略**: SNS影響力を持つことで、リクルート以外の選択肢を持つ
- **情報発信の使命感**: AI業界の最新情報を日本のCEO層に届けたい
- **スキル向上**: コンテンツ制作、マーケティング、データ分析のスキルを実践で磨く

## 5. 解決すべき本質的な課題（Core Problem）

### 🎯 コアプロブレム定義

> 「CEO層に深く刺さる投稿を、再現性高く、高頻度で作り続ける仕組みがない」

### 課題の構造

```
【表層】月間100万impに届かない（34.7%）
    ↓
【中層】投稿の再現性が低い × 頻度が低い × プラットフォーム最適化不足
    ↓
【深層】CEO層に特化した「勝ちパターン」の型化と運用システムがない
```

## 6. この課題を解決できた状態（Success Criteria）

### 6.1 定量的成功基準

| 指標 | 現状 | 3ヶ月後目標 | 6ヶ月後目標 |
|------|------|-----------|-----------|
| **月間インプレッション** | 346,766 | 600,000 | 1,000,000 |
| **CEO層フォロワー比率** | 4.3% | 7% | 10% |
| **投稿頻度（月）** | 26投稿 | 40投稿 | 50投稿 |
| **1投稿あたり制作時間** | 30分〜1時間 | 20分 | 15分 |
| **エンゲージメント率** | 1.2% | 2.5% | 3.5% |

### 6.2 定性的成功基準

- [ ] 投稿作成時に「どのテンプレートを使うか」を即座に判断できる
- [ ] 週次で「勝ちパターン」の効果を検証し、改善できている
- [ ] CEO層から「いつも参考にしています」という反応が増える
- [ ] Xでもバイラル投稿が月1回以上発生する
- [ ] noteサブスクへの導線が明確になり、登録者が増え始める

## 7. 課題解決のアプローチ

### 7.1 解決の方向性

| 課題 | 解決アプローチ |
|------|--------------|
| **A. 再現性の低さ** | 投稿テンプレートの型化、トピック選定ルールの明確化 |
| **B. プラットフォーム最適化不足** | SNS別戦略の策定、Xへのリソース再配分 |
| **C. ターゲット層への刺さり不足** | CEO層特化型コンテンツへの絞り込み、JTBDベースの投稿設計 |
| **D. 頻度と品質のトレードオフ** | AIツール導入、動画投稿の一時停止、テンプレート化 |

### 7.2 優先順位

| 優先度 | アプローチ | 期待効果 | 実行難易度 |
|--------|-----------|---------|-----------|
| **1** | 投稿テンプレート型化 | +30% imp | 低 |
| **2** | ロボット/半導体トピック強化 | +20% imp | 低 |
| **3** | 動画投稿停止→写真特化 | +15% imp | 低 |
| **4** | X戦略見直し（バイラル狙い） | +50% imp (X単体) | 中 |
| **5** | AIツール導入（Taplio等） | 制作時間半減 | 中 |

## 8. 制約条件

### 8.1 リソース制約

| 項目 | 制約 |
|------|------|
| **時間** | 週5〜7時間がSNS投稿に割ける上限 |
| **予算** | AIツールへの投資は月5,000円程度まで |
| **人員** | 1人（本人のみ） |

### 8.2 外部制約

| 項目 | 制約 |
|------|------|
| **アルゴリズム** | 各SNSのアルゴリズム変更に左右される |
| **競合** | AI業界のインフルエンサーが増加中 |
| **市場** | AIブームの持続性が不透明 |

## 9. リスクと対策

### 9.1 主要リスク

| リスク | 発生確率 | 影響度 | 対策 |
|--------|---------|--------|------|
| **施策を実行しても効果が出ない** | 中 | 高 | 2週間ごとに効果検証、ピボット前提 |
| **時間が確保できず投稿頻度が落ちる** | 高 | 中 | テンプレート化で時短、最低週3投稿を死守 |
| **CEO層以外のフォロワーが増える** | 中 | 中 | 投稿内容を定期的にCEO向けに修正 |
| **プラットフォームのアルゴリズム変更** | 低 | 高 | 複数SNSに分散、依存度を下げる |
