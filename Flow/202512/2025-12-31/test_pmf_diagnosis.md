### 2. 月次成長率

**選択指標**: MAU（月間アクティブユーザー）

**データ**:

| 月 | 実績（MAU） | 前月比 | 成長率 |
|----|------------|-------|--------|
| 2025-10 | 500人 | - | - |
| 2025-11 | 560人 | +60 | +12% |
| 2025-12 | 627人 | +67 | +12% |

**3ヶ月移動平均**: 12%/月

**トレンド**: 横ばい（安定成長）

**判定**: ✅ 合格

**分析**:
- 成長ドライバー: オーガニック流入（検索流入40%増）、既存ユーザーからの紹介（25%がリファラル経由）
- ボトルネック: 新規登録後の初回利用率が68%に留まる（目標80%）
- 改善提案: オンボーディングプロセスの最適化（チュートリアル短縮、クイックスタートガイド導入）

**計算式検証**:
```
10月→11月: (560 - 500) / 500 × 100 = 12%
11月→12月: (627 - 560) / 560 × 100 = 11.96% ≈ 12%
平均成長率: (12 + 12) / 2 = 12%
```

### 3. Churn Rate

**解約定義**: 30日間非アクティブ（ログインなし）

**Cohort分析**:

| 登録月 | 初月残存 | 2ヶ月残存 | 3ヶ月残存 | 月次Churn Rate |
|-------|---------|-----------|-----------|---------------|
| 2025-10 | 100人 | 97人 | 94人 | 3%/月 |
| 2025-11 | 120人 | 116人 | - | 3.3%/月 |
| 2025-12 | 150人 | 146人 | - | 2.7%/月 |

**3ヶ月平均Churn Rate**: 3%/月（目標: 5%以下）

**判定**: ✅ 合格

**解約理由分析**:
1. 機能理解不足（初期オンボーディング未完了）: 42%
2. 競合製品への移行: 28%
3. 利用シーン減少（ニーズ変化）: 20%
4. その他（価格、技術的問題等）: 10%

**改善提案**:
- 初回利用時のガイダンス強化（インタラクティブチュートリアル導入）
- 定期的なエンゲージメント施策（週次メール、プッシュ通知最適化）
- 非アクティブユーザーへの早期リーチアウト（14日非利用でアラート）

**計算式検証**:
```
10月Cohort: (100 - 97) / 100 × 100 = 3%/月（初月）
11月Cohort: (120 - 116) / 120 × 100 = 3.3%/月（初月）
12月Cohort: (150 - 146) / 150 × 100 = 2.7%/月（初月）
平均: (3 + 3.3 + 2.7) / 3 = 3%/月
```

### 4. NPS（Net Promoter Score）

**アンケート実施概要**:
- 配信日: 2025-12-20（Sean Ellisテストと同時配信）
- 有効回答数: 40人

**結果**:

| 分類 | スコア範囲 | 人数 | 割合 |
|-----|----------|------|------|
| Promoter（推奨者） | 9-10点 | 26人 | 65% |
| Passive（中立者） | 7-8点 | 10人 | 25% |
| Detractor（批判者） | 0-6点 | 4人 | 10% |

**NPS計算**:
```
NPS = Promoter% - Detractor% = 65% - 10% = 55
```

**判定**: ✅ 合格
