# Noteクリエイティブジャンル収益化事例 - 詳細分析レポート

**作成日**: 2026年1月8日
**対象事例**: 4名のクリエイター（クロネコ屋、ナナシ、toki、根田啓史）
**調査目的**: Note収益化の実態、戦略、再現性評価

---

## 1. クロネコ屋 - 年1億円超（ノウハウ販売）

### 1.1 プロフィール

| 項目 | 詳細 |
|-----|------|
| **属性** | 経営者・元作家・ブロガー |
| **開始時期** | 2014年（ブログ×Twitter起業）<br>2015年1月（アフィリエイト本格化） |
| **現在の規模** | - Twitterフォロワー: 非公開（推定10万人超）<br>- メルマガ読者: 3万人<br>- 有料note販売数: 1万部以上 |
| **実績** | - 最高年収: 1億200万円<br>- 年収5,000万円超: 7年連続<br>- 資産: 4億円（米国株運用3億円） |

### 1.2 収益モデル

#### 主要収入源（複合型）

1. **有料note販売**
   - 販売実績: 1万部以上
   - 主力商品: 「月1万円を稼ぐ有料noteの売り方×作り方講座」
   - 価格帯: 数千円～1万円
   - 推定売上: 数千万円規模

2. **Brain販売**
   - 最新作: 「1日3時間労働で稼ぐ！X運用＋有料noteマネタイズ講座」
   - 価格: 39,800円（発売記念29,800円）
   - 3日間売上: 700万円超
   - 2025年1月発売時点で好調

3. **ブログアフィリエイト**
   - 開始1年目: 月収250万円達成
   - 安定収益: 月数百万円規模

4. **コンサルティング・メンバーシップ**
   - メルマガ読者3万人への有料コンテンツ配信
   - 個別コンサル（推定）

#### 年間収益構造

```
総売上: 1億円超
├─ 有料note/Brain: 3,000万～5,000万円
├─ ブログアフィリエイト: 3,000万円
├─ コンサルティング: 1,000万～2,000万円
└─ その他（投資収益等）: 1,000万円

実質年収: 約1,200万円（法人化後の役員報酬）
```

### 1.3 価格設定

| 商品タイプ | 価格帯 | 販売実績 |
|-----------|--------|---------|
| note単品教材 | 3,000円～10,000円 | 1万部以上 |
| Brain教材 | 29,800円～39,800円 | 発売3日で700万円 |
| ブログ記事（無料） | 0円 | 集客用 |
| メルマガ | 一部有料 | 読者3万人 |

### 1.4 成功戦略

#### 差別化ポイント

1. **実績の透明性**
   - 年収1億200万円という具体的数字を公開
   - 失敗談も含めた全プロセス開示

2. **完全1人運営**
   - 外注なし
   - 再現性の高さを強調

3. **マルチプラットフォーム戦略**
   - note、Brain、メルマガ、ブログを連携
   - noteのBAN対策として複数プラットフォーム活用

4. **段階的価格設定**
   - 初心者向け: 数千円のnote
   - 中級者向け: 1万円前後のnote
   - 本気組向け: 3万円超のBrain教材

#### 集客方法

1. **SNS戦略**
   - Twitter（現X）での情報発信
   - ターゲット: 副業希望者、起業志望者
   - 投稿内容: 実績報告、ノウハウティザー、成功事例

2. **SEO対策**
   - ブログ「クロネコ屋の超ブログ術」での検索流入
   - キーワード: 「note 稼ぎ方」「アフィリエイト 初心者」等

3. **信頼構築**
   - メルマガでの継続的価値提供（3万人）
   - 購入者レビューの積極的公開

4. **影響力拡大**
   - Brain公式メディアでのインタビュー掲載
   - 他クリエイターとの対談・コラボ

### 1.5 成長過程

| 時期 | 段階 | 実績 |
|-----|------|------|
| **2014年** | 起業初期 | ブログ×Twitter起業、元作家から転身 |
| **2015年1月** | アフィリエイト開始 | ブログ・アフィリエイト本格化 |
| **2016年** | 急成長期 | 開始1年で月収250万円達成 |
| **2017年～** | 安定期 | 年収5,000万円超を7年連続キープ |
| **最盛期** | 頂点 | 年収1億200万円達成 |
| **2023年頃** | 多角化 | noteで1万部以上販売、複数プラットフォーム展開 |
| **2025年1月** | 最新 | Brain教材発売3日で700万円売上 |

**収益化までの期間**:
- 初収入（月数万円）: 6ヶ月以内
- 月収100万円超: 1年

- 年収1億円超: 5～6年

### 1.6 他プラットフォーム連携

| プラットフォーム | 活用目的 | 規模 |
|---------------|---------|------|
| **Twitter（X）** | 集客・ブランディング | フォロワー推定10万人超 |
| **ブログ** | SEO集客・信頼構築 | 月数万PV |
| **note** | 教材販売（過去メイン） | 1万部以上販売（現在一部非公開） |
| **Brain** | 高単価教材販売（現メイン） | 新作3日で700万円 |
| **メルマガ** | 読者育成・販売 | 読者3万人 |
| **投資（米国株）** | 資産運用 | 運用資産3億円 |

### 1.7 収益規模と戦略の再現性

#### 収益規模評価

```
年収レベル: ★★★★★（1億円超）
安定性: ★★★★☆（複数収入源で安定）
成長性: ★★★★☆（プラットフォーム多角化で継続成長）
```

#### 再現性評価

| 項目 | 評価 | 理由 |
|-----|------|------|
| **初心者への再現性** | ★★☆☆☆（低） | - 年収1億円は極めて困難<br>- 7年以上の継続的努力が必要<br>- マーケティングスキル必須 |
| **中級者への再現性** | ★★★☆☆（中） | - 月収10万～30万円は再現可能<br>- note販売の基本戦略は有効<br>- SNS運用との連携が鍵 |
| **成功に必要な期間** | 3～5年 | 年収100万円: 1年<br>月収30万円: 2～3年<br>年収1,000万円超: 5年以上 |
| **必須スキル** | 高度 | - ライティング<br>- マーケティング<br>- SNS運用<br>- 継続力 |

#### 初心者への推奨度

**推奨度: ★★☆☆☆（2/5）**

**理由**:
- ✅ メリット: 完全1人運営、初期投資ゼロ、場所を選ばない
- ❌ デメリット: 高度なスキル必須、収益化まで長期間、競合多数
- ⚠️ 注意点: クロネコ屋レベルは例外的成功例、月収10万円を目標とすべき

**推奨ターゲット**:
- すでに副業で月収10万円超を達成している中級者
- マーケティング・ライティングの実務経験者
- 3～5年の長期視点で取り組める人

## 2. ナナシ - 月収5万円（イラスト・素材販売）

### 2.1 プロフィール

| 項目 | 詳細 |
|-----|------|
| **属性** | イラストレーター・副業クリエイター |
| **開始時期** | 2024年頃（推定） |
| **現在の規模** | - フォロワー数: 非公開<br>- 有料note記事: 複数公開<br>- 月収: 5万円（2025年4月時点） |
| **作業時間** | - 週4～5日<br>- 1日2～3時間（副業として） |

### 2.2 収益モデル

#### 主要収入源

1. **有料noteでイラスト集販売**
   - 単発イラスト集: 500円～1,500円
   - 大規模素材集: 1,500円～3,000円
   - 月収内訳: 有料note販売で月5万円

2. **月額有料マガジン**
   - 価格: 500円～1,000円
   - 更新頻度: 月3回程度
   - コンテンツ: イラスト差分2枚程度
   - 目標: 45人で月収2万円（手数料引き後）

3. **ノウハウ販売**
   - 「イラスト販売で月5万円稼ぐ!有料note活用の裏ワザ公開」
   - 「月5万円稼ぐnote有料マガジンの作り方と設定手順」
   - 「イラストで月10万円稼ぐ!nanashi リコメンドを活用したマネタイズ術」

#### 月収5万円の内訳（推定）

```
月収合計: 5万円
├─ 有料note単品販売: 2万～3万円
│   └─ 単価500～3,000円 × 10～30部/月
├─ 月額マガジン: 1万～2万円
│   └─ 500～1,000円 × 20～40人
└─ ノウハウnote販売: 数千円
    └─ 1,000～2,000円 × 数部/月
```

### 2.3 価格設定

| 商品タイプ | 価格帯 | 販売戦略 |
|-----------|--------|---------|
| **単発イラスト集（小規模）** | 500円～1,500円 | 初回購入のハードルを下げる |
| **大規模素材集・ブラシセット** | 1,500円～3,000円 | 高価値コンテンツとして訴求 |
| **月額有料マガジン** | 500円～1,000円 | 継続課金で安定収益 |
| **ノウハウnote** | 1,000円～2,000円 | 実体験に基づく付加価値 |

### 2.4 成功戦略

#### 差別化ポイント

1. **「一般的な絵師でも稼げる」という再現性の強調**
   - プロレベルでなくても月5万円は可能
   - 副業として現実的な作業量（週10～15時間）

2. **ノウハウの透明性**
   - 自身の実体験をnoteで公開
   - 具体的な数字（月収5万円、作業時間等）を開示

3. **低負担の月額マガジン運営**
   - 月3回更新、差分2枚程度
   - 継続可能なペース設定

4. **複数収入源の組み合わせ**
   - 単発販売 + 月額課金 + ノウハウ販売
   - リスク分散と収益安定化

#### 集客方法

1. **note内SEO対策**
   - タイトル: 「月5万円」「副業」「裏ワザ」等のキーワード
   - ハッシュタグ活用: #イラスト #副業 #note収益化

2. **無料記事での信頼構築**
   - 一部ノウハウを無料公開
   - 有料note購入への導線設計

3. **リコメンド機能の活用**
   - note公式リコメンド掲載を狙う
   - 他クリエイターの相互紹介

4. **SNS連携（推定）**
   - Twitter/Instagramでのイラスト投稿
   - note記事への誘導

### 2.5 成長過程

| 時期 | 段階 | 実績 |
|-----|------|------|
| **開始期（0～3ヶ月）** | アカウント育成 | - 無料記事投稿<br>- フォロワー獲得<br>- 初収入: 数千円 |
| **成長期（3～6ヶ月）** | 有料化開始 | - 初有料note公開<br>- 月収1万円達成<br>- 月額マガジン開始 |
| **安定期（6～12ヶ月）** | 収益安定化 | - 月収3万円到達<br>- 複数商品展開<br>- ノウハウnote販売開始 |
| **現在（12ヶ月以上）** | 月収5万円達成 | - 安定的に月収5万円<br>- ノウハウ販売も好調<br>- 次の目標: 月収10万円 |

**収益化までの期間**:
- 初収入（数千円）: 3ヶ月以内
- 月収1万円: 3～6ヶ月
- 月収5万円: 6～12ヶ月
- 次目標（月収10万円）: 1～2年

### 2.6 他プラットフォーム連携

| プラットフォーム | 活用目的 | 推定規模 |
|---------------|---------|---------|
| **note** | メイン販売プラットフォーム | 有料記事複数 |
| **Twitter（X）** | 集客・作品投稿 | 推定数千～1万フォロワー |
| **Instagram** | ポートフォリオ・集客 | 推定数千フォロワー |
| **Pixiv（可能性）** | 作品投稿・知名度向上 | 非公開 |

### 2.7 収益規模と戦略の再現性

#### 収益規模評価

```
月収レベル: ★★☆☆☆（5万円）
安定性: ★★★☆☆（副業として安定、複数収入源）
成長性: ★★★☆☆（月収10万円まで成長可能）
```

#### 再現性評価

| 項目 | 評価 | 理由 |
|-----|------|------|
| **初心者への再現性** | ★★★★☆（高） | - イラストスキルがあれば実現可能<br>- 作業時間が現実的（週10～15時間）<br>- 具体的なノウハウを公開中 |
| **中級者への再現性** | ★★★★★（非常に高） | - すでにイラストポートフォリオがあれば即開始可能<br>- 月収5万円→10万円への拡大も視野 |
| **成功に必要な期間** | 6～12ヶ月 | 月収1万円: 3～6ヶ月<br>月収5万円: 6～12ヶ月<br>月収10万円: 1～2年 |
| **必須スキル** | 中程度 | - イラスト制作（中級レベル）<br>- 基本的なSNS運用<br>- note記事作成 |

#### 初心者への推奨度

**推奨度: ★★★★☆（4/5）**

**理由**:
- ✅ メリット:
  - 現実的な月収目標（5万円）
  - 副業として継続可能な作業量
  - 初期投資ほぼゼロ
  - 複数収入源で安定化
- ⚠️ 注意点:
  - イラスト制作スキルが必須
  - 収益化まで6ヶ月程度の継続が必要
  - 月収10万円以上は競争激化

**推奨ターゲット**:
- イラスト制作経験者（趣味レベル可）
- 副業で月5万円を目指す会社員
- 在宅ワークを希望する主婦・主夫
- 長期的にイラストレーターとして活動したい人

## 3. toki - 約10万円（写真・旅行）

### 3.1 プロフィール

| 項目 | 詳細 |
|-----|------|
| **属性** | 旅行写真愛好家・副業フォトグラファー |
| **開始時期** | 2021年11月（本格化） |
| **現在の規模** | - フォロワー数: 非公開<br>- 累計収益: 約10万円（2025年12月時点）<br>- 投稿写真数: 推定数百枚 |
| **作業時間** | 趣味レベル（旅行の延長線上） |
| **使用プラットフォーム** | PIXTA、Adobe Stock、note |

### 3.2 収益モデル

#### 主要収入源

1. **ストックフォト販売**
   - **PIXTA**: 主力プラットフォーム
   - **Adobe Stock**: サブプラットフォーム
   - 累計収益: 約10万円（2025年12月時点）
   - 手数料: 一般的に22～58%（プラットフォームにより異なる）

2. **note販売（サブ）**
   - 写真販売手数料: 5%（クレジット決済）
   - 他プラットフォームより低手数料が魅力

#### 累計収益の推移

```
累計収益: 約10万円（4年間）
├─ 2021年: 4,316.5ポイント（初年度、手探り）
├─ 2022年: 推定5,000～6,000ポイント
├─ 2023年: 推定7,000～8,000ポイント
├─ 2024年: 11,594.45ポイント（安定期）
└─ 2025年（12月28日時点）: 16,394.24ポイント
```

**年平均収益**: 約2.5万円/年
**月平均収益**: 約2,000円/月（趣味レベルの作業量）

### 3.3 価格設定

ストックフォトの価格設定（tokiの場合は受動的）：

| 写真タイプ | 販売価格（買い手側） | クリエイター取り分 |
|-----------|-------------------|-----------------|
| **小サイズ（Web用）** | 550円～1,100円 | 150円～500円 |
| **中サイズ（印刷用）** | 1,650円～3,300円 | 500円～1,500円 |
| **大サイズ（高解像度）** | 3,850円～5,500円 | 1,500円～2,500円 |
| **定額制ダウンロード** | 買い手は月額制 | 1DL = 25～100円 |

**note販売の場合**:
- 写真単品: 100円～500円
- 写真集: 500円～3,000円
- 手数料: 5%（クレジット決済）

### 3.4 成功戦略

#### 差別化ポイント

1. **既存資産の活用**
   - 趣味で撮影した旅行写真を再利用
   - 追加投資ゼロでスタート

2. **AI編集の活用（2021年11月～）**
   - **ゲームチェンジャー**: 写真編集ソフト導入で収益が急上昇
   - AI機能活用:
     - 空をAIで変更（1枚から複数素材作成）
     - AIが最適プリセットを自動提案
     - 「撮ったまま」→「編集済み高品質」に変化
   - ダウンロード数と単価が明確に向上

3. **継続的な資産化**
   - 一度投稿した写真が継続的に売れる
   - 「不労所得」的な収益構造
   - 2024年以降、安定的に推移

4. **低リスク・低ストレス**
   - 趣味の延長線上で収益化
   - ノルマなし、自分のペース
   - 失敗しても金銭的損失なし

#### 集客方法（ストックフォトの場合）

1. **プラットフォーム内SEO**
   - タグ付け: 「旅行」「風景」「日本」「観光地」等
   - タイトル最適化: 検索されやすいキーワード
   - 説明文: 具体的な撮影場所・シーン説明

2. **定期的な新規投稿**
   - 毎月50～100枚の追加が理想
   - tokiの場合: 趣味レベルで不定期投稿

3. **季節性の活用**
   - 春: 桜、新緑
   - 夏: 海、花火、夏祭り
   - 秋: 紅葉、収穫
   - 冬: 雪景色、イルミネーション

4. **複数プラットフォーム展開**
   - PIXTA + Adobe Stock + note
   - リーチの最大化

### 3.5 成長過程

| 時期 | 段階 | 実績 |
|-----|------|------|
| **2021年（開始期）** | 手探り投稿 | - 撮ったまま投稿<br>- 年間4,316ポイント<br>- 写真編集ソフト導入（11月） |

## 6. 結論・提言

### 6.1 Note収益化の現実

1. **月収5万円は現実的な目標**
   - ナナシの事例が示すとおり、6～12ヶ月で達成可能
   - イラスト・写真などのクリエイティブジャンルで実現性高い

2. **月収10万円以上は競争激化**
   - 全noteクリエイターの0.2%未満
   - 高度なマーケティング・スキルが必須
   - 3～5年の継続的努力が必要

3. **年収1,000万円超は例外的成功例**
   - クロネコ屋・根田啓史は極めて稀な例
   - 商業実績・SNS影響力・マーケティングスキルが必須
   - 初心者が目指すべき目標ではない

### 6.2 初心者への提言
