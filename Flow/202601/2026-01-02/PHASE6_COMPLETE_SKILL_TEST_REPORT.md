## テストビジネスアイデア

**名称**: ContentGenie（AIコンテンツ自動生成SaaS）

### 基本情報
- **ターゲット**: ソロプレナー、インディーハッカー、Micro-SaaS創業者（推定10万人）
- **課題**: X/Twitter、LinkedIn、ブログ記事作成に週10-15時間（年間520-780時間）
- **ソリューション**: AIで1週間分のコンテンツ（X投稿14本、LinkedIn記事2本、ブログ1本）を10分で一括生成
- **収益モデル**: サブスクリプション
  - Starter: $9/月（X投稿のみ、月間50本）
  - Pro: $29/月（X + LinkedIn、月間無制限）
  - Premium: $99/月（全プラットフォーム + カスタムブランディング + API連携）
- **技術スタック**: Claude API、Next.js 14、Vercel、Stripe、Supabase
- **開発期間**: 6-8週間（1人開発、ShipFast活用で3週間短縮可能）
- **初期コスト**: $500（ドメイン + Vercel Pro + Claude API credits）
- **運用コスト**: $200/月
