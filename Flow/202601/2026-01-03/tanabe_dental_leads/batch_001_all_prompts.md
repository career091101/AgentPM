## 医院1: 源内歯科クリニック

```
以下の歯科医院のWebサイトを分析し、必要情報を抽出してください。

**医院名**: 源内歯科クリニック
**WebサイトURL**: https://www.gennai-dental.com/

**抽出項目**:
1. SNS連携（Instagram, Facebook, LINE, Twitter）: boolean
2. ブログ更新日（最新記事の日付）: YYYY-MM-DD形式またはnull
3. 子ども向けコンテンツ（キッズページ等）: boolean
4. 待合室・設備の写真: boolean
5. 営業時間: テキスト形式（例: "月-金 9:00-18:00"）
6. **医院長名**: フルネーム（姓名）またはnull

**医院長名抽出方法**:
1. トップページから「院長」「医院長」「理事長」「代表」などのキーワードを探す
2. 見つからない場合、以下のリンクを探索（最大3ページ）:
   - 「院長紹介」「ドクター紹介」「スタッフ紹介」「医師紹介」「院長あいさつ」

3. 各ページで肩書きと共に記載されている名前を抽出
4. フルネーム（姓名）で出力（例: "田中太郎"）

**出力形式** (JSON):
```json
{
  "clinic_name": "源内歯科クリニック",
  "website_url": "https://www.gennai-dental.com/",
  "sns_instagram": true,
  "sns_facebook": false,
  "sns_line": true,
  "sns_twitter": false,
  "blog_updated": "2025-12-20",
  "kids_content": true,
  "waiting_room_photo": true,
  "operating_hours": "月-金 9:00-18:00, 土 9:00-13:00",
  "director_name": "源内太郎"
}
```

**制約事項**:
- タイムアウト: 5分
- 医院長名が見つからない場合は null
- リンク先探索は最大3ページまで
- JavaScriptアラート・モーダルダイアログを発生させる操作は禁止
```

**結果保存先**: `analysis_001_源内歯科クリニック.json`
