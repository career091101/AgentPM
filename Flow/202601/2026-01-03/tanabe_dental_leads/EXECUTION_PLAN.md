## 実行手順（各バッチ）

### STEP 1: Google Maps API実行

```bash
cd /Users/yuichi/AIPM/aipm_v0/Flow/202601/2026-01-03/tanabe_dental_leads
python extract_batch_50.py --batch [バッチ番号]
```

**出力**: `batch_[バッチ番号]_raw_data_[タイムスタンプ].json`

### STEP 2: Webサイト分析（手動プロンプト実行）

バッチごとに生成される`batch_[バッチ番号]_all_prompts.md`を参照し、以下を実行:

1. WebサイトありのリストをTaskツールで並列分析（5-10件ずつ、Haikuモデル）
2. 各医院のJSON出力（`analysis_XXX_[医院名].json`）

### STEP 3: データ統合＆CSV出力

Pythonスクリプトで統合（LLM処理）:
- Google Maps APIデータ + Webサイト分析データ
- スコアリング（最大135点）
- CSV出力: `dental_leads_production_batch_[バッチ番号].csv`

## 効率化戦略

### 1. バッチ並列実行（推奨）

複数バッチを同時実行して時間短縮:

```bash
# 例: バッチ2-6を並列実行
python extract_batch_50.py --batch 2 &
python extract_batch_50.py --batch 3 &
python extract_batch_50.py --batch 4 &
python extract_batch_50.py --batch 5 &
python extract_batch_50.py --batch 6 &
wait
```

**注意**: Google Maps APIのレート制限（1秒あたり50リクエスト）に注意。最大5バッチまで並列推奨。

### 2. Webサイト分析の自動化（今後検討）

現在は手動プロンプト実行だが、将来的にPythonスクリプトで自動化可能:
