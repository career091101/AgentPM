**現状**: 平均0.5回/投稿（ほぼシェアされていない）

**原因分析**:
- シェア動機が弱い（個人的な感想が多い）
- データ提供型コンテンツ不足
- 拡散フレーズ欠如（「この情報、広めたい」等）

**改善策**:
- 高野式パターン4（リスト型衝撃ファクト）を多用
- 具体的数値データを3倍増加（現4個 → 目標10-12個）
- シェア動機を明示（「この情報、業界の〇〇さんに共有したい」等）

- First Commentで議論促進（「あなたの業界ではどうですか？」等）
