```

「作業開始」
```

## 共通操作

### 確定反映
```
「確定反映して」
```
→ Flow配下のdraft_*.md を Stock配下に移動

### フェーズメニュー表示
```
「Discovery」
「Research」
「Planning」
```
→ 利用可能なテンプレート一覧を表示
