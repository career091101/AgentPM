# Five Perspectives (5つの眼)

> **出典**: 田所雅之「起業参謀の教科書」第1章
> **関連**: [[CONCEPT_IDEA_001]], [[CONCEPT_CPF_001]]

---

## 1. 定義

**5つの眼（Five Perspectives）** は、起業アイデアの検証や市場分析を多角的に行うための5つの視点フレームワーク。

鳥・虫・魚・医者・人という5つの異なる視点から物事を観察することで、盲点を減らし、より堅牢なビジネス判断を可能にする。

**5つの視点**:
1. **鳥の眼 (Bird's Eye / Macro)**: マクロ視点でメガトレンドを俯瞰
2. **虫の眼 (Bug's Eye / Micro)**: ミクロ視点で顧客の現場を観察
3. **魚の眼 (Fish's Eye / Trend)**: 流れを読み市場の潮流を把握
4. **医者の眼 (Doctor's Eye / Diagnostic)**: 診断視点でビジネスの健康状態をチェック
5. **人の眼 (Human's Eye / Humanity)**: 人間視点でステークホルダーの感情を理解

## 2. なぜ重要か

スタートアップの失敗原因の多くは「視野の狭さ」に起因する：
- ❌ マクロトレンドを無視して時代遅れのビジネスを始める
- ❌ 顧客の現場を見ずに机上の空論でプロダクト設計
- ❌ 市場の流れに逆らって無駄な努力を重ねる
- ❌ ビジネスの問題点を早期発見できず手遅れになる
- ❌ 人の感情を無視して反発を招く

**5つの眼を使うことで**:
- ✅ 複数の視点から盲点を減らせる
- ✅ マクロとミクロの両方を押さえられる
- ✅ データと感情の両面を理解できる
- ✅ 早期に問題を発見して軌道修正できる

## 3. 5つの視点の詳細

### 3.1 鳥の眼 (Macro Perspective)

**定義**: 上空から俯瞰するように、マクロトレンドや市場全体を見渡す視点。

**目的**:
- メガトレンド（技術・社会・経済・政治）を把握
- 10年後の未来を予測
- 業界全体の構造変化を理解

**質問例**:
- 今後10年で世界はどう変わるか?
- テクノロジーの進化はこの市場にどう影響するか?
- 規制の変化はビジネスチャンスか脅威か?
- 人口動態・経済成長の変化は?

**分析手法**:
- PEST分析（Political, Economic, Social, Technological）
- メガトレンド分析
- 業界レポート・マッキンゼー等のリサーチ参照

**事例（AI市場）**:
- メガトレンド: 生成AI革命、クラウドコンピューティングの普及
- 10年後: あらゆる業務がAI化、ホワイトカラー生産性10倍
- 規制: AI規制法（EU AI Act等）、倫理問題

---

### 3.2 虫の眼 (Micro Perspective)

**定義**: 地面を這う虫のように、顧客の現場に密着して細部を観察する視点。

**目的**:
- 顧客の日常行動を理解
- 現場の課題・ペインポイントを発見
- 机上の空論を排除

**質問例**:
- 顧客は実際にどう行動しているか?
- どんな不満・苦労を抱えているか?
- 既存ソリューションの何が不満か?
- 1日の行動パターンは?

**観察手法**:
- 現場観察（Shadowing）
- 顧客インタビュー
- ユーザーテスト
- 実際に顧客として体験（Mystery Shopping）

**事例（Uber）**:
- 観察: タクシーが捕まらない、料金が不透明、支払いが面倒
- 現場: 雨の日・夜中に困る人が多い
- ペイン: 待ち時間、ぼったくりリスク、現金払いの手間

---

### 3.3 魚の眼 (Trend Perspective)

**定義**: 川の流れを読む魚のように、市場の潮流・変化の兆しを捉える視点。

**目的**:
- 市場の成長/衰退トレンドを読む
- 早期シグナルを察知
- 時流に乗る/逆張りの判断

**質問例**:
- この市場は成長しているか、衰退しているか?
- 5年前と比べて何が変わったか?
- 先行指標は何を示しているか?
- 消費者の関心は増えているか減っているか?

**分析手法**:
- Google Trends分析
- 市場成長率（CAGR）チェック
- SNS話題量の推移
- VC投資額の変化

**事例（リモートワーク市場）**:
- 2019年: 一部の先進企業のみ
- 2020年コロナ: 急激に普及（Zoom株価10倍）
- 2023年: ハイブリッド定着、オフィス回帰の兆し
- 先行指標: Slack/Zoom/Notion等の成長率

---

### 3.4 医者の眼 (Diagnostic Perspective)

**定義**: 医者が患者を診断するように、ビジネスの健康状態をチェックする視点。

**目的**:
- ビジネスの問題点を早期発見
- データに基づく診断
- 手遅れになる前に治療（改善）

**質問例**:
- このビジネスは健全か?
- どこに病巣（問題）があるか?
- 数字の異常値は何を示すか?
- 早期警戒シグナルは?

**診断項目**:
- 財務: キャッシュフロー、バーンレート、ランウェイ
- 顧客: Churn Rate, NPS, CAC/LTV
- プロダクト: MAU, Retention, Engagement
- 組織: 離職率、採用難易度、文化

**健康チェックリスト**:
- [ ] キャッシュは6ヶ月以上あるか?
- [ ] Churn Rateは5%以下か?
- [ ] LTV/CACは3倍以上か?
- [ ] 主要メンバーの離職はないか?

**事例（不健康なスタートアップ）**:
- 症状: 売上は伸びているが赤字拡大
- 診断: CAC > LTV、ユニットエコノミクスが破綻
- 処方: マーケティング費用削減、プライシング見直し

---

### 3.5 人の眼 (Humanity Perspective)

**定義**: 人間の感情・価値観・動機を理解する視点。

**目的**:
- ステークホルダーの感情を理解
- 共感を生むストーリー作り
- モチベーション・恐怖・願望の把握

**質問例**:
- 顧客は何を恐れているか?
- 何に喜びを感じるか?
- なぜこの行動をとるのか（動機）?
- チームメンバーは何にモチベーションを感じるか?

**対象者**:
- 顧客: 何を求め、何を恐れるか
- チーム: モチベーション、不安、願望
- 投資家: 何に価値を感じるか
- パートナー: Win-Winの関係構築

**感情マップ**:
```
┌────────────────┬────────────────┬────────────────┐
│ 恐怖           │ 願望           │ 喜び           │
├────────────────┼────────────────┼────────────────┤
│ 失敗・損失     │ 成功・利益     │ 承認・達成感   │
│ 不確実性       │ 安心・確実性   │ 新しい発見     │
│ 社会的孤立     │ つながり       │ 仲間との絆     │
└────────────────┴────────────────┴────────────────┘
```

**事例（Airbnb）**:
- 顧客（ゲスト）の感情:
  - 恐怖: 知らない人の家に泊まる不安
  - 願望: 安くてローカルな体験
  - 喜び: 現地の人とのつながり
- 対策: レビューシステムで安心、ホスト写真で親近感

## 4. 使い方

### 4.1 アイデア検証時（STEP 0）

**目的**: アイデアが時流に乗っているか、顧客ニーズがあるかを多角的に検証

**手順**:
1. **鳥の眼**: メガトレンド調査（1-2時間）
   - Google Trends, 業界レポート, PEST分析
2. **虫の眼**: 顧客インタビュー10人（2-3日）
   - 現場観察、ペインポイント発見
3. **魚の眼**: 市場成長率・投資動向調査（1時間）
   - Crunchbase, PitchBook, Google Trends
4. **医者の眼**: 競合分析（半日）
   - 既存プレイヤーの財務・健全性チェック
5. **人の眼**: ペルソナ感情マップ作成（1時間）
   - 恐怖・願望・喜びを整理

### 4.2 CPF検証時（STEP 4）

**目的**: 課題の深さ・広さを確認

**優先順位**:
1. **虫の眼**: 顧客インタビューで課題深堀り（最重要）
2. **人の眼**: 顧客の感情・動機を理解
3. **魚の眼**: この課題を持つ人は増えているか?
4. **鳥の眼**: マクロトレンドと整合するか?
5. **医者の眼**: 既存ソリューションの問題点診断

### 4.3 PMF検証時（STEP 6-7）

**目的**: プロダクトの健全性と成長可能性を確認

**優先順位**:
1. **医者の眼**: リテンション・Churn Rate診断（最重要）
2. **人の眼**: ユーザーの熱狂度・NPS測定
3. **魚の眼**: 成長トレンド（MRR成長率）
4. **虫の眼**: ユーザー行動ログ分析
5. **鳥の眼**: TAM拡大可能性

## 5. よくある間違い

| 間違い | 症状 | 対策 |
|--------|------|------|
| **鳥の眼だけ** | 机上の空論、顧客不在の戦略 | 必ず虫の眼（現場）とセットで |
| **虫の眼だけ** | 近視眼的、市場トレンド無視 | 鳥の眼でマクロ確認 |
| **魚の眼だけ** | 流行に流される、芯がない | 人の眼で本質的ニーズ確認 |
| **医者の眼だけ** | 数字に溺れる、感情無視 | 人の眼で定性情報も重視 |
| **人の眼だけ** | 感情論、客観性欠如 | 医者の眼でデータ確認 |

**重要**: 5つの眼を**バランスよく使う**ことが鍵。どれか1つに偏らない。

## 6. 事例

### 6.1 成功事例: Zoom

| 視点 | 分析内容 |
|------|---------|
| 🦅 鳥の眼 | リモートワーク・グローバル化のメガトレンド |
| 🐛 虫の眼 | Skype/WebExの接続不安定・使いにくさを観察 |
| 🐟 魚の眼 | ビデオ会議市場が年率20%成長 |
| 🩺 医者の眼 | 既存ツールのChurn Rate高い、NPS低い |
| 👤 人の眼 | ユーザーは「つながらない不安」を恐れる |

**結果**: 5つの眼すべてがポジティブ → PMF達成

### 6.2 失敗事例: Quibi

| 視点 | 分析内容 |
|------|---------|
| 🦅 鳥の眼 | モバイル動画市場は成長（OK） |
| 🐛 虫の眼 | **実際の顧客インタビュー不足**（NG） |
| 🐟 魚の眼 | YouTubeがシェア独占、新規参入困難（NG） |
| 🩺 医者の眼 | リテンション率が異常に低い（NG） |
| 👤 人の眼 | ユーザーは「短編専用」を求めていない（NG） |

**結果**: 虫の眼（顧客理解）が欠如 → $1.75B調達も6ヶ月で終了
