# Case Study: Cheapskate London - 無料イベントNewsletter 36K購読者

**Category**: Niche - Free Events Curation
**Revenue Model**: Sponsorship + Ads + Affiliates
**Subscribers**: 36,000+
**Region**: London (LA/Kidsへ展開中)

---

## 📊 Overview

### Why Niche?
- **超ニッチ**: "ロンドンの無料イベント"のみに特化
- **明確な価値提案**: 毎週14の無料イベント厳選（月-日×2件/日）
- **口コミ成長**: 有料広告ゼロで36,000購読者達成
- **受賞歴**: Women in Journalism's Georgina Henry Award (2022)

### 創業者
- **Kate Samuelson**: ジャーナリスト
- **Georgia Weisz**: グラフィックデザイナー
- **設立**: 2019年5月

---

## 🎯 Business Model詳細

### 1. 収益構造
```
収益源（優先順位順）:
1. ブランドパートナーシップ
   └─ Southbank Center, Science Museum等
2. beehiiv Ads（広告ネットワーク）
3. Pay-per-click リンク
4. スポンサードイベント掲載

特徴:
✓ 無料イベント主催者も広告予算あり
✓ "人を集めたい"というニーズに応える
✓ 大手より低コスト運営で価格競争力
```

### 2. コンテンツ戦略
```
配信フォーマット:
- 毎週月曜配信
- 月-日の7日間 × 2イベント/日 = 14イベント
- 厳選キュレーション（質重視）

選定基準:
✓ 完全無料（事前登録のみOK）
✓ ロンドン市内アクセス可能
✓ 多様性（アート、音楽、ワークショップ等）
✓ 読者投稿も検討
```

### 3. 成長戦略
```
口コミエンジン:
1. 週次コンペティション企画
2. メール転送で応募資格獲得
3. バイラル拡散の仕組み化

結果:
→ 有料広告ゼロで5桁購読者達成
→ 口コミのみで成長
```

## 🚀 Growth Journey

### Phase 1: 立ち上げ (2019年5月)
1. **ニッチ発見**: ロンドン在住者の"高い生活費"ペインポイント
2. **MVP**: 手動キュレーション開始
3. **初期配信**: 友人・知人から開始
4. **品質重視**: 厳選された無料イベントのみ

### Phase 2: 口コミ成長 (2019-2021)
1. **転送コンペ**: 週次プレゼント企画で拡散
2. **5桁購読者**: 1-2年で10,000+達成
3. **収益化開始**: イベント主催者からの問い合わせ増
4. **パートナーシップ**: 大手文化施設との契約

### Phase 3: 拡大・受賞 (2022-2024)
1. **Cheapskate Kids**: 子供向けイベント版ローンチ
2. **Cheapskate LA**: ロサンゼルス版展開開始
3. **受賞**: Georgina Henry Award受賞
4. **36,000購読者**: ロンドン版で達成

## 💡 成功の鍵

### 1. 超明確な価値提案
```
"毎週14の無料イベントが届く"

シンプルさの威力:
✓ 何を得られるか一目瞭然
✓ 毎週のルーティン化
✓ 期待値コントロール容易
✓ 口コミで説明しやすい
```

### 2. 低コスト運営
```
チーム: 創業者2名のみ

コスト削減の工夫:
- 手動キュレーション（AIに頼らない品質）
- 大手メディアと異なる軽量運営
- 外注なし、内製化徹底

結果:
→ スポンサー価格を競合より低く設定可能
→ 利益率の高いビジネスモデル
```

### 3. ニッチ×都市の再現性
```
横展開戦略:
- ロンドン（成功実証）
- LA（米国テスト）
- Kids（セグメント拡大）

今後の可能性:
- NYC, Paris, Tokyo等
- 学生向け、シニア向け等
```

## 📈 Monetization詳細

### スポンサーシップ例
```
Tier 1: 大手文化施設
- Southbank Center
- Science Museum
- Natural History Museum
契約: 長期パートナーシップ、月額固定

Tier 2: 中小イベント主催者
- ローカルギャラリー
- 独立系シアター
- コミュニティセンター
契約: イベント単位、スポット掲載

Tier 3: beehiiv Ads
- 自動広告配信
- インプレッション課金
- 最低保証収益
```

### 収益推定
```
保守的推定（36,000購読者）:
- Open Rate: 40-50%（業界平均以上）
- 月間スポンサー: 5-10社
- 単価: £500-2,000/月
- beehiiv Ads: £500-1,000/月

月次収益: £3,000-£12,000 (¥60-240万円)
年間収益: £36,000-£144,000 (¥720万-2,880万円)

※2名運営のため実質的な利益率高い
```

## 🎓 学び・Takeaways

### 成功要因
1. **超ニッチ×明確**: "無料イベントのみ"という絞り込み
2. **口コミ設計**: 転送コンペによる自然成長
3. **低コスト運営**: 2名で36,000購読者対応
4. **スポンサーWin-Win**: "人を集めたい"ニーズに応える

### 日本での再現性
```
高い再現性の理由:
✓ 日本は無料イベント多数（文化施設、企業PR等）
✓ 節約志向の高まり
✓ 体験型消費のトレンド
✓ 情報過多で"厳選"ニーズ

課題:
△ 無料イベント情報の収集（英語圏より分散）
△ スポンサー文化の違い（日本は広告予算小）
△ Newsletter文化の浸透度

解決策:
✓ 自治体・文化施設との提携
✓ 低単価×多社数モデル
✓ SNS連携で認知拡大
```
