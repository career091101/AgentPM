## 1. 背景と目的

### 1.1 背景

近年、欧米を中心に「ソロプレナー（Solopreneur）」と呼ばれる一人起業家が急増している。特に、生成AI・自動化ツールの普及により、従来はチームを必要としていた事業運営を個人で完結できるようになった。

**注目すべきトレンド：**

| トレンド | 説明 |
|----------|------|
| **TikTok/Instagramマーケの爆発力** | 若い開発者がショート動画で爆発的にユーザー獲得 |
| **AIを活用した高速プロダクト開発** | 非エンジニアでもChatGPTでアプリ開発が可能に |
| **プロダクトの高額売却** | 48時間〜数週間で作ったツールを1000万〜1.5億円で売却 |
| **週1プロダクトローンチ戦略** | 大量ローンチで当たりを見つける戦略 |
| **グローバル市場へのアクセス** | 英語圏市場は日本市場の10倍以上の規模 |

## 2. スコープ

### 含む（In Scope）

- 海外（主に英語圏）のソロプレナー事例調査
- **重点調査**: AI活用プロダクト、TikTokマーケティング、高額売却事例
- **ニュースレター調査**: 1万ドル以上稼ぐニュースレター運営者の成功パターン分析
- **SNS集客調査**: 5プラットフォーム（Twitter/X、TikTok、Instagram、LinkedIn、Facebook）での成長戦略分析
  - 対象者: 既存調査対象＋SNS特化型成功者（Steven Cravotta、Brock氏等）
  - データ粒度: 概要〜ディープダイブ（フォロワー数、投稿頻度、エンゲージメント率、バズ投稿TOP10、成長曲線、コンテンツカテゴリ分類）
- IndieHackers、Twitter/X、Product Hunt、TikTok等の情報源調査
- 収益レポート・公開データの分析
- 調査レポート・事例集の作成

### 含まない（Out of Scope）

- 日本国内のソロプレナー調査（別プロジェクト）
- 実際の事業立ち上げ（本調査結果を基に別途実施）
