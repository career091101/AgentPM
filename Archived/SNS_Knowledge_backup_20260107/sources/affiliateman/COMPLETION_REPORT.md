## 🔧 使用したツール・スクリプト

### 1. メインスクレイパー（scraper.py）

- ホームページからブログ記事を収集
- `/movies/`ページから対談動画を収集
- YouTube URLを抽出してZOOMコンサルを特定
- パスワード認証に対応

### 2. 追加コンテンツ取得（fetch_missing_content.py）

- 特定URLリストからコンテンツを取得
- PDF資料のダウンロード
- メタデータの更新

### 3. 完全性検証（verify_completeness.py）

- サイト上の総コンテンツURLを収集
- 取得済みURLとの差分を検出
- 未取得コンテンツのリスト化

### 4. YouTube文字起こし（youtube_transcriber.py）

- YouTube URLから動画IDを抽出
- yt-dlpで字幕を取得（利用可能な場合）
- Whisper APIで音声文字起こし

### 5. RAGチャンク分割（chunker.py）

- Markdownファイルをセマンティックチャンキング
- 見出し単位でチャンク分割
- メタデータ付与してJSONL形式で出力

### 2. RAGシステムへの組み込み

```python
import json

# チャンクを読み込み
chunks = []
with open('chunks/all_chunks.jsonl', 'r', encoding='utf-8') as f:
    for line in f:
        chunks.append(json.loads(line))

print(f"総チャンク数: {len(chunks)}")

# カテゴリ別フィルタリング例
instagram_chunks = [c for c in chunks if c.get('category') == 'instagram']
twitter_chunks = [c for c in chunks if c.get('category') == 'twitter']
tiktok_chunks = [c for c in chunks if c.get('category') == 'tiktok']
```
