# Case Study 01: TikTokアルゴリズム完全攻略 - コメント数重視戦略

## 概要

TikTokで**6アカウントで100万再生を達成**した実績を持つK氏の、アルゴリズム理解に基づいた再現性の高いバズ戦略。特に**コメント数 = 滞在時間増加**に着目した独自手法。

## 実施主体

- **実施者**: K氏（SNS攻略サロン運営）
- **実績**: 6アカウントで100万再生達成
- **達成方法**: コメント数重視のコンテンツ設計
- **再現性**: 高い（明確なフレームワークあり）

## TikTokアルゴリズムの本質理解

### 初期アカウントのバズメカニズム

```
投稿（テーマA）
    ↓
テーマAに関心が近いユーザーにランダム表示
    ↓
エンゲージメント測定（いいね、コメント、保存、滞在時間、シェア）
    ↓
高エンゲージメント → さらにテーマA興味層へ拡散
    ↓
マス拡散（興味層全体にリーチ）
    ↓
飽和してバズ終了
```

**重要ポイント**: 初動はランダムだが、エンゲージメントが高いと雪だるま式に拡散

### フォロワー増加後のバズメカニズム

```
投稿
    ↓
まずフォロワーに表示
    ↓
フォロワーから一定のエンゲージメント獲得
    ↓
フォロワーと興味関心が近いアカウントへ拡散
    ↓
マス拡散
```

**超重要な洞察**:
> 「フォロワーから一定の評価をされないとマスにその投稿が表示されない」

**結論**: フォロワーのエンゲージメント率を高めることが最重要

## 特化アカウントの絶対性

### なぜ雑記ではなく特化なのか

#### ケース1: 雑記アカウントの失敗

```
美容投稿でバズ → 美容に興味あるフォロワー1,000人獲得
    ↓
次に暮らし系投稿
    ↓
フォロワー1,000人は興味なし → エンゲージメント低下
    ↓
アルゴリズムが低評価 → マス拡散されない
    ↓
バズらない
```

#### ケース2: 特化アカウントの成功

```
美容投稿でバズ → 美容に興味あるフォロワー1,000人獲得
    ↓
次も美容投稿
    ↓
フォロワー1,000人が高エンゲージメント
    ↓
アルゴリズムが高評価 → マス拡散
    ↓
さらにバズる
```

### 特化が重要な理由その2: 関連投稿表示

**TikTokの仕組み**:
```
ユーザーがあなたの投稿Aにいいね
    ↓
TikTokがあなたの他投稿（B, C, D）をそのユーザーのおすすめ欄に表示
    ↓
テーマが同じ → ユーザーがB, C, Dにもいいね
    ↓
エンゲージメント率向上 → アルゴリズムが高評価
```

**雑記だとこれが機能しない**:
- 投稿Aは美容、Bは暮らし、Cはグルメ
- ユーザーは美容にしか興味ない
- B, Cを見ても離脱 → エンゲージメント低下

**結論**: TikTokは必ず特化アカウントで運用すべき

## K氏の独自戦略: コメント数最重視

### なぜコメント数が最も重要なのか

K氏の発見:
> 「僕がこの中で1番意識していることがコメント数。6アカウントで100万再生叩き出した投稿には必ずコメント数が多い」

**メカニズム**:
```
コメント数が多い
    ↓
ユーザーがコメント欄を読む
    ↓
コメントを読んでいる間、動画が2-3回ループ再生される
    ↓
滞在時間が劇的に増加
    ↓
TikTokアルゴリズムが超高評価
    ↓
バズる
```

**数値例**:
- いいね狙い: 平均滞在時間 5秒 → 評価スコア 60点
- コメント狙い: 平均滞在時間 20秒（コメント読む） → 評価スコア 95点

**K氏の哲学**:
> 「いいねを狙いにいくよりかはコメント欄を狙いに行って、その結果滞在時間が長くなり動画がバズる」

## コメント数を増やす3つの施策

### 施策1: ランキング形式で1位をコメント欄に記載

**フレームワーク**:
```
[タイトル]
「〇〇してはいけないもの 5選！」

[動画内容]
5位: A（理由説明）
4位: B（理由説明）
3位: C（理由説明）
2位: D（理由説明）

「1番〇〇してはいけないものはコメント欄に書いたよ！」
```

**実例**:
```
「ダイエットで食べてはいけないもの 5選！」

5位はポテトチップス！これ食べるとほんと太ります！
理由は...（説明）

そして1番食べてはいけないものはコメント欄に書いたよ！
```

**効果**:
- ユーザーの95%以上がコメント欄を見る
- コメント欄滞在時間: 10-20秒
- 動画が2-3回ループ再生
- 結果: 滞在時間が通常の3-4倍

#### 重要なコツ: デメリットランキングが伸びる

**人間心理**: 得より損の回避を優先

**比較**:
| タイプ | 例 | 興味度 |
|--------|-----|--------|
| メリット | 「お金が増える習慣 5選」 | 60点 |
| デメリット | 「絶対にやってはいけないお金が減る習慣 5選」 | **95点** |

**他の例**:
```
メリット型: 「初デートでやると印象がよい行動 5選」
デメリット型: 「初デートでやるとマジでドン引きされるNG行動 5選」
→ デメリット型のほうが3倍バズる
```

### 施策2: 意外性を使ってコメント欄に誘導

**フレームワーク**:
```
[一般論と真逆のことを言う]
→ え！？なんで？（興味喚起）
→ 「詳しくはコメント欄を見てね」
```

**実例1: 就活**:
```
[動画]
就職活動の面接でやってはいけないことはたくさんあります。

その中でも個別面談では堅苦しい言葉や敬語は
絶対に使ってはいけないんです！！

一般論: 敬語使わないと評価下がる
↓
真逆の主張: 敬語使うと評価下がる
↓
これ説明するとかなり長くなるので詳しくはコメント欄を見てね
```

**効果**: 「え！？答え気になる」→ 95%がコメント欄へ

**実例2: 恋愛**:
```
一般論: 初日のデートでは健全であるべき
意外性: 初デートで絶対に手を繋がないといけない！
その理由はコメント欄に書いたよ
```

**実例3: ダイエット**:
```
一般論: ダイエットの時に野菜を食べる
意外性: ダイエットの時には絶対に野菜を食べてはいけない
その理由はコメント欄に書いたよ
```

**実例4: 仕事**:
```
一般論: 仕事ができる人はまともな人が多い
意外性: 仕事ができる人に会ったけど、みんなマジでキ○ガイ！
その理由はコメント欄に書いたよ！
```

**重要ポイント**: 一般論を知った上で、それと真逆のことを言う

### 施策3: 補足情報をコメント欄で提供

**フレームワーク**:
```
[動画で基本情報を提供]
→ さらに詳しい情報（リスト、具体例など）をコメント欄に記載
```

**実例1: ダイエット**:
```
[動画]
ダイエットでは糖質を取るとかなり太ってしまいます。
逆にいうと運動をまったくせずとも糖質の量を制限すると
体重がかなり減ります！

糖質が多い食べ物はご飯と麺、パンなどが多いので
極力ダイエット中に避けるようにしましょう！

[コメント欄誘導]
逆にダイエット中に食べても太りにくい
糖質量が少ない食べ物を20個コメント欄に書いたよ！
```

**実例2: 仕事術**:
```
[動画]
仕事ができる人は絶対に相手の期待を上回ってきます。
仕事ができない人は100%求められているところを80%で答えます。
仕事ができる人は100%ではなく120%と期待を超えるので
信頼を得られるのです。

[コメント欄誘導]
相手の期待を超える具体的な方法10個をコメント欄に書いたよ！
```

**効果**:
- 動画で興味を引く
- コメント欄で詳細を知りたい欲求
- コメント欄滞在時間: 15-30秒
- 結果: 超高評価

## エンゲージメント指標の優先順位

K氏の重要度ランキング:

| 順位 | 指標 | 重要度 | 理由 |
|------|------|--------|------|
| 1位 | **コメント数** | ★★★★★ | 滞在時間を最も増やす |
| 2位 | **滞在時間** | ★★★★★ | TikTokが最重視 |
| 3位 | **シェア数** | ★★★★☆ | 二次拡散を生む |
| 4位 | **保存数** | ★★★☆☆ | 後で見たい = 価値が高い証拠 |
| 5位 | **いいね数** | ★★☆☆☆ | 最も容易だが効果は低い |

**戦略的含意**:
- いいねを狙うのは初心者の発想
- プロはコメント数と滞在時間を狙う
- 100いいね < 50コメント

## コンセプト設計のフレームワーク

### 参入ジャンル選定の3条件

#### 条件1: 自分が発信できるジャンル

- 経験したことがあるジャンル
- 今後経験するために勉強中のジャンル
- 情熱を持って語れるジャンル

#### 条件2: 競合が伸びているか確認

**調査方法**:
1. TikTok検索で参入ジャンルのキーワードを入力
2. 親和性の高い競合を洗い出す
3. 各アカウントの「▼」ボタンを押す → 近しいアカウント表示
4. 競合を10-20アカウント抽出

**判断基準**:
- フォロワー1万人以上のアカウントが複数ある → 参入OK
- 最大フォロワー数 = そのジャンルの市場規模

**例**:
- ダイエットジャンル: 最大50万フォロワー → 超巨大市場
- ニッチなビジネスジャンル: 最大3万フォロワー → 小規模だが稼げる

#### 条件3: マネタイズ可能なジャンル

**確認方法**:
1. ASP管理画面でアフィリエイト案件を確認
2. note/Brain等でコンテンツ販売ニーズを確認
3. 悩みが深いジャンルか確認

**悩みが深い = 収益化しやすいジャンル**:
- 恋愛（復縁、モテたい、彼氏・彼女が欲しい）
- 仕事（年収アップ、転職、副業）
- ダイエット（痩せたい、筋肉つけたい）
- コンプレックス系全般（ハゲ、ワキガ、低身長等）

## 再現性のあるステップ

### Phase 1: アカウント設計（1週間）

**タスク**:
1. 参入ジャンル選定（3条件クリア）
2. 競合10-20アカウントリサーチ
3. 差別化ポイント設定
4. プロフィール最適化

**成果物**:
- ジャンル選定シート
- 競合分析リスト
- アカウントコンセプト1枚

### Phase 2: コンテンツ設計（1週間）

**タスク**:
1. コメント誘導パターン10個作成
   - ランキング形式 × 3
   - 意外性誘導 × 4
   - 補足情報提供 × 3

2. 投稿テーマ30個リストアップ

3. 競合のバズ投稿を20個分析

**成果物**:
- コメント誘導テンプレート10個
- 投稿テーマリスト30個

### Phase 3: 初期投稿（2週間、10投稿）

**目標**: アルゴリズムテスト

**実行**:
1. コメント誘導施策を必ず入れる
2. 2日に1投稿ペース
3. 各投稿のエンゲージメント分析
4. 高評価パターンを見極める

**成功指標**:
- 10投稿中2-3投稿が1万再生超え
- コメント数 50以上/投稿

### Phase 4: 爆速成長（1-3ヶ月、30-50投稿）

**目標**: フォロワー1万人、100万再生動画複数

**実行**:
1. 高評価パターンを再現
2. 毎日1投稿（週5-7投稿）
3. コメント欄での双方向コミュニケーション
4. フォロワーの反応を見て改善

**成功指標**:
- フォロワー1万人到達
- 3投稿に1投稿が10万再生超え
- コメント数平均100以上/投稿

### Phase 5: マネタイズ（3ヶ月目〜）

**実行**:
1. プロフィールにアフィリエイトリンク設置
2. 定期的に商品紹介投稿
3. ライブ配信でファン化
4. 外部サービス（note/コンサル）への誘導

**目標月収**: 10-100万円

## 成功要因の分析

### 1. アルゴリズムの本質理解

- 表面的なテクニックではなく、アルゴリズムの仕組みを理解
- 「なぜコメント数が重要か」を論理的に説明できる
- フォロワーのエンゲージメント率が全ての鍵

### 2. 再現性の高いフレームワーク

- 「なんとなくバズった」ではなく、明確な型がある
- 3つのコメント誘導施策は誰でも実行可能
- 6アカウントで再現 = 偶然ではない

### 3. 特化アカウントの徹底

- 雑記は絶対にやらない
- フォロワーの興味と投稿テーマを一致させる
- アルゴリズムに評価される設計

## 重要な教訓

### TikTokは「いいね」ではなく「コメント」を狙う

> 「僕がこの中で1番意識していることがコメント数。6アカウントで100万再生叩き出した投稿には必ずコメント数が多い」

- いいねは瞬間的、コメントは滞在時間増加
- コメント欄を読んでいる間に動画が2-3回ループ
- 滞在時間 = TikTokアルゴリズムの最重要指標

### 特化アカウント以外ありえない

> 「フォロワーから一定の評価をされないとマスに投稿が表示されない」

- 雑記アカウントはフォロワーのエンゲージメント率が低い
- エンゲージメント率が低いとマス拡散されない
- 特化アカウントだけがバズり続けられる

### アルゴリズム理解が全て

- 小手先のテクニックは通用しない
- 仕組みを理解して、アルゴリズムに評価される投稿を作る
- 「なぜバズるのか」を論理的に説明できることが重要

## まとめ

K氏のTikTok戦略は、**アルゴリズムの本質理解 × 再現性の高いフレームワーク**で構成されている。

成功の方程式:

1. **特化アカウントで運用** - 雑記は絶対NG
2. **コメント数を最重視** - 滞在時間増加のメカニズム
3. **3つのコメント誘導施策** - ランキング、意外性、補足情報
4. **フォロワーのエンゲージメント率を高める** - これがマス拡散の鍵

**最重要ポイント**:
> 「いいねを狙うのではなく、コメント欄を狙う。その結果滞在時間が長くなり動画がバズる」

6アカウントで100万再生達成という実績が、このフレームワークの再現性を証明している。
