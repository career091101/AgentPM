# Case Study 02: 1ツイートで20万円売上を達成したマネタイズ戦略

## 概要

フォロワー1万人程度のアカウントで、**1投稿でサロン売上12万円 + コンサル依頼3件（6万円）+ note売上（2.8万円）= 合計約20万円**を1日で達成した事例。

## 実施主体

- **実施者**: K氏（SNS攻略サロン運営）
- **アカウント規模**: フォロワー約1万人
- **商品**: オンラインサロン（2,980円）、コンサル（2万円/時）、note（3,980円）
- **期間**: 1日
- **達成成果**: 総売上約20万円

## 売上内訳

1. **サロン入会**: 約12万円（フォロリツ企画経由）
2. **コンサル依頼**: 3件 × 2万円/時 = 6万円（DM経由）
3. **note販売**: 3,980円 × 7部 = 約2.8万円
4. **合計**: 約20万円

## 実施した戦略

### 1. フォロリツ企画の設計

**企画の本質**:
> 「フォロリツ企画はフォロワーを増やす目的もあるが、一番は売上を爆速に生み出す売上拡大施策」

**設計ポイント**:
- 無料プレゼント + 有料商品誘導のセット
- プレゼント内容と売る商品の内容を近づける
- 自分の発信テーマに合致したターゲットを集客

**具体例**:
```
無料プレゼント: 「インスタ稼げるジャンル30選」
有料商品誘導: 「インスタで月100万円稼ぐストーリーの型50選（note）」

無料プレゼント: 「コンビニで食べても太らないお菓子30選」
有料商品誘導: 「2ヶ月で20kg痩せたダイエット方法（note）」
```

### 2. 事前告知ツイートの活用

**重要な発見**:
> 「フォロリツ企画をする前の事前告知ツイートは本企画のフォロリツより3倍バズる」

**実績**:
- 事前告知ツイート: 1,000いいね/1,000リツイート超え（フォロワー1万人で達成）
- 本企画ツイート: 約300いいね/300リツイート
- **事前告知で20人がサロン入会**

**心理的メカニズム**:
- 人は「リリース」より「告知」に反応する
- 鬼滅の刃の新作も、公開6ヶ月前の告知時が最も話題になる
- 期待感 > 実物という人間心理の活用

**事前告知の投稿例**:
```
明日、SNS攻略の超有益コンテンツを無料配布します！

過去最大級のプレゼント企画になるので、
絶対に見逃さないでください🔥

[告知画像]

※コメント欄に商品紹介を記載
```

### 3. コメント欄での商品誘導

**テクニック**:
- 事前告知ツイートのコメント欄に有料商品を紹介
- 「袋とじ戦法」: 無料部分で興味を引き、有料部分で本命を見せる
- サロンメンバーからの「これサロンで見れますか？」という自然な質問を活用

**袋とじ戦法の本質**:
> 「エロ本で下着が見えそうな写真（無料部分）で興味を湧かせて、1番見たい下着を脱いだ部分を有料で見せる。いかに続きの有料部分を見たい！と無料部分で思わせるかが大切」

**実例**:
```
[投稿]
凄すぎる...初期垢が1日でインスタフォロワー1000人増えてる。
僕のサロン限定施策です。

[コメント欄]
→ サロンメンバーA: これサロンで見れますか？
→ K氏: 見れます！さらに詳しい解説もあります
→ 第三者: 気になる！入会検討します
```

### 4. LINE公式アカウントとの連携

**レバレッジ戦略**:
- SNS投稿: フォロワーの20-30%にリーチ
- LINE公式アカ: 開封率70%超え
- フォロリツ企画をLINEでも告知することで、リーチを最大化

**効果**:
- フォロワー1万人 × 30% = 3,000人
- LINE登録者3,000人 × 70% = 2,100人
- 合計5,100人にリーチ（通常の2倍以上）

### 5. 商品が売れる投稿パターン

#### パターンA: 実績訴求
```
凄すぎる...初期垢が1日でインスタフォロワー1000人増えてる。
僕のサロン限定施策です。

すごい大手アカからメンションもらえます
しかもメンションから獲得するフォロワーは質が高いのでおすすめ

インスタ伸ばしたい人はぜひ
最大10万人のアカからメンションもらえるかもしれません

#けいサロン実績
```

#### パターンB: テーマ訴求
```
1ツイートで16万円一撃で稼いだTwitter施策知りたい人いますか？？

これ上手くハマればフォロワー5000人とかでも月100も可能です

いいねリツ多いとサロン限定で公開します

さすがこれは無料で出せません。
そんくらい有益かつ、みんなやると稼げなくなるので、
クローズでシェアします
```

#### パターンC: 希少性訴求
```
【残り3枠】個別コンサル募集

先月コンサルした方が1ヶ月でフォロワー5,000人増えました

本気で伸ばしたい方のみDMください
※冷やかし厳禁
```

## 戦略的フレームワーク

### 投稿の3分類

K氏の重要な指摘:
> 「投稿には3つの種類がある。(1)新規フォロワーを集める投稿、(2)フォロワーをファン化する投稿、(3)商品を売るための投稿。特に(3)が売上増に大切」

| 投稿種別 | 目的 | ターゲット | 頻度 |
|---------|------|-----------|------|
| (1) 新規集客投稿 | フォロワー増加 | 初心者、マス層 | 70% |
| (2) ファン化投稿 | 信頼関係構築 | 脱初心者〜中級者 | 20% |
| (3) 販売投稿 | 売上最大化 | 既存ファン層 | 10% |

**多くの人が失敗する理由**:
- (1)と(2)ばかりで、(3)をやらない
- (3)の投稿パターンを知らない
- 売り込みを恐れて遠慮してしまう

## 再現性のあるステップ

### Phase 1: 事前準備（1週間前）
1. 無料プレゼント作成（PDF/スライド形式）
2. 無料プレゼント内に有料商品の誘導文を挿入
3. 商品の内容と無料プレゼントのテーマを近づける
4. LINE公式アカウントの準備（未開設の場合）

### Phase 2: 事前告知（3日前）
1. 事前告知ツイート投稿（画像付き）
2. コメント欄に商品情報を記載
3. LINE公式アカで事前告知を配信
4. いいね/RTが伸びたら追加情報を投下

### Phase 3: 本企画実施（当日）
1. フォロリツ企画ツイート投稿
2. LINE公式アカで本企画告知
3. DMで無料プレゼント配布 + 有料商品誘導
4. コメント欄で自然な質問→回答の流れを作る

### Phase 4: フォローアップ（翌日〜）
1. 企画参加者へのお礼DM
2. LINE公式アカで追加コンテンツ配信
3. サロン/商品購入者への特典提供
4. 次回企画の予告

## 成功要因の分析

### 1. 事前告知による期待値の醸成
- 本企画の3倍のエンゲージメント
- 告知段階で20名のサロン入会
- 「期待感」というフックの活用

### 2. 無料と有料のシームレスな導線
- 無料プレゼントで価値提供
- 有料商品は「さらに深い内容」として自然に誘導
- 押し売り感なく購入意欲を喚起

### 3. マルチチャネル活用
- Twitter + LINE公式アカの組み合わせ
- リーチ最大化による売上拡大
- LINE開封率70%の威力

### 4. コメント欄での社会的証明
- サロンメンバーからの自然な質問
- 第三者からの「気になる！」コメント
- 購入への心理的ハードル低下

## 重要な教訓

### フォロリツ企画の本質

> 「フォロリツの目的は売上拡大！必ず無料プレゼントの中に有料商品を入れましょう」

- フォロワー増加は副次的効果
- 本命は売上最大化
- 無料プレゼント = 有料商品への導線

### 事前告知の威力

- 本企画より3倍バズる
- 告知段階で売上が立つ
- 必ず告知を挟むこと

### 商品と無料コンテンツの親和性

良い例:
```
無料: インスタ稼げるジャンル
有料: インスタで月100万円稼ぐストーリーの型50選
```

悪い例:
```
無料: インスタ稼げるジャンル
有料: Twitter運用マニュアル（ジャンル不一致）
```

## 応用可能性

### 月100万円への拡張

K氏の提言:
> 「僕はフォロリツで2,980円を売ったが、有料コンサル6ヶ月50万円を二人に売ったらそれで月100万円の達成」

**拡張戦略**:
1. フォロリツ企画で無料相談会への誘導
2. 無料相談会（またはワンコイン500円コンサル）で信頼関係構築
3. 相談会内で高額商品（50万円）を提案
4. 2件成約で月100万円達成

**ワンコインコンサルの意義**:
- 無料だと成約率の低い層が大量に来る
- 500円という少額でスクリーニング
- 申込者の質が高まり、高額商品の成約率向上

### 他ジャンルへの応用

| ジャンル | 無料プレゼント | 有料商品 |
|---------|--------------|---------|
| 恋愛 | 「大好きな彼氏ができた事例」 | 恋愛攻略note（9,800円） |
| マッチングアプリ | 「経験人数が10人増えた施策」 | モテるプロフ作成コンサル（3万円） |
| ダイエット | 「コンビニで太らないお菓子30選」 | 3ヶ月ダイエットコンサル（10万円） |
| ビジネス | 「副業で月5万円稼ぐ方法」 | 副業スタートアップ講座（5万円） |

## まとめ

この事例は、**フォロワー1万人でも20万円の売上を1日で達成できる**ことを証明している。

鍵となるのは：

1. **事前告知による期待値醸成** - 本企画の3倍バズる
2. **無料と有料のシームレスな導線** - 袋とじ戦法
3. **LINE公式アカとの連携** - リーチ2倍化
4. **商品を売る投稿の型** - 実績訴求/テーマ訴求/希少性訴求

**最重要ポイント**:
> 「フォロリツの目的は売上拡大。必ず無料プレゼントの中に有料商品を入れる」

この原則を守れば、フォロワー1万人で月20万円、5万人で月100万円も十分射程圏内となる。
