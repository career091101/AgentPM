# 事例調査:日清食品ホールディングス株式会社(食品製造)

## 1. 基本情報

- **企業名**: 日清食品ホールディングス株式会社
- **業界**: 即席麺・食品製造業
- **事業規模**: 連結売上高5,912億円(2023年3月期)
- **従業員数**: 連結約13,000名、国内約4,800名
- **導入AIツール**:
  - NISSIN AI-chat powered by GPT-4 Turbo/GPT-4o
  - Azure OpenAI Service(基盤技術)
  - Microsoft 365 Copilot

## 2. AI導入サマリー

日清食品グループは、2023年入社式での安藤宏基CEO発言をきっかけに、わずか3週間で独自の生成AIシステム「NISSIN AI-chat」を構築し全社展開。国内約4,800名が日々利用し、年間32,591時間の作業工数削減を達成。営業部門では利用率が28%(2023年5月)から約70%(2024年3月)に急上昇し、営業担当者1人あたり年間250時間の効率化を見込む。2024年11月からはMicrosoft 365 Copilotを約1,700人に導入し、利用率85%、1人あたり年間79.4時間・18万2,620円の削減効果を実現。投資対効果は3.9倍に達している。

## 3. 定量的効果

### NISSIN AI-chat(GPT-4ベース)
- **総削減時間**: 年間32,591時間の作業工数削減
- **営業部門利用率**: 28%(2023年5月)→約70%(2024年3月)
- **営業担当者の効率化**: 1人あたり年間250時間の削減見込み
- **プロンプトテンプレート**: 14部門で100種類超を作成
- **利用者数**: 国内約4,800名

### Microsoft 365 Copilot
- **導入対象**: 日清食品ホールディングス+日清食品約1,700人
- **月間利用率**: 85%
- **削減時間**: 1人あたり年間79.4時間
- **削減費用**: 1人あたり年間18万2,620円
- **投資対効果**: 3.9倍(年間ライセンス料4万6,440円に対し)

## 4. 導入背景・課題

### 経営トップのメッセージ(2023年4月入社式)
安藤宏基CEO発言:
「ChatGPTのような新しいテクノロジーを賢く駆使することで、短期間に多くの学びを得ていって欲しい」

### 組織課題
1. **業務効率化の必要性**: グローバル競争激化による生産性向上要請
2. **情報漏洩リスク**: 一般のChatGPT利用によるセキュリティ懸念
3. **デジタル人材不足**: 全社員のAIリテラシー向上

### 営業部門の具体的課題
1. 提案資料作成の時間不足
2. 市場分析・競合調査の負荷
3. 顧客対応の効率化
4. 約250人の営業担当者への展開

## 5. ソリューション詳細

### NISSIN AI-chat powered by GPT-4 Turbo/GPT-4o

**技術基盤**:
- Azure OpenAI Serviceを活用
- 日清食品グループ専用のセキュアな環境構築
- ユーザー入力内容はOpenAIの学習に利用されない

**モデル更新履歴**:
- 2023年4月25日: GPT-4で公開
- 2023年11月: GPT-4 Turboへ更新
- 2024年6月: GPT-4oへ更新

**開発期間**: プロジェクトチーム結成から3週間(2023年4月入社式後)

**セキュリティ対策**:
- Microsoft Azure環境での専用構築
- 入力データのOpenAI学習利用禁止
- 社内ネットワークからのみアクセス可能

### 100種類超のプロンプトテンプレート

**カバー部門**: 14部門
**主要カテゴリ**:
1. 営業: 提案資料作成、顧客分析
2. マーケティング: 市場調査、競合分析
3. 研究開発: 技術情報検索、アイデア発想
4. 人事: 人材育成プログラム設計
5. 経営企画: 戦略立案支援
6. 法務: 契約書レビュー
7. その他: 翻訳、文書要約等

### 営業部門での活用促進施策

**推進体制**:
- 全国約250人の営業担当者から約20人を選抜
- 業務効率化のユースケース検討チーム結成
- 効果検証と横展開

**具体的活用例**:
- 提案資料のドラフト作成
- 顧客企業の市場分析
- 競合製品情報の整理
- 営業メールの作成支援

**効果**: 1人あたり年間250時間の削減見込み

### Microsoft 365 Copilot(2024年11月導入)

**導入規模**:
- 日清食品ホールディングス全社員
- 日清食品(製造部門除く)
- 合計約1,700人にライセンス付与

**主な活用シーン**:
1. PowerPoint: Wordの企画書から自動スライド作成
2. Excel: データ分析、グラフ作成の自動化
3. Outlook: メール下書き、スケジュール調整
4. Teams: 会議要約、アクションアイテム抽出

**成果**:
- 月間利用率: 85%(業界平均を大きく上回る)
- 費用対効果: 3.9倍(投資額4.6万円に対し削減額18.3万円)

## 6. 導入プロセス

### NISSIN AI-chat構築(2023年4月)

**Week 1(入社式翌週)**:
1. 安藤CEO発言を受けてプロジェクトチーム結成
2. Azure OpenAI Service採用決定
3. セキュリティポリシー策定

**Week 2**:
1. 専用環境の構築開始
2. 社内ネットワーク接続設定
3. 初期テストユーザー選定

**Week 3**:
1. 全社員向けロールアウト準備
2. 使い方ガイドライン作成
3. 2023年4月25日公開(入社式から3週間後)

### 全社展開・活用促進(2023年5月〜2024年)

**Phase 1: 初期展開(2023年5月〜)**
- 全社員へのアクセス権付与
- 基本的な使い方の周知
- 営業部門利用率28%(2023年5月時点)

**Phase 2: 活用促進(2023年後半〜2024年)**
1. 営業部門での選抜メンバーによるユースケース開発
2. 14部門でのプロンプトテンプレート作成(100種類超)
3. デジタル化推進室によるてこ入れ
4. 営業部門利用率約70%達成(2024年3月)

**Phase 3: Microsoft 365 Copilot導入(2024年11月)**
1. 約1,700人へのライセンス付与
2. 効果検証: 利用率85%、ROI 3.9倍確認
3. 継続的な活用推進
