# 592_イオン_AIアシスタント

## 1. 企業基本情報

| 項目 | 内容 |
|------|------|
| 企業名 | イオン株式会社（イオンリテール株式会社） |
| 業種 | 小売業 |
| 設立 | 1926年（イオンリテール：2011年） |
| 本社所在地 | 千葉県千葉市美浜区 |

| 主要事業 | 総合小売業、ショッピングセンター運営 |

## 2. AI/生成AI導入の背景

### 課題
- 膨大な業務マニュアル（数千～数万ページ）の検索困難
- 従業員の問い合わせ対応に時間がかかる
- 新入社員の教育コスト増大
- 店舗業務の効率化ニーズ

### 経営判断
- デジタル技術を活用した従業員支援の強化
- Microsoft AIとの戦略的パートナーシップ
- 約390店舗への先行導入決定

## 3. 導入したAI技術の詳細

### AIツール名称
**AIアシスタント（次世代型従業員マニュアル）**

### 技術仕様
- **基盤**: Microsoft AI（Azure OpenAI Service）
- **機能**: 音声・文字による質問対応、自動回答システム
- **学習データ**: 数千～数万ページの業務マニュアル、法律情報
- **デバイス**: オールインワンデバイスに実装

### 開発パートナー
- Microsoft（AI技術提供）

## 4. 具体的な活用領域

### 店舗従業員向け支援
- イオンラウンジの利用方法・場所案内
- 免税対応の手順確認
- 免税対象商品の確認
- 拾得物対応マニュアル検索

### 社内業務効率化
- よくある質問への自動回答
- パソコン・複合機の設定サポート
- 社宅利用に関する問い合わせ対応
- 各種手当て・休暇に関する情報提供

### 対応策
- Microsoft AI技術による高精度回答システム構築
- オールインワンデバイスによるユーザビリティ向上
- 継続的なデータ学習によるAI精度向上
