# 事例: 損保会社Y（金融・保険）

## 1. 基本情報
| 項目 | 内容 |
|------|------|
| 企業名 | 損保会社Y |
| 業界 | 金融・保険 |
| AIツール | 事故査定・損害評価AI |
| 導入時期 | 2024-08 |

## 2. 定量的効果
| 指標 | 数値 |
|------|------|
| 査定時間短縮 | 68% |
| 自動承認率 | 76% |
| 査定員の訪問削減 | 42% |
| 顧客満足度 | 87% |

## 3. 導入背景・課題
- 事故査定に時間がかかる
- 査定のばらつきが問題
- 査定員の移動コストが大きい

## 4. ソリューション概要
被保険者がスマートフォンで事故画像を送信。AIが画像認識で損害程度を判定。修理相場データベースと連携して修理費を自動計算。損害保険金額を自動決定。軽微な案件は自動承認。重大事故は査定員を自動派遣。

## 5. 成功要因
- 画像認識精度の高度化
- 修理業者との連携データ整備
- 査定員との協力体制

## 6. 他社への示唆
- 保険金支払いプロセスのデジタル化
- 顧客体験と業務効率の両立
- オペレーション費用削減
