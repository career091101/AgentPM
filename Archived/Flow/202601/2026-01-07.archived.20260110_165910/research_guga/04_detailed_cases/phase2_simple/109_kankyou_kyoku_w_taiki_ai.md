# 事例: 環境局W（公共・自治体）

## 1. 基本情報
| 項目 | 内容 |
|------|------|
| 企業名 | 環境局W |
| 業界 | 公共・自治体 |
| AIツール | 大気汚染予測・対策AI |
| 導入時期 | 2024-09 |

## 2. 定量的効果
| 指標 | 数値 |
|------|------|
| 汚染予測精度 | 79% |
| 注意報発令精度 | 84% |
| PM2.5削減 | 14% |
| 市民健康被害軽減 | 22% |

## 3. 導入背景・課題
- 大気汚染の予測が困難
- 健康被害の予防対応が遅延
- 汚染原因の特定が難しい

## 4. ソリューション概要
気象データ、各地点のセンサー値、工場排気量をAIで分析し、大気汚染を事前予測。注意報・警報発令タイミングを提案。市民の外出控え呼びかけを自動生成。汚染原因の特定と対策工場への指導を提案。

## 5. 成功要因
- 気象機関との連携
- 企業の排気量データ共有
- 市民への継続的な情報提供

## 6. 他社への示唆
- 環境保全へのAI活用で市民健康を守れる
- 多元的データの統合で精度向上
- 予防的環境対策の実現
