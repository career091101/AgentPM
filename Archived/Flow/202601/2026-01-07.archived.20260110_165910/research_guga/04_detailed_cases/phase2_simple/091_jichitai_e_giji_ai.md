# 事例: 自治体E（公共・自治体）

## 1. 基本情報
| 項目 | 内容 |
|------|------|
| 企業名 | 自治体E |
| 業界 | 公共・自治体 |
| AIツール | 議事録自動生成・要約AI |
| 導入時期 | 2024-04 |

## 2. 定量的効果
| 指標 | 数値 |
|------|------|
| 議事録作成時間 | 75%削減 |
| 月間会議数 | 45件 |
| 自動要約精度 | 89% |
| 職員削減相当 | 1.2名分 |

## 3. 導入背景・課題
- 議事録作成に週10時間以上要している
- 文字起こしの誤字脱字が多発
- 会議後の要約依頼が遅延

## 4. ソリューション概要
音声をリアルタイムで認識し、自動で議事録を生成。決定事項・アクション項目を抽出し、自動要約を提供。複数言語対応で外国籍住民対応会議にも対応。

## 5. 成功要因
- 職員向けの導入研修を充実させた
- 音声品質の改善（マイク配置等）
- 議事録フォーマットの標準化

## 6. 他社への示唆
- 公共機関の事務効率化はAIの重要活用領域
- 透明性と効率性の両立が重要
- 導入前の業務流程設計が成功鍵
