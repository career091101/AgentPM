# 事例: 銀行L（金融・保険）

## 1. 基本情報
| 項目 | 内容 |
|------|------|
| 企業名 | 銀行L |
| 業界 | 金融・保険 |
| AIツール | 顧客生涯価値・離顧予測AI |
| 導入時期 | 2024-08 |

## 2. 定量的効果
| 指標 | 数値 |
|------|------|
| 離顧予測精度 | 76% |
| 顧客保持率向上 | 8.5% |
| クロスセル成功率 | 34%向上 |
| 営業効率向上 | 42% |

## 3. 導入背景・課題
- 顧客の離顧が増加している
- クロスセル機会を見落としている
- 営業活動が非効率である

## 4. ソリューション概要
顧客の取引履歴、保有商品、ライフステージをAIで分析し、離顧リスクを予測。サービス改善提案や個別営業を自動提案。次に購入しそうな商品を提案し、クロスセル機会を自動抽出。顧客の生涯価値を自動算定。

## 5. 成功要因
- 顧客データの整理と活用
- 営業チームとの連携
- AI提案を営業活動に統合

## 6. 他社への示唆
- 顧客維持と拡大がAIで実現
- 営業効率化で売上向上
- データドリブン営業への転換
