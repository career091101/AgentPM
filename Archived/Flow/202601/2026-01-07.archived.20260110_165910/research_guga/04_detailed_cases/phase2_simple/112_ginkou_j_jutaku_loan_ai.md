# 事例: 銀行J（金融・保険）

## 1. 基本情報
| 項目 | 内容 |
|------|------|
| 企業名 | 銀行J |
| 業界 | 金融・保険 |
| AIツール | 住宅ローン自動査定AI |
| 導入時期 | 2024-05 |

## 2. 定量的効果
| 指標 | 数値 |
|------|------|
| 審査時間短縮 | 71% |
| 月間申込数 | 38%増加 |
| 不動産鑑定費削減 | 45% |
| 顧客満足度 | 89% |

## 3. 導入背景・課題
- 住宅ローン審査に多くの時間と人手が必要
- 不動産鑑定に費用がかかる
- 顧客の申込から融資実行までが長い

## 4. ソリューション概要
申込者の信用情報、勤務先、年収、物件情報をAIで分析し、融資可否と金利を自動判定。Google Earth、土地情報、周辺相場価格をAIが学習し、不動産鑑定を自動化。事前審査結果を即座に通知。融資実行後のローン監視も自動化。

## 5. 成功要因
- 信用情報機関との連携
- 不動産データベース整備
- 融資担当者への十分な説明

## 6. 他社への示唆
- 金融プロセスのデジタル化で顧客満足度向上
- 鑑定業務の自動化でコスト削減
- スピード感のある金融サービスを実現
