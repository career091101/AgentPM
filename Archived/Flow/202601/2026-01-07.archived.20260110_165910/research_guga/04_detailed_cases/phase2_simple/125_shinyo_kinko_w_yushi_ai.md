# 事例: 信用金庫W（金融・保険）

## 1. 基本情報
| 項目 | 内容 |
|------|------|
| 企業名 | 信用金庫W |
| 業界 | 金融・保険 |
| AIツール | 中小企業融資提案AI |
| 導入時期 | 2024-08 |

## 2. 定量的効果
| 指標 | 数値 |
|------|------|
| 融資提案効率化 | 56% |
| 融資実行額 | 32%増加 |
| デフォルト率低減 | 5.8% |
| 中小企業貸出増加 | $75M |

## 3. 導入背景・課題
- 中小企業の融資判定が複雑
- 経営者との信頼構築に時間がかかる
- 返済能力の予測が不十分

## 4. ソリューション概要
決算書、銀行取引履歴、経営者情報をAIで分析し、融資可否と適切な融資額を提案。経営課題を自動抽出。経営改善提案を個別生成。返済シミュレーションを提供。経営支援プランを自動提案。

## 5. 成功要因
- 地元中小企業のデータ整備
- 融資担当者との密接な連携
- 経営支援体制の整備

## 6. 他社への示唆
- 地域金融機関でのAI活用で競争力強化
- 融資量増加で地域経済に貢献
- 経営支援で顧客ロイヤルティ向上
