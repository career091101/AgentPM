# 事例: 国機関L（公共・自治体）

## 1. 基本情報
| 項目 | 内容 |
|------|------|
| 企業名 | 国機関L |
| 業界 | 公共・自治体 |
| AIツール | 法令検索・整合性確認AI |
| 導入時期 | 2024-09 |

## 2. 定量的効果
| 指標 | 数値 |
|------|------|
| 法令検索時間 | 72%短縮 |
| 矛盾検出精度 | 87% |
| 立法補佐官の業務効率 | 38%向上 |
| 法律案修正回数 | 41%削減 |

## 3. 導入背景・課題
- 膨大な法律・政令・省令から該当部分を見つけるのに時間
- 新規法律案と既存法令との矛盾チェックが不十分
- 立法補佐官の負担が過重

## 4. ソリューション概要
全国の法律・政令・省令・通達をベクトル化し、検索・整合性確認を自動化。新規法律案のドラフトに対して、矛盾箇所を自動検出。過去の類似立法例から参考になる条文を提案。

## 5. 成功要因
- 法務省との密接な連携
- 法律知識ベースの整備
- ユーザーインタフェースの工夫

## 6. 他社への示唆
- 法律・規制領域でのAI活用が進展
- 複雑な文書処理の効率化に有効
- 専門家の創造的業務へのシフト実現
