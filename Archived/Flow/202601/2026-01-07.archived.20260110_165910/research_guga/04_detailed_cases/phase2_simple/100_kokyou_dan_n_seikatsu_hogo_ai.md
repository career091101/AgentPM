# 事例: 公共団体N（公共・自治体）

## 1. 基本情報
| 項目 | 内容 |
|------|------|
| 企業名 | 公共団体N |
| 業界 | 公共・自治体 |
| AIツール | 生活保護申請・支給管理AI |
| 導入時期 | 2024-04 |

## 2. 定量的効果
| 指標 | 数値 |
|------|------|
| 申請処理時間 | 61%短縮 |
| 不正受給検出精度 | 83% |
| 申請承認率適正化 | 15% |
| 職員業務効率化 | 32時間/週削減 |

## 3. 導入背景・課題
- 生活保護申請の処理に時間がかかる
- 不正受給の検出が難しい
- 福祉職員の負担が過重

## 4. ソリューション概要
申請者の収入、資産、家族構成をAIが自動確認し、支給判定を提案。不正受給リスクを検出。定期的な支給再判定を自動スケジュール。面接記録から支援計画を自動提案。

## 5. 成功要因
- 福祉事務所との十分な協力
- 申請者プライバシー保護
- 定期的な運用改善

## 6. 他社への示唆
- セーフティネット構築にAI活用は不可欠
- 不正検出と利用者支援の両立が重要
- 透明で公正な給付判定が実現できる
