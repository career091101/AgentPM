# 632_三井住友フィナンシャルグループ_SMBC-GAI / OpenAI / RAG

## 1. 企業概要
- **企業名**: 株式会社三井住友フィナンシャルグループ (SMFG/SMBCグループ)
- **業種**: 金融・銀行
- **従業員数**: 約46,000人（グループ全体）
- **事業内容**: 商業銀行、リース、証券、クレジットカード、投資銀行等の総合金融サービス

## 2. AI導入の背景・課題
### ビジネス課題
- 社内情報約130万件の効率的活用
- 全社員の生産性向上とデジタルスキル強化
- 顧客体験(CX)の革新的向上
- 経営層の知見の全社共有（AI-CEO構想）
- 金融業務における迅速な意思決定支援

### 技術的課題
- 大規模RAG（Retrieval-Augmented Generation）システムの構築
- セキュアな生成AI環境の実現
- 既存業務システムとのシームレスな統合
- リアルタイムでの情報検索・生成機能

## 3. 導入したAIツール
### 主要ツール
1. **SMBC-GAI（社内開発生成AIアシスタント）**
   - 開発時期: 2023年7月
   - 基盤技術: Microsoft Teams統合、Azure OpenAI Service
   - 利用状況: 2024年3月時点で「2秒に1回」利用される頻度

2. **OpenAI API**
   - 契約時期: 2024年10月（日本の大手金融機関初）
   - 利用開始: 2024年9月
   - 対象: 全行員

3. **RAG技術統合SMBC-GAI**
   - 実装時期: 2025年10月
   - 対象データ: 社内情報約130万件（国内最大規模）

4. **AI-CEO（AI中島達CEO）**
   - 導入時期: 2025年8月5日
   - 基盤技術: GPT-4o + AIアバター
   - 対象: 三井住友銀行全従業員

5. **AI上司（開発中）**
   - 予定: 2025年度内試験提供開始
   - 機能: 顧客ニーズ推定、ソリューション提案

## 4. 導入プロセス
### 導入スケジュール
- **2023年7月**: SMBC-GAI開発・リリース
- **2024年3月**: 高頻度利用の実現（2秒に1回）
- **2024年9月**: OpenAI API利用開始
- **2024年10月**: OpenAI社との正式契約締結
- **2025年8月**: AI-CEO導入
- **2025年10月**: RAG技術統合（130万件）

### 実装方法
1. **段階的展開**
   - Phase 1: Microsoft Teams内でのパイロット運用
   - Phase 2: 全社員への展開
   - Phase 3: RAG技術統合による高度化

2. **セキュリティ重視の設計**
   - Azure OpenAI Serviceによるセキュアな環境
   - 社内データの外部流出防止

3. **ユーザー中心設計**
   - Microsoft Teams統合で利用障壁を最小化
   - 直感的なインターフェース

## 5. 主な機能・活用シーン
### SMBC-GAIの機能
1. **基本機能（2023年7月～）**
   - 文章作成・要約
   - 情報検索
   - アイデア出し
   - Microsoft Teams常駐型チャットボット

2. **RAG統合機能（2025年10月～）**
   - 社内情報約130万件からの高精度検索
   - コンテキストを考慮した回答生成
   - リアルタイム情報更新

### AI-CEOの活用
- 中島達CEO過去の発言データベース参照
- AIアバターによる対話形式
- 経営方針・戦略の全社共有
- 従業員との「気軽な相談」の疑似体験

### AI上司（開発中）
- 商品データ分析
- 顧客取引実績分析
- 銀行員の暗黙知活用
- 顧客ニーズ推定・ソリューション提案

## 6. 導入効果・成果
### 定量的効果
| 指標 | 実績 |
|------|------|
| 利用頻度 | 2秒に1回（2024年3月時点） |
| RAGデータ規模 | 約130万件（国内最大） |
| AI-CEO導入規模 | 三井住友銀行全従業員 |

### 定性的効果
- **生産性向上**: 文書作成・情報検索の大幅な時間短縮
- **意思決定高速化**: 社内130万件の情報への即座アクセス
- **経営層の知見共有**: AI-CEOによるトップの思考の民主化
- **従業員エンゲージメント向上**: 気軽なCEO相談が可能に
- **顧客体験向上**: 迅速で的確なソリューション提案

## 7. 技術スタック
### アーキテクチャ
- **クラウド基盤**: Microsoft Azure
- **LLMモデル**:
  - Azure OpenAI Service
  - GPT-4o（AI-CEO）
  - OpenAI API（各種GPTモデル）
- **RAG技術**:
  - ベクトルデータベース
  - 社内文書検索システム統合

### 統合システム
- **Microsoft Teams**: ユーザーインターフェース
- **社内文書管理システム**: 130万件の情報源
- **顧客情報データベース**: AI上司連携（開発中）
- **AIアバターシステム**: AI-CEO用
