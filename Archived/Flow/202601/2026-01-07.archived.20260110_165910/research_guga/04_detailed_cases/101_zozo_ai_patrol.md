## 5. ソリューション詳細

### 100本ノック研修
- 全社員対象の独自生成AI研修
- 約1,300名が受講
- 成果を上げている複数部門から収集した100件の実践的活用パターン
- オンライン形式で使用プロンプトと出力例を紹介・解説
- 活用率34%→78.8%への大幅向上に貢献

### アイテムレビューパトロール（2024年4月運用開始）
- AIがZOZOTOWN上のレビューを自動確認
- ガイドライン違反を検出する独自ツール
- 4か月間の運用で業務時間67.7%削減
- チェック件数68.5%削減を達成

### モデル画像検出機能
- ZOZOTOWN掲載アイテム画像からモデル着用画像を検出
- モデル身長と着用サイズが未入力の画像を特定
- ブランドへの個別依頼業務を年間約1,500時間削減見込み

### AIツール・基盤の活用
- **コーディングAI**: GitHub Copilot、Claude Code、Gemini CLI、Codex CLI
- **モデル基盤**: Amazon Bedrock、Gemini / Vertex AI Model Garden、OpenAI API
- 2024年に41件のツール・機能開発
- 2025年3月時点で59個の業務ツールが稼働
