# 640_641_MS&ADインシュアランスグループ / 損保ジャパン_CAT.AI / おしそんLLM

## 1. 企業概要
### MS&ADインシュアランスグループ
- **業種**: 損害保険
- **主要企業**: 三井住友海上、あいおいニッセイ同和、MS&ADグランアシスタンス

### 損保ジャパン
- **企業名**: 損害保険ジャパン株式会社
- **従業員数**: 約23,000人
- **事業内容**: 損害保険全般

## 2. AI導入の背景・課題
### MS&AD
- ロードサービス受付の業務効率化・品質向上
- 繁閑差に応じた人材確保の課題
- 応対の自動化ニーズ

### 損保ジャパン
- 照会回答業務の効率化（約60%の照会に対応）
- 全国営業店での情報検索支援
- 火災保険業務の効率化

## 3. 導入したAIツール
### MS&ADグループ
**CAT.AI CX-Bot** (2024年7月導入)
- ロードサービス分野で国内初導入
- ナビゲーション型AIボット
- ナンバープレート撮影で情報取得
- 位置情報マップ連携

### 損保ジャパン
1. **おしそんLLM** (2025年6月全国展開)
   - 社内専用大規模言語モデル
   - RAG検索精度: 約29% → 約71%に向上
   - 約60%の照会で約40%の作業時間削減

2. **AI Agent "Heylix"** (2024年4月導入)
   - AI insideが提供
   - 火災保険業務効率化
   - 固定資産台帳から情報抽出・転記を95%精度で自動化

3. **全社生成AIチャットシステム** (前年度リリース)
   - Azure OpenAI Service ChatGPT活用
   - 全社員利用可能

4. **TASUKI Annotation** (2024年度試行)
   - ソフトバンク提供
   - RAG検索精度向上支援

## 4. 導入プロセス
### MS&AD
- **2024年7月**: CAT.AI CX-Bot導入（3社共同）

### 損保ジャパン
- **前年度**: 全社生成AIチャット構築
- **2024年4月**: Heylix導入
- **2024年度**: TASUKI Annotation試行
- **2025年6月**: おしそんLLM全国展開

## 5. 主な機能・活用シーン
### MS&AD CAT.AI
- ナンバープレート撮影→自動情報取得
- マップ連携での位置特定
- 緊急時のナビゲート

### 損保ジャパン
- おしそんLLM: 社内マニュアル・Q&A自動検索・回答生成
- Heylix: 固定資産台帳からの情報抽出・転記自動化（95%精度）
- 全社チャット: 全社員の生成AI活用

## 6. 導入効果・成果
### MS&AD
- ロードサービス応対自動化
- 繁閑差への柔軟な対応
- 顧客体験向上

### 損保ジャパン
| 指標 | 成果 |
|------|------|
| RAG検索精度 | 約29% → 約71% |
| 照会作業時間削減 | 約40%（対応可能な60%の照会で） |
| Heylix転記精度 | 95% |

## 7. 今後の展開予定
- おしそんLLMの機能拡張
- Heylixの適用業務拡大
- 全社的なAI活用深化
