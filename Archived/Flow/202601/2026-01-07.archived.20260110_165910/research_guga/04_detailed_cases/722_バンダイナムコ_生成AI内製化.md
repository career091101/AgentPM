# 722_バンダイナムコホールディングス_生成AI内製化

## 1. 企業概要

**企業名**: 株式会社バンダイナムコホールディングス
**業種**: 総合エンターテインメント（ゲーム、玩具、アミューズメント）
**従業員規模**: 約24,000名（連結）
**主要事業**: バンダイナムコエンターテインメント、バンダイ、バンダイナムコアミューズメント

## 2. AI導入の背景・目的

### 背景
- 生成AI技術の急速な発展
- ゲーム開発における効率化ニーズ
- 社内のナレッジ蓄積・体系化の必要性
- メタバース時代を見据えた3Dキャラクターモーション生成の重要性

### 目的
- 生成AI活用に向けた社内ナレッジの蓄積・体系化
- ゲーム開発でのLLM（大規模言語モデル）実践的利用
- 開発環境の効率化（Cloud Workstations、Gemini Code Assist）
- AIを活用したキャラクターモーション生成

## 3. 採用したAIツール・技術

### AWS Generative AI Use Cases JP（2024年3月）
- **概要**: バンダイナムコアミューズメントが導入した社内向けユースケースサイト
- **技術**: AWS提供の生成AIユースケース集
- **機能**:
  - LLMと対話ができるチャット
  - RAG（検索拡張生成）チャット
  - 文章生成、画像生成
  - 要約、校正、翻訳
- **目的**: 今後の生成AI活用に向けた社内ナレッジ蓄積・体系化

### Google Cloud開発環境（2024年）
- **Cloud Workstations**: マネージドサービスによる運用効率化
- **Gemini Code Assist**: 生成AIを利用した開発効率化
- **Gemini Pro**: 情報漏洩リスクを抑えた低コストチャットボット（1時間で構築）
- **Looker Studio**: ダッシュボード構築

### バンダイナムコ研究所のAI技術（CEDEC 2024発表）
- **ゲーム実況AIキャラクター**: LLMを活用した自動実況
- **ゲーム内テキスト生成**: LLMによる動的テキスト生成
- **キャラクターモーション生成**: AIを活用した多彩なモーション作成（共同研究）

### バンダイナムコエンターテインメント（2025年8月）
- **Google Cloudの生成AI活用**: システム開発事例公開

## 4. 導入プロセス・期間

### フェーズ1: 社内環境整備（2024年3月）
- AWS Generative AI Use Cases JPの導入（バンダイナムコアミューズメント）
- 社内ユースケースサイトの構築
- 生成AI主要ユースケースの体験環境整備

### フェーズ2: 開発環境刷新（2024年）
- Google Cloud WorkstationsとGemini Code Assistの導入（バンダイナムコネクサス）
- Gemini ProとLooker Studioを活用した低コストチャットボット構築
- マネージドサービスによる運用効率化

### フェーズ3: ゲーム開発への適用（2024年）
- LLMのゲーム開発での実践的利用研究（バンダイナムコ研究所）
- CEDEC 2024での事例発表（2024年9月）
- LLM活用時の「5つの落とし穴」と対策の共有

### フェーズ4: 全社展開（2025年）
- バンダイナムコエンターテインメントでの生成AI活用事例公開（2025年8月）
- グループ横断的な生成AI活用推進

## 5. 導入効果・ROI

### 定量的効果
- **チャットボット構築時間**: 1時間で構築完了
- **開発環境運用コスト削減**: マネージドサービス活用により削減（具体的な数値は非公開）
- **低コスト運用**: 情報漏洩リスク抑制と低コストの両立

### 定性的効果
- 社内のナレッジ蓄積・体系化の実現
- エンジニア向け開発環境の刷新
- ゲーム開発における生産性向上
- LLM活用の課題と対策の体系化
- AIを活用したキャラクターモーション生成の実現

## 7. 直面した課題と解決策

### 課題1: LLM活用の「5つの落とし穴」（CEDEC 2024）
- **課題**: LLMのゲーム開発での実用化における課題
- **解決策**:
  1. 回答の一貫性の担保
  2. ハルシネーション対策
  3. コンテキスト管理
  4. レスポンス速度の最適化
  5. コスト管理
- **結果**: 実践的なLLM活用ガイドラインの確立

### 課題2: 社内ナレッジの蓄積
- **課題**: 生成AI活用のノウハウが属人化
- **解決策**: AWS Generative AI Use Cases JPの導入
- **結果**: 社内で体系的に生成AIユースケースを体験可能

### 課題3: 開発環境の運用負荷
- **課題**: 従来型開発環境の運用コストと負荷
- **解決策**: Cloud WorkstationsとGemini Code Assistの導入
- **結果**: マネージドサービスによる運用効率化
