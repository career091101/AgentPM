# 事例調査:キリンホールディングス株式会社(飲料・食品)

## 1. 基本情報

- **企業名**: キリンホールディングス株式会社
- **業界**: 飲料・食品製造業
- **事業規模**: 連結売上収益2兆2,287億円(2023年12月期)
- **従業員数**: 連結約28,000名
- **導入AIツール**:
  - KIRIN BuddyAI(独自構築の業界特化型生成AI)
  - Azure OpenAI Service(基盤技術)
  - Vendy(ソフトバンク開発のAI自動販売機最適化サービス)

## 2. AI導入サマリー

キリンホールディングスは「KIRIN BuddyAI Project」として、業務担当者が容易かつ迅速に生成AIを活用できる独自の社内環境を構築。2024年11月からマーケティング領域で先行導入し、100種類を超える業務特化型プロンプトテンプレートを実装することで、年間約3万9,000時間の時間創出を見込んでいる。2025年末までに国内全従業員約1万5,000人への展開を目指し、最終的には年間約31万時間の時間創出を計画。自動販売機事業では、AIを活用したオペレーション最適化サービスを2025年9月までに約8万台へ導入予定。

## 3. 定量的効果

### KIRIN BuddyAI導入効果

**マーケティング領域(約400名)**:
- **時間創出**: 年間約3万9,000時間(初期見込み2万9,000時間から増加)
- **プロンプトテンプレート**: 100種類超(7カテゴリ)
- **導入時期**: 2024年11月先行導入

**全社展開目標(国内約1万5,000名)**:
- **時間創出**: 年間約31万時間
- **アクティブユーザー率**: 70%前後で安定推移(月1回以上利用)
- **展開完了予定**: 2025年末

### 自動販売機AI「Vendy」

- **導入台数**: 約8万台(2025年9月までに順次)
- **導入開始**: 2024年10月
- **効果**: オペレーション効率化、商品ラインアップ最適化

## 4. 導入背景・課題

### ビジネス環境の変化
1. **DX推進の必要性**: 長期経営構想「KV2027」における「価値創造を加速するICT」の推進
2. **業務効率化の要請**: グローバル競争激化による生産性向上の必要性
3. **人材不足**: 国内人口減少に伴う労働力不足への対応

### 組織課題
1. **業務の属人化**: 専門知識が個人に集中し、ナレッジ共有が不十分
2. **マーケティング業務の負荷**: リサーチ、分析、企画立案等の時間的制約
3. **自動販売機運営の非効率**: 補充・ルート配送の最適化不足

## 5. ソリューション詳細

### KIRIN BuddyAI Project

**技術基盤**:
- Azure OpenAI Serviceを活用した社内環境構築
- セキュアなクローズド環境(データは学習に利用されない)
- 業務領域特化型プロンプトテンプレート実装

**7つのカテゴリ×100種類超のテンプレート**:
1. **情報収集**: 市場情報、競合分析等
2. **市場環境分析**: トレンド分析、消費者動向把握
3. **マーケティング戦略立案**: ブランド戦略、商品企画
4. **エグゼキューション開発**: 施策立案、プロモーション企画
5. **製品開発支援**: 商品コンセプト検討、パッケージデザイン
6. **お客さま理解**: 顧客インサイト分析、ペルソナ作成
7. **汎用業務**: メール作成、文書要約、翻訳等

**特徴**:
- 業務データとの連携機能
- ユーザー教育プログラムの提供
- 継続的な機能改良とテンプレート追加

### Vendy(自動販売機AI)

**開発元**: ソフトバンク
**導入対象**: キリンビバレッジの自動販売機約8万台

**主要機能**:
1. **オペレーション業務効率化**:
   - 補充最適化
   - ルート配送計画の自動化
   - 在庫管理の高度化

2. **商品ラインアップ最適化**:
   - 設置先の特性に応じた品揃え提案
   - 季節・時間帯別の需要予測
   - 売上最大化のための商品配置最適化

**導入スケジュール**:
- 2024年10月: 導入開始
- 2025年9月: 約8万台への導入完了予定

### AIペルソナ(商品開発支援)

**対象商品**: 「キリン 氷結」ブランド等のRTD商品
**目的**: お客様理解の質向上と新たな価値創造

**機能**:
- 生成AIによる顧客ペルソナ構築
- 商品コンセプト検証
- ターゲット層のニーズ分析

## 6. 導入プロセス

### KIRIN BuddyAI導入ステップ

**Phase 1: 準備・構築(2024年前半)**
1. Azure OpenAI Service基盤構築
2. セキュリティポリシー策定
3. プロンプトテンプレート開発(7カテゴリ)

**Phase 2: 先行導入(2024年11月〜)**
1. マーケティング領域約400名への展開
2. ユーザー教育・トレーニング実施
3. フィードバック収集と機能改良

**Phase 3: 全社展開(2025年)**
1. 営業部門への展開
2. R&D部門への展開
3. その他部門への順次展開
4. 2025年末に国内約1万5,000人へ完全展開

### 人材育成の取り組み

**AVILEN社との連携**:
- 生成AI研修プログラム提供
- 効果: 月2,200時間の労働時間削減に成功
- 社員のAI活用スキル向上

**MANA社による支援**:
- 全社DX推進ロードマップ策定から実行まで支援
- 1万5,000人への展開計画策定

## 7. 成功要因分析

### 1. 業務特化型アプローチ
- 汎用生成AIではなく、業務領域ごとに100種類超のテンプレートを用意
- 社員が即座に使える実用的な設計
- 7カテゴリで網羅的に業務をカバー

### 2. 段階的展開戦略
- マーケティング(約400名)→営業・R&D→全社(1万5,000名)の順次拡大
- 先行部門での効果検証と改善を経て展開
- 過度な一斉導入を避けたリスク管理

### 3. 経営層のコミットメント
- 長期経営構想「KV2027」での位置づけ
- 全社横断的なDX推進体制
- 明確な時間創出目標(年間31万時間)

### 4. 継続的な改善文化
- ユーザーフィードバックによるテンプレート追加
- 月1回以上利用率70%という高いエンゲージメント維持
- 機能改良とユーザー教育の両輪

## 8. 課題・対策

### 現在の課題

**技術面**:
- プロンプトの品質向上
- 業務データ連携の拡充
- 生成結果の精度改善

**組織面**:
- 全社展開における教育リソースの確保
- 部門間での活用レベルの差
- 既存業務プロセスとの統合

### 対策

**技術改善**:
- テンプレートの継続的なブラッシュアップ
- ユーザーの使用ログ分析による改善
- 外部パートナー(AVILEN、MANA等)との協業継続

**組織対応**:
- 部門別のアンバサダー育成
- 成功事例の社内共有強化
- 業務プロセス改革との同時推進

3. 導入後のサポート不足(継続的な教育・改善が鍵)
4. 効果測定の曖昧さ(時間創出等の定量的KPIが必要)
