# 670_INPEX_Microsoft_365_Copilot

## 1. 企業名・業界
- **企業名**: 株式会社INPEX
- **業界**: 石油・天然ガス開発業
- **企業規模**: 大企業（従業員数: 約3,200人）

## 2. 導入AIツール名・提供企業
- **ツール名**:
  - Microsoft 365 Copilot
  - AIR ChatGPT（社内生成AIシステム）
  - Windows 11（AI活用基盤）
- **提供企業**:
  - Microsoft（Microsoft 365 Copilot、Azure OpenAI Service）
  - HP（デバイス・システム導入支援）
- **AI分類**: 生成AI

## 3. 導入目的・解決すべき課題
- 業務の高度効率化
- モダンデバイス管理
- 強固なセキュリティの確保
- デジタル化の加速
- 責任あるエネルギー・トランジション推進

## 4. 導入プロセス
1. **2024年**: 社内生成AI推進チーム「AIR」立ち上げ
2. **2024年**: 有償Microsoft 365 Copilot 650ライセンス導入（日本国内従業員の約1/3）
3. **2022年**: Windows 11早期移行プロジェクト検討開始
4. **2024年1月**: ノートPC 2,000台超調達
5. **2024年3月**: キッティング開始
6. **2025年度**: 海外拠点へのWindows 11展開開始

## 5. 具体的な活用内容
### Microsoft 365 Copilot
- **導入規模**: 650ライセンス（日本国内従業員の約1/3）
- **主な機能**:
  - Microsoft TeamsでのAIメモ
  - メールのドラフティング
  - 資料のサマリー作成
- **効果**: 業務効率化

### AIR ChatGPT
- **基盤**: Azure OpenAI Service
- **特徴**: セキュアな環境でGPT利用可能
- **展開**: 社内全体に展開
- **名称由来**: 「AIが空気のように、自然にある職場へ」

### ミニアプリ開発
- **開発者**: インターン社員（日本郵便、ゆうちょ銀行、かんぽ生命保険）
- **開発数**: 20個のミニアプリ作成

### Windows 11展開
- **調達**: 2024年1月にノートPC 2,000台超
- **キッティング開始**: 2024年3月
- **海外展開**: 2025年度から海外拠点への展開

## 6. 技術的詳細
- **Microsoft 365 Copilot**: Azure OpenAI Service基盤
- **AIR ChatGPT**: Azure OpenAI Service活用
- **Windows 11**: AI活用の基盤OS

- **Microsoft 365 Copilot**: 650ライセンス導入
