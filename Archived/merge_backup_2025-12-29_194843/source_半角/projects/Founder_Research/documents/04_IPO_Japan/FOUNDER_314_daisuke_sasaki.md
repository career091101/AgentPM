## 2. 創業ストーリー

### 2.1 課題発見（需要発見）

**着想源**:
- Google時代、中小企業のオンライン広告支援で全国を飛び回る
- 「中小企業の経営者は、会計・経理に膨大な時間を取られている」ことに気づく
- 自身も副業で確定申告を経験し、「なぜこんなに複雑なのか？」と痛感
- 日本の会計ソフトは「簿記を知っている人」前提で、非常に使いにくい
- 「簿記を知らなくても使える会計ソフト」があれば、中小企業の生産性が劇的に向上するのでは？

**需要検証方法**:
- 個人事業主・中小企業150社にヒアリング
- 「会計・経理に困っているか？」→ 90%が「困っている」
- 「簿記を知っているか？」→ 70%が「知らない」
- 「簡単に使える会計ソフトがあれば使うか？」→ 85%が「使いたい」
- フリーランス1,000人にアンケート
  - 確定申告に要する時間: 平均20時間/年
  - 税理士に依頼する費用: 平均10-30万円/年

### 2.2 CPF検証（Customer Problem Fit）

**インタビュー/顧客検証**:
- 実施数: 150社以上（個人事業主・中小企業）
- 手法: 訪問インタビュー、オフィス観察、確定申告同行
- 発見した課題の共通点:
  - **個人事業主**: 「確定申告が面倒」「簿記が分からない」「時間がない」
  - **中小企業**: 「経理担当者の採用が困難」「会計ソフトが使いにくい」「税理士費用が高い」

**3U検証**:
- Unworkable（現状では解決不可能）:
  - 既存の会計ソフト（弥生、TKC等）は簿記知識が必須
  - 手作業でExcel管理は限界（時間がかかる、ミスが多い）
  - 税理士に依頼すると費用が高い（年間10-30万円）
- Unavoidable（避けられない）:
  - 確定申告は法的義務（個人事業主・中小企業にとって避けられない）
  - 決算書・試算表は経営判断に必須
  - 税務調査リスク（正確な記帳が必要）
- Urgent（緊急性が高い）:
  - 確定申告期限（毎年3月15日）は絶対
  - 経理業務の遅れは資金繰り悪化に直結
  - 経営判断の遅れは競争力低下に

**支払い意思（WTP）**:
- 確認方法: 有料ベータ版を100社に提供
- 結果:
  - 月額980円で80社が契約
  - 年間契約率70%（継続意思の確認）
  - 「税理士費用の1/10で済む」との声が多数

### 2.3 PSF検証（Problem Solution Fit）

**10倍優位性**:

| 軸 | 従来の解決策 | 自社ソリューション | 倍率 |
|---|------------|-----------------|------|
| 経理作業時間 | 手作業（月20時間） | 自動化（月2時間） | 10倍 |
| 簿記知識 | 必須（簿記3級以上） | 不要（誰でも使える） | ∞倍 |
| 導入時間 | 数日（設定複雑） | 数分（銀行連携のみ） | 100倍 |
| 自動化率 | 手動入力（100%） | 自動連携（80%） | 5倍 |
| 税理士費用 | 年間10-30万円 | 月額980円（年1.2万円） | 10-25倍 |

**MVP**:
- タイプ: Prototype（銀行口座連携 + 自動仕訳）
- 初期反応:
  - ベータ版100社、うち80社が有料契約
  - NPS（推奨度）: 80（業界平均20-30）
  - 「これがないと確定申告できない」との声多数
- CVR: 初期15% → 改善後30%（2年後）

**UVP（独自の価値提案）**:
- 「スモールビジネスを、世界の主役に。」
- 簿記知識ゼロでも使える「誰でも会計」
- 銀行口座・クレジットカード連携で自動仕訳
- 確定申告書の自動生成

**競合との差別化**:
- 既存会計ソフト（弥生、TKC等）との差別化:
  - 簿記知識不要（UI/UX設計が革命的）
  - クラウド型（インストール不要、どこでもアクセス）
  - 自動連携（銀行・クレカ・電子マネー等）
- 税理士との差別化:
  - 圧倒的低価格（月額980円 vs 年間10-30万円）
  - リアルタイム（税理士は月1回報告）
  - セルフサービス（自分で経営状況を把握）

## 3. ピボット/失敗経験

### 3.1 初期の失敗

- **UX設計の試行錯誤（2012-2013年）**:
  - 当初、「簿記を知らない人でも使える」UIを実現できず
  - ユーザーテスト50回以上実施し、徹底的に改善
  - 「仕訳」という言葉を使わず、「支出」「収入」という日常語に

- **銀行連携の技術的困難（2013年）**:
  - 日本の銀行は API を提供しておらず、スクレイピングで対応
  - セキュリティ・安定性の問題で苦戦
  - 3ヶ月かけて全国主要銀行との連携を実現

- **価格設定の試行錯誤（2013-2014年）**:
  - 初期価格: 月額500円 → 安すぎて価値が伝わらない
  - 月額980円に値上げ → 解約率低下、満足度向上
  - 「安すぎると『本当に使えるのか？』と疑われる」学び

### 3.2 ピボット（該当する場合）

- ピボットは実施していない
- 一貫して「簿記知識ゼロでも使える会計ソフト」を追求

## 4. 成長戦略

### 4.1 初期トラクション獲得

- **個人事業主向けマーケティング（2012-2014年）**:
  - SEO/SEM集中投資（「確定申告 簡単」「会計ソフト おすすめ」で上位獲得）
  - Google Ads の ROAS 300-500%を維持
  - 確定申告シーズン（1-3月）に集中マーケティング

- **口コミ戦略**:
  - 「確定申告が3時間で終わった！」という熱狂的ファンを獲得
  - 紹介プログラム（紹介で1ヶ月無料）
  - NPS 80を維持し、オーガニックグロース

- **会計事務所との提携**:
  - 税理士・会計士との提携プログラム
  - 顧問先に freee を推奨してもらう

### 4.2 フライホイール

1. **簡単なUI/UXで個人事業主獲得**
   → 2. **口コミで新規ユーザー増加**
   → 3. **データ蓄積で自動仕訳精度向上**
   → 4. **さらに使いやすくなる**
   → 5. **中小企業にも拡大**
   → 6. **会計事務所との提携拡大**
   → 1に戻る

- **ネットワーク効果**:
  - 利用者が増えるほど、自動仕訳の精度が向上（機械学習）
  - 会計事務所との提携が増えるほど、顧問先への導入が進む

### 4.3 スケール戦略

- **中小企業進出（2014年〜）**:
  - 個人事業主向け「freee会計」に加え、中小企業向け「freee会計プラス」
  - 人事労務（給与計算）、請求書管理、経費精算等の機能追加

- **周辺領域への拡大（2016年〜）**:
  - freee人事労務（給与計算・社会保険）
  - freee申告（確定申告・法人税申告）
  - freee会社設立（起業支援）

- **エンタープライズ進出（2018年〜）**:
  - 大企業向けプラン（セキュリティ強化、API連携）
  - 会計事務所向けプラン（顧問先管理機能）

- **海外展開（2020年〜）**:
  - 東南アジア（シンガポール、インドネシア）で展開準備

### 4.4 バリューチェーン

```
顧客（銀行口座・クレカ連携）
  → freee（自動仕訳・データ蓄積）
  → AI/機械学習（仕訳精度向上）
  → 顧客（試算表・確定申告書自動生成）
  → 税理士（顧問サポート）
```

- **自動仕訳の仕組み**:
  - 銀行口座・クレジットカード・電子マネーと自動連携
  - AI/機械学習で取引を自動分類
  - ユーザーが修正すると、次回から学習して精度向上

### 6.3 差別化要因
