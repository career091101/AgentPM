## 概要

「インタビューを記事にするニュースレターが人気を独占する理由」を分析。対談・インタビュー形式のコンテンツで読者エンゲージメントを最大化する戦略。

---

## 核心となるフレームワーク

### インタビューNLの5つの優位性

| # | 優位性 | 説明 |
|---|--------|------|
| 1 | **権威性の借用** | 著名人の知見を自分の媒体で発信 |
| 2 | **ネットワーク効果** | ゲストのファンベースにリーチ |
| 3 | **コンテンツの差別化** | 一次情報のみ発信可能 |
| 4 | **関係構築** | 業界人脈の拡大 |
| 5 | **継続性** | ゲストを変えることで無限のネタ |

### インタビュー記事の構成パターン

```
┌─────────────────────────────────────┐
│ 1. ゲスト紹介（実績・背景）         │
│    → 読者に「読む価値」を示す       │
├─────────────────────────────────────┤
│ 2. キーインサイト（核心部分）       │
│    → 最も価値のある発言を先出し     │
├─────────────────────────────────────┤
│ 3. 対談詳細（Q&A形式）              │
│    → 読みやすく編集                 │
├─────────────────────────────────────┤
│ 4. 読者への示唆（アクションアイテム）│
│    → 「明日から使える」を意識       │
└─────────────────────────────────────┘
```

---

## 具体的な実践ステップ

### Phase 1: ゲスト獲得
1. Xで業界のキーパーソンをリストアップ
2. 価値提供型のアプローチ（プロモーション提案）
3. 既存ゲストからの紹介連鎖

### Phase 2: インタビュー実施
1. 事前リサーチ徹底（過去発言、著作）
2. オープンクエスチョン中心
3. 録音→文字起こし→編集

### Phase 3: コンテンツ化
1. キーインサイトを冒頭に配置
2. 読みやすいQ&A形式に編集
3. ゲストのSNSでシェアを依頼
