# Customer Feedback Command

**コマンド**: `/customer-feedback`

**説明**: NPS・コホート・センチメント分析、フィードバック優先順位付け

**エージェント**: Customer Feedback Agent

---

## 使用方法

```
/customer-feedback
```

または

```
NPSを分析してください
フィードバックの優先順位を付けてください
```

---

## 実行内容

1. **データ読み込み・検証**: フィードバックデータ（CSV/JSON/API）の読み込み、欠損値処理
2. **NPS計算**: Net Promoter Score算出、推奨者・批判者の特定、セグメント別NPS
3. **コホート分析**（オプション）: 登録月別・プラン別・流入元別の満足度・チャーン率分析
4. **センチメント分析**（オプション）: フィードバックの感情分析（ポジティブ/ネガティブ/ニュートラル）
5. **優先順位付け**: 影響度・緊急度に基づく改善項目のランキング
6. **レポート生成**: nps_report.md、cohort_analysis.json、sentiment_analysis.json、priority_feedback.md

---

## 入力パラメータ

1. **フィードバックデータ**（必須）: CSVファイルパスまたはAPI接続
2. **分析タイプ**（必須）: nps / cohort / sentiment / prioritization / all
3. **NPS基準値**（オプション）: ForSolo: 40、ForStartup: 50、ForRecruit: 60（デフォルト: 50）
4. **コホート定義**（オプション）: registration_month / plan / source（デフォルト: registration_month）
5. **センチメントカテゴリ**（オプション）: UI/UX、機能、価格、サポート等（デフォルト: 自動抽出）

## 実行例

### 例1: NPS分析 + 優先順位付け

```
ユーザー: /customer-feedback

エージェント: フィードバックデータのファイルパスを教えてください
ユーザー: Flow/202601/2026-01-03/feedback_data.csv

エージェント: 分析タイプを選択してください
ユーザー: nps, prioritization

エージェント: NPS基準値を選択してください
ユーザー: 50（ForStartup基準）

エージェント: フィードバック分析中...

[25分後]

エージェント: ✅ フィードバック分析完了

**全体NPS**:
- **NPSスコア**: 45（目標50、達成率90%）
- **推奨者**: 55%（550人）
- **中立者**: 30%（300人）
- **批判者**: 15%（150人）
- **総回答数**: 1,000人

**セグメント別NPS**:
| セグメント | NPS | 推奨者% | 批判者% | 回答数 |
|-----------|-----|---------|---------|--------|
| 無料プラン | 30 | 40% | 10% | 400 |
| 有料プラン | 60 | 70% | 10% | 600 |

**推奨者の主なコメント**:
1. 「使いやすい、直感的」（85人）
2. 「機能が豊富」（62人）
3. 「サポートが手厚い」（48人）

**批判者の主なコメント**:
1. 「価格が高い」（42人）
2. 「動作が遅い」（38人）
3. 「機能が不足」（25人）

**フィードバック優先順位**:

### 高優先度（即座に対応）
1. **動作が遅い**（優先度スコア: 85）
   - 影響人数: 380人（38%）
   - センチメント: -0.72（非常にネガティブ）
   - 推奨対処: パフォーマンス改善（バックエンド最適化）
   - 期待効果: チャーン率-5%、NPS +8ポイント

2. **価格が高い**（優先度スコア: 78）
   - 影響人数: 280人（28%）
   - センチメント: -0.65（ネガティブ）
   - 推奨対処: 無料プラン機能拡充、年間プラン割引
   - 期待効果: コンバージョン率+15%、NPS +5ポイント

### 中優先度（次のスプリントで対応）
3. **〇〇機能がない**（優先度スコア: 62）
   - 影響人数: 180人（18%）
   - 推奨対処: ロードマップに反映
   - 期待効果: リテンション+8%

**次のアクション**:
1. パフォーマンス改善を即座に着手
2. 価格戦略の見直し（無料プラン強化）
3. NPS目標達成に向けた施策実行

レポート:
- Flow/202601/2026-01-03/feedback/nps_report.md
- Flow/202601/2026-01-03/feedback/priority_feedback.md
```
