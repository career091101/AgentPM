# ForSolo Design Micro-SaaS Model Command

このコマンドはForSolo特化版のMicro-SaaS収益化モデル設計スキルを実行します。

## 実行内容

`/design-micro-saas-model` スキル（ForSolo Edition）を起動し、以下を自動実行:

1. 価格設定戦略（月額$10-$50の低価格帯）
2. ティアリング設計（Free → Starter → Pro）
3. MRRロードマップ作成（$1K → $5K → $10K）
4. アップセル/クロスセル戦略
5. コスト構造最適化（月$50以下）

## ForSolo特化要素

- **価格帯**: 月額$10-$50（ソロプレナー向け低価格戦略）
- **LTV/CAC**: 10.0以上（広告費ゼロ前提）
- **利益率**: 80%以上（コスト最小化）
- **MRRフェーズ**:
  - Phase 1: $1K（3-6ヶ月）
  - Phase 2: $5K（6-12ヶ月）
  - Phase 3: $10K+（12ヶ月以降、フルタイム転換）
- **ツールコスト**: 月$50以下（Vercel、Supabase無料枠活用）
- Solopreneur Research統合: 85件 + Tier 2ケーススタディ13件

## 使用タイミング

- PMF検証完了後
- 収益化戦略を本格化したい
- MRR $1K達成を目指す

## 出力

- `Stock/programs/創業支援・新規事業開発（AIエージェント）/projects/Founder_Agent_ForSolo/documents/solopreneur/micro_saas_model.md`

## 次のコマンド

設計完了後:
- `/for-solo-create-bip-strategy` - Build in Public戦略作成へ
- `/for-solo-build-flywheel` - 成長フライホイール構築へ
- `/for-solo-measure-aarrr` - AARRR測定へ
