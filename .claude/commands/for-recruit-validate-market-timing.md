# for-recruit-validate-market-timing

市場タイミングの適切性を5次元評価し、「早すぎる（Too Early）」「遅すぎる（Too Late）」リスクを判定します。

## 使い方

```
/for-recruit-validate-market-timing
```

## 実行内容

@/Users/yuichi/AIPM/aipm_v0/.claude/skills/for_recruit/validate-market-timing/SKILL.md を参照し、以下を実行：

1. プロジェクト情報のヒアリング（対話形式）
2. 5次元市場タイミング評価（技術、顧客、競合、規制、成長）
3. 早すぎる vs 遅すぎるの判定
4. タイミングリスク軽減策の提示
5. 評価レポート（YAML形式）の出力

## 評価次元

| 次元 | 評価ポイント | スコア範囲 |
|-----|------------|----------|
| **技術成熟度** | 必要技術の実用段階、コスト、リスク | 0-10点 |
| **顧客準備度** | 課題認識、解決策探索、支払意思 | 0-10点 |
| **競合状況** | 先行者有無、市場シェア分布、後発参入余地 | 0-10点 |
| **規制・社会環境** | 規制障壁、社会的受容性、法制度変化見込み | 0-10点 |
| **市場成長性** | 成長率、TAM拡大見込み、持続性 | 0-10点 |

**総合スコア**: 5次元の平均（0-10点）

## 判定基準

| スコア範囲 | タイミング評価 | リスクレベル |
|-----------|--------------|------------|
| **8.0-10.0** | 適切なタイミング（Sweet Spot） | Green |
| **6.0-7.9** | 概ね適切（一部リスクあり） | Yellow |
| **4.0-5.9** | タイミングリスク高い | Orange |
| **0.0-3.9** | 不適切なタイミング | Red |

## データソース

- Airレジ v3調査（適切なタイミング成功事例）
- スタディサプリ v3調査（適切なタイミング成功事例）
- エリクラ v3調査（早すぎる失敗事例）
- CAREER CARVER（遅すぎる失敗事例）
- integrated_analysis_report.md（市場タイミング統計21%）

## 所要時間

20-40分（対話形式、評価レポート生成含む）
