# ForSolo Create BIP Strategy Command

このコマンドはForSolo特化版のBuild in Public戦略作成スキルを実行します。

## 実行内容

`/create-bip-strategy` スキル（ForSolo Edition）を起動し、以下を自動実行:

1. X/Twitter戦略設計（透明性重視）
2. フォロワー獲得計画（1K → 10K）
3. コンテンツカレンダー作成
4. エンゲージメント施策
5. コミュニティ構築戦略

## ForSolo特化要素

- **X/Twitterフォロワー目標**:
  - 6ヶ月: 1K
  - 24ヶ月: 10K
- **投稿頻度**: 週5-7回（進捗、失敗、学び）
- **透明性**: 収益、MRR、コスト構造の公開
- **エンゲージメント率**: 5%以上
- **コンバージョン**: フォロワー→初期ユーザー変換率10%
- Solopreneur Research統合: 85件 + Tier 2ケーススタディ21件

## 使用タイミング

- PMF検証完了後
- マーケティング戦略を本格化したい
- MRR $1K達成を目指す

## 出力

- `Stock/programs/創業支援・新規事業開発（AIエージェント）/projects/Founder_Agent_ForSolo/documents/solopreneur/bip_strategy.md`

## 次のコマンド

戦略作成後:
- `/for-solo-create-content-flywheel` - コンテンツフライホイール作成へ
- `/for-solo-build-flywheel` - 成長フライホイール構築へ
- `/for-solo-measure-aarrr` - AARRR測定へ
