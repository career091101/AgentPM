# Analytics Dashboard Command

**コマンド**: `/analytics-dashboard`

**説明**: KPIダッシュボード・A/Bテスト結果の自動分析

**エージェント**: Analytics Dashboard Agent

---

## 使用方法

```
/analytics-dashboard
```

または

```
KPIダッシュボードを生成してください
A/Bテスト結果を分析してください
```

---

## 実行内容

1. **データ読み込み・検証**: 分析データ（CSV/JSON/API）の読み込み、欠損値・異常値処理
2. **KPI計算**: AARRR指標の自動計算（Acquisition、Activation、Retention、Revenue、Referral）
3. **ダッシュボード生成**: インタラクティブHTMLダッシュボード作成（Plotly.js使用）
4. **A/Bテスト分析**（オプション）: 統計的有意性検定、効果量計算、推奨アクション提示
5. **トレンド分析**（オプション）: 移動平均、季節性分解、異常値検出、成長率計算
6. **予測モデル構築**（オプション）: Prophet使用、次30-180日の予測、Pythonコード自動生成

---

## 入力パラメータ

1. **分析データ**（必須）: CSVファイルパスまたはAPI接続
2. **KPI定義**（必須）: acquisition / activation / retention / revenue / referral / all
3. **A/Bテスト設定**（オプション）: テスト名、バリアントA/B、指標、期間
4. **ダッシュボードタイプ**（オプション）: html / pdf / notion（デフォルト: html）
5. **トレンド分析期間**（オプション）: 7days / 30days / 90days / 12months（デフォルト: 30days）
6. **予測期間**（オプション）: 7-180日（デフォルト: 30日）

### 例2: A/Bテスト結果分析

```
ユーザー: ボタン色のA/Bテスト結果を分析してください

エージェント: 分析データのファイルパスを教えてください
ユーザー: Flow/202601/2026-01-03/ab_test_data.csv

エージェント: A/Bテスト設定を入力してください
テスト名: ボタン色変更
バリアントA: 赤
バリアントB: 緑
指標: ctr
期間: 2026-01-01 ~ 2026-01-15

エージェント: A/Bテスト分析中...

[15分後]

エージェント: ✅ A/Bテスト結果分析完了

**結果サマリー**:
- **勝者**: 緑ボタン（統計的有意差あり、p < 0.01）
- **CTR改善**: 赤 3.2% → 緑 4.1%（+28%改善）
- **信頼区間**: [+18%, +38%]（95%信頼区間）
- **効果量**: Cohen's d = 0.45（中程度の効果）
- **サンプルサイズ**: 赤 1,250人、緑 1,280人（十分）

**統計分析**:
| 指標 | バリアントA | バリアントB | 改善率 | p値 | 統計的有意 |
|------|-----------|-----------|--------|-----|----------|
| CTR | 3.2% | 4.1% | +28% | 0.003 | ✅ 有意 |
| コンバージョン率 | 1.5% | 1.8% | +20% | 0.08 | ❌ 有意でない |

**推奨アクション**:
1. **即座に緑ボタンに切り替え**（CTRで統計的有意な改善、実務的にも意味あり）
2. **コンバージョン率は継続観察**（サンプルサイズ拡大で有意差が出る可能性）
3. **次のA/Bテスト**: ボタンサイズ最適化（大 vs 中 vs 小）

**レポート**:
- Flow/202601/2026-01-03/ab_test/ab_test_report.md
- Flow/202601/2026-01-03/ab_test/ab_test_stats.json
```

- **季節性**: 12月、6月にピーク（年末・中間決算期）

**成長率**:
- 前月比: +12%（目標+10%達成）
- 前年同月比: +180%（急成長）
- CAGR: +145%/年

**予測（次6ヶ月）**:
| 月 | 予測MRR | 95%信頼区間 |
|----|---------|-----------|
| 2026-01 | $12,000 | [$10,500, $13,500] |

| 2026-05 | $21,000 | [$17,400, $24,600] |

**モデル評価**:
