# Command: /for-recruit-inventory-internal-resources

## Trigger

```
/for-recruit-inventory-internal-resources
```

## Description

企業内新規事業において、既存の社内リソース6カテゴリ（既存顧客基盤、営業チャネル、ブランド力、技術インフラ、人的リソース、データ資産）を体系的に棚卸しし、新規事業へのシナジー効果を定量化します。

リクルート製品研究（31製品）から、成功製品の85%が既存顧客基盤を活用し、リソース3種以上活用でROI 1000%以上達成（例: Airペイ ROI 11,450%）という知見に基づいています。

## Usage

```
/for-recruit-inventory-internal-resources
```

実行すると、以下の対話形式で新規事業のリソース棚卸しを行います:

1. 新規事業アイデアの入力（事業名、ターゲット、課題、ソリューション）
2. ビジネスモデルの入力（月額SaaS、決済手数料、初期費用等）
3. 既存のリクルート製品・サービスとの関連性確認
4. リソース6カテゴリの自動棚卸し・定量化
5. 総合ROI計算（コスト削減効果 + 売上増加効果）
6. リソース活用戦略の提案（Phase 1-3、KPI設定）
7. リスク評価と対策

## Output

- **棚卸しレポート**: Markdown形式、リソース6カテゴリ詳細、ROI計算、活用戦略
- **Quality Gate**: リソース活用数3種以上、ROI 1000%以上を推奨
- **Next Actions**: 具体的なアクションプラン

## Examples

### Example 1: Airスケジュール（美容室向け予約管理システム）

```
新規事業: Airスケジュール
ターゲット: 美容室・サロン 10万店舗
課題: 予約管理の手作業、予約ミス、ダブルブッキング
ソリューション: クラウド予約管理システム、顧客管理、売上分析

【リソース活用】
- 既存顧客基盤: ホットペッパービューティー10万店舗（クロスセル率15% → 1.5万店舗）
- 営業チャネル: ホットペッパービューティー営業500名（CAC削減 5万円 → 1万円）
- ブランド力: リクルート + ホットペッパービューティー（CAC削減40%）
- 技術インフラ: Airペイ決済基盤転用（開発コスト4,500万円削減）
- 人的リソース: Ring制度社内公募5-10名（採用コスト3,500万円削減）
- データ資産: ホットペッパービューティー予約データ（需要予測精度向上）

【ROI計算】
- コスト削減効果: 26.57億円
- 売上増加効果: 7.2億円
- 投資額: 1.3億円
- ROI: 2,498%（約25倍）
```

### Example 2: Airペイ（実際の事例）

```
新規事業: Airペイ
ターゲット: 小規模店舗（Airレジ既存顧客）
課題: キャッシュレス対応の高額初期費用（数十万円〜百万円）
ソリューション: 初期費用0円、決済手数料2.48-3.74%のキャッシュレス決済

【リソース活用】
- 既存顧客基盤: Airレジ90.4万顧客（クロスセル率57% → 51.5万加盟店）
- 営業チャネル: ホットペッパーグルメ営業2,000名（CAC削減 8万円 → 2万円）
- ブランド力: リクルート（CAC削減40%）
- 技術インフラ: リクルート決済基盤転用（開発コスト5,000万円削減）

【ROI計算】
- 投資: 1,000万円
- 効果: 11.45億円/年（クロスセル3億円 + 開発コスト削減5,000万円 + CAC削減3億円 + 手数料収益5.15億円）
- ROI: 11,450%（約115倍）
```
