# ForSolo Validate 10x Command

このコマンドはForSolo特化版の10倍優位性検証スキルを実行します。

## 実行内容

`/validate-10x` スキル（ForSolo Edition）を起動し、以下を自動実行:

1. 競合比較分析（機能、価格、UX）
2. 10倍優位性軸の特定
3. 1人実行可能性による差別化評価
4. 10倍検証スコア算出
5. 差別化戦略提案

## ForSolo特化要素

- **10倍優位性軸**: 2軸以上（ForStartup 3軸より緩和）
- **優先軸**:
  - 1人実行可能性（コスト最小化、スピード）
  - UX/シンプルさ（複雑な機能より使いやすさ）
  - ニッチ特化（大手が参入しない領域）
- **価格優位性**: 競合の1/2-1/3（月額$10-$50）
- Solopreneur Research統合: 85件

## 使用タイミング

- CPF検証完了後
- 競合調査完了後
- 差別化戦略を明確にしたい

## 出力

- `Stock/programs/創業支援・新規事業開発（AIエージェント）/projects/Founder_Agent_ForSolo/documents/2_discovery/10x_validation.md`

## 次のコマンド

10倍達成時:
- `/for-solo-validate-psf` - PSF検証へ
- `/for-solo-create-persona` - ペルソナ詳細化へ

要改善時:
- `/for-solo-research-competitors` - 競合再分析
- `/for-solo-discover-demand` - 別の課題を探索
