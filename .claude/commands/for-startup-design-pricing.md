# ForStartup Design Pricing Command

Invokes the `design-pricing` skill (ForStartup Edition).

## Usage

```bash
/for-startup-design-pricing
```

## Description

VC水準の価格戦略を自動設計する自律実行型スキル。競合ベンチマーク、価格感度分析、収益モデル最適化を提供します。

## Parameters

None required - the skill runs autonomously and will prompt for product information if needed.

## Output

`Stock/programs/創業支援・新規事業開発(AIエージェント)/projects/Founder_Agent_ForStartup/documents/3_planning/pricing_strategy.md`
