# ForStartup Validate 10x Command

Invokes the `validate-10x` skill (ForStartup Edition).

## Usage

```bash
/for-startup-validate-10x
```

## Description

10倍優位性検証スキル（ForStartup版）。3軸以上（速度・コスト・品質）での10倍優位性を定量的に証明します。

## Parameters

None required - the skill runs autonomously.

## Output

`Stock/programs/創業支援・新規事業開発（AIエージェント）/projects/Founder_Agent_ForStartup/documents/2_discovery/10x_validation.md`
