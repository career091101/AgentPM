# ForStartup Analyze Competitive Moat Command

Invokes the `analyze-competitive-moat` skill (ForStartup Edition).

## Usage

```bash
/for-startup-analyze-competitive-moat
```

## Description

競合優位性（経済的堀）をVC基準で分析する自律実行型スキル。ネットワーク効果、スイッチングコスト、ブランド力、データ資産、規制障壁を評価します。

## Parameters

None required - the skill runs autonomously and will prompt for business model information if needed.

## Output

`Stock/programs/創業支援・新規事業開発(AIエージェント)/projects/Founder_Agent_ForStartup/documents/2_discovery/competitive_moat_analysis.md`
