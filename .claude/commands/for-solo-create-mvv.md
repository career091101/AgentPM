# ForSolo Create MVV Command

このコマンドはForSolo特化版のMVV（Mission/Vision/Values）作成スキルを実行します。

## 実行内容

`/create-mvv` スキル（ForSolo Edition）を起動し、以下を自動実行:

1. Mission定義（解決したい課題）
2. Vision設定（3-5年後の姿）
3. Values設定（行動指針）
4. Build in Public原則との整合性確認
5. MVVドキュメント作成

## ForSolo特化要素

- **Mission**: ニッチ課題への集中
- **Vision**: MRR $10K達成、フルタイム転換
- **Values**: 透明性、コスト最小化、1人実行可能性
- **Build in Public**: X/Twitterでの透明性重視
- Solopreneur Research統合: 85件

## 使用タイミング

- PMF検証完了後
- ブランディング戦略を明確にしたい
- Build in Public戦略作成前

## 出力

- `Stock/programs/創業支援・新規事業開発（AIエージェント）/projects/Founder_Agent_ForSolo/documents/solopreneur/mvv.md`

## 次のコマンド

作成完了後:
- `/for-solo-create-bip-strategy` - Build in Public戦略作成へ
- `/for-solo-build-flywheel` - 成長フライホイール構築へ
