# ForStartup Measure AARRR Command

Invokes the `measure-aarrr` skill (ForStartup Edition).

## Usage

```bash
/for-startup-measure-aarrr
```

## Description

AARRR（Acquisition/Activation/Retention/Referral/Revenue）フレームワークでVC投資基準の成長ファネルを測定。月次成長率20%、年次3倍の厳格な目標設定、DAU/MAU・NRR・LTV/CAC等のVC判断指標を統合し、成長を阻害するボトルネックを定量分析します。

## Parameters

None required - the skill runs autonomously.

## Output

`Flow/{YYYYMM}/{YYYY-MM-DD}/aarrr_analysis_forstartup.md`
