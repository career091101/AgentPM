# ForStartup Startup Scorecard Command

Invokes the `startup-scorecard` skill (ForStartup Edition).

## Usage

```bash
/for-startup-startup-scorecard
```

## Description

スタートアップスコアカード評価（ForStartup版）。VC投資基準（CPF 70%、10倍優位性3軸、LTV/CAC 5.0以上）で総合評価を実施します。

## Parameters

None required - the skill runs autonomously.

## Output

`Stock/programs/創業支援・新規事業開発（AIエージェント）/projects/Founder_Agent_ForStartup/documents/6_monitoring/startup_scorecard.md`
