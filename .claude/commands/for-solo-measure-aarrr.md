# ForSolo Measure AARRR Command

このコマンドはForSolo特化版のAARRR（Pirate Metrics）測定スキルを実行します。

## 実行内容

`/measure-aarrr` スキル（ForSolo Edition）を起動し、以下を自動実行:

1. Acquisition（獲得）指標測定
2. Activation（活性化）指標測定
3. Retention（継続）指標測定
4. Referral（紹介）指標測定
5. Revenue（収益）指標測定

## ForSolo特化要素

- **Acquisition**: CAC $0-10（SEO・Build in Public重視）
- **Activation**: サインアップ→初回利用 70%以上
- **Retention**: 月次継続率 60%以上
- **Referral**: NPS 40以上（口コミ・バイラル）
- **Revenue**: MRR $1K → $5K → $10K、LTV/CAC 10.0以上
- Solopreneur Research統合: 85件

## 使用タイミング

- PMF検証完了後
- 成長戦略本格化時
- 定期測定（月次）

## 出力

- `Stock/programs/創業支援・新規事業開発（AIエージェント）/projects/Founder_Agent_ForSolo/documents/solopreneur/aarrr_report.md`

## 次のコマンド

測定完了後:
- `/for-solo-monitor-burn-rate` - バーンレートモニタリングへ
- `/for-solo-startup-scorecard-forsolo` - スタートアップスコアカードへ
