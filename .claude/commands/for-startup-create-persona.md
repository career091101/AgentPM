# ForStartup Create Persona Command

Invokes the `create-persona` skill (ForStartup Edition).

## Usage

```bash
/for-startup-create-persona
```

## Description

VC投資水準のターゲットペルソナ作成スキル。起業の科学のペルソナ8要素に加え、市場規模評価（TAM/SAM/SOM）、支払い意思額（WTP）分析、ビジネスモデル別最適化を実行します。

## Parameters

None required - the skill runs autonomously.

## Output

`Stock/programs/創業支援・新規事業開発（AIエージェント）/projects/Founder_Agent_ForStartup/documents/2_discovery/persona.md`
