# ForStartup Build Flywheel Command

Invokes the `build-flywheel` skill (ForStartup Edition).

## Usage

```bash
/for-startup-build-flywheel
```

## Description

スケーラブルな成長フライホイールを設計する自律実行型スキル。VC投資基準（月次成長率20%+）を満たす自己強化ループ、ネットワーク効果、データフライホイールを提供します。

## Parameters

None required - the skill runs autonomously and will prompt for business model information if needed.

## Output

`Stock/programs/創業支援・新規事業開発(AIエージェント)/projects/Founder_Agent_ForStartup/documents/3_planning/flywheel_design.md`
