# ForStartup Validate PMF Command

Invokes the `validate-pmf` skill (ForStartup Edition).

## Usage

```bash
/for-startup-validate-pmf
```

## Description

PMF（Product Market Fit）検証スキル（ForStartup版）。月次成長率20%以上、LTV/CAC 5.0以上の厳格な基準で評価します。

## Parameters

None required - the skill runs autonomously.

## Output

`Stock/programs/創業支援・新規事業開発（AIエージェント）/projects/Founder_Agent_ForStartup/documents/2_discovery/pmf_validation.md`
