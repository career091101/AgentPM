# ForSolo Discover Demand Command

このコマンドはForSolo特化版の需要発見スキルを実行します。

## 実行内容

`/discover-demand` スキル（ForSolo Edition）を起動し、以下を自動実行:

1. 市場トレンド調査（AI、ソロプレナー向けツール重点）
2. ソーシャルリスニング（X/Twitter、Reddit、Indie Hackers）
3. 競合プロダクト分析（Product Hunt、Gumroad等）
4. 課題候補リスト作成（10-15件）
5. 1人実行可能性スクリーニング

## ForSolo特化要素

- **市場機会基準**: 4点以上（ニッチ市場OK、TAM 10億円以上でも可）
- **実行可能性重視**: 6点以上（1人実行可能性が最重要）
- **コスト最小化**: 初期投資$1,000以下、月次コスト$50以下
- Build in Public戦略: X/Twitterでの透明性重視
- Solopreneur Research統合: 85件 + Tier 2ケーススタディ20件

## 使用タイミング

- プロジェクト開始時（Phase 1: 需要発見）
- ピボット検討時
- 新規課題探索時

## 出力

- `Stock/programs/創業支援・新規事業開発（AIエージェント）/projects/Founder_Agent_ForSolo/documents/2_discovery/demand_discovery.md`

## 次のコマンド

需要発見後:
- `/for-solo-research-problem` - 課題調査へ
- `/for-solo-create-persona` - ペルソナ作成へ
