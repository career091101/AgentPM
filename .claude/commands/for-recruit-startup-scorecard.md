# ForRecruit Startup Scorecard Command

Runs the ForRecruit-specific startup scorecard skill.

**Usage**: `/for-recruit-startup-scorecard`

**Invokes**: `for_recruit/startup-scorecard`

**Prerequisites**:
- Phase1成果物（`documents/`, `mvp/`）
- Ring制度診断結果（`ring_criteria_diagnosis.md`）

**Output**: `Stock/programs/創業支援・新規事業開発（AIエージェント）/projects/Founder_Agent_Phase1/documents/5_monitoring/scorecard_forrecruit.md`

**Execution Time**: 30-50分（社内リソース評価含む）

**ForRecruit Adjustments**:
- 総合80点満点（4視点40点+社内リソース20点+シナジー20点）
- Ring制度評価項目の統合（Ring 1-3各段階の達成基準）
- 社内リソース活用評価（6カテゴリ20点満点）
- 既存事業シナジー評価（カニバリゼーション回避、クロスセル効果）

**Integrated Research**: Recruit_Product_Research 31製品分析（Airレジ、Airペイ、Geppo等）

**Scoring Targets**:
- Airレジ: 71/80点（Financial 9、Customer 9、Internal Process 10、L&G 9、社内リソース 16、シナジー 18）
- Airペイ: 76/80点（Financial 9、Customer 10、Internal Process 10、L&G 9、社内リソース 18、シナジー 20）
- Geppo: 73/80点（Financial 9、Customer 10、Internal Process 8、L&G 10、社内リソース 16、シナジー 20）
