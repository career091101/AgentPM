# ForStartup Validate Market Timing Command

Invokes the `validate-market-timing` skill (ForStartup Edition).

## Usage

```bash
/for-startup-validate-market-timing
```

## Description

市場参入タイミングをVC基準で検証する自律実行型スキル。TAM $1B+成長性、技術トレンド分析、競合環境評価、規制リスク分析を提供します。

## Parameters

None required - the skill runs autonomously and will prompt for market information if needed.

## Output

`Stock/programs/創業支援・新規事業開発(AIエージェント)/projects/Founder_Agent_ForStartup/documents/2_discovery/market_timing_validation.md`
