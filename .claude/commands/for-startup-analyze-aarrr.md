# ForStartup Analyze AARRR Command

Invokes the `analyze-aarrr` skill (ForStartup Edition).

## Usage

```bash
/for-startup-analyze-aarrr
```

## Description

AARRR（Acquisition, Activation, Retention, Revenue, Referral）メトリクスをVC投資基準で分析する自律実行型スキル。月次成長率20%基準でのボトルネック特定と改善施策を提供します。

## Parameters

None required - the skill runs autonomously and will prompt for metrics data if needed.

## Output

`Stock/programs/創業支援・新規事業開発(AIエージェント)/projects/Founder_Agent_ForStartup/documents/5_monitoring/aarrr_analysis.md`
