# ForStartup Prepare VC Meeting Command

Invokes the `prepare-vc-meeting` skill (ForStartup Edition).

## Usage

```bash
/for-startup-prepare-vc-meeting
```

## Description

VC Meeting準備を自動化する自律実行型スキル。想定Q&A 30問の生成、デモシナリオ作成、VC事前調査レポート、フォローアップ計画を提供します。

## Parameters

None required - the skill runs autonomously and will prompt for VC information if needed.

## Output

`Stock/programs/創業支援・新規事業開発（AIエージェント）/projects/Founder_Agent_ForStartup/documents/3_planning/vc_meeting_prep.md`
