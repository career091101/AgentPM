# ForStartup Build LP Command

Invokes the `build-lp` skill (ForStartup Edition).

## Usage

```bash
/for-startup-build-lp
```

## Description

VC水準のランディングページを自動構築する自律実行型スキル（HTML/CSS/JS完全生成）。Product Hunt戦略、デモ動画埋め込み、早期ユーザー獲得最適化を提供します。

## Parameters

None required - the skill runs autonomously and will prompt for product information if needed.

## Output

- LP: `Stock/programs/創業支援・新規事業開発(AIエージェント)/projects/Founder_Agent_ForStartup/documents/4_executing/lp/index.html`
- 戦略: `Stock/programs/創業支援・新規事業開発(AIエージェント)/projects/Founder_Agent_ForStartup/documents/3_planning/lp_strategy.md`
