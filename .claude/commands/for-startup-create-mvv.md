# ForStartup Create MVV Command

Invokes the `create-mvv` skill (ForStartup Edition).

## Usage

```bash
/for-startup-create-mvv
```

## Description

VC向けMVV（Mission/Vision/Values）作成スキル。投資家を惹きつける長期ビジョンと価値観を定義します。

## Parameters

None required - the skill runs autonomously.

## Output

`Stock/programs/創業支援・新規事業開発（AIエージェント）/projects/Founder_Agent_ForStartup/documents/1_initiating/mvv.md`
