# ForStartup Validate PSF Command

Invokes the `validate-psf` skill (ForStartup Edition).

## Usage

```bash
/for-startup-validate-psf
```

## Description

PSF（Problem Solution Fit）検証スキル（ForStartup版）。技術実現可能性とスケーラビリティを重視した評価を実施します。

## Parameters

None required - the skill runs autonomously.

## Output

`Stock/programs/創業支援・新規事業開発（AIエージェント）/projects/Founder_Agent_ForStartup/documents/2_discovery/psf_validation.md`
