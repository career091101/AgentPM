# /for-recruit-build-flywheel

ForRecruit特化版: 成長を加速させるフライホイールを設計する自律実行型コマンド。社内エコシステム連携（Airシリーズモデル）を重視し、リクルート資産3種以上活用でLTV 3-5倍向上、クロスセル効果定量化（Airレジ→Airペイ 57%）を目標に設計します。

## Skill情報

- **Skill Name**: `build-flywheel-for-recruit`
- **所要時間**: 30-50分（自動実行）
- **実行モード**: 自律実行型（対話なし）

## 入力要件

- `lean_canvas.md`: リーンキャンバス（必須）
- `persona.md`: ペルソナ定義（必須）
- `resource_inventory.md`: 社内リソース棚卸し結果（ForRecruit必須）

## ForRecruit固有の特徴

1. **エコシステム連携評価**: 社内製品連携でChurn率1/2〜1/3
2. **クロスセル効果定量化**: Airレジ→Airペイ クロスセル率57%ベンチマーク
3. **LTV向上目標**: 単体製品比3-5倍（エコシステム連携効果）
4. **社内リソース活用**: 営業網・データ資産・ブランドの統合活用
5. **Recruit Product Research統合**: 15-20事例統合

## 出力成果物

- `{IDEA_FOLDER}/documents/3_planning/flywheel.md`
  - フライホイール構造（Mermaid図）
  - エコシステム連携評価
  - クロスセル効果シミュレーション
  - NE（ネガティブ要因）と対策
  - KPI設定（クロスセル率、LTV/CAC比、NRR等）

## 次のステップ

- `/validate-cpf`: CPF検証（ForRecruit版、基準50%）
- `/validate-10x`: 10倍優位性検証（ForRecruit版、1軸以上）
- `/validate-psf`: PSF検証（ForRecruit版、ROI 1000%以上）

## 使用例

```bash
/for-recruit-build-flywheel
```
