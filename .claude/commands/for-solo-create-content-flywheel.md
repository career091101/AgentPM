# ForSolo Create Content Flywheel Command

このコマンドはForSolo特化版のコンテンツフライホイール作成スキルを実行します。

## 実行内容

`/create-content-flywheel` スキル（ForSolo Edition）を起動し、以下を自動実行:

1. コンテンツ戦略設計（ブログ、X/Twitter、Product Hunt）
2. SEOキーワード選定
3. コンテンツカレンダー作成
4. フライホイール加速施策
5. トラフィック獲得戦略

## ForSolo特化要素

- **ブログ記事**: 50+本（SEO重視）
- **X/Twitter投稿**: 週5-7回（Build in Public）
- **Product Hunt準備**: ローンチ戦略
- **コスト**: $0（広告費ゼロ）
- Solopreneur Research統合: 85件

## 使用タイミング

- PMF検証完了後
- SEO戦略を本格化したい
- トラフィック獲得を強化したい

## 出力

- `Stock/programs/創業支援・新規事業開発（AIエージェント）/projects/Founder_Agent_ForSolo/documents/solopreneur/content_flywheel.md`

## 次のコマンド

作成完了後:
- `/for-solo-build-flywheel` - 成長フライホイール統合へ
- `/for-solo-measure-aarrr` - AARRR測定へ
