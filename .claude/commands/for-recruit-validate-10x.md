# /for-recruit-validate-10x

**Domain**: ForRecruit (企業内新規事業)
**Skill Path**: `.claude/skills/for_recruit/validate-10x/SKILL.md`

## 概要

既存代替案と比較して10倍優位性を検証するForRecruit特化版スキル。8軸（時間/コスト/成果/使いやすさ/導入障壁/営業網/データ資産/エコシステム連携）で定量比較し、1軸以上で10倍達成がRing 2通過条件。リクルート製品研究31件の10倍優位性パターン（コスト100倍、時間7倍、手数料6-20倍等）を統合。

## 使用タイミング

- リーンキャンバス作成後
- PSF検証の一環として
- Ring 2申請前の競合優位性確認

## 前提条件

- lean_canvas.md 作成済み（必須）
- competitor_research.md 作成済み（推奨）
- inventory_internal_resources.md 作成済み（推奨）

## 実行

```
/for-recruit-validate-10x
```

`Stock/programs/創業支援・新規事業開発（AIエージェント）/projects/Founder_Agent_Phase1/documents/3_planning/10x_validation.md`

## 判定基準（ForRecruit調整）

### 10倍優位性軸数

| 軸数 | 判定 | 次のアクション | 成功率 |
|-----|------|---------------|-------|
| **2軸以上** | ✅ 10倍達成（Ring 2推奨） | PSF検証/LP構築へ | 80-100% |
| **1軸のみ** | ⚠️ 部分達成（Ring 2最低） | 強化軸を特定し再設計 | 60% |
| **0軸** | ❌ 未達成 | ソリューション根本見直し | 25% |

**ForRecruit調整**:
- Origin版: 2軸以上必須
- ForRecruit版: **1軸以上でRing 2通過可**（社内リソース活用で差別化可能）
- 推奨: 2軸以上（成功率80%以上）

### 評価8軸

| 軸 | 定義 | ベンチマーク |
|----|------|------------|
| **コスト削減** | 初期費用・ランニングコスト削減 | 10-100倍（Airレジ100倍） |
| **時間短縮** | 作業時間・プロセス時間削減 | 5-10倍（Airレジ7倍） |
| **成果向上** | 効果・パフォーマンス向上 | 2-5倍 |
| **使いやすさ** | 学習コスト・操作性向上 | 2-3倍 |
| **導入障壁** | 導入ステップ数・期間削減 | 5-10倍 |
| **営業網活用** | 既存営業チャネル活用によるCAC削減 | 5-10倍（Airレジ1/5〜1/10） |
| **データ資産活用** | 既存データによる精度向上・コスト削減 | 3-20倍（Airキャッシュ6-20倍） |
| **エコシステム連携** | 複数サービス統合による利便性向上 | 3-5倍（AirシリーズLTV 3倍向上） |

## ForRecruit Benchmark（統合研究より）

### 成功事例（10倍優位性達成）

**Airレジ（4軸10倍）**:
- コスト削減: 50-100万円 → 基本無料（**100倍**）
- 時間短縮: 導入1週間 → 1日以内（**7倍**）
- 営業網活用: CAC 5万円 → 1-2万円（**5倍**）
- エコシステム連携: LTV **3倍向上**

**Airペイ（4軸10倍）**:
- 初期費用削減: 数十万円〜百万円 → 0円（**100倍**）
- 対応ブランド数: 競合5-10種 → 81種（**8-16倍**）
- 営業網活用: CAC 8万円 → 2-3万円（**3-4倍**）
- データ資産活用: 手数料0.5%（業界平均の1/6〜1/20、**6-20倍**）

**Airキャッシュ（3軸10倍）**:
- 入金スピード: 1-2週間 → 最短翌日（**7倍**）
- 手数料率: 3-10% → 0.5%〜（**6-20倍**）
- データ資産活用: Airペイ決済データで信用スコアリング、審査自動化

**Geppo（2軸10倍）**:
- 回答負荷軽減: 30分 → 1分（**10倍**）
- 回答率向上: 競合50-70% → 96%（**1.4-2倍**）

**スタディサプリ（2軸10倍）**:
- コスト削減: 予備校年間50-100万円 → 月額2,178円（**20-40倍**）
- アクセス性: 地理的制約あり → どこでも学習可能（**10倍のアクセス向上**）

### 失敗事例（10倍優位性なし）

**エリクラ（0軸10倍）**:
- 10分単位ワークシェア、地産地消（差別化弱い）
- 競合タイミーに100倍差（10万人 vs 1,000万人）
- 教訓: 最低1軸で10倍優位性を構築、模倣困難な領域を確保

**CODE.SCORE（0軸10倍）**:
- エンジニアスキル可視化 vs 競合AI個別最適化学習
- 教育市場で劣位
- 教訓: 競合が10倍以上優れている市場には参入しない

### 社内リソース活用と成功率

| 資産活用数 | PMFスコア | 成功率 | 代表製品 |
|----------|---------|-------|---------|
| **3種以上** | 8.8 | **100%** | Airレジ、Airペイ、Geppo |
| **1-2種** | 7.5 | 80% | SUUMO、じゃらん、スタディサプリ |
| **0種** | 5.2 | 25% | CODE.SCORE、termhub |

## 次のステップ

### 10倍達成時（2軸以上）
- `/for-recruit-validate-ring-criteria` でRing 2基準チェック
- `/build-lp` でランディングページ構築
- `/for-recruit-validate-psf` でPSF総合検証

### 部分達成時（1軸のみ）
- 未達成軸の改善アクション実行
- 社内リソース活用を3種以上に増やす
- 1-2ヶ月後に再検証

### 未達成時（0軸）
- リーンキャンバス再設計
- 競合が10倍優れている市場への参入を再検討
- `/pivot-decision` でPivot判断

50-80分（競合調査、社内リソース分析含む）
