# ForStartup Validate Unit Economics Strict Command

Invokes the `validate-unit-economics-strict` skill (ForStartup Edition).

## Usage

```bash
/for-startup-validate-unit-economics-strict
```

## Description

ユニットエコノミクスを厳格なVC投資基準で検証する自律実行型スキル。LTV/CAC 5.0以上、CAC回収期間12ヶ月以内、Gross Margin 70%以上、NRR 100%以上、Magic Number 0.75以上を評価します。

## Parameters

None required - the skill runs autonomously and will prompt for metrics data if needed.

## Output

`Stock/programs/創業支援・新規事業開発(AIエージェント)/projects/Founder_Agent_ForStartup/documents/2_discovery/unit_economics_validation.md`
