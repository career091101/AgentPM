# ForSolo Build Flywheel Command

このコマンドはForSolo特化版の成長フライホイール構築スキルを実行します。

## 実行内容

`/build-flywheel` スキル（ForSolo Edition）を起動し、以下を自動実行:

1. フライホイール設計（SEO、Build in Public、Product Hunt）
2. 各ステージの連携定義
3. 成長指標設定（MRR、フォロワー、トラフィック）
4. フライホイール加速施策
5. フライホイール図作成

## ForSolo特化要素

- **主要エンジン**:
  - SEO（ブログ記事50+本）
  - Build in Public（X/Twitterフォロワー1K → 10K）
  - Product Hunt（#1獲得戦略）
- **成長指標**: MRR $1K → $5K → $10K
- **コスト**: 広告費ゼロ（SEO・オーガニック重視）
- Solopreneur Research統合: 85件

## 使用タイミング

- PMF検証完了後
- 成長戦略を本格化したい
- MRR $5K達成を目指す

## 出力

- `Stock/programs/創業支援・新規事業開発（AIエージェント）/projects/Founder_Agent_ForSolo/documents/solopreneur/flywheel.md`

## 次のコマンド

構築完了後:
- `/for-solo-create-content-flywheel` - コンテンツフライホイール作成へ
- `/for-solo-measure-aarrr` - AARRR測定へ
