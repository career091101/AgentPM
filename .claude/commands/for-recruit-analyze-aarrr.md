# ForRecruit Analyze AARRR Command

Runs the ForRecruit-specific AARRR analysis skill.

**Usage**: `/for-recruit-analyze-aarrr`

**Invokes**: `for_recruit/analyze-aarrr`

**Prerequisites**:
- PMF達成（`/validate-pmf` 完了、Ring 3承認可）
- 外部顧客獲得100社/人以上
- 収益化開始（初期売上発生）
- 3年黒字・5年累損解消計画策定済み

**Output**: `Flow/{YYYYMM}/{YYYY-MM-DD}/aarrr_analysis_forrecruit.md`

**Execution Time**: 40-60分（社内外データ統合含む）

**ForRecruit Adjustments**:
- Acquisition: 社内+外部顧客の2段階評価
- Activation: 社内実績基準（Geppo 96%、Airレジ80%）
- Retention: 継続率98%目標（Geppo基準）
- Revenue: 3年黒字計画必須
- Referral: 社内NPS基準（60-80）

**Integrated Research**: Recruit_Product_Research 31製品分析（Airレジ、Airペイ、Geppo、スタディサプリ等）
