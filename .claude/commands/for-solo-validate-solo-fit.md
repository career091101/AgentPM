# ForSolo Validate Solo-Fit Command

このコマンドはForSolo特化版の1人実行可能性検証（Solo-Fit）スキルを実行します。

## 実行内容

`/validate-solo-fit` スキル（ForSolo Edition）を起動し、以下を自動実行:

1. 必須スキルチェックリスト評価
2. 時間確保可能性分析
3. コスト構造最適化
4. Solo-Fit総合スコア算出
5. 実行計画作成

## ForSolo特化要素

- **Solo-Fit Score**: 6点満点（技術2点 + 時間2点 + コスト2点）
- **必須スキル**: フルスタック or No-Code、デザイン、マーケティング、サポート
- **時間確保**: 週20時間以上（6ヶ月継続可能）
- **初期投資**: $1,000以下
- **月次コスト**: $50以下
- **フルタイム転換マイルストーン**: MRR $10K達成
- Solopreneur Research統合: 85件 + Tier 2ケーススタディ11件

## 使用タイミング

- プロジェクト開始前（Phase 0: アイデア検証）
- CPF検証時（実行可能性スコア評価）
- ピボット検討時

## 出力

- `Stock/programs/創業支援・新規事業開発（AIエージェント）/projects/Founder_Agent_ForSolo/documents/solopreneur/solo_fit_assessment.md`

## 次のコマンド

Solo-Fit達成時:
- `/for-solo-discover-demand` - 需要発見へ
- `/for-solo-validate-cpf` - CPF検証へ

要改善時:
- `/for-solo-optimize-tool-stack` - ツールスタック最適化へ
- スキル不足の場合は外注検討
