# ForStartup Discover Demand Command

Invokes the `discover-demand` skill (ForStartup Edition).

## Usage

```bash
/for-startup-discover-demand
```

## Description

VC調達基準に準拠した厳格な市場機会発見スキル。TAM $1B以上、月次成長率20%以上、10倍優位性3軸必須化。スケーラビリティ・ネットワーク効果を重視した評価を実施します。

## Parameters

None required - the skill runs autonomously.

## Output

`Stock/programs/創業支援・新規事業開発（AIエージェント）/projects/Founder_Agent_ForStartup/documents/2_discovery/demand_analysis.md`

## Related Skills

- `/for-startup-validate-cpf` - CPF検証（70%基準）
- `/for-startup-research-problem` - 課題の深刻さ調査
- `/for-startup-validate-10x` - 10倍優位性3軸検証
