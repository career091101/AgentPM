# ForStartup Inventory Internal Resources Command

Invokes the `inventory-internal-resources` skill (ForStartup Edition).

## Usage

```bash
/for-startup-inventory-internal-resources
```

## Description

スタートアップリソース棚卸しを実施する自律実行型スキル。創業チームスキル、技術資産、初期顧客基盤、コミュニティ資産を評価・可視化します。

## Parameters

None required - the skill runs autonomously and will prompt for team information if needed.

## Output

`Stock/programs/創業支援・新規事業開発(AIエージェント)/projects/Founder_Agent_ForStartup/documents/2_discovery/resource_inventory.md`
