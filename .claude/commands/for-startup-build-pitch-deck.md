# ForStartup Build Pitch Deck Command

Invokes the `build-pitch-deck` skill (ForStartup Edition).

## Usage

```bash
/for-startup-build-pitch-deck
```

## Description

VC向けピッチデッキ自動生成（10-15スライド構成）。VC投資判断に最適化し、定量データ重視、10倍優位性の可視化、Traction可視化を実現します。

## Parameters

None required - the skill runs autonomously.

## Output

`Stock/programs/創業支援・新規事業開発（AIエージェント）/projects/Founder_Agent_ForStartup/documents/3_planning/pitch_deck.md`

## Related Skills

- `/for-startup-validate-cpf` - CPF検証
- `/for-startup-validate-10x` - 10倍優位性検証
- `/for-startup-prepare-vc-meeting` - VC Meeting準備
