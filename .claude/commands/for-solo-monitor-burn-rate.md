# ForSolo Monitor Burn Rate Command

このコマンドはForSolo特化版のバーンレートモニタリングスキルを実行します。

## 実行内容

`/monitor-burn-rate` スキル（ForSolo Edition）を起動し、以下を自動実行:

1. 月次コスト構造分析
2. バーンレート算出
3. ランウェイ予測
4. コスト最適化提案
5. 収益化マイルストーン評価

## ForSolo特化要素

- **月次コスト目標**: $50以下
- **初期投資**: $1,000以下
- **バーンレート**: 月$50 × ランウェイ20ヶ月
- **利益率**: 80%以上
- **収益化マイルストーン**: MRR $1K（3-6ヶ月）、$5K（6-12ヶ月）、$10K（12ヶ月以降）
- Solopreneur Research統合: 85件

## 使用タイミング

- プロジェクト開始時（初期コスト設定）
- 定期測定（月次）
- コスト最適化が必要な時

## 出力

- `Stock/programs/創業支援・新規事業開発（AIエージェント）/projects/Founder_Agent_ForSolo/documents/solopreneur/burn_rate_report.md`

## 次のコマンド

測定完了後:
- `/for-solo-startup-scorecard-forsolo` - スタートアップスコアカードへ
- `/for-solo-validate-unit-economics` - ユニットエコノミクス検証へ
