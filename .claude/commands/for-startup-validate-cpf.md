# ForStartup Validate CPF Command

Invokes the `validate-cpf` skill (ForStartup Edition).

## Usage

```bash
/for-startup-validate-cpf
```

## Description

CPF（Customer Problem Fit）検証スキル（ForStartup版）。VC調達基準70%以上のスコアを目指し、課題の深刻さを定量的に評価します。

## Parameters

None required - the skill runs autonomously.

## Output

`Stock/programs/創業支援・新規事業開発（AIエージェント）/projects/Founder_Agent_ForStartup/documents/2_discovery/cpf_judgment.md`

- `/for-startup-research-problem` - 課題調査
- `/for-startup-validate-psf` - PSF検証
