# ForStartup Build Synergy Map Command

Invokes the `build-synergy-map` skill (ForStartup Edition).

## Usage

```bash
/for-startup-build-synergy-map
```

## Description

投資家・パートナー企業とのシナジーマップを作成する自律実行型スキル。VC提携効果、顧客紹介機会、戦略的提携価値を可視化します。

## Parameters

None required - the skill runs autonomously and will prompt for investor/partner information if needed.

## Output

`Stock/programs/創業支援・新規事業開発(AIエージェント)/projects/Founder_Agent_ForStartup/documents/3_planning/synergy_map.md`
