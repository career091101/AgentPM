# ForStartup Orchestrate Review Loop Command

Invokes the `orchestrate-review-loop` skill (ForStartup Edition).

## Usage

```bash
/for-startup-orchestrate-review-loop
```

## Description

Phase 1-5全体のレビューループを統括する自律実行型スキル。各フェーズの品質評価、エラーハンドリング、リトライ制御、統合レポート作成を提供します。

## Parameters

None required - the skill runs autonomously and orchestrates the full review cycle.

## Output

`Stock/programs/創業支援・新規事業開発(AIエージェント)/projects/Founder_Agent_ForStartup/documents/5_monitoring/review_loop_report.md`
