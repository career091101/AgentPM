# ForStartup Research Problem Command

Invokes the `research-problem` skill (ForStartup Edition).

## Usage

```bash
/for-startup-research-problem
```

## Description

課題の深刻さ調査スキル（ForStartup版）。定量データを重視し、VC投資家が納得する課題の深刻さを証明します。

## Parameters

None required - the skill runs autonomously.

## Output

`Stock/programs/創業支援・新規事業開発（AIエージェント）/projects/Founder_Agent_ForStartup/documents/2_discovery/problem_research.md`
