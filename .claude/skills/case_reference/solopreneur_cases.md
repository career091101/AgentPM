# Solopreneur Case Reference Skill

ソロプレナー成功事例85件の参照・検索。

---

## Skill情報

- **コマンド**: `/reference-cases`
- **目的**: 類似事例から学ぶ
- **事例数**: 85件

---

## このSkillでできること

1. **条件検索**: 業種・収益・成長戦略で絞り込み
2. **事例詳細閲覧**: 成功要因・失敗・学びの抽出
3. **パターン分析**: 共通する成功法則の発見

### パターン分析

```markdown
## パターン分析: 成功の共通法則

### [$1M+ ARR達成者の共通点]

**分析対象**: 12事例

#### 1. Build in Publicの実施
- 12事例中11事例（92%）がBuild in Public
- Twitter平均フォロワー: 50K+

#### 2. 早期収益化
- 平均: ローンチ後2週間で初収益
- 無料期間: 最大14日

#### 3. ニッチ特化
- 広く浅くではなく、狭く深く
- 例: 「ノーコードツール全般」→「Twitter自動化」

#### 4. 高速イテレーション
- 平均: 週1-2回のアップデート
- MVP → PMF期間: 平均3ヶ月

#### 5. SEO投資
- 10事例（83%）がSEOに注力
- ブログ記事平均: 50+本
```
