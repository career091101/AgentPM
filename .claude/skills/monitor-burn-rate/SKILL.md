# Monitor Burn Rate

## 目的

スタートアップの資金状況を常時監視し、**バーンレート（月次消費金額）**と**ランウェイ（資金が尽きるまでの期間）**を自動計算。18ヶ月ルールに基づき、資金ショートを未然に防ぎ、最適な資金調達タイミングを提案します。

**重要な原則**（起業の科学）:
> 「ランウェイが18ヶ月を切ったら、資金調達の準備を開始すべきである」
> 「ランウェイが6ヶ月以下になったら、事業継続が危機的状況にある」
> — David Skok

## 実行ステップ

### STEP 1: 現在残高の入力（1分）

**質問**:
1. 現在の手元現金残高（銀行口座残高）
2. 最終更新日

**入力例**:
```
現在残高: ¥5,000,000
更新日: 2025-12-31
```

---

### STEP 2: Gross Burn Rate計算（2分）

**計算方法**: 月次総支出の算出

**必要データ**:
- 人件費（給与、社会保険）
- オフィス・設備費
- マーケティング費
- 開発費（外注、ツール費）
- その他経費

**計算式**:
```
Gross Burn Rate = 月次総支出 / 月
```

**計算例**:
```
人件費: ¥800,000/月
オフィス費: ¥150,000/月
マーケティング費: ¥200,000/月
開発費: ¥300,000/月
その他: ¥50,000/月

Gross Burn Rate = ¥1,500,000/月
```

**業種別目安**:
| ステージ | 目安バーンレート |
|---------|----------------|
| シード期（チーム3-5人） | ¥500,000-1,500,000/月 |
| アーリー期（チーム10人） | ¥2,000,000-5,000,000/月 |
| グロース期（チーム30人） | ¥10,000,000-30,000,000/月 |

---

### STEP 3: Net Burn Rate計算（2分）

**計算方法**: 収益を差し引いた実質消費額

**必要データ**:
- 月次収益（売上、契約金）
- 月次総支出（STEP 2から）

**計算式**:
```
Net Burn Rate = (月次総支出 - 月次収益) / 月
```

**計算例**:
```
月次収益: ¥300,000
月次総支出: ¥1,500,000

Net Burn Rate = ¥1,200,000/月
```

**注意**:
- Net Burn Rateがマイナス（収益 > 支出）の場合 → キャッシュフロー黒字 ✅
- Net Burn Rateがプラスの場合 → 赤字運営、資金枯渇リスクあり ⚠️

---

### STEP 4: ランウェイ計算（1分）

**計算方法**: 現在の消費ペースで資金が何ヶ月持つか

**計算式**:
```
Runway（月） = 現在残高 / Net Burn Rate
```

**計算例**:
```
現在残高: ¥5,000,000
Net Burn Rate: ¥1,200,000/月

Runway = 5,000,000 / 1,200,000 = 4.2ヶ月 🚨
```

**判定基準**（起業の科学準拠）:

| ランウェイ | 判定 | 状態 | 推奨アクション |
|-----------|------|------|--------------|
| **18ヶ月以上** | ✅ 安全 | 余裕あり | 継続監視 |
| **12-18ヶ月** | ⚠️ 警告 | 資金調達準備開始 | 投資家リスト作成、ピッチ準備 |
| **6-12ヶ月** | 🚨 緊急 | 資金調達必須 | 即座に調達活動開始 |
| **6ヶ月以下** | 🔴 危機的 | 倒産リスク | 緊急ブリッジ調達またはコスト削減 |

---

### STEP 5: キャッシュフロー予測（6ヶ月先まで）（2分）

**予測方法**: 現在のバーンレートで6ヶ月先までの残高推移を計算

**計算式**:
```
N ヶ月後の予測残高 = 現在残高 - (Net Burn Rate × N)
```

**予測表（Markdown形式）**:

| 月 | 予測残高 | 判定 |
|----|---------|------|
| 2026-01 | ¥3,800,000 | 🚨 |
| 2026-02 | ¥2,600,000 | 🚨 |
| 2026-03 | ¥1,400,000 | 🔴 |
| 2026-04 | ¥200,000 | 🔴 危機的 |
| 2026-05 | ¥0（資金枯渇） | ⚠️ 倒産リスク |
| 2026-06 | - | - |

**可視化（簡易グラフ）**:
```

```

### STEP 6: シナリオ分析（3シナリオ）（3分）

**目的**: 楽観・現実・悲観の3パターンでランウェイを試算

#### シナリオ1: 楽観シナリオ（収益+20%、支出-10%）

```
月次収益: ¥300,000 × 1.20 = ¥360,000
月次支出: ¥1,500,000 × 0.90 = ¥1,350,000
Net Burn Rate: ¥990,000/月
Runway: 5,000,000 / 990,000 = 5.0ヶ月 ⚠️
```

#### シナリオ2: 現実シナリオ（現状維持）

```
Net Burn Rate: ¥1,200,000/月
Runway: 5,000,000 / 1,200,000 = 4.2ヶ月 🚨
```

#### シナリオ3: 悲観シナリオ（収益-20%、支出+10%）

```
月次収益: ¥300,000 × 0.80 = ¥240,000
月次支出: ¥1,500,000 × 1.10 = ¥1,650,000
Net Burn Rate: ¥1,410,000/月
Runway: 5,000,000 / 1,410,000 = 3.5ヶ月 🔴
```

**シナリオ比較表**:

| シナリオ | Net Burn Rate | Runway | 判定 |
|---------|--------------|--------|------|
| 楽観 | ¥990,000 | 5.0ヶ月 | ⚠️ |
| 現実 | ¥1,200,000 | 4.2ヶ月 | 🚨 |
| 悲観 | ¥1,410,000 | 3.5ヶ月 | 🔴 |

**インサイト**:
- いずれのシナリオでも6ヶ月以下 → **緊急対応必須**
- 悲観シナリオでは3ヶ月で資金枯渇 → **即座に資金調達またはコスト削減**

---

### STEP 7: 資金調達タイミング・調達額提案（3分）

**判定ロジック**:
- ランウェイ < 18ヶ月 → 資金調達準備開始
- ランウェイ < 12ヶ月 → 資金調達必須
- ランウェイ < 6ヶ月 → 緊急ブリッジ調達

#### 現状判定（例）

```
現在Runway: 4.2ヶ月 🚨 緊急
→ 即座に資金調達活動を開始すべき
```

#### 目標調達額の自動計算

**計算式**（18ヶ月ルール適用）:
```
目標調達額 = Net Burn Rate × 18ヶ月
```

**計算例**:
```
Net Burn Rate: ¥1,200,000/月
目標調達額 = ¥1,200,000 × 18 = ¥21,600,000
```

**調達方法の提案**:

| 調達額 | 推奨方法 | 適した投資家 |
|-------|---------|------------|
| ~¥10,000,000 | エンジェル投資 | 個人投資家、エンジェルネットワーク |
| ¥10,000,000~¥50,000,000 | シードVC | シードステージVC、アクセラレーター |
| ¥50,000,000~¥200,000,000 | シリーズA | アーリーステージVC |
| ¥200,000,000~ | シリーズB以降 | グロースファンド、CVC |

**本ケースの提案**:
```
目標調達額: ¥21,600,000
推奨方法: シードVC
投資家候補: シードステージVC、エンジェルネットワーク

調達活動計画:
- Week 1-2: 投資家リスト作成、ピッチデッキ準備
- Week 3-4: ウォームイントロ、初回ミーティング
- Month 2-3: デューデリジェンス、条件交渉
- Month 4: クロージング
```

### STEP 8: コスト最適化提案（2分）

**目的**: 資金調達と並行してコスト削減を実行

**削減優先順位**（起業の科学推奨）:

#### 優先度1: 即効性のある削減（今月実行可能）

| 項目 | 現状コスト | 削減案 | 削減額 | 実行難易度 |
|-----|-----------|-------|-------|----------|
| オフィス費 | ¥150,000 | コワーキング移行 | -¥50,000 | ★☆☆ |
| ツール費 | ¥50,000 | 無料プラン切替 | -¥20,000 | ★☆☆ |
| 外注費 | ¥300,000 | 内製化 | -¥100,000 | ★★☆ |
| **合計削減** | - | - | **-¥170,000/月** | - |

**削減後のバーンレート**:
```
Net Burn Rate: ¥1,200,000 → ¥1,030,000/月（-14%）
新Runway: 5,000,000 / 1,030,000 = 4.9ヶ月
```

#### 優先度2: 中期的な削減（3ヶ月以内）

- マーケティング費の見直し（ROI < 1.0のチャネル停止）
- 採用計画の延期（PMF達成後に再開）

#### 優先度3: 収益改善施策（並行実行）

- 既存顧客へのアップセル
- 価格改定（値上げ）
- 高単価プラン追加
