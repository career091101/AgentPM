## 附録B: よくある用語

| 用語 | 定義 | 計算式 |
|-----|------|-------|
| **Gross Burn** | 月次総支出（収益除く） | 人件費 + 固定費 + 変動費 |
| **Net Burn** | 実質消費額（収益差引後） | Gross Burn - 月次収益 |
| **Runway** | 資金枯渇までの期間 | 現在残高 / Net Burn |
| **ARPU** | 顧客当たり月次平均売上 | 月次収益 / アクティブ顧客数 |
| **Churn Rate** | 月次解約率 | 解約顧客数 / 月初顧客数 |
| **CAC** | 顧客獲得単価 | マーケティング費 / 新規顧客数 |
| **LTV** | 顧客生涯価値 | ARPU × 平均継続月数 |
