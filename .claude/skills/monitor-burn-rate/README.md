# Monitor Burn Rate - README

## スキル概要

### 目的

スタートアップの資金状況を月単位で監視し、**バーンレート（月次消費金額）**と**ランウェイ（資金が尽きるまでの期間）**を自動計算します。18ヶ月ルール（起業の科学）に基づいて資金ショートを未然に防ぎ、最適な資金調達タイミングと調達額を提案します。

### バーンレート・ランウェイの重要性

スタートアップの資金管理で最も重要な指標がバーンレートです。なぜならば：

- **予測可能性**: 現在の消費ペースで「何ヶ月資金が持つのか」を正確に把握できる
- **早期警戒**: ランウェイが18ヶ月を切ったら、資金調達準備を開始すべき時点
- **意思決定の根拠**: 採用・マーケティング・プロダクト開発の優先順位を決める基準
- **資金調達交渉**: VCに提示する「経営状況の透明性」

### 18ヶ月ルール（起業の科学）

> 「ランウェイが18ヶ月を切ったら、資金調達の準備を開始すべきである」
> — David Skok

このルールの理由：

1. **資金調達に要する期間**: 初接触から調達完了まで平均4-6ヶ月
2. **交渉のレバレッジ**: ランウェイに余裕があると、VCとの交渉で有利
3. **緊急感の回避**: 「今すぐ金が必要」という状態では悪い条件を受け入れやすくなる

## 使用タイミング

### 推奨実行頻度

- **毎月末日**: 月次決算後に実行（月1回、定例化推奨）
- **大型支出前**: オフィス移転、採用開始、新規プロダクト立ち上げなど
- **資金調達計画策定時**: 調達額・タイミングを検討する際
- **四半期ごとの経営会議**: CEO・CFOが経営判断をする際

### 実行条件

以下の情報が用意されていれば実行可能：

- 現在の銀行口座残高
- 月次の詳細な支出内訳（人件費、オフィス費、マーケティング費など）
- 月次の収益（売上・契約金など）

## 18ヶ月ルール - 詳細解説

### ランウェイの判定基準（4段階）

| ランウェイ | 判定 | 状態 | 推奨アクション |
|-----------|------|------|--------------|
| **18ヶ月以上** | ✅ 安全 | 余裕あり | 月次監視継続、成長投資検討 |
| **12-18ヶ月** | ⚠️ 警告 | 準備期間 | 投資家リスト作成、ピッチ準備、コスト最適化開始 |
| **6-12ヶ月** | 🚨 緊急 | 調達必須 | 即座に調達活動開始、採用延期、コスト削減実行 |
| **6ヶ月以下** | 🔴 危機的 | 倒産リスク | 緊急ブリッジ調達、ドラスティックコスト削減、Pivot検討 |

### 各段階での具体的アクション

#### ✅ 安全（≥18ヶ月）

- 月次でKPI監視（バーンレート、ARPU、Churn率）
- PMF達成に向けたプロダクト開発に集中
- マーケティング・採用を強化してもOK
- ただし「バーンレートが月5%以上増加」したら要注意

#### ⚠️ 警告（12-18ヶ月）

- **Now**: コスト最適化プロジェクト開始
  - ROI < 1.0のマーケティング施策を停止
  - オフィス費・ツール費を見直し
  - 目標: 月次10-15%のコスト削減
- **Next 3ヶ月**: 資金調達準備
  - ピッチデッキ・データルーム準備
  - 投資家10社以上にウォームイントロ
  - 初回ミーティング5社以上設定

#### 🚨 緊急（6-12ヶ月）

- **今月**: 投資家との初回ミーティング開始（最低5社）
- **翌月**: デューデリジェンス開始（2-3社）
- **並行**: 即効性のあるコスト削減実行
  - オフィス規模縮小
  - 採用計画の全停止
  - ツール・外注費の削減
- **代替案**: ブリッジラウンド、コンバーティブルノート検討

#### 🔴 危機的（<6ヶ月）

- **緊急対応**:
  - 既存投資家への緊急相談
  - ブリッジラウンド（簡易な条件で短期資金調達）
  - 取引先・顧客からの前払い依頼
  - 緊急経営会議（全チーム）
- **並行**:
  - 人員削減検討（最小限の人員で運営）
  - オフィス撤退
  - 非コア事業の停止
  - Pivot検討（`/pivot-decision` スキル実行）

## 使用例（実践シナリオ）

### シナリオ1: ランウェイ4ヶ月（危機的ケース）

**入力**:
```
現在残高: ¥5,000,000
月次収益: ¥300,000
月次支出: ¥1,500,000
```

**実行**:
1. スキル実行 → 計算
2. Net Burn Rate = ¥1,200,000/月
3. Runway = 5,000,000 / 1,200,000 = **4.2ヶ月 🔴**

**出力 - 推奨アクション**:
```
判定: 🔴 危機的 → 緊急対応必須

推奨アクション:
1. 既存投資家へのブリッジラウンド依頼（今週中）
2. 即効性のあるコスト削減：
   - オフィス費 ¥150,000 → ¥50,000（-¥100,000）
   - 外注費 ¥300,000 → ¥150,000（-¥150,000）
   合計: -¥250,000/月
3. 削減後Runway = 5,000,000 / 950,000 = 5.3ヶ月（やや改善）
4. それでも不足 → Pivot検討 or 資金調達必須
```

### シナリオ2: ランウェイ15ヶ月（警告ケース）

**入力**:
```
現在残高: ¥30,000,000
月次収益: ¥500,000
月次支出: ¥2,500,000
```

**実行**:
1. Net Burn Rate = ¥2,000,000/月
2. Runway = 30,000,000 / 2,000,000 = **15.0ヶ月 ⚠️**

**出力 - 推奨アクション**:
```
判定: ⚠️ 警告 → 資金調達準備開始

推奨アクション:
1. コスト最適化（今月実行）：
   - ROI < 1.0のマーケティング施策停止: -¥300,000/月
   - ツール費見直し: -¥50,000/月
   合計: -¥350,000/月 = バーンレート-17.5%

2. 削減後Runway = 30,000,000 / 1,650,000 = 18.2ヶ月（安全圏へ）

3. 資金調達準備（並行）：
   - 目標調達額 = ¥2,000,000 × 18 = ¥36,000,000
   - 推奨方法: シリーズA
   - Week 1-2: ピッチデッキ準備、投資家リスト作成
   - Week 3-4: ウォームイントロ、初回ミーティング
   - Month 2-3: デューデリジェンス
```

### シナリオ3: ランウェイ25ヶ月（安全ケース）

**入力**:
```
現在残高: ¥50,000,000
月次収益: ¥1,000,000
月次支出: ¥3,000,000
```

**実行**:
1. Net Burn Rate = ¥2,000,000/月
2. Runway = 50,000,000 / 2,000,000 = **25.0ヶ月 ✅**

**出力 - 推奨アクション**:
```
判定: ✅ 安全 → 成長投資検討

推奨アクション:
1. 月次監視継続（バーンレート増加に注意）
2. 成長投資の検討：
   - マーケティング予算を月+¥500,000へ増加
   - 採用を年2人ペースで加速
   - R&Dへの投資増加
3. 注意: バーンレート増加が月5%を超えたら再度ランウェイを計算
```

## よくある質問（FAQ）

### Q1. バーンレートと赤字は同じですか？

**A**: 異なります。

- **バーンレート**: 現在の消費ペース（計画無関係）
- **赤字**: 収入 < 支出の経営状況

バーンレート計算では、現在の消費率で「いつ資金が尽きるか」を判定します。一方、赤字は会計的な損益を示しています。

**例**:
```
バーンレート: ¥1,200,000/月 → 4.2ヶ月で資金枯渇
赤字: ¥1,200,000/月 → 既に損失を計上
```

### Q2. Runway計算に現在の支出増減を反映すべきですか？

**A**: 基本は「現状の支出ペース」で計算します。ただし：

- **予測可能な変動**: 採用予定がある場合は反映すべき
- **不確実な削減計画**: 含めない（実現率が低いため）

**保守的アプローチ**:
```
現在のバーンレート → Runway計算
削減後のバーンレート → 別途「シナリオ分析」で試算
```

### Q3. 18ヶ月ルール以外の判定基準はありますか？

**A**: はい。段階別に異なる判定基準があります：

| ステージ | 目安Runway |
|----------|-----------|
| シード期（プレSeries A） | 12ヶ月以上 |
| Series A | 18ヶ月以上（起業の科学基準） |
| Series B以降 | 24ヶ月以上 |

ただし、本スキルは「起業の科学」の18ヶ月ルールを基準にしています。

### Q4. 資金調達額は「18ヶ月 × バーンレート」で計算すべきですか？

**A**: これは**最小限の目安**です。実際の調達額は以下を考慮すべき：

```
調達額 = バーンレート × 18ヶ月 + 成長投資額 + 緊急予備費

成長投資額: マーケティング・採用の加速予算
緊急予備費: 想定外の支出に対応（通常は6-12ヶ月分）
```

**実践例**:
```
最小調達額 = ¥1,200,000 × 18 = ¥21,600,000
+ 成長投資 = ¥5,000,000
+ 予備費 = ¥3,000,000（3ヶ月分）
─────────────────────────
推奨調達額 = ¥29,600,000 → ¥30,000,000（四捨五入）
```

### Q5. キャッシュフロー黒字の場合（Net Burn Rateがマイナス）はどうする？

**A**: **ランウェイは無限大（∞）** = 自走可能な状態です。

```
月次収益: ¥5,000,000
月次支出: ¥3,000,000
Net Burn Rate: -¥2,000,000/月（黒字）

判定: ✅ 優秀 → 成長投資の加速
推奨: マーケティング・採用を強化して成長速度を上げる
```

この場合、資金調達は「成長加速目的」（従来のVC調達）に切り替わります。

### Q6. 月次バーンレートが変動する場合はどう対応？

**A**: **過去3ヶ月の平均バーンレート**を使用してください。

```
11月: ¥1,000,000
12月: ¥1,200,000
1月: ¥1,400,000

平均 = (¥1,000,000 + ¥1,200,000 + ¥1,400,000) / 3 = ¥1,200,000
```

単月での変動を反映させると、Runway計算が不安定になります。
