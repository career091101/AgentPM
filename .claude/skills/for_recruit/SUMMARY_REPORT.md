## エグゼクティブサマリー

Recruit_Product_Researchの86件の需要発見・課題検証事例から、ForRecruit向けに最適化された2つのスキルを作成しました。既存顧客基盤活用により、Origin基準よりも**低コスト・低リスク**で検証可能な評価基準を設定しています。

### 主要成果

1. **評価基準の最適化**:
   - 市場規模（TAM）: 100億円 → **50億円**（社内リソース活用で小規模市場も参入可能）
   - インタビュー数: 20人 → **10人**（既存顧客基盤活用で効率化）
   - 課題共通率: 70% → **60%**（社内PoC前提で緩和）

2. **社内リソース活用パターンの体系化**:
   - 営業網活用: ホットペッパーグルメ2,000名規模
   - 既存顧客活用率: 成功製品85%（12/16製品）
   - 社内先行導入率: 31%（5/16製品）

3. **定量ベンチマークの確立**:
   - User Research Count: 成功製品平均35.2回 → ForRecruit推奨15回
   - Problem Commonality: 成功製品平均72.9% → ForRecruit推奨60%
   - LTV/CAC比: 成功製品平均15-48倍（Airレジ48倍、Geppo6.0倍）

| 指標 | 数値 | 手法 |

| CAC削減効果 | 1/10（50万円→5万円） | 営業網2,000名活用 |
| 初期導入率 | 50% | 既存顧客からの転換 |
| スケール速度 | 3年で10万店 | 営業網による直販 |

**再現可能な手法**:
- ホットペッパーグルメ営業網2,000名規模で飲食店への直販
- Airレジ既存顧客90.4万店舗へのクロスセル
- リクルートブランド信頼性で初期獲得コスト30-50%削減

### パターン2: 社内先行導入型（5/16製品、31%）

**代表事例**: Geppo、エリクラ

| 指標 | Geppo（成功） | エリクラ（失敗） |
|------|-------------|---------------|
| 社内運用期間 | 4年間（サイバーエージェント） | 6年間（実証実験レベル） |
| 回答率/活用率 | 96% → 98% | 10万人（低活用） |
| 外販後の成果 | 継続率98%、25名～数万名導入 | 2025年6月撤退 |
| PMF判断期間 | 4年後に外販判断 | 6年継続も撤退 |

**成功条件**:
- 1-2年以内にPMF判断（Geppoは4年だが、これは長すぎる）
- 社内での高い活用率（Geppo 96%、エリクラは低かった）
- 定量データ取得（離職率改善等）で営業資料化

**失敗条件**:
- 6年間「実証実験レベル」継続は異常（エリクラ）
- 差別化要素が弱い（10分単位、地産地消はニーズ弱い）
- 競合に100倍差をつけられる（タイミー1,000万人 vs エリクラ10万人）

## 失敗パターン分析

### 失敗要因TOP3

1. **競合優位性の欠如**（5製品、33%）
   - CODE.SCORE: 課題共通率38%、競合優位性不足
   - エリクラ: 差別化要素が弱い、タイミーに100倍差

2. **市場ニーズの過大評価**（5製品、33%）
   - リクルートDMPフォロー: 採用担当者のDMP活用意欲を過大評価
   - 軽いヒアリングで済ませず、定量調査併用が必要

3. **ビジネスモデルの構造的欠陥**（3製品、20%）
   - スタサプ個別指導: 月額10,780円では講師人件費を賄えない
   - エリクラ: 廃棄物処理責任を外部転嫁、法的リスク

### 教訓

| 失敗パターン | 教訓 | ForRecruit対策 |
|------------|------|--------------|
| 競合優位性欠如 | 競合が10倍優れている市場には参入しない | 既存ブランド力で差別化、5社以下の競合環境を選択 |
| ニーズ過大評価 | 軽いヒアリングは危険、定量調査併用 | 既存顧客ヒアリングでもバイアス注意、10人以上インタビュー必須 |
| Unit Economics破綻 | 収益性犠牲の成長は持続せず | LTV/CAC > 3、CAC回収期間 < 12ヶ月を厳守 |
| 撤退判断の遅れ | 6年実証実験は異常 | 社内PoC 1-2年以内にPMF判断、成長曲線が見えなければ撤退 |

## ForRecruit固有の評価基準

### 市場機会評価（調整版）

| 指標 | Origin | ForRecruit | 調整理由 |
|------|--------|----------|---------|
| TAM | 100億円以上 | **50億円以上** | 社内リソース活用で小規模市場も参入可能 |
| 成長率 | 10%/年以上 | **5%/年以上** | 既存顧客基盤で安定成長可能 |
| 競合飽和度 | 3社以下 | **5社以下** | 既存ブランド力で差別化可能 |

### 課題検証基準（緩和版）

| 指標 | Origin | ForRecruit | 調整理由 |
|------|--------|----------|---------|
| インタビュー数 | 20人以上 | **10人以上** | 社内ネットワーク活用、既存顧客基盤活用 |
| 課題共通率 | 70%以上 | **60%以上** | 社内PoC前提での段階的検証 |
| 緊急性スコア | 7/10以上 | **6/10以上** | 社内リソース活用で解決可能 |

### 社内リソース活用評価（新規追加）

| 評価項目 | 配点 | 評価基準 |
|---------|------|---------|
| 既存顧客基盤との関連性 | 5点 | 既存事業（ホットペッパー、じゃらん、SUUMO、リクナビ等）の顧客層と重なるか |
| 社内営業網活用可能性 | 5点 | 営業網（ホットペッパー営業2,000名等）で販売可能か |
| ブランド力転用可能性 | 3点 | リクルートブランドの信頼性が活かせるか |
| 既存データ資産活用 | 3点 | 決済データ、顧客データ等の既存資産が活用できるか |
| クロスセル機会 | 4点 | 既存製品とのエコシステム統合が可能か |

**ForRecruit適合性スコア**: 20点満点
- 15点以上: ✅ 既存資産活用の好機
- 10-14点: ⚠️ 一部資産活用可能
- 9点以下: ❌ 資産活用困難、Origin基準で再評価

## 成功事例詳細（統合10件）

### 1. Airレジ（CORP_S009）
- **市場ニーズ**: 中小飲食店・小売店の75%がPOSレジ未導入
- **User Research**: 30回（ホットペッパーグルメ加盟店）
- **10倍優位性**: 初期費用100倍削減（50-100万円 → 0円）
- **リクルート資産**: 営業網2,000名、既存顧客8万店
- **成果**: 90.4万アカウント、市場シェア44%

### 2. Airペイ（CORP_S001）
- **市場ニーズ**: 小規模店舗の85%がキャッシュレス未対応
- **User Research**: 100回（Airレジ既存顧客）
- **10倍優位性**: 初期費用100倍削減、対応決済77種類（3倍）
- **リクルート資産**: Airレジ顧客基盤90.4万店、営業網2,000名
- **成果**: 51.5万店舗、市場シェア35%

### 3. Geppo（CORP_M001）
- **市場ニーズ**: 離職の兆候を早期発見する手段がない
- **User Research**: 50回以上（社内検証含む）

- **リクルート資産**: サイバーエージェント社内4年運用、リクルート1,200名導入
- **成果**: 継続率98%

### 4. レストランボード

## 失敗事例詳細（統合4件）

### 1. CODE.SCORE（CORP_F002）
- **失敗要因**: 課題共通率38%、既存事業シナジー不足
- **教訓**: 競合が10倍優れている市場には参入しない

### 2. エリクラ
- **失敗要因**: 6年実証実験も撤退、差別化要素が弱い
- **教訓**: 早期撤退判断の重要性、1-2年でPMF判断

### 3. リクルートDMPフォロー
- **失敗要因**: ニーズ過大評価、定量検証不足
- **教訓**: 既存顧客ヒアリングでもバイアス注意

### 4. スタディサプリ個別指導
- **失敗要因**: Unit Economics破綻、自社製品カニバリゼーション
- **教訓**: 収益性犠牲の成長は持続せず、価格差に見合う価値提供
