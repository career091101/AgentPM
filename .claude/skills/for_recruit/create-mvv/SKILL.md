# Create MVV Skill (ForRecruit Edition)

MVV（Mission/Vision/Values）を早期定義する自律実行型Skill。**ForRecruit特化版**では、リクルート企業価値観との整合性チェックを必須化し、既存事業とのシナジー評価を追加します。

---

## このSkillでできること

1. **Mission定義**: 何のために存在するか（存在意義）を明確化
2. **Vision定義**: どこを目指すか（目指す未来）を描画
3. **Values定義**: どう行動するか（行動指針）を策定
4. **企業価値観整合性チェック**: リクルート6つの価値観との整合性検証（ForRecruit追加）
5. **既存事業シナジー評価**: 既存事業との相互補完性、カニバリゼーション回避（ForRecruit追加）
6. **MVV整合性検証**: MVV 3要素の相互整合性を確認
7. **Lean Canvas整合**: リーンキャンバスとの整合性を検証

## Domain-Specific Knowledge (from Recruit_Product_Research)

### Success Patterns

#### 1. Geppo - リクルート価値観「個の尊重」との整合性

**MVV構造**:
- **Mission**: 従業員一人ひとりの声を大切にし、組織の健全性を高める
- **Vision**: すべての企業で従業員エンゲージメントが可視化され、離職率が半減する世界
- **Values**:
  1. 個の尊重: 従業員の本音を尊重し、匿名性を保証
  2. 継続的改善: 月1回の短時間サーベイでリアルタイム改善
  3. データドリブン: 定量データで組織課題を可視化

**リクルート企業価値観との整合性**:
- ✅ **個の尊重**: 従業員エンゲージメント向上により、個の力を最大化
- ✅ **新しい価値の創造**: 回答負荷10倍軽減（30分→1分）、継続率98%の革新
- ✅ **社会への貢献**: 離職率改善（20%→10%）により、企業と社会の健全性向上

**社内先行導入の効果**:
- リクルート1,200名で4年間先行運用（2018年〜）
- 離職防止効果を実証後、外販開始
- 自社での実証データが営業資料になる

#### 2. Airシリーズ - リクルート価値観「新しい価値の創造」「社会への貢献」

**MVV構造（Airレジ例）**:
- **Mission**: 中小企業のDXを支援し、地域経済を活性化する
- **Vision**: 全国すべての中小店舗がデジタル化され、経営効率が向上する世界
- **Values**:
  1. 中小企業ファースト: 初期費用0円、使いやすさ重視
  2. エコシステム連携: Airペイ、Airキャッシュ等と統合、LTV 3-5倍向上
  3. データドリブン: 売上データ可視化、経営判断支援

**リクルート企業価値観との整合性**:
- ✅ **新しい価値の創造**: 初期費用100倍削減（50-100万円→0円）
- ✅ **社会への貢献**: キャッシュレス推進、中小企業DX支援
- ✅ **個の尊重**: 店舗オーナーの経営自律性向上

**既存事業とのシナジー**:
- ホットペッパーグルメ営業網2,000名 → Airレジ直販、CAC 1/5〜1/10削減
- ホットペッパーグルメ顧客基盤 → Airレジベータテスト、クロスセル促進
- エコシステム連携: ホットペッパーグルメ予約データ × Airレジ売上データ統合

#### 3. SUUMO - 既存事業統合によるシナジー最大化

**MVV構造**:
- **Mission**: すべての人に最適な住まいを見つける機会を提供する
- **Vision**: 日本一信頼される不動産情報プラットフォーム
- **Values**:
  1. 情報の透明性: 掲載物件数最大化、比較しやすい情報設計
  2. ユーザーファースト: 検索体験の最適化、地域情報充実
  3. 不動産事業者との共創: Win-Winの関係構築

**既存事業とのシナジー**:
- リクルート住宅情報 + 住宅情報ナビ統合
- ブランド統合によりマーケティング効率**3倍向上**
- 広告費**40%削減**（ブランド認知度向上）
- 不動産仲介会社との既存関係活用 → 営業網拡大

### Common Pitfalls

#### 1. スタディサプリ個別指導 - 既存製品カニバリゼーション

**MVV構造の問題**:
- **Mission**: すべての学習者に個別最適化された教育を提供する
- **Vision**: 教育の機会均等、オンライン個別指導の普及
- **Values**: 個別最適化、低価格化、アクセシビリティ向上

**失敗の本質**:
- ベーシックコース（2,178円）が優秀すぎて、個別指導（10,780円）が**5倍の価格差に見合う価値を提供できず**
- Mission「個別最適化」はベーシックでも実現可能、個別指導の独自価値が不明確
- 既存製品との**補完関係**でなく**代替関係**になり、カニバリゼーション発生
- 結果: 1.5年で撤退

**教訓**:
- MVV策定時に**既存製品との差別化**を明確化
- 価格差に見合う**10倍優位性**をValuesに反映
- 既存製品を**補完**する価値提案、**代替**しない

#### 2. エリクラ - 社会的責任との不整合

**MVV構造の問題**:
- **Mission**: 地域の人手不足解消、廃棄物処理のスピード向上
- **Vision**: 10分単位で働ける世界、地産地消の雇用創出
- **Values**: スピード、柔軟性、地域貢献

**失敗の本質**:
- ビジネスモデルが**廃棄物処理責任をワーカーに転嫁**、不法投棄を誘発
- リクルート企業価値観「社会への貢献」との**矛盾**
- 法的コンプライアンスを軽視、社会的批判を招く
- 結果: 6年で撤退

**教訓**:
- MVV策定時に**法的コンプライアンス、社会的責任**を検証
- リクルート企業価値観「社会への貢献」との整合性を厳格にチェック
- 短期的利益と長期的社会価値のバランス

#### 3. CODE.SCORE - ターゲット市場とのミスマッチ

**MVV構造の問題**:
- **Mission**: エンジニアスキルを可視化し、教育の質を向上させる
- **Vision**: すべての教育機関でエンジニアスキル評価が標準化される
- **Values**: 技術力重視、客観的評価、教育効果最大化

**失敗の本質**:
- ターゲット市場（教育機関・塾）は**AI個別最適化学習（atama+, Monoxer）**を求めていた
- エンジニアスキル可視化は**ニーズとミスマッチ**
- Mission「エンジニアスキル可視化」は自社視点、顧客視点が不足
- 結果: 2年で撤退

**教訓**:
- Mission/Vision策定時に**顧客ニーズとの整合性**を厳密に検証
- 「自社が提供したいもの」でなく「顧客が本当に求めるもの」を起点に
- 競合が10倍優れている領域への参入は避ける

### Best Practices

#### 1. 企業価値観整合性チェックの実施

**チェック方法**:
```markdown
## リクルート企業価値観整合性チェック（6項目）

| 価値観 | 整合性 | 根拠 |
|-------|:------:|------|
| 新しい価値の創造 | ✅/❌ | [Mission/Vision/Valuesのどこに反映されているか] |
| 個の尊重 | ✅/❌ | [ユーザー/従業員の個をどう尊重するか] |
| 社会への貢献 | ✅/❌ | [どの社会課題を解決するか] |
| 最高へのこだわり | ✅/❌ | [品質基準、10倍優位性はどこか] |
| 当事者意識 | ✅/❌ | [自社先行導入、リスクテイクはあるか] |
| チームワーク | ✅/❌ | [社内連携、エコシステム構築はあるか] |

**合格基準**: 6項目中**4項目以上**がと整合性があること
```

#### 2. 既存事業シナジー評価の実施

**評価軸**:
1. **補完関係の強さ**: 既存事業を強化するか、代替するか（補完関係が必須）
2. **リソース活用可能性**: 営業網、顧客基盤、データ資産を活用できるか
3. **カニバリゼーションリスク**: 既存事業の売上を侵食しないか
4. **ブランド整合性**: リクルートブランドと整合するか

**評価フォーマット**:
```markdown
## 既存事業シナジー評価

| 既存事業 | 補完関係 | リソース活用 | カニバリリスク | ブランド整合 | 総合評価 |
|---------|:-------:|:----------:|:------------:|:----------:|:--------:|
| [事業A] | ✅ | ✅ | ❌ | ✅ | **高** |
| [事業B] | ⚠️ | ✅ | ⚠️ | ✅ | **中** |

**総合シナジースコア**: [高/中/低]
**カニバリゼーション対策**: [対策内容]
```

#### 3. 社内先行導入前提のMVV設計

**Geppoモデル**:
1. **自社先行導入**: リクルート1,200名で4年間運用
2. **実証データ蓄積**: 離職率改善（20%→10%）を定量化
3. **営業資料化**: 自社実証データが信頼性の源泉
4. **外販開始**: 実証済みの価値提案で顧客獲得

**MVVへの反映**:
- **Mission**: 自社で実現済みの社会価値を明記（例: 離職率半減）
- **Vision**: 自社実証データをベースにした定量目標（例: すべての企業で従業員エンゲージメント可視化）
- **Values**: 自社先行導入の経験から抽出した行動指針（例: データドリブン、個の尊重）

## Instructions

**実行モード**: 自律実行（対話なし）
**推定所要時間**: 20-40分

### 自動実行ステップ

1. **ビジネスアイデア・リーンキャンバス読み込み**
2. **ベンチマーク企業MVV調査**（Geppo、Airシリーズ等）
3. **Mission草案作成**（1-2文、30秒で読める、社会的意義明確）
4. **Vision草案作成**（3-5年後の理想状態、定量目標含む）
5. **Values草案作成**（3-5個の行動指針）
6. **企業価値観整合性チェック**（6項目、4項目以上合格必須）（ForRecruit追加）
7. **既存事業シナジー評価**（補完関係、リソース活用、カニバリゼーション）（ForRecruit追加）
8. **MVV整合性検証**（Mission→Vision→Values）
9. **Lean Canvas整合性検証**（Problem/UVP/Solution整合）
10. **成果物出力**

### 判定基準（7項目チェック、ForRecruit調整版）

| 項目 | 合格条件 |
|------|----------|
| Mission明確性 | 30秒で読め、社会的意義が明確 |
| Vision具体性 | イメージ可能、測定可能要素あり |
| Values実用性 | 3-5個、具体的行動につながる |
| MVV整合性 | Mission→Vision→Values整合（3/3） |
| Lean Canvas整合 | Problem/UVP/Solution整合（3/3） |
| **企業価値観整合性** | **6項目中4項目以上** |
| **既存事業シナジー** | **補完関係あり、カニバリゼーション対策済み** |

**総合判定**:
- 7/7: ✅ MVV完了 → 次のステップへ
- 5-6/7: ⚠️ 要改善 → 不合格項目を再作成
- 0-4/7: ❌ 再定義 → ビジネスアイデア見直し
