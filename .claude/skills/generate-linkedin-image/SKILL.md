# LinkedIn投稿用画像生成スキル

LinkedIn投稿の本文から、高エンゲージメント仕様のOGP画像を自動生成するスキルです。

## 概要

このスキルは、LinkedIn投稿テキストを入力として、以下を生成します：

1. **フォトリアルな写真**: 映画的ドキュメンタリースタイル
2. **断定型キャッチコピー**: 投稿内容から抽出した数値・企業名を含む訴求力の高いテキスト
3. **最適化されたレイアウト**: 1:1正方形、スクリム効果、WCAG基準準拠

## 使用方法

### 基本的な実行

```bash
# スキル起動
/generate-linkedin-image
```

実行すると、以下の選択肢が表示されます：

```
📋 利用可能なサンプルプロンプト:
  1. AI BPO（LayerX福島氏）
  2. AGI到来予測（Google DeepMind）
  3. Claude Opus 4.5（5時間業務）
  4. AI開発工数80%削減（トランスコスモス）
  5. カスタムプロンプトを入力

選択してください (1-5):
```

### カスタムプロンプトでの実行

選択肢5を選択し、LinkedIn投稿テキストを入力してください：

```
選択してください (1-5): 5

📝 カスタムプロンプトを入力してください:
> [LinkedIn投稿の本文をペースト]
```

## 生成される画像の仕様

### 基本設定

| 項目 | 仕様 | 根拠 |
|------|------|------|
| **サイズ** | 1080×1080px | LinkedIn推奨 |
| **アスペクト比** | 1:1（正方形） | 2026年調査で最高エンゲージメント |
| **スタイル** | 映画的ドキュメンタリー写真 | フォトリアル、8K UHD |
| **ファイル形式** | PNG | 高品質、透過対応 |
| **ファイルサイズ** | 5MB以下 | LinkedIn制限 |

### テキストオーバーレイ

#### レイアウト
1. **ヘッダー**: 'AI Update'（固定）
2. **メインコピー**: 投稿から抽出したキャッチコピー（40-60文字、2-3行）

#### デザイン
- **背景グラデーション**: 画像下部30-40%に黒→透明のスクリム効果
- **テキスト色**: 白（#FFFFFF）
- **コントラスト比**: 4.5:1以上（WCAG 2.1 AA基準）
- **フォント**: 太字（Bold）、モバイル可読性重視

### キャッチコピーの特徴

生成されるキャッチコピーは、以下の要素を含みます：

- ✅ **断定型表現**（「〜だ」「〜である」「〜を意味する」）
- ✅ **具体的数値**（削減率、倍率、金額、期間など）
- ✅ **企業名・人名**（信頼性を高める固有名詞）
- ✅ **2-3行構成**（1行20-30文字、モバイル最適）

#### 例

```
❌ 「AIツールが普及しているそうです」（伝聞型、数値なし）
✅ 「AI導入で人員80%削減。2人で10人分の仕事が可能に」（断定型+数値）

❌ 「AGIが近い将来到来すると言われています」（抽象的）
✅ 「AGI到来は2028年。確率50%とDeepMind創設者が断言」（具体+企業名）
```

## 制約事項

### 肖像権・商標権保護

生成される画像には以下が含まれません：

- ❌ 実在の人物の顔・肖像
- ❌ 実在企業のロゴマーク
- ❌ 特定製品のデザイン

### 推奨ビジュアル

- ✅ 後ろ姿・シルエット
- ✅ 手元クローズアップ
- ✅ 抽象的メタファー（AI、データフロー、ネットワーク）
- ✅ オフィス環境、テクノロジー要素

## トラブルシューティング

### APIキーエラー

```
❌ エラー: 環境変数 GOOGLE_GEMINI_API_KEY が設定されていません
```

**解決方法**:
```bash
export GOOGLE_GEMINI_API_KEY='your-api-key-here'
source ~/.zshrc
```

### 画像生成エラー

```
❌ エラー: 画像データが生成されませんでした
```

**原因と解決方法**:
- プロンプトが不適切 → より具体的な説明を追加
- API制限 → 無料版は1日3枚まで（有料アカウントに移行）
- ネットワークエラー → インターネット接続を確認

### ファイルサイズ超過

```
⚠️  警告: ファイルサイズが5MBを超えています（LinkedIn制限）
```

**解決方法**: 自動で最適化済み（通常1-2MB）
