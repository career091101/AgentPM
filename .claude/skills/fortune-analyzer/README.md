# 運勢分析エージェント（fortune-analyzer）v2.0

四柱推命と九星気学を使って、今月・今年の運勢を自動分析するClaude Codeスキルです。

## 概要

このスキルは、指定された生年月日の人の**今月（2026年1月）**と**今年（2026年）**の運勢を、**35-45の信頼性の高いサイト**から情報を収集し、信頼性スコアリング付きMarkdownレポートとして出力します。

### v2.0の主な強化点

- **情報源を5倍に拡大**: 3-10サイト → 35-45サイト
- **信頼性スコアリング**: ★1-5の5段階評価システム導入
- **情報整合性チェック**: 複数サイト間での情報一致度を検証
- **カテゴリ別信頼性表示**: プロ占い師、専門サイト、メディアなど6段階に分類

### なぜ35-45サイトなのか？

**v1.0の課題**:
- 3-10サイトでは情報の偏りや矛盾を検証できない
- 特定の占い師の見解に依存してしまう
- 吉方位や吉日が異なる場合、どれを信じるべきか不明

**v2.0の解決策**:
- **統計的信頼性**: 5サイト以上で一致する情報は極めて信頼できると判断
- **多様な視点**: プロ占い師、専門サイト、メディアなど複数カテゴリから収集
- **矛盾検出**: 異なる見解がある場合、その内容と理由を明示
- **透明性**: 全42サイトの出典を信頼性カテゴリ別に明記

以下のいずれかのワードでスキルが起動します：

- 今月の運勢
- 今年の運勢
- 月運
- 年運
- 運勢分析

## パラメータ

### 必須パラメータ

- `birth_date`: 生年月日（YYYY/MM/DD形式）
  - 例: `1990/11/01`

### オプションパラメータ

- `output_path`: レポート保存先
  - デフォルト: `/Users/yuichi/AIPM/aipm_v0/Flow/202601/2026-01-01/`

## 出力形式

### ファイル名

```
運勢分析_YYYY-MM-DD_2026年1月&2026年.md
```

例: `運勢分析_1990-11-01_2026年1月&2026年.md`

### レポート構成

1. **基本情報**
   - 生年月日
   - 本命星・月命星
   - 分析日

2. **2026年1月の運勢**
   - 全体運
   - 仕事運・金運
   - 恋愛運・対人運
   - 健康運
   - 吉方位・ラッキーアイテム
   - 注意事項

3. **2026年の運勢**
   - 年間の運気の流れ
   - 重要な時期
   - 仕事・キャリア
   - 恋愛・結婚
   - 健康・生活
   - 年間吉方位
   - 開運アクション

4. **参考情報**
   - 調査に使用したサイトのURL

## 技術仕様

### 使用ツール

- **WebSearch**: 運勢情報の検索
- **WebFetch**: サイトからの情報抽出
- **Read**: 既存レポートの参照
- **Write**: 新規レポート作成
- **Edit**: レポートの更新

### 情報源（v2.0強化）

**35-45サイト**から以下のカテゴリ別に情報を収集：

#### 信頼性評価基準（優先順位順）

1. **プロの占い師の公式サイト（最優先）**
   - 実名・実績が明記されている
   - 長年の運営実績
   - 例: 八雲院、銀座の母、等

2. **九星気学専門サイト**
   - 長年運営されている老舗サイト
   - 理論的背景が充実
   - 例: kigaku.co.jp、yakumoin.info 等

3. **運勢カレンダー専門サイト**
   - 日別の詳細情報を提供
   - 吉方位・吉日カレンダー完備
   - 例: 2996.info（福来る）等

4. **大手メディアの占いコーナー**
   - Oggi、ELLE、Domani 等
   - 編集部の品質チェックあり

5. **占い館の公式サイト**
   - 実店舗を持つ占い館
   - 鑑定実績が明記されている

6. **気学教室・講座サイト**
   - 教育機関として運営
   - 理論解説が詳細

#### 情報収集規模

- **今月の運勢**: 15-20サイト（12種類の検索パターン）
- **今年の運勢**: 20-25サイト（18種類の検索パターン）
- **合計**: 35-45サイトから情報を統合

#### 信頼性スコアリング

- **5サイト以上で一致** → 信頼度: 極めて高い ★★★★★
- **3-4サイトで一致** → 信頼度: 高い ★★★★☆
- **2サイトで一致** → 信頼度: 中 ★★★☆☆
- **1サイトのみ** → 信頼度: 低（参考情報として扱う）★★☆☆☆

## 注意事項

1. **日付の正確性**
   - 今月 = 2026年1月
   - 今年 = 2026年
   - 過去の運勢情報は除外されます

2. **情報の信頼性（v2.0強化）**
   - **35-45サイト**から情報を収集
   - 情報整合性を★1-5の5段階で評価
   - 矛盾する情報は信頼度と共に併記
   - 全ての出典を信頼性カテゴリ別に明記

3. **表現の配慮**
   - ネガティブな運勢でも前向きな表現
   - 具体的なアドバイスを提供
