# Merriman Weekly Outlook - 週次金融占星術分析

MMA（Merriman Market Analyst）週次レポートから金融占星術（Geocosmic）分析を自動抽出。

---

## このSkillでできること

1. **自動レポート取得**: MMA公式サイトから最新週次レポートをWebFetch
2. **構造化抽出**: 重要天体配置、市場見通し、反転日（CRD）を自動抽出
3. **Markdown生成**: 読みやすい週次分析レポートを自動生成
4. **トレード統合**: TradingAgents Phase1との連携可能
5. **過去レポート保存**: 週次ディレクトリに自動保存

        print("⚠️ レポートが見つかりません（404 Not Found）")
        print("対処: URLを確認してください")
        # 最新週のURLを自動検索
        latest_url = find_latest_mma_weekly_url()

    elif "timeout" in str(e).lower():
        print("⚠️ タイムアウトエラー（サーバー応答なし）")
        # 10秒待機後に再試行
        time.sleep(10)
        report_content = WebFetch(url=mma_url, prompt=extraction_prompt)
    else:
        print(f"❌ WebFetchエラー: {e}")
