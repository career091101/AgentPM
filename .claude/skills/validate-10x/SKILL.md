# Validate 10x Skill

既存代替案と比較して10倍優位性を検証する自律実行型Skill。

---

## このSkillでできること

1. **競合・代替案の特定**: 直接競合/間接競合/既存対処法を洗い出し
2. **5軸比較**: 時間/コスト/使いやすさ/成果/導入障壁で定量比較
3. **10倍判定**: 各軸で10倍以上の優位性があるか判定
4. **改善アクション策定**: 未達成軸の改善案を提示
5. **ベンチマーク調査**: 成功企業の10倍優位性を参照

## Instructions

**実行モード**: 自律実行（対話なし）
**推定所要時間**: 40-70分

### 自動実行ステップ

1. リーンキャンバス読み込み
2. 競合・代替案の特定（Web調査）
3. **【v2新規】競合リサーチ成果物読込**（`competitor_research.md`）
4. 既存最良ソリューション選定
5. **【v2新規】Top 3競合特定**（`/research-competitors`成果物から）
6. 5軸比較表作成
7. **【v2新規】競合対比10倍検証**（vs. Top 3 Competitors）
8. 10倍判定（各軸）
9. 総合判定
10. 改善アクション策定
11. ベンチマーク調査
12. 成果物出力

### 5軸比較

| 軸 | 計算式 |
|----|--------|
| 時間効率 | 既存の所要時間 / 新の所要時間 |
| コスト | 既存の総コスト / 新の総コスト |
| 使いやすさ | 既存の学習コスト / 新の学習コスト |
| 成果・効果 | 新の成果 / 既存の成果 |
| 導入障壁 | 既存の導入ステップ数 / 新の導入ステップ数 |

### 判定基準

**10倍達成軸数による総合判定**:
- **2軸以上**: ✅ 10倍達成 → PSF検証/LP構築へ進む
- **1軸のみ**: ⚠️ 部分達成 → 強化軸を特定し再設計
- **0軸**: ❌ 未達成 → ソリューション根本見直し

### STEP 3: 競合優位性の根拠記述

**各10倍達成軸について、競合対比での優位性を明記**:

#### 軸1: 時間効率10倍（10時間 vs 競合A 100時間）

**根拠**:
- **自社**: AIエージェントが24時間自動実行（人的作業0時間）
- **競合A**: 手動データ収集 + 分析に40時間 + レポート作成に60時間
- **差異要因**: 完全自動化 vs 手作業

**持続可能性**: ✅ 高い（AI精度向上で時間効率さらに改善）

---

#### 軸2: コスト10倍（$5,000 vs 競合A $50,000）

**根拠**:
- **自社**: SaaS月額課金（変動費モデル）+ 自動化による人件費削減
- **競合A**: コンサルタント派遣（固定費 + 人件費高騰）
- **差異要因**: スケールメリット + 人的コスト削減

**持続可能性**: ✅ 高い（顧客数増加でコストさらに削減）

---

#### 軸3: 導入障壁10倍（3日 vs 競合A 30日）

**根拠**:
- **自社**: ノーコード設定（3ステップ、計3日）
- **競合A**: カスタムコーディング必須（10-15ステップ、計30日）
- **差異要因**: ノーコード vs フルコーディング

**持続可能性**: ✅ 高い（テンプレート拡充で導入さらに短縮）

---

### STEP 4: 未達成軸の改善提案（競合対比）

**成果軸（1.2倍、10倍未達）**:

**現状分析**:
- 自社: 95%改善
- 競合Best: 85%改善（競合B）
- Gap: +10%（相対的には優位だが絶対値で10倍未達）

**改善案**:
1. **アルゴリズム強化**: 機械学習モデル精度向上で95% → 98%
2. **エッジケース対応**: 残り5%の未対応ケースを解決
3. **ベンチマーク見直し**: 「改善率」ではなく「絶対効果（売上増加額）」で測定
   - 例: 自社 $100K売上増 vs 競合A $10K → 10倍達成 ✅

**期待効果**: 成果軸で10倍達成 → **4軸10倍**へ向上

---

**使いやすさ軸（1.4倍、10倍未達）**:

**現状分析**:
- 自社: 9/10
- 競合Best: 7/10（競合B）
- Gap: +2ポイント（1.3倍程度）

**改善案**:
1. **学習コスト削減**: オンボーディング時間を1/10に短縮（10分 → 1分）
2. **UI簡素化**: ワンクリック実行（現状3クリック）
3. **音声UI追加**: 音声コマンド対応で操作0秒化

**期待効果**: 使いやすさで10倍達成 → **4軸10倍**へ向上
