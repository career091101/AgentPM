## 附録: 計算式リファレンス

### Sean Ellisテスト

```
Sean Ellisスコア = （「非常に残念」回答数 / 有効回答数） × 100

注）「よくわかからない」は分母から除外
```

### 月次成長率

```
月次成長率 = （当月値 - 前月値） / 前月値 × 100

3ヶ月移動平均 = （1ヶ月目の成長率 + 2ヶ月目の成長率） / 2
```

### Churn Rate

```
月次Churn Rate = （月初顧客数 - 月末顧客数） / 月初顧客数 × 100
```

### NPS

```
NPS = （Promoter数 / 総回答数 × 100） - （Detractor数 / 総回答数 × 100）
```

### ICEスコア

```
ICEスコア = Impact × Confidence × Ease

Impact (1-10): この施策で指標が改善される度合い
Confidence (1-10): この施策が成功する確実性
Ease (1-10): 実装にかかる時間・リソース（少ないほど高い）
```
