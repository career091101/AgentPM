## II. 失敗パターン分類（撤退理由別）

### 撤退理由1: 競合優位性の欠如（5製品）

#### CORP_F003: エリクラ

**概要**: スキマバイトマッチング、6年実証実験後撤退（2018-2025）

| 項目 | 数値/内容 |
|-----|---------|
| **CPFスコア** | 推定52% |
| **Problem Commonality** | 50%未満 |
| **社内先行導入** | サイバーエージェント社内6年間実証実験レベル |
| **10倍優位性** | 0軸（2-3倍程度、差別化弱い） |
| **外部スケール** | 失敗（10万人、タイミー1,000万人と100倍差） |
| **サービス期間** | 6年（異常な長期化） |

**失敗要因**:
1. 差別化要素「10分単位」「地産地消」は顧客ニーズが弱い
2. プラットフォーム事業は初速が命、100倍差を覆せない
3. 6年間「実証実験」レベル継続は異常、早期撤退判断すべき
4. 廃棄物処理責任をワーカーに転嫁する法的リスク

**教訓**:
- 社内先行導入は1-2年以内にPMF判断（6年は異常）
- 競合に100倍差をつけられたら撤退
- 法的リスクを外部転嫁するビジネスモデルは持続不可能

**推奨スキル（反面教師）**:
- `/for-recruit-research-problem` (社内先行導入の失敗事例)
- `/for-recruit-validate-ring-criteria` (Ring 3失敗パターン)
- `/for-recruit-validate-10x` (10倍優位性0軸の失敗事例)

**Ring制度**: Ring 3外部スケール失敗パターン

### 撤退理由2: 市場構造変化への対応遅れ（4製品）

#### CORP_F007: エイビーロード

**概要**: 海外旅行情報誌・サイト、37年継続後撤退（1984-2021）

| 期間 | 市場環境 | 対応 | 結果 |
|-----|---------|------|------|
| 1984-2000年代 | 旅行代理店経由が主流 | メタサーチ成功 | 圧倒的シェア |
| 2010年代 | 直接予約増加、OTA台頭 | 情報提供型のまま | シェア低下 |
| 2020年 | COVID-19、海外旅行84.2%減 | - | 2021年3月終了 |

**失敗要因**:
1. 市場変化（直接予約増加）を認識しながらピボットできず
2. 情報提供型から取引型（OTA）への転換が必要だったが実行されず
3. COVID-19は最後の一押し、構造変化への対応遅れが本質

**教訓**:
- 市場変化の兆候を常時モニタリング、大胆なピボットを躊躇しない
- 「継続できる」≠「継続すべき」、成長曲線が描けない事業は早期撤退
- プラットフォーム進化の必要性: 情報提供→取引→エコシステム

**推奨スキル（反面教師）**:
- `/for-recruit-validate-pmf` (市場変化対応失敗の事例)
- `/for-recruit-startup-scorecard` (成長曲線停滞の失敗事例)

#### CORP_F011: スタディサプリ個別指導塾オンライン中学講座

**概要**: オンライン個別指導、1.5年で撤退

| 項目 | 数値/内容 |
|-----|---------|
| **月額料金** | 10,780円 |
| **ベーシックコース** | 2,178円（5倍の価格差） |
| **Unit Economics** | 講師人件費を賄えない |
| **サービス期間** | 約1.5年（超短期撤退） |

**失敗要因**:
1. 月額10,780円では講師人件費を賄えない
2. ベーシックコース2,178円が優秀すぎて高額版が売れない
3. 自社製品カニバリゼーション
4. LTV/CAC比 推定1-2倍（赤字または微益）

**教訓**:
- Unit Economicsを厳密に計算、収益性を犠牲にした成長は持続しない
- 自社製品カニバリゼーション回避: 既存製品が優秀すぎると高額版が売れない
- 価格差に見合う価値提供が必要

**推奨スキル（反面教師）**:
- `/for-recruit-design-pricing` (Unit Economics破綻の失敗事例)
- `/for-recruit-build-flywheel` (カニバリゼーションの失敗事例)

**Ring制度**: Ring 2-3での超短期撤退判断（1.5年で撤退は正しい判断）
