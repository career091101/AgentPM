# 有料note構成テンプレート

## 概要

このファイルは、有料note生成スキル（generate-note-premium-pro）で使用する構成テンプレートを定義します。

## 各セクションの詳細ガイドライン

### セクション1: 表紙（300文字）

#### タイトル作成の公式

1. **数字型**
   - 「○○で月○万円稼ぐ」
   - 「○日で○○をマスターする」
   - 「○○の○つの法則」

2. **問いかけ型**
   - 「なぜ○○だけが成功するのか？」
   - 「あなたは○○していませんか？」

3. **断定型**
   - 「○○完全攻略」
   - 「○○の教科書」
   - 「○○バイブル」

4. **逆説型**
   - 「○○するな」
   - 「○○は間違いだった」

#### サブタイトル作成の公式

- **ターゲット明示型**: 「○○な人のための」
- **成果明示型**: 「○○を実現する」
- **時間明示型**: 「○日で身につく」
- **レベル明示型**: 「初心者からプロまで」

### セクション2: このnoteで得られること（500文字）

#### 箇条書きの公式

```markdown
## このnoteで得られること

- ○○の基本原則が**完全に理解**できる
- ○○を**今日から実践**できるようになる
- ○○で**月○万円**の収益が得られる具体的手順がわかる
- ○○の**失敗パターン**を事前に回避できる
- ○○を**3倍効率化**するテンプレートが手に入る
- ○○の**次のステージ**への道筋が明確になる
```

#### ポイント
- 動詞で終わる（「わかる」「できる」「手に入る」）
- 数値を入れる（「月10万円」「3倍」「30日」）
- ベネフィットを明確に（「時間短縮」「収益化」「自動化」）

## 有料級にするためのチェックリスト

### 必須項目（全て満たす必要あり）

#### 具体性

- [ ] 「何を」「いつ」「どうやって」が全て明記されている
- [ ] 数値目標が3つ以上含まれている
- [ ] 具体的な手順（ステップ）が5つ以上ある
- [ ] 「〜と思います」ではなく「〜です」「〜してください」で断言している

#### ツール提供

- [ ] そのまま使えるテンプレートが2つ以上ある
- [ ] チェックリストが1つ以上ある
- [ ] 30日プラン or ロードマップがある
- [ ] Q&Aが5つ以上ある

#### 事例

- [ ] 成功事例が3つ以上ある（数字付き）
- [ ] 失敗事例が2つ以上ある（回避策付き）
- [ ] ビフォーアフターが明確に描かれている

#### 差別化

- [ ] 無料記事では得られない独自の視点がある
- [ ] 他のnoteとの違いが明確
- [ ] 筆者の実体験・オリジナルメソッドが含まれている

### 推奨項目（できるだけ満たす）

- [ ] 図解・表が5つ以上ある
- [ ] 引用・データソースが3つ以上ある
- [ ] 読者参加型のワークが各章にある
- [ ] 段階別（初心者・中級者・上級者）の読み方ガイドがある
- [ ] 購入後のサポート情報がある
