# Analyze AI Competitors Skill - ForGenAI Edition

GenAI製品向け競合分析の完全自律実行型Skill。ChatGPT、Claude、Gemini等の主要AI製品を**12軸で定量・定性比較**し、差別化ポイント特定、ポジショニング戦略立案、市場セグメント分析を実行。

---

## 1. Overview

### このSkillでできること

1. **競合製品リストアップ**: 主要5-10製品（ChatGPT、Claude、Gemini等）+ 垂直特化3-5製品（Cursor、Perplexity等）
2. **12軸比較分析**: AI精度、応答速度、価格、API、長文処理、マルチモーダル、ハルシネーション率、エコシステム、データプライバシー、カスタマイズ性、ユーザー体験、開発者ツール
3. **定量比較**: ベンチマークスコア（MMLU、HumanEval等）、API価格、応答速度実測
4. **定性比較**: UX、ブランド、コミュニティ、ドキュメント品質
5. **SWOT分析**: 自社 vs 競合トップ3
6. **ポジショニングマップ作成**: 2軸（汎用性 vs 特化性、価格 vs 性能）
7. **差別化ポイント特定**: 3-5ポイントの具体的な差別化戦略
8. **競合の弱点分析**: 攻撃可能な領域の特定
9. **市場セグメント分析**: どのセグメントで勝てるか
10. **成果物出力**: 競合分析レポート、ポジショニングマップ、差別化戦略

### 既存スキルとの差分

| 要素 | 汎用競合分析 | analyze-ai-competitors |
|------|------------|----------------------|
| **対象製品** | 一般SaaS | **AI製品特化（LLM、画像生成、コード生成等）** |
| **比較軸** | 5-7軸 | **12軸（AI特化指標含む）** |
| **AI精度評価** | なし | **ベンチマーク（MMLU、HumanEval等）** |
| **ハルシネーション** | なし | **誤情報生成率（%）** |
| **長文処理** | なし | **コンテキストウィンドウ（tokens数）** |
| **マルチモーダル** | なし | **テキスト・画像・音声・動画対応** |
| **API価格** | 月額課金のみ | **$/1M tokens + 月額サブスク** |
| **エコシステム** | アプリ統合 | **プラグイン、RAG、ファインチューニング** |
| **成功事例参照** | 一般SaaS | **ChatGPT、Claude、Gemini、Cursor、Perplexity等** |

### STEP 6: SWOT分析（自社 vs 競合トップ3）

**自社製品（仮: AI文書生成ツール）vs ChatGPT/Claude/Gemini**

#### Strengths（強み）

- **特化性**: 文書生成に特化、ChatGPTより高精度
- **価格**: API $3/1M（ChatGPT $6.25の半額）
- **RAG統合**: 独自RAGエンジン、検索精度98%

#### Weaknesses（弱み）

- **ブランド認知**: ChatGPTの1/100
- **エコシステム**: プラグイン10個（ChatGPT 1,000+）
- **マルチモーダル**: テキストのみ（画像・音声未対応）

#### Opportunities（機会）

- **ニッチ市場**: 文書生成専業ツールは少ない
- **企業向け**: ChatGPT Enterpriseは高額、中堅企業向けに展開可能
- **日本語特化**: 日本語文書生成でChatGPTより高精度

#### Threats（脅威）

- **ChatGPT機能拡張**: GPT-4o文書生成機能強化
- **価格競争**: Geminiの低価格攻勢
- **統合戦略**: Microsoft Copilot（Office統合）の台頭

### STEP 8: 差別化ポイント特定（3-5ポイント）

## Domain-Specific Knowledge (from Research)

### Success Patterns（GenAI_research統合）

1. **ChatGPT（圧倒的ブランド、低CAC）**:
   - **ブランド力**: "AI = ChatGPT"の認知、マーケティングコストほぼ不要
   - **汎用性**: 幅広い用途対応、ニッチ特化製品が差別化余地あり
   - **エコシステム**: プラグイン1,000+、サードパーティツール統合最大規模
   - **弱点**: 汎用すぎて特化性欠如、価格やや高い（API $6.25/1M）

2. **Claude（安全性・倫理、長文処理）**:
   - **差別化軸**: 安全性・倫理的AI、Constitutional AI
   - **長文処理**: 200K tokens、論文・法律文書等で優位
   - **ハルシネーション率**: 3%（業界最低水準）
   - **弱点**: マルチモーダル制限（画像のみ）、エコシステム小さい

3. **Gemini（Googleエコシステム、マルチモーダル）**:
   - **統合戦略**: Google Workspace統合、企業向け展開
   - **マルチモーダル**: テキスト+画像+音声+動画、最大規模
   - **長文処理**: 2M tokens、業界最大
   - **弱点**: AI精度やや低い（MMLU 85.9%）、開発者ツールやや劣る

4. **Cursor（コード生成特化、IDE統合）**:
   - **垂直特化戦略**: コード生成に特化、ChatGPTより高精度
   - **IDE統合**: VSCode統合、開発者ワークフローに溶け込む
   - **ターゲット**: 開発者市場、NPS 70（高満足度）
   - **価格**: $20/月（ChatGPT Plusと同額、コード生成特化で正当化）

   - **検索特化**: ChatGPTとの明確な差別化、引用精度98%

   - **価格**: $20/月（ChatGPT Plusと同額、検索特化で正当化）

### Common Pitfalls（競合分析での失敗パターン）

1. **汎用性の罠**: ChatGPTと同じ汎用性で勝負しようとする → 差別化失敗、ブランド力で敗北
2. **価格競争の罠**: Geminiの低価格に対抗して価格競争 → 収益性悪化、持続不可能
3. **全方位戦略の罠**: すべての軸で優位性を狙う → リソース分散、どれも中途半端
4. **ベンチマーク過信**: MMLU等のベンチマークスコアに固執 → 実用性軽視、ユーザー満足度低下
5. **マルチモーダル追随**: Geminiのマルチモーダルに追随 → 開発コスト増、差別化不明確

### Best Practices

1. **差別化軸の明確化**: 汎用性を捨て、特化性で勝負（Cursor、Perplexity事例）
2. **価格戦略の正当化**: 低価格または高ARPU、中途半端な価格は避ける
3. **ニッチ市場狙い**: TAM $100M-1Bの市場で#1を目指す（汎用市場でChatGPTに勝てない）
4. **ベンチマーク + 実用性**: ベンチマークスコアだけでなく、実用性・ユーザー満足度重視
5. **垂直統合戦略**: IDE統合（Cursor）、Google Workspace統合（Gemini）等、ワークフローに溶け込む
6. **セキュリティ訴求**: Claude並みのプライバシー + 価格優位性で金融・医療・法律業界狙い
