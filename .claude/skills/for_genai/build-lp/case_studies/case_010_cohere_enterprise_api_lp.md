## 概要
- **製品名**: Cohere

- **URL**: https://cohere.ai

## 学び

### 成功要因

1. **プライベートデプロイ訴求**:
   - 「Your data stays yours」を冒頭で明記
   - VPC、オンプレミス、エアギャップ環境の選択肢提示

2. **業界特化ソリューション**:
   - 金融、医療、製造業ごとの課題・解決策を明示
   - 規制遵守（GDPR、HIPAA等）を業界別に訴求

3. **エンタープライズ顧客事例**:
   - Oracle、Salesforce、McKinsey等の権威ある社会的証明
   - 具体的な数値（60%削減、40%向上等）とVP/CTOクォート

4. **セキュリティ・コンプライアンス**:
   - SOC 2、ISO 27001、GDPR、HIPAA認証を明示
   - 「Zero retention」「Never used for training」を強調

### 教訓

- **エンタープライズLPでは「プライベートデプロイ」が最優先**: クラウドAPI依存を嫌う企業層への訴求
- **業界特化**: 金融、医療等の規制業界ごとにコンプライアンス要件を明示
- **顧客事例の権威性**: Oracle、Salesforce等「エンタープライズで知られる企業」を社会的証明に活用
- **段階的価格設定**: Cloud API → Private Deployment → Enterpriseの明確な差別化

### 適用可能性

- **エンタープライズAI全般**: プライベートデプロイ、セキュリティ・コンプライアンスをLPのコアに配置
- **規制業界AI**: 業界別のコンプライアンス要件（GDPR、HIPAA等）を明示
- **B2B LLM Platform**: カスタムモデル、SLA保証、専用サポートの訴求
