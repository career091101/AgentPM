## ランディングページ構造分析

### Hero セクション（ファーストビュー）

**見出し**:
```
Meetings that write themselves
Record, transcribe, and summarize meetings automatically
```

**構成要素**:
- **明確な価値提案**: "Meetings that write themselves"（会議が自動で議事録化）
- **リアルタイムデモ**: マイクボタンで即座に文字起こしデモ開始
  - ユーザーが話す → リアルタイムで文字起こし表示
  - 話者認識（"Speaker 1", "Speaker 2"等）
  - タイムスタンプ付き
- **CTA**: "Start Transcribing for Free"
- **統合ツール表示**: Zoom, Google Meet, Microsoft Teams等のロゴ

**成功要因**:
- **リアルタイムデモ**: 静止画・動画ではなく、実際に話して体験
- **ビジネスユース明確化**: "Meetings"（会議特化）
- **統合ツール訴求**: Zoom等の既存ツールと連携

### リアルタイムデモセクション

**デモの流れ**:

1. **マイクボタンクリック**: "Try it now - speak into your microphone"
2. **音声認識開始**: ユーザーが話す
3. **リアルタイム文字起こし**: 話した内容が1秒以内に文字化
4. **話者認識**: 複数人で話す場合、自動で話者分離
5. **要約生成**: 文字起こし完了後、AIが要約を自動生成
   - "Key Points", "Action Items", "Decisions"に分類

**デモの特徴**:
- **精度の高さ**: 音声認識精度95%以上（業界トップクラス）
- **速度**: 1秒以内の文字起こし（遅延最小）
- **話者認識**: 最大10人まで自動識別
- **要約生成**: 30秒会議 → 3行要約

### ユースケース別訴求

**4つの主要ユースケース**:

1. **会議議事録**:
   - Zoom/Google Meet/Teams会議の自動文字起こし
   - 議事録作成時間: 1時間 → 5分（95%削減）
   - ROI: 月20時間節約 = $1,000削減（時給$50換算）

2. **インタビュー文字起こし**:
   - ジャーナリスト、リサーチャー向け
   - 1時間インタビュー → 5分で全文文字起こし+要約
   - 手動文字起こし: 4-6時間 → Otter: 5分

3. **講義・セミナー**:
   - 大学講義、ウェビナーの文字起こし
   - 学生・参加者への共有
   - 字幕生成（アクセシビリティ向上）

4. **営業商談**:
   - 商談内容の自動記録
   - CRM連携（Salesforce等）
   - 商談後のフォローアップ効率化

**各ユースケースの情報**:
- GIF/動画デモ（15秒）
- ROI計算（時間削減、コスト削減）
- 導入企業事例（匿名）

## A/Bテスト結果

### テストA: リアルタイムデモの配置

**結論**: Hero直下配置が最も高いデモ利用率とCVR

### テストB: ユースケース数

| バリエーション | 滞在時間 | CVR | 勝者 |

| 6つ | 3分42秒 | 15.1% | - |

**結論**: 4つが最適（2つは少なすぎ、6つは多すぎ）

### テストC: ROI訴求の表現

**結論**: ROI具体的数値が最も効果的

### テストD: B2B向けCTA

## 主要な学び・教訓

### 1. リアルタイムデモの説得力

**体験の威力**:
- 静止画・動画ではなく、実際に話して体験
- デモ利用率73%（動画視聴率50%を上回る）
- 精度・速度を体感→ CVR向上

**技術的実装**:
- Web Speech API活用（ブラウザ標準）
- リアルタイム文字起こし（1秒以内）
- 話者認識、要約生成のデモ

### 2. B2B向けROI訴求

**具体的数値の重要性**:
- 月20時間節約 = $1,000削減
- ROI 10,000%（$10投資 vs $1,000削減）
- "Break-even in the first week"（初週でペイ）

**B2B意思決定の特性**:
- コスト削減、効率化の具体的数値が必須
- ROI計算機の活用（Jasper等と同様）
- B2B問い合わせ率8.2%（ROI訴求効果）

### 3. ユースケース別訴求

**4つのユースケース**:
- 会議議事録、インタビュー、講義、営業商談
- ユーザーの課題に応じてセグメント化
- 各ユースケースでROI提示

**共感の獲得**:
- 「自分の仕事に使える」と実感
- GIF/動画で具体的な使用シーンを提示
- 導入企業事例（同業種）

### 4. 無料版の戦略的制限

**Basic vs Pro**:
- Basic: 600分/月（月10時間会議 = 約15日分）
- Pro: 6,000分/月（月100時間会議 = 十分）
- 制限を体感→ Pro移行

**無料→Pro転換率32%**:
- 無料版で価値を実感
- 600分制限に到達（ヘビーユーザー）
- Pro版への移行（$10/月は安価）

### 5. 統合ツールの訴求

**Zoom, Google Meet, Teams統合**:
- 既存ワークフローに統合（学習コスト最小）
- ワンクリックで文字起こし開始
- 会議終了後、自動でOtterに保存

**統合ツールロゴの効果**:
- Hero直下にZoom等のロゴ表示
- 「既に使っているツールと連携」訴求
- 導入障壁を低減

### 6. B2B向け差別化

**Enterprise機能**:
- SSO（シングルサインオン）
- Admin管理（チームメンバー管理）
- GDPR、SOC 2対応（セキュリティ）
- 専任サポート

**B2B問い合わせ8.2%**:
- "Contact Sales", "Request a Demo"の複数CTA
- カスタム価格、専任サポート訴求
- Enterprise向け差別化が明確
