## 概要
- **製品名**: GitHub Copilot
- **カテゴリ**: AI Pair Programming
- **URL**: https://github.com/features/copilot
- **関連性**: エンタープライズLP設計により、ROI 3.5倍・導入事例を訴求。Fortune 500企業の78%が導入検討、エンタープライズ契約1,000+達成。

## 背景
GitHub Copilotは2021年に個人開発者向けで公開。2023年にCopilot for Business（$19/ユーザー/月）、Copilot Enterprise（$39/ユーザー/月）を追加し、エンタープライズ市場に拡大。

## 学び

### 成功要因

1. **ROI実証の徹底**:
   - 「$18,750/年/開発者」という具体的な金額訴求
   - ROI計算機ダウンロードで意思決定支援

2. **エンタープライズ固有ニーズの明示**:
   - プライベートコードベース学習（汎用AIとの差別化）
   - セキュリティ・コンプライアンス（IP保護、監査ログ）

3. **顧客事例の権威性**:
   - Shopify、Stripe、Duolingo等の著名企業
   - 具体的な数値（40%向上、70%削減等）とCTO/VPクォート

4. **段階的価格設定**:
   - Individual（$10）→ Business（$19）→ Enterprise（$39）
   - 機能差別化（プライベートコードベース学習、カスタムAI等）

### 教訓

- **エンタープライズLPでは「ROI実証」が最優先**: $18,750/年の具体的金額、ROI計算機提供
- **セキュリティ・コンプライアンス**: IP保護、監査ログ、SOC 2等の明示が契約条件
- **顧客事例の具体性**: 「生産性向上」ではなく「40%向上」「70%削減」の定量データ
- **段階的価格設定**: Individual → Business → Enterpriseの明確な機能差別化

### 適用可能性

- **エンタープライズAI全般**: ROI実証、セキュリティ保証、顧客事例をLPのコアに配置
- **開発者ツールLP**: プライベートコードベース学習、カスタムAI訴求
