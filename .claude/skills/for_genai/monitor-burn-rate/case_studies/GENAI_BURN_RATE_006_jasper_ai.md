## 詳細分析

| 項目 | 月次コスト | 構成比 |

| **Net Burn Rate** | **$5.2M/月** | - |

**AI固有コスト**: $4M + $1M = $5M（全体の36%、健全範囲）

### 資金調達タイミング

- **2022年10月**: Series A $125M（Insight Partners主導、評価額$1.5B）
- **調達理由**: MRR $1M → $5M達成、成長加速
- **ランウェイ**: $125M ÷ $5.2M/月 = 24.0ヶ月

### ランウェイ延長施策

1. **PLG戦略**（2021年1月）:
   - 無料トライアル → Pro課金（月額$49-119）
   - 初年度で10万人突破、月次収益$1M

2. **Enterprise Plan**（2022年5月）:
   - 月額$499-999/ユーザー
   - Fortune 500の10%が利用開始

3. **API費用最適化**:
   - OpenAI GPT-3.5への移行でコスト50%削減
   - キャッシング強化でAPI呼び出し30%削減

4. **コンテンツテンプレート**:
   - 50+テンプレート追加で離脱率50%削減
   - 平均契約期間12ヶ月 → 18ヶ月

### 成長 vs 効率のバランス

**Rule of 40評価**:
```
年間成長率: 100%（MRR $1M → $5M、12ヶ月で5倍）
利益率: ($8.8M - $14M) / $8.8M = -59%
Rule of 40 = 100% + (-59%) = 41% ✅
```

**成長効率**:
- CAC: $100（コンテンツマーケティング重視）
- LTV: $882（平均契約期間18ヶ月、月額$49）
- LTV/CAC: 8.8（優秀）
