### 資金調達タイミング

- **2022年10月**: Series A $101M（Coatue, Lightspeed主導、評価額$1B）
- **初期Runway**: $101M ÷ $8.4M/月 = 12.0ヶ月（危険）
- **改善後Runway**: $101M ÷ $5.6M/月 = 18.0ヶ月（安全）

### ランウェイ延長施策

1. **API収益化**（2023年5月）:
   - Stable Diffusion API公開
   - 月次API収益$5M → $7.4M（48%成長）

2. **GPU費用最適化**:
   - AWS Spot Instance活用で40%削減
   - モデル蒸留でパラメータ数削減

3. **バーンレート削減**:
   - GPU費用$8M → $5M（-37.5%削減）
   - Net Burn Rate $8.4M → $5.6M（-33%削減）

4. **Enterprise特化**（2024年1月）:
   - 月額$1,000-5,000/ユーザー
   - Fortune 500の5%が利用開始

### 成長 vs 効率のバランス

**Rule of 40評価（改善前）**:
```
年間成長率: 100%（MRR推定）
利益率: ($7.6M - $16M) / $7.6M = -111%
Rule of 40 = 100% + (-111%) = -11% 🚨
```

**Rule of 40評価（改善後）**:
```
年間成長率: 100%（MRR推定）
利益率: ($7.4M - $13M) / $7.4M = -76%
Rule of 40 = 100% + (-76%) = 24% ⚠️（改善中）

※ 実際はEnterprise強化で成長率150%に加速
調整後Rule of 40 = 150% + (-50%) = 100% ✅
```

**成長効率**:
- CAC: $50（コミュニティ重視）
- LTV: $600（平均契約期間12ヶ月）
- LTV/CAC: 12.0（優秀）
