# Adept - 資金調達後もバーンレート課題

## Burn Rate指標サマリー

| 指標 | 実績 | ForGenAI基準 | 判定 |
|------|------|-------------|:----:|
| ランウェイ | 15ヶ月 | 18ヶ月以上 | ⚠️ |
| Net Burn Rate | $23.3M/月 | MRRの50%以内 | 🔴 |
| 総調達額 | $350M | N/A | - |
| Rule of 40 | -233% | 40%以上 | 🔴 |
| AI固有コスト比率 | 55% | 30-60% | ✅ |

## 詳細分析

### バーンレート内訳（2024年推定）

| 項目 | 月次コスト | 構成比 |
|-----|-----------|-------|
| 人件費（120人） | $12M | 40% |
| GPU費用（AWS） | $15M | 50% |
| API推論費用 | $2M | 7% |
| マーケティング費 | $0.5M | 2% |
| その他 | $0.5M | 2% |
| **合計支出** | **$30M/月** | **100%** |
| **月次収益（MRR）** | **$6.7M** | - |
| **Net Burn Rate** | **$23.3M/月** | - |

**AI固有コスト**: $15M + $2M = $17M（全体の57%、やや高い）

### 資金調達タイミング

- **2023年3月**: Series B $350M（General Catalyst, Spark Capital主導、評価額$1B）
- **調達理由**: Action Transformer開発、GPU投資
- **ランウェイ**: $350M ÷ $23.3M/月 = 15.0ヶ月（警告）

### バーンレート課題

1. **収益化遅延**:
   - 月次収益$6.7M（低い）
   - API公開から6ヶ月経過も成長鈍化

2. **GPU費用増加**:
   - 月$15M（全体の50%）
   - モデル改善でGPU費用が想定の2倍に増加

3. **Net Burn Rate高すぎ**:
   - $23.3M/月はMRR $6.7Mの348%
   - ForGenAI基準50%を大幅超過

4. **ランウェイ15ヶ月**:
   - ForGenAI基準18ヶ月未達
   - 資金調達または大幅コスト削減が必要

### 成長 vs 効率のバランス

**Rule of 40評価**:
```
年間成長率: 100%（MRR推定）
利益率: ($6.7M - $30M) / $6.7M = -348%
Rule of 40 = 100% + (-348%) = -248% 🔴
```

**成長効率**:
- CAC: $100（Enterprise営業費用含む）
- LTV: $240（平均契約期間12ヶ月）
- LTV/CAC: 2.4（低い、ForGenAI基準5.0未達）

## 教訓

### 失敗要因

1. **収益化遅延**: 月次収益$6.7M（低い）
2. **GPU費用増加**: 月$15M（全体の50%）、想定の2倍
3. **Net Burn Rate高すぎ**: $23.3M/月はMRR $6.7Mの348%
4. **ランウェイ15ヶ月**: ForGenAI基準18ヶ月未達

### ForGenAI製品への示唆

- **収益化最優先**: GPU費用増加前に収益化達成が必須
- **GPU費用管理**: 月$15M（全体の50%）は上限、超過は危険
- **Net Burn Rate**: MRRの50%以内に抑える（Adeptは348%で論外）
- **ランウェイ18ヶ月以上確保**: 15ヶ月は警告、早期資金調達検討

### 注意点

- Adeptは現在も存続中（失敗ではない）
- ただし、バーンレート改善が急務（次ラウンド調達に影響）
- GPU費用$15M/月は高すぎ、$10M以下に削減必要
