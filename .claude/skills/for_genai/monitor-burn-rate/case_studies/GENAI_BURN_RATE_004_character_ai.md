2. **バイラル成長**:

3. **GPU費用最適化**:
   - Google TPU活用でコスト30%削減

4. **Google買収**（2024年8月）:
   - $2.7Bライセンス契約
   - 創業者はGoogle AI部門に参加

### 成長 vs 効率のバランス

**Rule of 40評価**:
```
年間成長率: 200%（MRR $5M → $30M、12ヶ月で6倍）
利益率: ($30M - $50M) / $30M = -67%
Rule of 40 = 200% + (-67%) = 133% ✅
```
