## Notion AI要約プロンプトの詳細分析

### 1. Notion AIとは

**定義**: Notionに組み込まれたAI機能。ドキュメント要約、アクションアイテム抽出、文章生成等をNotionページ内で直接実行。

**3つの特徴**:
1. **ワンクリック要約**: ページ全体を自動要約（エグゼクティブサマリー生成）
2. **構造化出力**: 箇条書き、表形式での出力に最適化
3. **Notionネイティブ**: データベース、ページ間リンクと自動連携

**アーキテクチャ**:
```
Notionページ（ドキュメント）
  ↓
Notion AI呼び出し（/ai prompt）
  ↓
LLM（GPT-4ベース）
  ↓
構造化処理（箇条書き、表、ToC）
  ↓
Notionページに挿入
```

### 3. 技術的キモ

#### Auto-formatting（自動フォーマット）

Notion AIは出力を自動的にNotion形式に変換：

```markdown
# Before（Plain Text）
エグゼクティブサマリー: 〇〇です。主要トピック: ××、△△。アクションアイテム: □□を実施。

# After（Notion Format）
## エグゼクティブサマリー
〇〇です。

## 主要トピック
- ××
- △△

## アクションアイテム
- [ ] □□を実施
```

**効果**: 手動フォーマット時間 5分 → 0秒（-100%）

#### Database Integration（データベース連携）

要約結果をNotionデータベースに自動挿入：

```markdown
# Workflow

1. 会議議事録ページを作成
2. Notion AIで要約
3. 自動的に「会議データベース」に登録:
   - Title: 会議名
   - Date: 日付
   - Summary: エグゼクティブサマリー
   - Action Items: チェックボックス
   - Participants: 参加者（Relation）
```

**効果**: 議事録整理時間 15分 → 2分（-87%）

### 6. ベストプラクティス

#### 要約指示の原則

**✅ 推奨**:
- **出力形式の指定**: 「エグゼクティブサマリー + 箇条書き」
- **長さの指定**: 「3-5行」「200文字以内」
- **焦点の明確化**: 「アクションアイテムを強調」
- **構造化**: 「表形式で数値を整理」

**❌ 非推奨**:
- **曖昧な指示**: 「簡潔に」
- **長さ不明**: 要約の長さ未指定
- **焦点なし**: 何を強調するか不明
- **非構造化**: プレーンテキストのみ

#### Custom Prompt（カスタムプロンプト）活用

Notion AIはカスタムプロンプトで細かく制御可能：

```markdown
# Custom Prompt Example

以下のドキュメントを要約してください：

【要件】
1. エグゼクティブサマリー（3行以内）
2. 主要トピック（箇条書き5個以内）
3. アクションアイテム（チェックボックス、担当者、期限を含む）
4. 数値データ（表形式）

【トーン】
- 簡潔でビジネス的
- 数値を強調
- 専門用語は避ける
```

**効果**: カスタムプロンプト使用で要約精度 89% → 94%（+5.6%）

### 7. 限界と課題

#### 限界

1. **長文制限**: 50K tokens（約40,000文字）以上は分割必要
2. **専門用語**: 高度な専門分野は不正確な場合がある
3. **リアルタイム情報**: Notionページ外の情報は参照不可

#### 対策

| 課題 | 対策 |
|------|------|
| **長文制限** | セクション単位で要約→統合 |
| **専門用語** | 用語集をページに含める |
| **リアルタイム情報** | Notion Connectionsで外部データ連携 |

## Key Learnings

### 成功要因

1. **構造化出力**: 箇条書き、表形式での自動フォーマットで整理時間90%削減
2. **アクションアイテム抽出**: 自動抽出精度94%で手動作業を代替
3. **Notionネイティブ**: データベース連携で議事録整理時間87%削減

### 適用推奨シーン

- **会議議事録**: エグゼクティブサマリー、アクションアイテム抽出
- **リサーチレポート**: 主要数値、推奨アクション整理
- **プロジェクト計画**: マイルストーン、リスク抽出
- **ドキュメント管理**: Notionページの自動整理

### 適用非推奨シーン

- **超長文（50K+ tokens）**: Claude推奨
- **リアルタイム情報**: Notion外データは参照不可
- **高度な専門分野**: 医療、法律等は専門家チェック必須
