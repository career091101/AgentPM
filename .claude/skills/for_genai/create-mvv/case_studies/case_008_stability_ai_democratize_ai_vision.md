# case_008_stability_ai_democratize_ai_vision.md

## 概要
- **製品名**: Stability AI
- **カテゴリ**: Open Source AI Models
- **URL**: https://stability.ai
- **関連性**: MVV策定におけるオープンソース戦略Vision事例

## 背景
Stability AIは2020年創業、2022年8月にStable Diffusion（オープンソース画像生成）をリリース。DALL-E（クローズド）対抗で、完全オープンソース戦略。2024年時点でStable Diffusion 10億回以上ダウンロード。

## MVV（Mission/Vision/Values）

### Mission
**「AI by the people, for the people」**
- 人々によるAI、人々のためのAI
- 大企業独占ではなく、AI民主化を実現

### Vision
**「世界最大のオープンソースAIエコシステム」**
- 2025年までに、全てのクリエイターがオープンソースAIを使える世界
- DALL-E、Midjourneyのクローズドモデルに対抗

### Values
1. **Open Source First**: 全モデルをオープンソース公開
2. **Community-driven**: 開発者コミュニティ主導
3. **Accessibility**: 誰でも無料で高性能AIにアクセス
4. **Transparency**: モデルの訓練データ、手法を完全公開

## MVVの実装

### Mission実装
- **Stable Diffusion公開**: 2022年8月、完全オープンソース
- **ローカル実行可能**: GPU 8GB以上で個人PC実行可能
- **商用利用可能**: 企業も無料で利用可能（クローズドモデルと差別化）

### Vision実装
- **10億回ダウンロード**: Stable Diffusion v1/v2/v3合計
- **開発者コミュニティ**: GitHub 50,000 stars、Discord 50万人
- **派生モデル**: ControlNet、LoRA等、コミュニティが拡張

### Values実装
- **完全公開**: モデルウェイト、訓練コード、データセット全公開
- **無料アクセス**: Hugging Face、Replicateで誰でも実行可能
- **透明性**: LAION-5Bデータセット詳細公開（著作権議論も含む）

## 定量データ
- **ダウンロード数**: 10億回以上 (Stable Diffusion v1-v3)
- **GitHub Stars**: 50,000以上
- **Discord参加者**: 50万人以上
- **派生モデル数**: 10,000以上（ControlNet、LoRA等）
- **評価額**: $1B (2022年10月)
- **調達額**: $101M (Coatue、Lightspeed)
- **ARR**: 非公開（B2B API中心）

## 学び

### 成功要因
1. **Mission明確化**: "AI民主化"でオープンソースコミュニティ結集
2. **Vision実装**: DALL-E対抗で、完全オープンソース戦略
3. **Values徹底**: 透明性、アクセシビリティで開発者支持獲得

### 教訓
- MVVは「AI民主化」のような理念で、コミュニティを動員可能
- Vision「オープンソース」は、クローズドモデル対抗の最強戦略
- Values「Transparency」は、訓練データ公開等で技術的に実装

### 適用可能性
- **オープンソースAI全般**: LLM、音声、動画生成
- **コミュニティ重視製品**: GitHub、Discord活用
- **クローズドモデル対抗**: OpenAI、Anthropic対抗戦略
