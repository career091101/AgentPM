## 詳細分析

### LTV計算
```
ARPU: $17/月（Pro Plan）
平均継続月数: 12ヶ月
Gross Margin: 68%（Whisper API課金 + 独自モデル）

LTV = $17 × 12 × 0.68 = $139
Churn率計算: $17 × 0.68 / 0.083 = $139
```

### AI固有コスト

| コスト項目 | 月次コスト/ユーザー | 割合 |
|----------|------------------|------|
| Whisper API（音声認識） | $4.8 | 28.2% |

| GPU費用 | $0.1 | 0.6% |
| **合計** | **$5.4** | **31.7%** |

**Gross Margin計算**:
```
Gross Margin = ($17 - $5.4) / $17 = 68%
```

**注**: Otter.aiはWhisper API（音声認識）と独自モデル（要約）を併用。Gross Margin 68%（目標70%にやや届かず）。
