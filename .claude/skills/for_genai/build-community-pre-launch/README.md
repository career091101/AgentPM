# Build Community Pre-Launch - ForGenAI Edition

## Overview

GenAI製品向けプレローンチコミュニティ構築スキル。Product Huntローンチの3-6ヶ月前から段階的にコミュニティを構築し、初期ユーザー獲得・フィードバック収集・Product Hunt準備を実現。

## 3-Phase Strategy

### Phase 1: アカウント育成（3-6ヶ月前）
- X/Twitter #BuildInPublic 毎日投稿
- Discord/Slackコミュニティ設計
- LinkedIn週次投稿
- 目標: X 100 followers、Discord 30 members

### Phase 2: エンゲージメント構築（1-3ヶ月前）
- X/Twitterスレッド投稿週1回
- Discordイベント企画（Weekly Office Hours）
- Hacker News Show HN投稿
- インフルエンサー交流開始
- 目標: X 200 followers、Discord 50 members、HN 50+ upvotes

### Phase 3: ベータプログラム（1ヶ月前）
- ベータテスターTier定義（VIP、Active、Passive）
- Product Hunt準備（Discord専用チャンネル）
- コミュニティ動員計画
- 目標: Beta signups 200、Influencer mentions 5

## Key Features

1. **プラットフォーム戦略**: X/Twitter、Discord、LinkedIn、Hacker News、Reddit
2. **#BuildInPublic戦略**: 透明性、失敗共有、コミュニティ巻き込み
3. **コンテンツカレンダー**: 3-6ヶ月分の投稿計画（曜日別テーマ）
4. **インフルエンサー関係構築**: Tech YouTubers、AI研究者へのアプローチ
5. **ベータプログラム設計**: Tier別特典、招待メールテンプレート
6. **メトリクス追跡**: 8つのKPI（X followers、Discord active rate等）

## Success Cases

1. **Cursor**: #BuildInPublic 3ヶ月で500 followers → Discord 150 members → PH #1獲得（12K signups）
2. **Perplexity**: AI研究者コミュニティ構築 → HN 300 upvotes → PH #2獲得（5K signups）
3. **Notion AI**: 既存ユーザー2,000万人基盤活用 → PH #3獲得（1,500+ upvotes）
4. **Midjourney**: Discord-first戦略 → 200万+ members → バイラル成長
5. **ChatGPT Plugins**: 開発者エコシステム動員 → PH #1獲得（3,000+ upvotes）
6. **Jasper AI**: マーケター向けFacebookグループ5,000人 → PH #5獲得（1,100+ upvotes）
