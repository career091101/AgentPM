## 4. API価格変更分析

### 月次コスト試算 (1000万 tokens/月)

#### Claude 3 Sonnetでの月間コスト
```
入力: 6M tokens × $3 / 1M = $18
出力: 4M tokens × $15 / 1M = $60
合計: $78/月
```

#### Claude 3 Haikuでの月間コスト
```
入力: 6M tokens × $0.25 / 1M = $1.50
出力: 4M tokens × $1.25 / 1M = $5
合計: $6.50/月
```

#### 削減効果
```
月間削減: $78 - $6.50 = $71.50 (91.7%削減)
年間削減: $858
```
