# Create Persona Skill (ForGenAI Edition)

AI技術リテラシー別ペルソナを作成する自律実行型Skill。**ForGenAI特化版**では、開発者、ビジネスユーザー、エンドユーザーの3層ターゲット設計を実現し、AI製品特有のペインポイント（プロンプト失敗、API料金不安、Hallucination恐怖等）を統合します。

---

## このSkillでできること

1. **AI技術リテラシー3層ペルソナ**: 開発者/ビジネスユーザー/エンドユーザーの特性分析
2. **AI特化ペインポイント抽出**: プロンプト失敗、API料金不安、Hallucination恐怖等
3. **Product Hunt Early Adopter特性**: Product Huntコミュニティ、Hunter確保戦略
4. **LLM選定基準**: 各ペルソナに最適なLLM（OpenAI/Claude/Gemini）選定
5. **プロンプトパターン最適化**: 各ペルソナ向けChain-of-Thought、Few-shot設計

## Domain-Specific Knowledge (from GenAI_Research)

### Success Patterns

#### 1. ChatGPT - エンドユーザー特化ペルソナ

**ターゲット層**: 非技術者エンドユーザー（プログラミング不要）

**ペルソナ**:
- **名前**: 佐藤花子（32歳、マーケティング担当）
- **技術リテラシー**: 低（プログラミング経験なし）
- **ペインポイント**:
  1. プログラミングできないのでAI利用できない
  2. API料金が高すぎる（無料で使いたい）
  3. Hallucination が怖い（間違った情報を信じてしまう）
- **ゴール**: メール作成、企画書作成を10倍高速化したい
- **LLM選定**: ChatGPT（無料、プログラミング不要）

**成功要因**:
- プログラミング不要（ブラウザで即利用開始）
- 無料プラン提供（月額$0）
- Hallucination削減（RLHF強化学習）

#### 2. GitHub Copilot - 開発者特化ペルソナ

**ターゲット層**: ソフトウェアエンジニア（開発速度10倍化）

**ペルソナ**:
- **名前**: 田中太郎（28歳、ソフトウェアエンジニア）
- **技術リテラシー**: 高（Python、JavaScript熟練）
- **ペインポイント**:
  1. 単純なコーディングに時間がかかる
  2. ジュニアエンジニアを雇う予算がない
  3. コード品質を保ちながら開発速度を上げたい
- **ゴール**: コーディング速度10倍向上、開発コスト削減
- **LLM選定**: GitHub Copilot（IDE統合、コード特化）

**成功要因**:
- IDE統合（VS Code、JetBrains等）
- コード生成速度10倍（単純コーディングで開発時間1/10）
- $10/月で収益化成功（ジュニアエンジニア外注$2,000-5,000の1/200〜1/500）

#### 3. Perplexity AI - ビジネスユーザー特化ペルソナ

**ターゲット層**: ビジネスパーソン（情報信頼性重視）

**ペルソナ**:
- **名前**: 山田健一（45歳、コンサルタント）
- **技術リテラシー**: 中（Excel、PowerPoint熟練、プログラミング経験少）
- **ペインポイント**:
  1. Google検索は広告が多すぎる
  2. ChatGPT はHallucination が怖い（引用元不明）
  3. 情報収集に時間がかかりすぎる
- **ゴール**: 信頼性の高い情報を5倍高速で収集したい
- **LLM選定**: Perplexity AI（Citation機能、検索特化）

**成功要因**:
- Citation機能（引用元明示で信頼性10倍）
- Google検索より5倍高速（複数検索→集約の時間削減）
- Product Hunt #1獲得（ビジネスユーザー高評価）

### Best Practices

#### 1. AI技術リテラシー3層ターゲット設計

**実践方法**:
- **開発者層**: IDE統合、API利用、Fine-tuning経験あり
- **ビジネスユーザー層**: プロンプト利用、ブラウザベース、無料/低額プラン
- **エンドユーザー層**: プログラミング不要、ワンクリック実行、無料必須

**効果**:
- 各層に最適化したUX/料金設計
- TAM（対象市場）最大化（開発者100M、ビジネスユーザー1B、エンドユーザー5B）

#### 2. AI特化ペインポイントの深堀り

**ペインポイント5類型**:
1. **プロンプト失敗**: 望んだ結果が得られない（Few-shot例示不足）
2. **API料金不安**: 高額請求が怖い（従量課金のリスク）
3. **Hallucination恐怖**: 誤情報を信じてしまう（Citation機能不足）
4. **技術的障壁**: プログラミング必須で使えない（ノーコード需要）
5. **速度不満**: 遅すぎて使い物にならない（Streaming API需要）
