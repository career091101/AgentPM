# GENAI_PERSONA_001: ChatGPT Free - 大学生Alex

## ペルソナ概要
- 名前: Alex（アレックス、20歳、男性）
- 年齢: 20歳
- 職業: 大学生（経済学部3年、私立大学）
- AI習熟度: 初心者（プロンプト1行、Few-shot未経験）

## AI利用パターン
- プロンプト頻度: 3-5回/日（レポート提出前は10-15回/日）
- 主要利用シーン: レポート作成（下書き、要約、校正）、試験勉強（知識確認、問題作成）
- 支払い意欲: 無料版固執（$20/月Pro版は高すぎる）
- 幻覚許容度: 中（ファクトチェックはするが完璧求めず、下書き前提）

## 課題・ニーズ

### 課題1（最重要）: レポート作成に時間がかかる
- **詳細**: 週2-3本のレポート（各2000-3000字）、構成に3時間、執筆に5時間
- **頻度**: 週2-3回（レポート提出前）
- **感情**: 焦り、プレッシャー、アルバイトとの両立が困難
- **AI利用で期待**: 構成案生成30分、下書き生成2時間 → **合計5時間削減**

### 課題2: 試験勉強の効率が悪い
- **詳細**: 教科書を読むだけで理解不足、問題演習が少ない
- **頻度**: 試験期間（年4回）
- **感情**: 不安、理解不足の自覚
- **AI利用で期待**: 知識確認、練習問題自動生成

### 課題3: 金銭的余裕がない
- **詳細**: 月収8万円（アルバイト）、生活費6万円、残2万円
- **頻度**: 毎月
- **感情**: コスト重視、無料ツール探し
- **AI利用での懸念**: Pro版$20/月は高すぎる、無料版で十分

## 製品選択理由

### なぜChatGPT無料版か
1. **無料**: 学生にとって最重要、$20/月は支払えない
2. **知名度**: 周囲の学生も全員使っている、情報共有しやすい
3. **汎用性**: レポート、試験勉強、就活ESなど幅広く使える
4. **日本語精度**: DeepL翻訳より自然な日本語
5. **スマホアプリ**: 移動中も利用可能

### 競合との比較
- **Claude Free**: 試したが知名度低い、日本語精度もChatGPTと同等
- **Gemini**: Google検索との統合は便利だが、レポート生成精度低い
- **Perplexity**: 検索特化で便利だが、レポート下書きには不向き
- **Bing Chat（Copilot）**: Edge必須が不便、ChatGPTで十分

## チャーンリスク

### リスク要因1: 幻覚による単位落とし（高リスク）
- **シナリオ**: ChatGPT生成のレポートに誤情報、教授に指摘され減点
- **発生確率**: 15%（年1-2回、幻覚遭遇）
- **対策**: ファクトチェック習慣化、教科書・論文での確認
- **影響**: 信頼低下、使用頻度減少

### リスク要因2: Pro版への移行圧力なし（低リスク）
- **シナリオ**: 無料版で十分、Pro版のメリット実感なし
- **発生確率**: 95%（ほとんどの学生が無料版のみ）
- **対策**: Pro版の価値訴求（レスポンス速度、GPT-4精度）
- **影響**: LTV $0、広告収益のみ

### リスク要因3: 卒業後の利用継続（中リスク）
- **シナリオ**: 就職後、業務でChatGPT使用禁止、個人利用も減少
- **発生確率**: 40%（企業によるAI利用制限）
- **対策**: 就活ES作成、ビジネスメール等の用途拡張
- **影響**: チャーン率40%

## マーケティング戦略

### このペルソナへのアプローチ

#### 1. 大学キャンパスでの普及
- **戦術**: 大学生協、学生団体との提携、ワークショップ開催

- **期待効果**: SEO流入、学生間での共有

- **戦術**: 学生証提示で50%OFF、無料トライアル30日
- **期待効果**: Pro版転換率5% → 10%

#### 4. 教育機関向けライセンス
- **戦術**: 大学全体でのPro版一括契約、学生無料提供
- **期待効果**: B2B2C戦略、大学経由でのユーザー獲得
