# case_006_midjourney_creator_persona.md

## 概要
- **製品名**: Midjourney
- **カテゴリ**: AI Image Generation
- **URL**: https://midjourney.com
- **関連性**: クリエイター向けペルソナ設計により、Discord中心のコミュニティとプロンプト共有文化を構築。月間アクティブユーザー15M+、プロンプト共有率78%を達成。

## 背景
Midjourneyは2022年にAI画像生成市場に参入。DALL-E 2、Stable Diffusionとの差別化として「アート品質」「Discord統合」「コミュニティ駆動」を打ち出し、クリエイティブ業界で急速に普及。

## クリエイター・ペルソナ

### プライマリペルソナ: "Professional Digital Artist"
- **デモグラフィック**:
  - 年齢: 25-45歳
  - 職業: フリーランスイラストレーター、コンセプトアーティスト、UI/UXデザイナー
  - 経験: 3-15年
  - 予算: $30-$60/月（Basic/Standard/Pro）

- **サイコグラフィック**:
  - **ペインポイント**:
    - クライアント要望の初期イメージ作成に時間がかかる
    - コンセプトアートのイテレーション速度が遅い
  - **意思決定基準**:
    1. アート品質（「AI臭さ」がないか）
    2. プロンプトのコントロール性（細部の調整が可能か）
    3. 商用利用可能性（ライセンス明確か）
  - **行動パターン**:
    - Discordで他クリエイターのプロンプトを研究
    - ArtStation、Behanceに作品を公開
    - プロンプトライブラリを個人管理
  - **情報源**: ArtStation、CreativeBloq、YouTube（プロンプトチュートリアル）

- **Jobs-to-be-Done**:
  1. **Functional Job**: クライアント要望の初期コンセプトを30分で10バリエーション作成
  2. **Emotional Job**: 「AIに仕事を奪われる」不安と「AIを創作ツールとして使いこなす」プライドの両立
  3. **Social Job**: 「最新AI技術を使いこなすクリエイター」としてコミュニティで認知される

### セカンダリペルソナ: "Hobbyist Creator"
- **特徴**: 趣味でイラスト、ファンアート作成、SNS共有が主目的
- **ニーズ**: 簡単操作、コミュニティ参加、プロンプト共有
- **購買行動**: 無料トライアル（25枚） → Basic（$10/月、200枚） → 継続率72%

### ターシャリペルソナ: "Enterprise Creative Director"
- **特徴**: 広告代理店、ゲーム会社のクリエイティブディレクター
- **ニーズ**: チームプラン、商用利用、プライベートチャンネル
- **購買行動**: Pro（$60/月） → Mega（$120/月、企業ライセンス）

## 定量データ
- **MAU**: 15M+（2023年）
- **プロンプト共有率**: 78%（Discordパブリックチャンネル経由）
- **平均セッション時間**: 38分（プロンプト試行錯誤とコミュニティ閲覧）
- **商用利用率**: Pro/Mega会員の82%が商用プロジェクトに使用
- **リテンション**: Basic会員の7日間継続率88%、30日間72%
- **コミュニティ成長**: Discord 15M+メンバー（2023年10月時点）

## 学び

### 成功要因
1. **Discord-First戦略**: Web UIではなくDiscordを主要プラットフォームとし、コミュニティ形成を促進
2. **プロンプト共有文化**: パブリックチャンネルで他ユーザーのプロンプトが見える設計（学習促進 + バイラル効果）
3. **アート品質へのこだわり**: 「AI臭さ」を排除し、プロフェッショナルに耐える品質を追求
4. **段階的価格設定**: Basic（$10、趣味）→ Standard（$30、プロ）→ Pro（$60、商用）→ Mega（$120、企業）

### 教訓
- **コミュニティ・ファースト**: ペルソナに「コミュニティ参加動機」「プロンプト共有意欲」を含める
- **Discord統合の戦略**: 既存クリエイターコミュニティが集まるプラットフォーム（Discord、ArtStation）を活用
- **「AI vs 人間」の対立回避**: 「AIは創作ツール」と位置づけ、クリエイターのプライドを尊重
- **商用利用の明確化**: ペルソナに「ライセンス要求」「著作権不安」を含め、Pro/Megaで解決

### 適用可能性
- **クリエイティブAI全般**: ペルソナに「アート品質要求」「プロンプトコントロール性」を含める
- **コミュニティ駆動AI**: Discord、Slack等の既存コミュニティプラットフォームを統合
- **AI検証**: プロンプト共有率、コミュニティエンゲージメントをKPIとして測定
