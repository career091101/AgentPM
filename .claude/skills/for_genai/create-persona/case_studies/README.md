## 利用方法

### ペルソナ作成時の参照
1. 対象GenAI製品タイプを特定（Text生成、Image生成、Code生成等）
2. 類似ケーススタディを3-5件選定
3. AI習熟度、プロンプト頻度、支払い意欲、幻覚許容度を参考
4. ドメイン固有のカスタマイズ（Product Hunt戦略等）

### 品質チェック
- AI習熟度: 3段階明記（初心者/中級/上級）
- プロンプト頻度: 定量化（回数/日）
- 支払い意欲: Tier+金額+根拠
- 幻覚許容度: 低/中/高+用途別
- チャーンリスク: 5要因評価
