# case_007_runway_creator_persona.md

## 概要
- **製品名**: Runway ML
- **カテゴリ**: AI Video Generation & Editing
- **URL**: https://runwayml.com
- **関連性**: プロクリエイター→一般ユーザーへのペルソナ拡張により、ユーザー10倍増加。Gen-2で月間動画生成10M+を達成。

## 背景
Runway MLは2018年に研究者向けAIツールとして創業。2022年にGen-1（テキストto動画）、2023年にGen-2（改良版）をリリースし、プロフェッショナル映像制作からクリエイターエコノミーまで拡大。

## ペルソナ進化: Pro Creator → General Creator

### Phase 1: プライマリペルソナ "Professional Filmmaker"
- **デモグラフィック**:
  - 年齢: 28-50歳
  - 職業: 映像ディレクター、VFXアーティスト、ポストプロダクション
  - 経験: 5-20年
  - 予算: $100-$500/月（Standard/Pro/Unlimited）

- **サイコグラフィック**:
  - **ペインポイント**:
    - VFX制作に数週間〜数ヶ月かかる
    - クライアントプレビューのための粗編集が時間浪費
  - **意思決定基準**:
    1. Hollywood品質（4K、リアリスティック）
    2. 既存ワークフロー統合（Adobe Premiere、DaVinci Resolve）
    3. 商用利用権の明確性
  - **行動パターン**:
    - Premiere Pro/After Effectsを毎日使用
    - Vimeo、Behanceでポートフォリオ公開
  - **情報源**: American Cinematographer、Frame.io、NAB Show

- **Jobs-to-be-Done**:
  1. **Functional Job**: クライアントプレビュー用のVFXを1週間→1日に短縮
  2. **Emotional Job**: 「AIが創造性を奪う」不安と「最新技術を使う先進性」のバランス
  3. **Social Job**: 業界で「Hollywood級のAI VFXを使いこなすクリエイター」として認知される

### Phase 2: セカンダリペルソナ "Content Creator (YouTube/TikTok)"
- **デモグラフィック**:
  - 年齢: 18-35歳
  - 職業: YouTuber、TikTokクリエイター、インフルエンサー
  - フォロワー: 10K-1M+
  - 予算: $12-$35/月（Basic/Standard）

- **サイコグラフィック**:
  - **ペインポイント**:
    - 週3-5本の動画投稿ペースを維持するのが困難
    - ストック映像のコストが高い（$50-$200/本）
  - **意思決定基準**:
    1. 「バズる」ビジュアル（視聴維持率向上）
    2. 制作時間の短縮（1本あたり10時間→5時間）
    3. サブスク価格の手頃さ（$50以下/月）
  - **行動パターン**:
    - CapCut、Premiere Rush等の簡易編集ツール使用
    - TikTok/Instagram Reelsでトレンドリサーチ
  - **情報源**: YouTube Creator Academy、TikTok Creator Portal

- **Jobs-to-be-Done**:
  1. **Functional Job**: ストック映像を使わず、オリジナルAI動画で差別化
  2. **Emotional Job**: 「他のクリエイターより先進的」「バズる映像を作れる」自信
  3. **Social Job**: フォロワーから「映像クオリティが高い」と評価される

### Phase 3: ターシャリペルソナ "Hobbyist Experimenter"
- **特徴**: AI技術への興味、趣味での動画作成、学習目的
- **ニーズ**: 低価格、簡単操作、コミュニティ学習
- **購買行動**: 無料トライアル → Basic（$12/月、125クレジット） → 継続率58%

## 定量データ
- **ユーザー成長**: Phase 1（2022年）100K → Phase 2（2023年）1M+（10倍増加）
- **月間動画生成**: 10M+（2023年10月時点）
- **Pro Creator比率**: 全体の18%（Standard/Pro/Unlimited会員）
- **Content Creator比率**: 全体の52%（Basic/Standard会員）
- **平均セッション時間**: Pro Creator 48分、Content Creator 22分
- **リテンション**: Basic会員の30日間継続率58%、Pro会員82%

## 学び

### 成功要因
1. **ペルソナ拡張戦略**: プロ特化 → コンテンツクリエイター → 一般ユーザーへ段階的拡大
2. **Hollywood実績の活用**: 映画「Everything Everywhere All at Once」等での使用を訴求し、品質証明
3. **価格帯の多様化**: $12（Basic） → $35（Standard） → $76（Pro） → $188（Unlimited）
4. **簡易化UI**: プロ向けの複雑機能と、初心者向けの1クリック生成を並存

### 教訓
- **ペルソナ拡張のタイミング**: プロ市場で実績確立 → 品質証明を武器にマス市場へ
- **Hollywood効果**: 有名作品での採用実績をペルソナ訴求に活用
- **クリエイターエコノミー**: YouTube/TikTokクリエイターは「ストック映像コスト削減」が購買動機
- **段階的UI複雑度**: ペルソナごとに異なるUI（プロ→フル機能、初心者→シンプル）

### 適用可能性
- **クリエイティブAI全般**: プロ実績 → マス市場拡張の2段階ペルソナ戦略
- **AI動画ツール**: ペルソナに「ストック映像コスト」「制作時間」を含める
- **AI検証**: ペルソナ別のリテンション率、セッション時間を測定し、最適化
