## Domain-Specific Knowledge (from Founder_Research)

### Success Patterns

**1. Airレジ（10倍優位性4軸達成）**
- **コスト削減**: 初期費用50-100万円 → 基本無料（**100倍**）
- **時間短縮**: 導入1週間 → 1日以内（**7倍**）
- **営業網活用**: CAC 5万円 → 1-2万円（**5倍**）、グルメサイト営業2,000名活用
- **エコシステム連携**: 単体利用 → Airペイ・Airキャッシュ・Airカード統合（**3倍のLTV向上**）
- **結果**: PMF 9/10、品質92点、90.4万アカウント、市場シェア44%

**2. Airペイ（10倍優位性4軸達成）**
- **初期費用削減**: 数十万円〜百万円 → 0円（**100倍**）
- **対応ブランド数**: 競合5-10種 → 81種（**8-16倍**）
- **営業網活用**: CAC 8万円 → 2-3万円（**3-4倍**）、スタートアップ成功製品既存顧客クロスセル率57%
- **データ資産活用**: Airキャッシュ信用スコアリング、手数料0.5%（業界平均の1/6〜1/20、**6-20倍**）
- **結果**: 1年で20万店舗、初年度売上5億円、3年黒字達成

**3. Airキャッシュ（10倍優位性3軸達成）**
- **入金スピード**: 1-2週間 → 最短翌日（**7倍**）
- **手数料率**: 3-10% → 0.5%〜（**6-20倍**）
- **データ資産活用**: Airペイ決済データで信用スコアリング、審査自動化
- **結果**: 中小企業の資金繰り課題解決、ファクタリング市場で差別化

**4. Geppo（10倍優位性2軸達成）**
- **回答負荷軽減**: 30分 → 1分（3問のみ、**10倍**）
- **回答率向上**: 競合50-70% → 96%（**1.4-2倍**）
- **継続率**: 98%（業界トップクラス）
- **結果**: 2年で300社、離職率改善効果実証、3年黒字達成

**5. スタディサプリ（10倍優位性2軸達成）**
- **コスト削減**: 予備校年間50-100万円 → 月額2,178円（年間26,136円、**20-40倍**）
- **アクセス性**: 地理的制約あり → どこでも学習可能（**10倍のアクセス向上**）
- **既存顧客基盤活用**: 進学ブック配布100万人 → 初年度30万ユーザー転換率30%
- **結果**: 5年で200万人、教育市場でNo.1オンライン学習サービス

### Common Pitfalls

**失敗パターン1: 10倍優位性なし（2-3倍程度）**
- **エリクラ**: 10分単位ワークシェア、地産地消（差別化弱い）→ 競合に100倍差（10万人 vs 1,000万人）→ 6年実証実験後撤退
- **教訓**: 最低1軸で10倍優位性を構築、競合が容易に模倣できない領域を確保

**失敗パターン2: 競合が10倍優れている市場への参入**
- **CODE.SCORE**: エンジニアスキル可視化 vs 競合企業（AI個別最適化学習）→ 教育市場で劣位 → 2年で撤退
- **教訓**: 競合が10倍以上優れている市場には参入しない、後発は圧倒的差別化が必須

**失敗パターン3: スタートアップリソース未活用（0-1種）**
- **CODE.SCORE**: 技術力のみ、既存顧客基盤・営業網未活用 → CAC高騰、スケール困難 → 2年で撤退
- **教訓**: スタートアップリソースを3種以上活用した製品はPMF 8.8点、成功率100%

| **負荷軽減** | ユーザー負荷・作業負荷軽減 | 10倍 | Geppo（回答負荷10倍削減、1分で完了） |

**10倍優位性軸数と成功率**:
- **0軸**: 成功率25%、PMFスコア5.2、撤退可能性高い
- **1軸**: 成功率60%、PMFスコア7.0、Series A Stage最低基準（ForStartup）
- **2軸以上**: 成功率80-100%、PMFスコア7.5-8.8、Series A Stage推奨基準

**スタートアップリソース活用と成功率**:
- **3種以上活用**: PMFスコア8.8、品質88.5、成功率100%（Airレジ、Airペイ、Geppo）
- **1-2種活用**: PMFスコア7.5、品質84.2、成功率80%（SUUMO、じゃらん、スタディサプリ）
- **0種活用**: PMFスコア5.2、品質78.0、成功率25%（CODE.SCORE、termhub）

### Best Practices

1. **コスト削減10倍の実現方法**: 既存インフラ転用（スタートアップ成功製品初期費用0円）、クラウドSaaS化、スケールメリット活用
2. **時間短縮10倍の実現方法**: プロセス自動化（Geppo回答負荷10倍削減）、クラウド即時提供（スタートアップ成功製品導入7倍短縮）
3. **営業網活用でCAC削減**: 営業網2,000名 → スタートアップ成功製品のCAC 1-2万円（競合の1/5〜1/10）
4. **データ資産活用で手数料最安化**: スタートアップ成功製品決済データ → 信用スコアリング → 手数料0.5%（業界平均の1/6〜1/20）
5. **エコシステム連携でLTV向上**: スタートアップ成功製品連携により、LTV 3倍以上向上
6. **ブランド統合でマーケティング効率3倍**: 統合によりブランド認知度向上、広告費40%削減


**2-2. スタートアップリソース棚卸し成果物読込**（`inventory_internal_resources.md`）

**抽出情報**:
```markdown
## リソース6カテゴリ活用

```

**2-3. WebSearchで競合調査**（未存在時のフォールバック）

未存在の場合、WebSearchで以下を調査:
- 直接競合Top 3
- 間接競合（代替ソリューション）
- 既存対処法（手作業、Excelベース等）

#### STEP 3: 既存最良ソリューション選定

競合A、B、Cの中から、各軸で最も優れているものを特定:
- コスト最安: 競合A
- 時間最短: 競合B
- 成果最大: 競合C
等
