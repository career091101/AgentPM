## 統合されているResearch Knowledge詳細

### Success Patterns (成功パターン)

各スキルに以下の成功パターンが統合されています:

#### Airbnb
- **CPF**: 「居場所がない」ペイン発見、写真改善で予約2-3倍
- **フライホイール**: ホスト⇔ゲスト相互成長、ネットワーク効果14/15
- **PMF**: Sean Ellis 45%, 月次成長率25-35%, NPS 60+
- **資金調達**: YC $20K → Sequoia $600K → Series A $7.2M
- **参照**: `@Founder_Research/documents/01_Legendary/FOUNDER_006_brian_chesky.md`

#### Freshworks
- **CPF**: Zendesk価格改定タイミング、Hacker News初期ユーザー
- **10x優位性**: 導入時間10倍速、価格10分の1
- **NRR**: 120%+達成、アップセル・クロスセル戦略
- **参照**: `@Founder_Research/documents/02_Unicorn/FOUNDER_060_girish_mathrubootham.md`

#### Box
- **ボトムアップ採用**: フリーミアム → チーム → エンタープライズ
- **ピボット**: コンシューマー → エンタープライズで年次500%+成長
- **LTV/CAC**: 4.0-6.0、CAC $0（フリーミアム）
- **参照**: `@Founder_Research/documents/02_Unicorn/FOUNDER_061_aaron_levie.md`

### Common Pitfalls (失敗パターン)

各スキルに以下の失敗パターン・教訓が統合されています:

1. **CPF未達での先走り**: 70%未達で開発着手 → リソース浪費
2. **単一軸10x優位性**: 1軸のみでは競合追従される
3. **ユニットエコノミクス未確立での調達**: 次ラウンドで資金枯渇
4. **ランウェイ6ヶ月以下での調達開始**: 交渉力低下、不利条件
5. **成長率未達成の放置**: 月次20%未達でVC調達困難化
