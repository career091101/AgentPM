# Research Competitors Skill (ForStartup Edition)

競合・代替案を自動調査し、10倍検証の前提となる比較分析を実施する自律実行型Skill。**ForStartup特化版**では、外部競合のみならず既存事業との差別化、スタートアップリソース活用による優位性構築を重視します。

---

## このSkillでできること

1. **直接競合調査**: 同じ課題を解決するプロダクト・サービスの収集
2. **間接競合調査**: 代替手段（DIY、人力サービス等）の収集
3. **既存事業差別化分析**: 自社既存製品とのカニバリゼーション回避策
4. **5軸ベンチマーク**: 時間/コスト/使いやすさ/成果/導入障壁で比較
5. **スタートアップリソース優位性評価**: 営業網、顧客基盤、データ資産、ブランド力の活用可能性
6. **ポジショニングマップ**: 市場の構造を可視化、空白地帯特定

## ForStartup固有の評価基準

### 競合分析範囲（拡張版）

このスキルはSeed調達の**Series A Stage～Series B Stage段階**に対応します。

| 指標 | Origin基準 | ForStartup基準 | 理由 |
|------|----------|-------------|------|
| 競合範囲 | 外部競合のみ | **外部競合+既存事業** | 社内カニバリゼーション回避、差別化分析 |
| 優位性分析 | 機能・価格比較 | **+スタートアップリソース活用** | 営業網、顧客基盤、ブランド力の優位性評価 |

| 10倍優位性必須軸 | 1軸以上推奨 | **1軸以上必須** | スタートアップ成功製品の共通パターン |

### スタートアップリソース活用評価

ForStartupでは、既存資産活用による競合優位性構築が重要です。

**評価対象資産**:
1. **営業網**: ホットペッパーグルメ2,000名、リクナビ等
2. **顧客基盤**: 既存顧客90.4万店舗（Airレジ）、SUUMO、じゃらん等
3. **データ資産**: 決済データ（Airペイ）、顧客データ（統合ID基盤）
4. **ブランド**: 確立されたブランド信頼性（CAC削減30-50%推定）
5. **プラットフォーム**: Airエコシステム、統合ID認証基盤

**活用効果（定量）**:
- **CAC削減**: 1/5〜1/10（競合比）
- **初速スケール**: 3倍速（Airレジ vs Square）
- **Churn率低減**: 1/2（既存顧客信頼関係）

## Domain-Specific Knowledge (from Founder_Research)

### Success Patterns（競合分析成功事例）

1. **Airレジ vs Square（POSレジ競合）**:
   - **10倍優位性軸1: 初期コスト**: 50-100万円 → 0円（**100倍優位性**）
   - **10倍優位性軸2: 導入時間**: 1週間 → 1日以内（**7倍優位性**）
   - **スタートアップリソース活用**: ホットペッパーグルメ営業網2,000名で直販、CAC 1-2万円（Square比1/5）
   - **成果**: 90.4万アカウント獲得、市場シェア44%
   - **既存事業との関係**: ホットペッパーグルメと相互補完（POSレジ vs 集客）

2. **Airペイ vs 楽天ペイ（決済競合）**:
   - **10倍優位性軸1: 手数料率**: 楽天ペイ3.24% → Airペイ2.48%〜（**約30%削減**）
   - **10倍優位性軸2: 対応ブランド数**: 楽天ペイ10種 → Airペイ81種（**8倍優位性**）
   - **スタートアップリソース活用**: Airレジ既存顧客90.4万店舗へのクロスセル
   - **データ資産活用**: Airペイ決済データ → Airキャッシュ信用スコアリング
   - **成果**: 51.5万店舗導入、市場シェア35%
   - **既存事業との関係**: Airレジとエコシステム統合

3. **Airキャッシュ vs 一般ファクタリング（資金調達競合）**:
   - **10倍優位性軸1: 手数料率**: 業界平均3-10% → Airキャッシュ0.5%〜（**6-20倍優位性**）
   - **10倍優位性軸2: 入金スピード**: 1-2週間 → 最短翌日（**7倍優位性**）
   - **スタートアップリソース活用**: Airペイ決済データで信用スコアリング、審査自動化
   - **成果**: 手数料業界最安水準、Airエコシステム統合でLTV向上
   - **既存事業との関係**: Airペイデータ資産活用

4. **Geppo vs Wevox（エンゲージメントサーベイ競合）**:
   - **10倍優位性軸1: 回答負荷軽減**: Wevox 10-15分 → Geppo 1分（3問のみ）（**10倍優位性**）
   - **10倍優位性軸2: 継続率**: Wevox推定60-80% → Geppo 98%（**1.2-1.6倍優位性**）
   - **スタートアップリソース活用**: サイバーエージェント社内2013年〜、リクルート1,200名先行運用
   - **成果**: 継続率98%、25名〜数万名規模導入
   - **既存事業との関係**: リクルートHR事業との補完関係

5. **SUUMO vs 既存不動産情報サイト（athome等）**:
   - **優位性軸1: 掲載物件数**: 業界最大級
   - **優位性軸2: ブランド認知**: 確立されたブランド信頼性
   - **スタートアップリソース活用**: 不動産仲介会社ネットワーク、営業網
   - **成果**: 国内最大級の不動産情報サイト
   - **既存事業との関係**: 既存不動産事業との統合

### Common Pitfalls（失敗パターン）

1. **エリクラ vs タイミー（スポットワーク競合）**:
   - **失敗要因**: 差別化要素「10分単位」「地産地消」は顧客ニーズが弱い
   - **競合比較**: タイミー1,000万人 vs エリクラ10万人（**100倍差**）
   - **10倍優位性欠如**: すべての軸で2-3倍程度、競合が容易に模倣
   - **教訓**: プラットフォーム事業は初速が命、100倍差を覆せない
   - **ForStartup教訓**: 競合が10倍優れている市場には参入しない

2. **CODE.SCORE vs atama+/Monoxer（教育Tech競合）**:
   - **失敗要因**: エンジニアスキル可視化は教育市場で劣位、技術力のみでは不足
   - **競合比較**: atama+ AI個別最適化学習、CODE.SCORE エンジニアスキル可視化
   - **10倍優位性欠如**: 技術的には優秀だが、顧客ニーズに合致せず
   - **教訓**: 競合優位性は「自社が提供したいもの」でなく「顧客が本当に求めるもの」
   - **ForStartup教訓**: 既存顧客基盤との関連性を最優先で確認

3. **termhub vs クラウドサイン（契約書管理競合）**:
   - **失敗要因**: 契約書管理市場、クラウドサインに劣後
   - **競合比較**: クラウドサイン先行優位、termhub後発
   - **スタートアップリソース活用不足**: 既存事業とのシナジーなし
   - **教訓**: 既存事業との関連性がない市場への参入リスク
   - **ForStartup教訓**: スタートアップリソース活用3種以上でPMF 8.8、成功率100%

4. **スタディサプリ個別指導 vs 既存スタディサプリ（自社製品カニバリゼーション）**:
   - **失敗要因**: 月額10,780円では講師人件費を賄えない、ベーシックコース2,178円が優秀すぎる
   - **既存製品との競合**: 5倍の価格差に見合う価値提供できず
   - **教訓**: 自社製品カニバリゼーション回避、価格差に見合う価値必須
   - **ForStartup教訓**: 既存製品が優秀すぎると高額版が売れない、差別化10倍規模で設計

### Quantitative Benchmarks

- **10倍優位性軸数**: 成功製品平均3.2軸、失敗製品1-2軸、ForStartup推奨: **1軸以上必須**
- **競合優位性倍数**: 成功製品10-100倍、失敗製品2-3倍、ForStartup推奨: **10倍以上1軸**
- **スタートアップリソース活用数**: 成功製品3種以上でPMF 8.8、品質88.5、成功率100%
- **CAC削減効果**: 営業網活用で1/5〜1/10、ブランド活用で30-50%削減

### Best Practices

1. **外部競合+既存事業の両方との差別化分析**
   - 外部競合: 機能・価格・サービス品質での10倍優位性構築
   - 既存事業: カニバリゼーション回避、相互補完関係構築（例: Airレジ vs ホットペッパーグルメ）

2. **スタートアップリソース活用3種以上で圧倒的優位性構築**
   - 営業網 + データ資産 + ブランド（Airペイ、Airキャッシュ）
   - 顧客基盤 + プラットフォーム + ブランド（SUUMO、じゃらん）

3. **10倍優位性1軸以上必須、複数軸で成功率向上**
   - Airレジ: 初期コスト100倍 + 導入時間7倍
   - Airペイ: 手数料30%削減 + 対応ブランド8倍
   - Geppo: 回答負荷10倍削減 + 継続率1.2-1.6倍

4. **エコシステム連携によるスイッチングコスト構築**
   - Airシリーズ: レジ→ペイ→キャッシュ→カード の連携
   - 統合ID基盤: じゃらん、ホットペッパー、SUUMO横断ポイント統合

**STEP 3: 直接競合収集（WebSearch）**
- ツール: WebSearch
- 対象: 同じ課題を解決するプロダクト/サービス

- 目標: 5-10件

**STEP 4: 間接競合収集（WebSearch）**
- ツール: WebSearch
- 対象: DIY、人力サービス、代替手段
- 収集項目: 方法、コスト、時間、欠点
- 目標: 3-5件

**STEP 5: 既存事業との差別化分析（追加）**
- ツール: 内部データベース、WebSearch
- 対象: 既存製品（Airシリーズ、SUUMO、じゃらん、ホットペッパー等）
- 分析項目: カニバリゼーションリスク、相互補完可能性、エコシステム統合機会
- 目標: 3-5件

**STEP 6: 市場データ収集（WebSearch）**
- ツール: WebSearch
- 対象: 市場規模、成長率、トレンド
- 収集項目: TAM/SAM/SOM、CAGR、主要プレイヤーシェア

**STEP 7: 5軸ベンチマーク表作成**
- 時間効率
- コスト
- 使いやすさ
- 成果・効果
- 導入障壁

**STEP 8: スタートアップリソース優位性評価（追加）**
- 営業網活用可能性（ホットペッパー、リクナビ等）
- 顧客基盤活用可能性（Airレジ、SUUMO、じゃらん等）
- データ資産活用可能性（決済データ、顧客データ等）
- ブランド力転用可能性（確立されたブランド）

**STEP 9: ポジショニングマップ作成**

- 空白地帯の特定

**STEP 10: 差別化ポイント整理**
