# ForStartup向けスキル選定: 優先度ランキング

**作成日**: 2026-01-02
**対象**: VC調達型スタートアップ向けスキル最適化
**総スキル数**: 26
**選定数**: 18

---

## 1. 優先度評価基準

### High（最優先: VC調達に直結）
- VC投資家の審査項目に直接対応
- 資金調達ロードマップの主要マイルストーン
- Series A以降のスケーリング検証

### Medium（重要: スケーラビリティ検証）
- 早期スケール段階での成長指標測定
- チーム規模・組織課題への対応
- 事業プロセス最適化

### Low（基礎: 全ドメイン共通）
- 検証の基礎フレームワーク
- 事業計画補助的役割
- 他優先度スキルのインプット

## 5. 選定スキルの実装順序（推奨フロー）

### **月1-2: Phase 1 検証（8-10週間）**

```
Week 1-2:
  → /create-mvv            (MVV定義)
  → /create-persona        (ペルソナ生成)

Week 2-3:
  → /discover-demand       (市場ニーズ)
  → /research-problem      (課題検証)

Week 3-4:
  → /simulate-interview    (顧客ニーズシミュレーション)
  → /research-competitors  (競合分析)

Week 4-5:
  → /validate-cpf          (CPF≥70% 判定)
  → /build-flywheel        (成長ループ設計)
  → /validate-10x          (10倍優位性)

Week 5-6:
  → /validate-psf          (PSF達成判定)

⬇️ [ステージゲート: CPF & PSF]
```

### **月3-6: MVP構築・PMF検証（16-20週間）**

```
Week 7-16:
  → MVP構築 (Build LP は Webflow 等外注推奨)
  → ユーザー獲得・初期テスト

Week 16-20:
  → /validate-pmf          (PMF≥80% 判定)
     - Sean Ellis ≥40%
     - 月次成長 ≥10%/月
     - Churn ≤5%/月
     - NPS ≥50

⬇️ [ステージゲート: PMF ≥80%]
```

### **月7-12: スケーリング・Series A準備（24-26週間）**

```
Week 21-24:
  → /measure-aarrr         (AARRR分析)
  → /validate-unit-economics (LTV/CAC, CAC回収期間)
  → /monitor-burn-rate     (ランウェイ・資金調達タイミング)

Week 24+:
  → ピッチデッキ作成（/create-mvv ベース）
  → Series A 融資申請

⬇️ [Series A 投資決定]
```

### **全フェーズ横断**

```
常時:
  → /startup-mentor        (全フェーズガイダンス)
  → /pivot-decision        (必要に応じて戦略転換検討)
```

## 6. 除外スキルの理由

| スキル | 除外理由 | 代替戦略 |
|--------|---------|--------|
| **Build LP** | MVP 開発コスト最小化が VC 向けスタートアップの優先 | Webflow, Bubble等ローコード ツール推奨。開発チーム コア機能に集中 |
| **Create SNS Content** | Build in Public は PR 補助的。VC はプロダクト & 成長指標を重視 | 必要に応じて PR エージェンシー活用。SNS は 後段成長戦略（Phase 3+ AARRR） に含める |
| **Collect X Timeline** | SNS データ収集は主要 Traction 指標ではない | トレンド 追跡は オプション。マーケティングデータは AARRR に統合 |
| **Extract Top Tweets** | SNS 投稿最適化は B2C SaaS 向け。B2B では優先度低 | B2B: PR は既存顧客テスト証言に集中。B2C: AARRR Activation で対応 |
| **Scrape Tweet Details** | 同上 | 同上 |
| **Extract Content** | コンテンツ抽出は SNS 投稿自動化のコンポーネント。スケール段階に内在 | マーケティングチーム形成後に実装 |
| **Analyze Replies** | SNS 反応分析は デモンストレーション補助的 | カスタマーサクセスデータで代替 |
| **Research Topic** | 投稿トピック調査は SNS 運用効率化。スケール後 | 同上 |
| **Generate SNS Posts** | 投稿文自動生成は同上 | 同上 |
| **Approve and Schedule** | 投稿承認・スケジューリングは SNS 自動化。スケール後 | 同上 |

## 9. Summary: ForStartup向けスキル戦略

### 3つの検証段階

| 段階 | 目標 | スキル数 | 期間 | Go/No-Go |
|------|------|--------|------|----------|
| **Phase 1** | CPF & PSF達成 | 9個 | 8-10週 | CPF≥70% & PSF達成 |
| **Phase 2** | PMF達成 | 2個 | 12-16週 | PMF≥80% |
| **Phase 3** | Series A準備 | 3個 | 4-6週 | UE合格 & 資金計画確定 |
| **横断** | メンタリング | 1個 | 継続 | 重要決定時 |

### VC投資家が最重視する成功指標

1. **PMF達成**: Sean Ellisテスト ≥40% ← **Validate PMF**
2. **成長ファネル**: 月次成長 ≥10%/月 ← **Measure AARRR**
3. **ユニットエコノミクス**: LTV/CAC ≥5.0 ← **Validate Unit Economics**
4. **資金計画**: ランウェイ ≥24ヶ月 ← **Monitor Burn Rate**
