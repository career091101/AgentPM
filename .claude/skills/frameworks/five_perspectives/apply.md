# Five Perspectives Analysis Skill

5つの眼で市場を多角的に分析。

---

## Skill情報

- **コマンド**: `/apply-five-perspectives`
- **目的**: 市場・トレンド・課題の多角的分析
- **所要時間**: 45-60分

---

## プロンプト

### セッション開始

```markdown
# 5つの眼 市場分析セッション

5つの視点で市場を多角的に分析します。

## 5つの眼とは？

1. **鳥の眼（マクロ）**: 市場全体を俯瞰
2. **虫の眼（ミクロ）**: 顧客の細部を観察
3. **魚の眼（トレンド）**: 時流・変化を捉える
4. **医者の眼（診断）**: 課題の根本原因を特定
5. **人の眼（ヒューマン）**: 人間的共感・倫理

---

## あなたのビジネス領域

参考のため、ビジネス概要を教えてください:
- 何を作っていますか？
- 誰のどんな課題を解決しますか？
```

---

### 各眼の分析

#### 1. 鳥の眼（マクロ視点）

```markdown
## 1. 鳥の眼（マクロ視点）

市場全体を俯瞰し、メガトレンドを捉えます。

### 質問

#### Q1. 市場規模（TAM）
- この領域の市場規模は？
  - 世界: $[X]B
  - 日本: $[Y]B
- 成長率（CAGR）: [X]%

#### Q2. 主要プレイヤー
- 大手: [企業名3-5社]
- 新興: [スタートアップ3-5社]

#### Q3. 市場構造
- 集中度: [寡占 / 分散]
- 参入障壁: [高 / 中 / 低]
- 規制: [あり / なし]

---

### 分析結果

**市場機会**: [評価]
- 市場規模十分
- 成長市場
- 参入余地あり

**リスク**: [特定]
```

---

#### 2. 虫の眼（ミクロ視点）

```markdown
## 2. 虫の眼（ミクロ視点）

顧客の細部を観察し、ペインポイントを発見します。

### 質問

#### Q1. ペルソナの一日
- 朝: [何をしている？]
- 昼: [何をしている？]
- 夜: [何をしている？]

#### Q2. 課題に直面する瞬間
- いつ、どこで、課題に遭遇？
- その時の感情は？
- 現在の解決策は？

#### Q3. 意思決定プロセス
- 購買の意思決定者は？
- 検討期間: [X]日/週/月
- 判断基準: [価格 / 機能 / ブランド]

---

### 分析結果

**インサイト**: [発見]
- ペインポイント明確
- 購買プロセス理解

**改善点**: [提案]
```

---

#### 3. 魚の眼（トレンド視点）

```markdown
## 3. 魚の眼（トレンド視点）

時流・変化を捉え、波に乗ります。

### 質問

#### Q1. テクノロジートレンド
- 関連技術: [AI / ブロックチェーン / IoT / etc]
- 成熟度: [黎明期 / 成長期 / 成熟期]

#### Q2. 社会トレンド
- 関連する社会変化: [リモートワーク / 高齢化 / etc]
- コロナ後の変化: [加速 / 減速]

#### Q3. 規制・政策
- 関連法規制: [あり / なし]
- 補助金・支援: [あり / なし]

---

### 分析結果

**追い風**: [特定]
- トレンドに乗っている
- タイミング良い

**向かい風**: [特定]
```

---

#### 4. 医者の眼（診断視点）

```markdown
## 4. 医者の眼（診断視点）

課題の根本原因を診断します。

### 質問

#### Q1. 表面的な課題
- 顧客が言う課題: [X]

#### Q2. 根本原因（Why × 5）
- なぜその課題が起きる？
  - → なぜ？
    - → なぜ？
      - → なぜ？
        - → なぜ？
          - **根本原因**: [特定]

#### Q3. 既存ソリューションが解決できない理由
- なぜ既存品で解決できない？
  - [理由1]
  - [理由2]

---

### 分析結果

**根本原因**: [特定]

**あなたのソリューションが効く理由**: [説明]
```

---

#### 5. 人の眼（ヒューマン視点）

```markdown
## 5. 人の眼（ヒューマン視点）

人間的共感、倫理観で判断します。

### 質問

#### Q1. 共感
- この課題に共感できますか？
- 自分も困った経験ありますか？

#### Q2. 倫理
- 社会的に良いことですか？
- ダークパターン（悪用）の可能性は？

#### Q3. ミッション
- なぜこれをやりたいのですか？
- 10年後、何を実現したいですか？

---

### 分析結果

**情熱**: [評価]

**倫理的問題**: [なし / あり: 詳細]

**ミッション**: [明確化]
```
