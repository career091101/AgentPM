# Lean Canvas Application Skill

リーンキャンバス作成支援。対話形式で9要素を一緒に埋める。

---

## Skill情報

- **コマンド**: `/apply-lean-canvas`
- **目的**: リーンキャンバス9要素の完成
- **所要時間**: 60-90分

---

## プロンプト

### セッション開始

```markdown
# リーンキャンバス作成セッション

リーンキャンバスの9要素を対話形式で一緒に埋めていきます。

## リーンキャンバスとは？

1ページでビジネスモデルを可視化するフレームワーク。

**9要素**:
1. 課題（Problem）
2. 顧客セグメント（Customer Segments）
3. 独自の価値提案（Unique Value Proposition）
4. ソリューション（Solution）
5. チャネル（Channels）
6. 収益の流れ（Revenue Streams）
7. コスト構造（Cost Structure）
8. 主要指標（Key Metrics）
9. 圧倒的な優位性（Unfair Advantage）

---

所要時間: 60-90分
休憩しながら進めても大丈夫です。

準備はいいですか？

[はい] で開始します。
```

### 各要素の対話

#### 要素1: 課題

```markdown
## 1. 課題（Problem）

顧客が抱える上位3つの課題を教えてください。

**質問**:
- 顧客の一番の悩みは何ですか？
- なぜそれが問題なのですか？
- 現在どうやって解決していますか？（代替品）

**回答例**:
```
1. 予約管理が非効率（電話・ノート）
2. ノーショー（無断キャンセル）が多い
3. 予約データが蓄積されない
```

---

**あなたの回答**:
1. [課題1]
2. [課題2]
3. [課題3]

---

[回答を受けて]
素晴らしい。3つの課題、明確ですね。

特に[課題1]は、3Uスコアが高そうです。

では次へ進みます。
```

#### 要素2: 顧客セグメント

```markdown
## 2. 顧客セグメント（Customer Segments）

誰の課題を解決しますか？

**質問**:
- ターゲット顧客の属性（年齢、職業、業界、企業規模等）
- アーリーアダプターの特徴

**回答例**:
```
ターゲット: 従業員10人以下の飲食店オーナー、40-60代
アーリーアダプター: ITリテラシー高めの30-40代オーナー
```

---

**あなたの回答**:
- ターゲット:
- アーリーアダプター:
```
