# Monitor Burn Rate - ForSolo Edition

## 目的

ソロプレナーの資金状況を常時監視し、**バーンレート（月次消費金額）**と**ランウェイ（資金が尽きるまでの期間）**を自動計算。**18ヶ月ルール**（自己資金運営推奨基準）に基づき、資金ショートを未然に防ぎ、コスト最適化と収益化タイミングを提案します。

**重要な原則**（ソロプレナー特化）:
> 「ソロプレナーは、ランウェイが18ヶ月を切ったら、収益化を加速するか、コスト削減を徹底すべきである」
> 「ランウェイが12ヶ月以下になったら、事業継続が危機的状況にある」
> 「月次ツールコストは$100以下、外注費は必要最小限、人件費は自分の生活費のみ」
> — Marc Lou, Pieter Levels, Tony Dinh (Solopreneur Research)

**ForSolo版の追加機能**:
1. **月次ツールコスト制限チェック**: $100/月超過時の代替案提示
2. **1人実行可能性評価**: 外注依存度、固定費比率の分析
3. **Micro-SaaS収益化ロードマップ**: $1K → $5K → $10K MRRの段階的成長計画
4. **Solopreneur成功事例との比較**: 85件の事例から抽出したベンチマーク適用

### Common Pitfalls (失敗パターン)

1. **高額ツールへの過剰投資** → 月$500のツールで収益ゼロ、資金枯渇
2. **外注依存** → 1人で完結できない構造、コスト肥大化
3. **オフィス固定費** → 自宅/カフェで十分なのに賃貸オフィス契約
4. **早すぎるサブスク契約** → 年間契約で初期キャッシュ流出

### Best Practices

1. **毎月バーンレート・ランウェイを計測**: 個人事業主でも財務管理は必須
2. **月次ツールコスト$100以下を厳守**: 無料枠、Open Source代替を優先
3. **外注は必要最小限**: デザイン、マーケティングは自己学習で対応
4. **収益化を最優先**: ランウェイ18ヶ月を切ったら新機能より収益化

### STEP 2: Gross Burn Rate計算（2分）

**計算方法**: 月次総支出の算出

**必要データ**:
- 生活費（自分の最低生活費）
- ツール費（SaaS、開発ツール）
- マーケティング費（広告、SNS運用）
- 外注費（デザイン、ライティング等）
- その他経費（サーバー費、API費用）

**計算式**:
```
Gross Burn Rate = 月次総支出 / 月
```

**計算例**:
```
生活費: ¥200,000/月（最低限の固定費）
ツール費: ¥10,000/月（Hosting, Notion, Figma無料枠）
マーケティング費: ¥0/月（Twitter Build in Public）
外注費: ¥50,000/月（LP制作のみ）
その他: ¥20,000/月（API費用、ドメイン）

Gross Burn Rate = ¥280,000/月
```

**ソロプレナー別目安**（ForSolo基準）:

| ステージ | 目安バーンレート | 状況 | 推奨アクション |
|---------|----------------|------|--------------|
| **Pre-Launch（MVP開発）** | ¥100,000-300,000/月 | 無収益、自己資金 | ツールコスト最小化、外注ゼロ |
| **Post-Launch（初期トラクション）** | ¥200,000-500,000/月 | 初期収益あり | 収益化優先、マーケティング投資 |
| **Growth（MRR $1K-$5K）** | ¥300,000-800,000/月 | 収益増加中 | 再投資バランス、ツール有料化検討 |
| **Scale（MRR $5K-$10K）** | ¥500,000-1,500,000/月 | 安定収益 | 外注拡大、生活費向上OK |

---

### STEP 3: Net Burn Rate計算（2分）

**計算方法**: 収益を差し引いた実質消費額

**必要データ**:
- 月次収益（買い切り、サブスク、広告）
- 月次総支出（STEP 2から）

**計算式**:
```
Net Burn Rate = (月次総支出 - 月次収益) / 月
```

**計算例**:
```
月次収益: ¥150,000（サブスク$1,000）
月次総支出: ¥280,000

Net Burn Rate = ¥130,000/月
```

**注意**:
- Net Burn Rateがマイナス（収益 > 支出）の場合 → キャッシュフロー黒字 ✅
- Net Burn Rateがプラスの場合 → 赤字運営、資金枯渇リスクあり ⚠️

---

### STEP 4: ランウェイ計算（1分）

**計算方法**: 現在の消費ペースで資金が何ヶ月持つか

**計算式**:
```
Runway（月） = 現在残高 / Net Burn Rate
```

**計算例**:
```
現在残高: ¥5,000,000
Net Burn Rate: ¥130,000/月

Runway = 5,000,000 / 130,000 = 38.5ヶ月 ✅
```

**判定基準**（ForSolo版: 18ヶ月ルール適用）:

| ランウェイ | 判定 | 状態 | 推奨アクション |
|-----------|------|------|--------------|
| **18ヶ月以上** | ✅ 安全 | 余裕あり | 収益化加速、月次監視継続 |
| **12-18ヶ月** | ⚠️ 注意 | 収益化検討開始 | Pricing見直し、マーケティング強化 |
| **6-12ヶ月** | 🚨 警告 | 収益化必須 | コスト削減、収益化最優先 |
| **6ヶ月以下** | 🔴 緊急 | 倒産リスク | 緊急コスト削減、即座に収益化 |

**ソロプレナー特有の対応**:
- ランウェイ < 12ヶ月 → 副業化、収益化加速、ピボット検討
- ランウェイ < 6ヶ月 → プロダクト凍結、コンサル収入確保
