## プロジェクト概要

**Founder Agent ForSolo Edition** は、1人で事業を運営するソロプレナー、副業起業家、デジタルノマド向けの特化版エージェントです。コスト最小化、利益率重視、個人実行可能性を最優先し、**$10K+ MRR**（月間経常収益）を目指すインディーハッカーに最適化されています。

### 誰のためのものか

- **ソロプレナー**: 1人で事業を構築・運営する起業家
- **副業起業家**: 本業を持ちながら週20-40時間で事業を立ち上げたい人
- **デジタルノマド**: 場所に依存せず、リモートで収益化を目指す人
- **インディーハッカー**: VC資金調達なしで、小規模・高収益ビジネスを構築したい人

### 何が得られるか

- **20スキル**: 1人実行可能性に最適化された自律実行型スキルセット
- **29+ Tier 2ケーススタディ**: Marc Lou、Tony Dinh、Pieter Levels等の実証済みノウハウ
- **502件のSolopreneur Research**: App（187件）、Newsletter（58件）、SNS（141件）の成功パターン分析
- **実践的な成功パターン**: Build in Public、Micro-SaaS収益化、月額$100以下のツールスタック
- **段階的成長戦略**: $1K → $5K → $10K MRRの明確なロードマップ

## 主な特徴

### 1. 1人実行可能性重視の評価基準

**実行可能性スコア6点以上**を必須化（ForStartupの4点から強化）。必須スキルチェック（コーディング、マーケティング、CS）、時間確保計画（週20-40時間）、コスト制約（月額$100以下）を多角的に検証します。

### 2. 502件のSolopreneur Researchに基づく実証済みノウハウ

- **App**: 187件（Marc Lou、Tony Dinh、Pieter Levels等）
- **Newsletter**: 58件（Newsletterビジネス事例）
- **SNS**: 141件（SNS成長戦略事例）

全件ファクトチェック済み、YAML Front Matter v4.0形式、日本市場適用性スコア付き。

### 3. Build in Public × Micro-SaaS収益化パターン

- **Build in Public**: X/Twitterでの透明性マーケティング、30日間コンテンツカレンダー自動生成、フォロワー獲得ロードマップ（0→1K→10K）
- **Micro-SaaS**: Boilerplate、Template、Tool、Niche SaaSの4パターン、$9/$29/$99の3段階サブスクリプション設計
- **段階的成長**: $1K MRR（初月-3ヶ月） → $5K MRR（4-9ヶ月） → $10K MRR（10-18ヶ月）

### 4. 月額$100以下のツールスタック推奨

無料枠最大活用戦略（Vercel、Supabase、Railway）、代替ツール推薦（有料サービスのOSS代替）、コスト段階的拡大計画（$0→$1K→$10K収益時の切り替え）。

### 5. LTV/CAC 10.0以上の有機的成長戦略

広告費ゼロ前提、SEO・Build in Public・Product Huntでの有機的成長、CAC（顧客獲得コスト）$0-10、LTV/CAC比率10.0以上（ForStartup 5.0の2倍）。

## 評価基準の違い（ForStartup vs ForSolo）

| 基準 | ForStartup | ForSolo | 変更理由 |
|------|-----------|---------|---------|
| **実行可能性スコア** | 4点以上 | **6点以上** | 1人実行可能性が最重要、必須スキル充足度70%以上 |
| **市場機会スコア** | 6点以上 | **4点以上** | ニッチ市場OK、小規模でも高収益性を重視 |
| **CPF閾値** | 70%以上 | **60%以上** | 仮説検証重視、小規模検証で許容 |
| **LTV/CAC比率** | 3.0以上 | **10.0以上** | 有機的成長重視、広告費ゼロ前提 |
| **月額ツールコスト** | 無制限 | **$100以下** | コスト制約、無料枠活用85% |
| **初期投資額** | 無制限 | **$1,000以下** | リスク最小化、Boilerplate活用 |

## スキル一覧（20スキル）

### 新規スキル（5個）- ForSolo専用

1. **validate-solo-fit**: 1人実行可能性検証
   - 6軸評価（技術、スキル、時間、コスト、マーケ、サポート）
   - 必須スキルチェック（コーディング、マーケティング、CS）
   - 時間配分モデル（開発50%、マーケ30%、CS20%）
   - 総合判定（Solo Fit達成/要改善/見直し）

2. **create-bip-strategy**: Build in Public戦略
   - 30日間コンテンツカレンダー自動生成
   - フォロワー獲得ロードマップ（0→1K→10K）
   - エンゲージメント施策（リプライ戦略、スレッド作成）
   - KPI設定（フォロワー数、エンゲージメント率、プロダクト訪問率）

3. **design-micro-saas-model**: Micro-SaaS設計
   - 収益化パターン判定（Boilerplate、Template、Tool、Niche SaaS）
   - 段階的成長ロードマップ（$1K → $5K → $10K MRR）
   - サブスクリプション設計（$9/$29/$99の3段階プラン）
   - 収益シミュレーション（3ヶ月予測、6ヶ月予測）

4. **optimize-tool-stack**: ツールスタック最適化
   - 月額コスト$100以下の最適ツール選定
   - 無料枠活用戦略（Vercel、Supabase、Railway）
   - 代替ツール推薦（有料サービスのOSS代替）
   - コスト段階的拡大計画

5. **create-content-flywheel**: コンテンツフライホイール設計
   - コンテンツ再利用戦略（1記事→10投稿→1動画）
   - Newsletter/Blog/SNS/Video統合カレンダー
   - AIツール活用効率化（ChatGPT、Canva、CapCut）
   - 週次コンテンツ計画

### 調整済みスキル（15個）- ForSolo最適化

6. **discover-demand**: 需要発見（ニッチ市場OK、市場機会4点以上に緩和）
7. **validate-cpf**: 課題検証（CPF閾値70% → 60%に緩和、小規模検証で許容）
8. **validate-psf**: ソリューション検証（1人実行可能性を最重視）
9. **validate-pmf**: プロダクト市場適合検証（LTV/CAC 10.0以上を基準に）
10. **research-competitors**: 競合調査（ニッチ市場での差別化重視）
11. **measure-aarrr**: AARRRファネル分析（有機的成長チャネル重視）
12. **validate-10x**: 10倍優位性検証（コスト優位性、時間優位性を重視）
13. **create-persona**: ペルソナ作成（ニッチターゲットの深掘り）
14. **research-problem**: 課題リサーチ（1人で解決可能な課題に絞る）
15. **simulate-interview**: 顧客インタビュー（小規模検証、5-10人で判断）
16. **build-flywheel**: 成長ループ構築（有機的成長、バイラル係数重視）
17. **create-mvv**: ミッション・ビジョン・バリュー（個人の価値観重視）
18. **startup-scorecard**: 事業全体評価（1人実行可能性、利益率を最重視）
19. **monitor-burn-rate**: バーンレート監視（月次コスト$100以下を厳守）
20. **validate-unit-economics**: ユニットエコノミクス検証（LTV/CAC 10.0以上）

### スキップしたスキル（3個）- ソロプレナーには不要

- **prepare-vc-meeting**: VC面談準備（資金調達不要）
- **create-fundraising-plan**: 調達計画作成（自己資金前提）
- **build-pitch-deck**: ピッチデッキ作成（VC不要）

## 成功パターン（3つの代表事例）

### 1. Marc Lou（ShipFast）: MRR $135K、27個失敗

**パターン**: Boilerplateビジネスモデル

- **収益**: MRR $135,000（月間）
- **開発期間**: 1週間MVP
- **マーケティング**: Build in Public（X/Twitter 20万フォロワー）
- **失敗**: 27個のプロダクト失敗を経て成功
- **教訓**: 最もシンプルなBoilerplateモデルで成功、失敗談が最高エンゲージメント

### 2. Tony Dinh（TypingMind）: MRR $142.5K、7個失敗

**パターン**: Niche Tool（ChatGPT UI）

- **収益**: MRR $142,500（月間）
- **開発期間**: 数日でMVP
- **マーケティング**: Product Hunt #1複数回獲得
- **失敗**: 7個のプロダクト失敗
- **教訓**: ChatGPT API公開直後の速攻リリースでファーストムーバー獲得

### 3. Pieter Levels（PhotoAI）: MRR $237K、70+個失敗

**パターン**: AI画像生成SaaS

- **収益**: MRR $237,000（月間）
- **開発期間**: 複数プロダクト並行運営
- **マーケティング**: SEO特化、"12 Startups in 12 Months"でバイラル
- **失敗**: 70+個のプロダクト失敗（95%失敗率でも成功）
- **教訓**: 極限までコスト最小化（VPS $5/月等）、複数プロダクトポートフォリオ戦略
