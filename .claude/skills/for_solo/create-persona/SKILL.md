# Create Persona Skill (ForSolo Edition)

需要発見レポートからターゲットペルソナを自動生成する自律実行型Skill。ソロプレナー特有のペルソナパターン（技術者、副業起業家、デジタルノマド）を統合。

---

## このSkillでできること

1. **8要素ペルソナ生成**: 起業の科学準拠の詳細なペルソナ作成
2. **日常行動シナリオ**: 1日のスケジュールと課題発生タイミングの特定
3. **3U検証との紐付け**: Unworkable/Unavoidable/Urgentの事前評価
4. **WTP（支払い意思）推定**: 価格感度の仮説構築
5. **ForSolo特化**: ソロプレナーペルソナ特性、Build in Public戦略、SNS行動パターン

## Instructions

### セッション開始

ターゲットの説明を入力してください（省略時は `demand_discovery.md` を参照）:

**実行モード**: 自律実行（対話なし）
**推定所要時間**: 10-20分

### 自動実行フロー

**STEP 1: 入力情報の取得**
- `demand_discovery.md` が存在する場合: 最有望需要候補を読み込み
- 存在しない場合: ユーザー入力からターゲット情報を取得

**STEP 2: ペルソナ基本情報の生成（ForSolo特化）**
- 名前（仮名）
- 年齢（**ForSolo典型: 25-40歳**）
- 性別
- 職業（**具体的に: ソフトウェアエンジニア、デザイナー、マーケター等**）
- 年収（**ForSolo典型: $50K-150K**）
- 家族構成（**ForSolo典型: 独身または共働き**）
- 居住地（**ForSolo典型: 都市部またはリモート**）
- **働き方**: フルタイム/副業/フリーランス

**STEP 3: 日常行動（A Day in the Life）の生成（ForSolo特化）**
- 1日のスケジュール（起床〜就寝）
- **リモートワークパターン**: 在宅/カフェ/コワーキング
- **副業時間帯**: 早朝（6-8時）または夜間（20-24時）
- 課題が発生するタイミングの特定
- プロダクト利用シーンの想定
- **SNS利用時間**: X（Twitter）チェック、IndieHackers閲覧

**STEP 4: 課題・ペインの詳細化（トップ3、ForSolo視点）**
- 課題の具体的記述
- 発生頻度
- 感情（イライラ、ストレス等）
- 3U検証との紐付け
- **ForSolo特有課題**: 時間不足、技術サポート不在、予算制約

**STEP 5: ゴール・願望の定義（ForSolo特化）**
- 短期ゴール（3ヶ月）: **MRR $1K達成、Product Hunt上位ランクイン**
- 長期ゴール（1-3年）: **本業辞めて独立、MRR $10K達成**
- 理想の状態: **場所・時間の自由、収益の自動化**

**STEP 6: 現在の解決策と不満の整理（ForSolo視点）**
- 使用中のツール・サービス（**ForSolo典型: Notion, Figma, VS Code, ChatGPT**）
- 現在のワークフロー
- 既存解決策への不満と強度（1-10）
- **手動作業の多さ**: コピペ、繰り返し作業

**STEP 7: 情報源・価値観の定義（ForSolo特化）**
- 情報収集チャネル
  - **X（Twitter）**: Indie Hackerフォロー、Build in Public観察
  - **IndieHackers**: 月次収益レポート、成功事例
  - **Product Hunt**: 毎日チェック、新製品発見
  - **YouTube**: Pieter Levels、Marc Lou、Tony Dinh等
- 意思決定プロセス
  - **無料トライアル重視**: 即試せるものを好む
  - **コミュニティ評判**: X/IndieHackersの評価を信頼
- 価値観・動機
  - **自由**: 場所・時間の制約からの解放
  - **収益性**: 最小コストで最大利益
  - **社会貢献**: 同じ課題を持つ人を助ける

**STEP 8: 支払い意思（WTP）の推定（ForSolo特化）**
- 想定価格帯: **月額$10-50**（ForSolo典型）
- 支払い根拠: **時間節約 > 月額コスト**
- **価格感度**: 高い（$100/月超は厳しい）

**STEP 9: 成果物出力**
- ツール: Write
- パス: `Stock/programs/創業支援・新規事業開発（AIエージェント）/projects/Founder_Agent_Phase1/documents/2_discovery/persona.md`

## ドメイン知識

### ソロプレナーペルソナ特性

**3大タイプ**:

#### 1. 技術者型（Engineer Solopreneur）
- **職業**: ソフトウェアエンジニア、デザイナー
- **年齢**: 25-35歳
- **特徴**: 技術力高い、マーケティング苦手、1人で完結したい
- **成功例**: Marc Lou（ShipFast）、Tony Dinh（BlackMagic.so）

#### 2. 副業起業家型（Side Hustler）
- **職業**: 本業あり（エンジニア、マーケター等）
- **年齢**: 30-40歳
- **特徴**: 時間制約大、リスク回避、本業辞めたい願望
- **成功例**: Pieter Levels（Nomad List、副業→独立）

#### 3. デジタルノマド型（Digital Nomad）
- **職業**: フリーランス、リモートワーカー
- **年齢**: 25-40歳
- **特徴**: 場所の自由重視、収益の自動化追求、ミニマリスト
- **成功例**: Levels.io（Nomad List）、Remote OK

**共通特性**:
- X（Twitter）活発（週5-7投稿）
- IndieHackersコミュニティ参加
- Build in Public実践
- 月$100以下ツール制約
- 時間節約 > 機能豊富

### Build in Public戦略

**行動パターン**:
1. **毎日X（Twitter）投稿**: 進捗、学び、失敗を共有
2. **週次収益報告**: IndieHackersで月次MRR公開
3. **コンテンツ作成**: ブログ、YouTube、ポッドキャスト
4. **コミュニティ参加**: IndieHackers、Product Hunt、Discord

**情報収集チャネル**:
- **X（Twitter）**: リアルタイム情報、Indie Hackerフォロー
- **IndieHackers**: 成功事例、フォーラム、収益透明性
- **Product Hunt**: 新製品発見、トレンド把握
- **YouTube**: Pieter Levels、Marc Lou等のチャンネル
