# Estimate Market Size Skill

TAM/SAM/SOM 3層市場規模を計算し、投資判断の根拠を提供する自律実行型Skill。

---

## このSkillでできること

1. **TAM計算**: 理論上の最大市場規模を算出（トップダウン・ボトムアップ両方）
2. **SAM計算**: 実際にサービス可能な市場規模を特定（地理的・言語制約適用）
3. **SOM計算**: 現実的に獲得可能な市場規模を推定（1-5%シェア仮説）
4. **市場成長率推定**: 業界トレンド調査と5年後予測
5. **投資判断基準**: TAM/SAM/SOMが投資判断基準を満たすか評価

## 市場規模計算基準（起業の科学準拠）

### TAM（Total Addressable Market）- 理論上の最大市場

| アプローチ | 計算式 | データソース |
|-----------|--------|-------------|
| **トップダウン** | 業界全体規模 × セグメント割合 | 総務省統計、業界レポート、調査会社データ |
| **ボトムアップ** | 潜在顧客数 × 顧客単価 × 購買頻度 | ペルソナデータ、競合価格、市場調査 |

### SAM（Serviceable Available Market）- サービス可能な市場

| 制約要因 | 適用方法 |
|---------|---------|
| **地理的制約** | TAM × 対象地域の割合（例: 日本のみ → TAM × 2.5%） |
| **言語制約** | TAM × 対応言語圏の割合 |
| **ターゲット制約** | TAM × ターゲットセグメントの割合 |

### SOM（Serviceable Obtainable Market）- 獲得可能な市場

| 推定方法 | シェア仮説 | 根拠 |
|---------|-----------|------|
| **保守的** | SAM × 1% | 新規参入時の現実的シェア |
| **現実的** | SAM × 3-5% | 3-5年後の目標シェア |
| **楽観的** | SAM × 10% | 市場リーダー達成時 |

## 投資判断基準

| 指標 | 最低基準 | 理想基準 | 判定 |
|------|---------|---------|:----:|
| **TAM** | ¥100億以上 | ¥1,000億以上 | ✅/❌ |
| **SAM** | ¥50億以上 | ¥500億以上 | ✅/❌ |
| **SOM（5年後）** | ¥10億以上 | ¥50億以上 | ✅/❌ |
| **市場成長率** | 5%/年以上 | 20%/年以上 | ✅/❌ |

**総合判定**:
- 4指標すべて✅ → 十分な市場規模（CPF検証へ進む）
- 2-3指標✅ → 要検討（市場規模再評価）
- 1指標以下✅ → 市場が小さすぎる（別アイデア検討推奨）

## Instructions

### STEP 1: ビジネスアイデア・ターゲット市場の入力

**ユーザーからの情報収集**:
- ビジネスアイデアの概要
- ターゲット市場（業界、地域、顧客層）
- ペルソナ情報（作成済みの場合は `persona.md` を読み込み）

**自動実行**: 質問応答なしで、既存ドキュメントから情報を抽出

---

### STEP 2: TAM計算（トップダウンアプローチ）

**データソース**:
1. **総務省統計局**: 人口統計、産業別市場規模
2. **業界レポート**: Gartner, IDC, 矢野経済研究所
3. **WebSearch**: 「[業界名] 市場規模 2025」

**計算手順**:
```
TAM（トップダウン） = 業界全体規模 × ターゲットセグメント割合

例: SaaS市場規模 ¥5兆円 × 中小企業向け30% = ¥1.5兆円
```

**出力**:
- TAM（トップダウン）: ¥XX兆円
- データソース: [引用元URL]
- 推定精度: 高/中/低

---

### STEP 3: TAM計算（ボトムアップアプローチ）

**計算手順**:
```
TAM（ボトムアップ） = 潜在顧客数 × 顧客単価 × 購買頻度

例:
- 日本の中小企業数: 360万社
- 対象セグメント（従業員50-300人）: 30万社
- 想定単価: ¥50,000/月
- TAM = 30万社 × ¥50,000 × 12ヶ月 = ¥1,800億円
```

**出力**:
- TAM（ボトムアップ）: ¥XX億円
- 潜在顧客数: XX社/人
- 顧客単価: ¥XX,XXX
- 購買頻度: XX回/年

---

### STEP 4: TAM整合性確認

**2つのアプローチの比較**:
```
TAM（トップダウン） vs TAM（ボトムアップ）

差異が2倍以内 → ✅ 整合性あり
差異が2-5倍 → ⚠️ 要再検証
差異が5倍超 → ❌ データ見直し必須
```

**最終TAM採用**:
- 整合性ありの場合: 2つの平均値
- 整合性なしの場合: より保守的な値

---

### STEP 5: SAM計算（制約適用）

**地理的制約**:
```
日本のみ → TAM × 2.5%（世界GDP比）
日本 + アジア → TAM × 30%
グローバル → TAM × 100%
```

**言語制約**:
```
日本語のみ → TAM × 2.5%
英語対応 → TAM × 80%
多言語対応 → TAM × 100%
```

**ターゲット制約**:
```
特定業界のみ → TAM × 業界割合
企業規模制限 → TAM × 該当規模割合
```

**計算例**:
```
TAM: ¥1.5兆円
地理的制約（日本のみ）: ¥1.5兆円 × 2.5% = ¥375億円
ターゲット制約（IT業界）: ¥375億円 × 20% = ¥75億円

SAM = ¥75億円
```

---

### STEP 6: SOM計算（現実的シェア仮説）

**シェア仮説の設定**:

| 時期 | シェア仮説 | 計算 |
|------|-----------|------|
| **初年度** | 0.5% | SAM × 0.5% |
| **3年後** | 3% | SAM × 3% |
| **5年後** | 5% | SAM × 5% |

**計算例**:
```
SAM: ¥75億円

SOM（初年度）: ¥75億円 × 0.5% = ¥3,750万円
SOM（3年後）: ¥75億円 × 3% = ¥2.25億円
SOM（5年後）: ¥75億円 × 5% = ¥3.75億円
```

**現実性チェック**:
- 競合シェア分析: 既存トッププレイヤーのシェア確認
- 成長曲線: 現実的な成長率かどうか検証

---

### STEP 7: 市場成長率推定（WebSearch）

**検索クエリ**:
- 「[業界名] 市場成長率 2025-2030」
- 「[業界名] CAGR（年平均成長率）」

**データソース**:
- 調査会社レポート（Gartner, IDC, Forrester）
- 総務省統計
- 業界団体データ

**成長率分類**:
- 5%未満: 成熟市場
- 5-15%: 安定成長市場
- 15-30%: 高成長市場
- 30%超: 急成長市場

**5年後予測**:
```
TAM（5年後） = TAM（現在） × (1 + 成長率)^5

例: TAM ¥1.5兆円、成長率20%/年
TAM（5年後） = ¥1.5兆円 × (1.20)^5 = ¥3.73兆円
```

---

### STEP 8: 競合分析（市場の集中度）

**市場集中度の確認**:
- トッププレイヤー3社のシェア合計
- HHI（Herfindahl-Hirschman Index）算出（可能な場合）

**市場タイプ判定**:

| 集中度 | 判定 | SOM獲得難易度 |
|-------|------|-------------|
| トップ3社で80%超 | 寡占市場 | 高（差別化必須） |
| トップ3社で50-80% | 競争市場 | 中（ニッチ戦略） |
| トップ3社で50%未満 | 分散市場 | 低（参入余地あり） |

---

### STEP 9: 投資判断基準評価

**4指標の判定**:

| 指標 | 実績 | 最低基準 | 理想基準 | 判定 |
|------|------|---------|---------|:----:|
| TAM | ¥XX億円 | ¥100億 | ¥1,000億 | ✅/❌ |
| SAM | ¥XX億円 | ¥50億 | ¥500億 | ✅/❌ |
| SOM（5年後） | ¥XX億円 | ¥10億 | ¥50億 | ✅/❌ |
| 市場成長率 | XX%/年 | 5%/年 | 20%/年 | ✅/❌ |

**総合判定ロジック**:
```
if すべて✅:
    判定 = "✅ 十分な市場規模"
    次のアクション = "/validate-cpf でCPF検証へ"
elif 2-3個✅:
    判定 = "⚠️ 要検討"
    次のアクション = "市場セグメントの再定義を推奨"
else:
    判定 = "❌ 市場が小さすぎる"
    次のアクション = "別のアイデア探索を推奨"
```
