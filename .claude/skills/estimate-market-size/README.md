# Market Size Estimation - 市場規模推定スキル

TAM（Total Addressable Market）、SAM（Serviceable Available Market）、SOM（Serviceable Obtainable Market）の3層市場規模を計算し、投資判断の根拠となるレポートを自動生成します。

## スキル概要

### 目的

ビジネスアイデアの市場規模を体系的に検証し、投資判断基準（TAM/SAM/SOM）を満たしているか評価します。**起業の科学準拠**の計算手法で、トップダウン・ボトムアップ両方のアプローチから市場規模を推定し、データの信頼性を確保します。

### 対象ステージ

- **ステージ**: Phase1（アイデア検証）
- **タイミング**: アイデア初期段階、投資家ピッチ準備時
- **前提条件**: ビジネスアイデア、ターゲット市場が定義済み
- **所要時間**: 30-50分（WebSearch含む）

### このスキルでできること

1. **TAM計算（2アプローチ）** - トップダウン・ボトムアップの両面から理論上の最大市場規模を算出
2. **SAM計算** - 地理的・言語・ターゲット制約を適用し、実際にサービス可能な市場を特定
3. **SOM計算** - 現実的な市場シェア仮説（1-5%）に基づいて獲得可能な市場を推定
4. **市場成長率分析** - 業界トレンドを調査し、5年後の市場規模を予測
5. **投資判断基準評価** - TAM/SAM/SOM/成長率の4指標で総合判定

## 投資判断基準

### 最低基準 vs 理想基準

| 指標 | 最低基準 | 理想基準 | 判定 |
|------|---------|---------|:----:|
| **TAM** | ¥100億以上 | ¥1,000億以上 | ✅/❌ |
| **SAM** | ¥50億以上 | ¥500億以上 | ✅/❌ |
| **SOM（5年後）** | ¥10億以上 | ¥50億以上 | ✅/❌ |
| **市場成長率** | 5%/年以上 | 20%/年以上 | ✅/❌ |

### 総合判定ロジック

| 判定 | 達成条件 | 意味 | 次のアクション |
|------|---------|------|--------------|
| ✅ 十分な市場規模 | 4指標すべて✅ | 投資判断の基準を満たす | `/validate-cpf` へ進む |
| ⚠️ 要検討 | 2-3指標✅ | 改善の余地がある | 市場セグメント再定義 |
| ❌ 市場が小さすぎる | 1指標以下✅ | 事業化に懸念あり | 別アイデア探索推奨 |

## 主要な計算式

```
TAM（トップダウン） = 業界全体規模 × セグメント割合
TAM（ボトムアップ） = 潜在顧客数 × 顧客単価 × 購買頻度

SAM = TAM × 地理制約 × 言語制約 × ターゲット制約

SOM = SAM × シェア仮説（初年度0.5%, 3年後3%, 5年後5%）

TAM（5年後） = TAM（現在） × (1 + 成長率)^5
```

## よくある質問（FAQ）

### Q1: TAMが2つのアプローチで大きく異なる場合はどうする？

**A**: 差異が2倍以内なら整合性ありとして、2つの平均値を採用します。2-5倍の差異は要再検証（データ確認）、5倍超はデータ見直し必須です。保守的なアプローチとして、より小さい値を採用することも可能です。

### Q2: 市場成長率のデータが見つからない場合は？

**A**: Gartner、IDC、矢野経済研究所などの調査会社レポートを参照します。業界団体のデータも有効です。データが取得できない場合は、類似業界の成長率を参考値として使用し、その旨をレポートに記載してください。

### Q3: グローバル展開を考えている場合、SAM計算はどうする？

**A**: 段階的に評価してください：

- **初期（Year 1-2）**: 日本のみ（SAM × 2.5%）
- **成長期（Year 3-5）**: 日本 + アジア（SAM × 30%）
- **スケール期（Year 5+）**: グローバル（SAM × 100%）

各段階で別々のSOM目標を設定します。

### Q4: SOMのシェア仮説（5%）は根拠がありますか？

**A**: はい。起業の科学では新規参入企業の現実的シェアを以下のように定義しています：

- **保守的**: 1%（市場参入障壁が高い場合）
- **現実的**: 3-5%（競争市場で3-5年かけて達成）
- **楽観的**: 10%（市場リーダー達成時）

本スキルはデフォルトで3-5%を採用しますが、市場特性に応じて調整可能です。

## 注意事項

1. **データの信頼性**: 公的機関・調査会社のデータを優先。業界団体データも有効。
2. **保守的な推定**: 楽観的な仮定を避け、現実的なシェア仮説を採用。
3. **整合性チェック**: トップダウン・ボトムアップの差異が大きい場合は要再検証。
4. **市場定義の明確化**: TAMが大きすぎる場合はターゲット市場の再定義を検討。
