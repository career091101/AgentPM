### バイラル係数（K-factor）計算

```
K-factor = 紹介率 × 紹介先登録率

例：
- 現在のユーザー: 1,000人
- そのうち紹介した人: 200人（20%）
- 紹介先のうち実際に登録: 60人（30%）

K-factor = 0.20 × 0.30 = 0.06
平均紹介人数 = 60 / 200 = 0.3人
```

**現在のバイラル係数**: X.XX

**K-factorの意味**:
- \>1.0：指数関数的成長（自律的拡大）
- 0.5-1.0：緩やかな成長（外部マーケティング必要）
- \<0.5：限定的な成長（ほぼマーケティング依存）
