# Build LP Skill

MVP検証用のランディングページを構築する自律実行型Skill。

---

## このSkillでできること

1. **LP構成設計**: ヒーローセクション/機能/価格/CTA等の構成を設計
2. **UVP最適化**: 3秒で価値が伝わるキャッチコピー生成（3パターン）
3. **HTML/CSS/JS生成**: レスポンシブ対応の完全なLPコード生成
4. **フォーム統合**: メール登録/事前予約フォームを実装
5. **5項目チェック**: コンテンツ/UVP/CTA/レスポンシブ/フォームを検証

## Instructions

**実行モード**: 自律実行（対話なし）
**推定所要時間**: 40-80分

### 【v2新機能】MVP 10類型選択ロジック

**実装日**: 2025-12-29（Tier 1 Batch 2）
**目的**: 検証目的に応じて最適なMVP類型を自動選択（Gap 3解消）

---

#### STEP 1: 検証目的の特定

**PSF検証で必要な証拠**（`psf_diagnosis.md`から判定）:

```markdown
検証目的チェックリスト:
- [ ] **UVP刺さり度**（顧客反応の検証）
- [ ] **支払い意思**（Price Sensitivity測定）
- [ ] **技術的実現可能性**（実装難易度検証）
- [ ] **オペレーション負荷**（運用コスト検証）
```

**優先度判定**:
1. Enterprise B2B → UVP刺さり度 **最優先**（信頼構築が重要）
2. Consumer B2C → 支払い意思 **最優先**（価格感度高い）
3. 技術先行型 → 技術実現可能性 **最優先**（実装リスク高）
4. サービス型 → オペレーション負荷 **最優先**（手動対応コスト）

---

#### STEP 2: MVP 10類型マッピング

**検証目的別推奨MVP**（起業の科学準拠）:

| 検証目的 | 推奨MVP類型 | コスト | 期間 | 証拠 |
|----------|------------|--------|------|------|
| **UVP刺さり度検証** | **Landing Page** ✅ | $0-$500 | 1週間 | メール登録数、滞在時間、CTR |
| **支払い意思検証** | **Concierge MVP** | $1,000-$3,000 | 2-3週間 | 支払い完了率、継続率、NPS |
| **技術実現可能性** | **Prototype** | $3,000-$5,000 | 3-4週間 | 機能動作、性能指標、技術デモ |
| **オペレーション負荷** | **Wizard of Oz** | $500-$2,000 | 1-2週間 | 手動対応工数、自動化率、CS時間 |
| **ユーザー体験全体** | **Single Feature MVP** | $2,000-$4,000 | 2-3週間 | タスク完了率、満足度、離脱率 |
| **既存市場参入** | **Piecemeal MVP** | $1,000-$2,500 | 1-2週間 | 既存ツール組合せ、統合度、コスト削減 |
| **バイラル性検証** | **Explainer Video** | $500-$1,500 | 1週間 | 動画視聴率、共有数、SNS拡散 |
| **二面市場検証** | **Marketplace Mockup** | $1,500-$3,500 | 2-3週間 | 供給側/需要側マッチング率 |
| **PMF確信度高** | **Pre-order Page** | $500-$1,000 | 1週間 | 事前購入率、平均単価、キャンセル率 |
| **ハードウェア/物理製品** | **3D Rendering / CAD** | $2,000-$5,000 | 2-4週間 | デザイン評価、製造可能性、コスト試算 |

#### STEP 3: 自動選択ロジック

**決定木アルゴリズム**:

```
検証目的の優先度判定
    ↓
├─ UVP刺さり度 最優先？
│   ├─ Yes → **Landing Page** 選択 ✅
│   │   理由: コスト最小（$0-$500）、期間最短（1週間）
│   │   証拠: メール登録数、滞在時間、ヒートマップ
│   └─ No → 次へ
│
├─ 支払い意思 最優先？
│   ├─ Yes → **Concierge MVP** 選択
│   │   理由: 高単価B2Bで支払い意思を直接検証
│   │   証拠: 支払い完了率、継続契約率、LTV試算
│   └─ No → 次へ
│
├─ 技術実現可能性 最優先？
│   ├─ Yes → **Prototype** 選択
│   │   理由: 技術リスク高い場合、実装検証が必須
│   │   証拠: 機能動作、パフォーマンス、スケーラビリティ
│   └─ No → 次へ
│
└─ オペレーション負荷 最優先？
    ├─ Yes → **Wizard of Oz** 選択
    │   理由: 自動化前に手動対応で需要検証
    │   証拠: 手動工数、顧客満足度、自動化余地
    └─ No → **Landing Page**（デフォルト）
```

#### STEP 4: 選択結果と実装計画

**出力フォーマット**:

```markdown
## MVP類型選択結果【v2新規】

### 検証目的
**優先順位**:
1. ✅ UVP刺さり度（最優先）
2. ⚠️ 支払い意思（次優先）
3. ❌ 技術実現可能性（優先度低）
4. ❌ オペレーション負荷（優先度低）

### 推奨MVP類型
**選択**: **Landing Page** ✅

**選定理由**:
- 検証目的: UVP刺さり度（最優先）
- Enterprise B2Bでは信頼構築が重要
- コスト最小（$500）、期間最短（1週間）
- 早期フィードバック獲得に最適

**代替案**:
- 案2: Concierge MVP（支払い意思検証なら推奨）
- 案3: Explainer Video（バイラル性検証なら推奨）

### MVP実装計画（Landing Page）

**Week 1: 設計・デザイン**
- Day 1-2: ワイヤーフレーム作成（7セクション構成）
- Day 3-4: UVP最適化（3パターン生成・選定）
- Day 5: デザインカンプ作成

**Week 2: 実装・公開**
- Day 1-2: HTML/CSS/JS実装
- Day 3: フォーム統合（Google Forms / Typeform）
- Day 4: レスポンシブ対応・5項目チェック
- Day 5: ドメイン設定・公開

**Week 3: 測定開始**
- 目標CVR: 5%以上（メール登録）
- トラフィック: 200-500 visitors（広告配信）
- 測定期間: 2週間

### 証拠収集計画

**測定指標**（Landing Page型）:
| 指標 | 目標値 | 測定方法 |
|------|--------|----------|
| **CVR**（メール登録率） | 5%以上 | Google Analytics / Mixpanel |
| **平均滞在時間** | 60秒以上 | GA / Hotjar |
| **ヒートマップ** | CTA 50%クリック | Hotjar / Crazy Egg |
| **離脱率** | Hero 30%以下 | GA |
| **デバイス比率** | Mobile 60%+ | GA |

**成功判定**（2週間後）:
- CVR 5%以上 → ✅ PSF合格候補（UVP刺さり度確認）
- CVR 3-5% → ⚠️ LP改善実施後に再測定
- CVR 3%未満 → ❌ UVP再定義、またはPivot検討
```

### 自動実行ステップ（v2更新）

1. **【NEW】検証目的の特定**（PSF診断結果から）
2. **【NEW】MVP 10類型から最適選択**（決定木アルゴリズム）
3. **【NEW】実装計画生成**（Week 1-3スケジュール）
4. リーンキャンバス・ペルソナ読み込み
5. ベンチマークLP調査（選択MVP類型に基づく）
6. LP構成設計（7セクション）
7. UVP最適化（3パターン生成）
8. HTML生成（セマンティックHTML）
9. CSS生成（レスポンシブデザイン）
10. JavaScript生成（フォームバリデーション等）
11. フォーム実装（Google Forms等）
12. 5項目チェック
13. **【NEW】証拠収集計画出力**（測定指標・成功判定）
14. 成果物出力

### 7セクション構成

1. **Hero**: UVP + CTA
2. **Problem**: ペルソナの困りごと
3. **Solution**: どう解決するか
4. **Features**: 3つの核心機能
5. **How it Works**: 3ステップ
6. **Pricing**: 価格プラン（1-3個）
7. **CTA**: メール登録/事前予約

### 判定基準（5項目チェック）

| 項目 | 合格条件 |
|------|----------|
| **コンテンツ** | 7セクションすべて実装 |
| **UVP明確性** | 3秒で価値が伝わる（30文字以内） |
| **CTA配置** | 2箇所以上、目立つ配置 |
| **レスポンシブ** | モバイル/タブレット/PC対応 |
| **フォーム動作** | 送信テスト成功 |

**総合判定**:
- 5/5: ✅ 完了 → LP公開、広告配信開始
- 3-4/5: ⚠️ 要修正 → 不合格項目を修正
- 0-2/5: ❌ 再構築 → LP設計から見直し
