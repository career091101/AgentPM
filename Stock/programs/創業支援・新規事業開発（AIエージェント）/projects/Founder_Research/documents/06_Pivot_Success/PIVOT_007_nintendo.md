# 横井軍平・山内溥 - Nintendo（Multiple Pivots）

## 1. 基本情報

| 項目 | 内容 |
|------|------|
| 主要人物 | 横井軍平（製品開発）、山内溥（経営） |
| 生年 | 横井: 1941年、山内: 1927年 |
| 企業名 | 任天堂株式会社 |
| 創業年 | 1889年（明治22年） |
| 業界 | ゲーム / エンターテインメント / 玩具 |
| 現在の状況 | 東証プライム上場、世界的ゲーム企業 |
| 時価総額 | $60B+（2024年） |

## 2. 複数ピボットの歴史

### 2.1 創業〜戦前: 花札製造（1889-1940s）

**創業者**: 山内房治郎
- 京都で花札製造開始
- 手作り花札の品質で評価
- 主に賭博用具として流通

**市場**: 限定的（賭博、正月遊び）

### 2.2 Pivot 1: トランプ製造（1950s）

**山内溥社長就任（1949年、22歳）**:
- 祖父の急病で早稲田大学中退
- 3代目社長に就任

**トランプへのピボット**:
- 戦後、花札市場縮小
- 西洋カード（トランプ）市場に参入
- **1959年**: Disneyとライセンス契約
- Disneyキャラクター付きトランプが大ヒット

**成功**: 年間60万枚→100万枚→600万枚

**問題**: 市場飽和
- トランプ市場は限界
- 新規顧客獲得困難

### 2.3 Pivot 2: 玩具製造（1960s）

**横井軍平入社（1965年）**:
- 同志社大学電子工学科卒
- 製造ライン保守担当として入社

**ウルトラハンド開発（1966年）**:
- 横井が暇つぶしに作った伸縮アーム
- 山内社長が偶然発見、製品化
- **120万個販売**

**玩具シリーズ**:
- ラブテスター（1969年）
- 光線銃（1970年代）
- 様々なヒット商品

**問題**: 玩具市場の競争激化
- バンダイ、タカラ（現タカラトミー）等の競合
- ヒット商品依存

### 2.4 Pivot 3: 電子ゲーム（1970s-1980s初期）

**ゲーム&ウオッチ（1980年）**:
- 横井軍平のアイデア
- 新幹線でサラリーマンが電卓で遊んでいるのを見て着想
- 携帯型液晶ゲーム
- **全世界4,340万台販売**

**成功要因**:
- 携帯性
- 低価格
- シンプルなゲーム性

**影響**:
- 十字キー発明（後のファミコン、ゲームボーイに継承）

### 2.5 Pivot 4: 家庭用ゲーム機（1980s〜）

**アタリショック（1983年）**:
- 北米で家庭用ゲーム市場崩壊
- 粗悪ソフトの氾濫
- 消費者の信頼喪失

**ファミリーコンピュータ（ファミコン、1983年）**:
- 山内社長の決断: ゲーム市場への本格参入
- 価格: 14,800円（当時の競合は3-5万円）
- 品質管理（任天堂承認制）

**成功**:
- 日本: 1,935万台
- 全世界: 6,191万台
- ゲーム産業の復活

**主要タイトル**:
- スーパーマリオブラザーズ（1985年）
- ゼルダの伝説（1986年）
- ドラゴンクエスト（1986年、エニックス）

## 3. ピボット成功要因

### 3.1 経営者の決断力

**山内溥の特徴**:
- リスクを恐れない
- 市場飽和を早期に察知
- 新規事業への果敢な投資

### 3.2 技術者の創造性

**横井軍平の哲学**:
- 「枯れた技術の水平思考」
- 既存技術を新しい用途に
- 低コストで高付加価値

### 3.3 品質管理

**任天堂クオリティ**:
- ソフトウェアの承認制
- サードパーティへの厳格な審査
- ブランド価値の維持

### 3.4 IP（知的財産）の構築

**キャラクター資産**:
- マリオ、ゼルダ、ドンキーコング等
- 長期的なフランチャイズ
- 映画、テーマパーク等への展開

## 4. その後の展開

### 4.1 携帯ゲーム機（1989年〜）

**ゲームボーイ（1989年）**:
- 横井軍平の最後の傑作
- モノクロ液晶（枯れた技術）
- テトリス同梱
- **全世界1億1,869万台**

### 4.2 3D時代への移行（1996年〜）

**NINTENDO64（1996年）**:
- 3Dゲームへの挑戦
- スーパーマリオ64

**課題**: CD-ROM不採用でサードパーティ離れ

### 4.3 復活と革新（2000s-2020s）

**Nintendo DS（2004年）**:
- タッチスクリーン
- 1億5,402万台

**Wii（2006年）**:
- モーションコントロール
- カジュアル層の開拓
- 1億163万台

**Nintendo Switch（2017年）**:
- 据置・携帯ハイブリッド
- 1億4,000万台+（2024年）

## 5. 教訓

### 5.1 複数ピボットの重要性

**市場変化への適応**:
- 花札 → トランプ → 玩具 → 電子ゲーム → 家庭用ゲーム機
- 各ピボットで生き残り

**100年企業の秘訣**:
- 1つの市場に固執しない
- 常に次の成長機会を探す

### 5.2 技術と創造性のバランス

**枯れた技術の水平思考**:
- 最先端技術ではなく、既存技術の新しい使い方
- コスト削減と革新の両立

### 5.3 IPの重要性

**キャラクター資産**:
- マリオは世界で最も有名なゲームキャラクター
- 長期的な収益源
- ブランド価値

## 6. 日本市場への示唆

**日本企業のピボット事例**:
- 富士フイルム（フィルム → 化粧品・医療）
- オムロン（電気機器 → ヘルスケア）
- ヤマハ（楽器 → 音響機器 → 半導体）

**示唆**:
- 長期的視点
- コア技術の活用
- 市場変化への柔軟な対応
