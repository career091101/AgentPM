```
STEP 7: Rule of 40評価（SaaS専用、3分）
  公式: YoY成長率(%) + EBITDA Margin(%)
  目標: 40%以上
```

#### 業種別計算対応

**SaaS（サブスクリプション）**:
```
LTV = MRR × 粗利率 / Churn率
または
LTV = ARPU × 平均継続月数 × 粗利率
平均継続月数 = 1 / Churn率
```

**Eコマース（リピート型）**:
```
LTV = 平均購入単価 × 購入回数 × 粗利率
購入回数 = 1 / (1 - リピート率)
```

**マーケットプレイス（両面市場）**:
```
LTV = GMV × Take Rate × 平均取引回数 × 粗利率
GMV: 流通総額
Take Rate: 手数料率
```

- 期待効果: LTV $560K → $933K (+67%)

**施策2: CAC削減（効率改善）**
- 目標: CAC -30%削減
- 施策: 紹介プログラム導入、コンテンツマーケ強化、パートナーチャネル開拓
- 期待効果: CAC $66K → $46K (-30%)

**施策3: ARPU向上（単価改善）**
- 目標: ARPU +20%向上
- 施策: Enterprise Plan追加、既存顧客アップセル、バリューベースプライシング
