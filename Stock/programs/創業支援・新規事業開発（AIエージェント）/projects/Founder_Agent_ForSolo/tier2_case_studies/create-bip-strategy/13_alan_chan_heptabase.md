## 2. 思考プロセス可視化BIP戦略

### コンテンツタイプ
| 種類 | 詳細 |
|------|------|
| **思考マップ公開** | Heptabase上での製品企画、問題解決プロセスをスクリーンショット公開 |
| **学習ノート公開** | 読んだ本、論文の思考整理プロセスを可視化 |
| **開発意思決定** | "なぜこの機能を実装するか"の思考過程を図解 |

### コンテンツ例
```
"Heptabaseでプロダクト戦略を整理した図を公開します。

```
