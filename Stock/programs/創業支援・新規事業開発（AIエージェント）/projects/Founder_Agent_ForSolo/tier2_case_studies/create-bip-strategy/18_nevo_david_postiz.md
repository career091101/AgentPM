## 2. OSS開発BIP戦略

### GitHub中心の透明性

| 要素 | 詳細 |
|------|------|
| **コード公開** | 100% OSSとして GitHub公開 |
| **Issue透明性** | バグ、機能要望、ロードマップをすべてGitHub Issueで管理 |
| **PR歓迎** | コミュニティからのPull Request積極受け入れ |
| **開発ログ** | GitHub Actionsで自動デプロイ、進捗可視化 |
