## 1. Build in Public の核心: 17歳起業家の爆速バイラル戦略

### プロフィール
- **年齢**: 17歳（2025年時点）
- **国籍**: キプロス
- **X**: [@danvsi](https://x.com/danvsi)
- **特徴**: 9歳から動画制作、完全オーガニック成長（広告費$0）

### 収益実績
- **Crayo.AI**: MRR $500K-600K/月
- **ビジネス全体**: $1M/月（Snapchat、YouTube Shorts、教育コミュニティ含む）
- **開発コスト**: わずか$10,000
- **成長期間**: ローンチから6ヶ月で月商$600K達成

## 2. Build in Public 戦略

### TikTok/Reddit 主導のバイラル成長

| 要素 | 詳細 |
|------|------|
| **主要チャネル** | TikTok（メイン）、Reddit、X |
| **投稿頻度** | TikTok: 日3-5回、X: 日1-2回 |
| **コンテンツ** | Before/After動画、AI生成プロセス、10代起業家ストーリー |
| **エンゲージメント** | TikTok平均50万再生、X 5-10% |

### コンテンツカレンダー（週間サンプル）
```
Mon: "17歳で月商$600K達成した理由（TikTok 1分動画）"
Tue: "Crayo.AIで作ったショート動画のBefore/After"
Wed: "完全オーガニック成長の秘訣（Reddit AMA）"
Thu: "AI動画生成の裏側（開発進捗スクリーンショット）"
Fri: "今週の収益: $150K（製品別内訳）"

Sun: "次週の目標: 新機能3つリリース"
```
