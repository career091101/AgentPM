## ユニットエコノミクス

### LTV（顧客生涯価値）

| 顧客セグメント | 平均利用期間 | 月額 | チャーン率 | LTV |
|--------------|------------|------|----------|-----|
| **無料ユーザー** | 3ヶ月 | $0 | 70%/月 | $0 |
| **Pro（月払い）** | 12ヶ月 | $29 | 15%/月 | **$193.33** |
| **Pro（年払い）** | 18ヶ月 | $24.17 | 10%/月 | **$435.06** |
| **Team（月払い）** | 18ヶ月 | $99 | 10%/月 | **$1,485** |
| **Team（年払い）** | 24ヶ月 | $82.50 | 7%/月 | **$2,828.57** |
| **Enterprise** | 30ヶ月 | $500 | 5%/月 | **$15,000** |

**加重平均LTV**（有料ユーザー全体）:
- Pro月払い（40%）: $193.33 × 0.4 = $77.33
- Pro年払い（30%）: $435.06 × 0.3 = $130.52
- Team月払い（15%）: $1,485 × 0.15 = $222.75
- Team年払い（10%）: $2,828.57 × 0.1 = $282.86
- Enterprise（5%）: $15,000 × 0.05 = $750
- **合計**: **$1,463.46**

### CAC（顧客獲得コスト）

| チャネル | 月間コスト | 新規有料ユーザー | CAC |
|---------|---------|----------------|-----|
| **Product Hunt** | $0 | 100人 | **$0** |

| **コンテンツマーケ** | $2,000（外注） | 30人 | **$67** |
| **合計** | **$7,000** | 230人 | **$30.43** |

**CAC削減の秘訣**:
- Product Huntで初期トラクション獲得（無料）
- 最新AIモデル活用が話題性を生む
- 競合出現前の先行者利益

### LTV/CAC比率

```
LTV/CAC = $1,463.46 / $30.43 = 48.1倍
```

**評価**:
- **業界標準**: LTV/CAC 3.0以上で健全
- **AIウェブアプリ構築ツールの実績**: **48.1倍**（超優秀）
- **理由**: 最新AIモデル活用と先行者利益
- **成長性**: MRR成長率が高く、売却倍率30倍で評価

### CAC回収期間

```
Payback Period = CAC / 月次収益
                = $30.43 / $50（平均単価）
                = 0.6ヶ月（即座）
```

## 日本市場適用

### 文化的適応
日本市場で本事例を適用する際の調整ポイント:

1. **言語・ローカライゼーション**
   - UI/UX: 日本語対応必須（機械翻訳ではなくネイティブレビュー）
   - カスタマーサポート: 日本語対応（英語のみは避ける）
   - マーケティング: 日本語コンテンツ作成（X/Twitter、ブログ等）

2. **決済・価格設定**
   - クレジットカード: JCB対応追加
   - 価格表示: 税込表示（消費税10%）
   - 決済手段: 銀行振込、コンビニ決済の検討

3. **マーケティングチャネル**
   - SNS: X/Twitter中心（70%）、LinkedIn補完（20%）、Facebook/Instagram（10%）
   - Build in Public: 日本では慎重（失敗の公開に抵抗あり）
   - コミュニティ: Slack/Discord活用（日本語チャンネル）

4. **タイムライン調整**
   - フォロワー獲得: 米国の1.5倍時間（日本市場は保守的）
   - PMF達成: 米国の1.2-1.5倍時間
   - MRR成長: 米国比80%ペース（市場規模差）
