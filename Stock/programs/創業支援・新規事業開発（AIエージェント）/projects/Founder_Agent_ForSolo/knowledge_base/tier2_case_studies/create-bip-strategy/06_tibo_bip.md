# Tier 2 Case Study: Tibo - Build in Public Strategy

**Skill**: create-bip-strategy
**Source Case**: APP_013 (Tibo - Tweet Hunter / Taplio)
**Category**: App
**Created**: 2026-01-02

## 1. Build in Public Overview

### Basic Strategy
- **Transparency Level**: 完全公開（収益、失敗、ローンチ過程すべて）
- **Primary Platform**: X/Twitter（メイン）+ LinkedIn（Taplio拡張）
- **Content Strategy**: インフルエンサーエクイティ戦略、失敗談公開、透明性
- **Community Engagement**: インフルエンサー12人に株式配布、相互支援エコシステム

### Key Metrics
- **X Followers**: 15万人以上（推定、Tweet Hunter/Taplio時代）
- **Influencer Equity Recipients**: 12人（Arvid Kahl、JK Molina等）
- **Revenue Transparency**: ARR $1.4M達成時に公開
- **Exit**: $10M+（キャッシュ+Earn out、Lempireへ売却）

## 4. Influencer Equity Strategy（革新的マーケティング）

### 戦略概要

| 要素 | 詳細 |
|------|------|
| **配布先** | インフルエンサー12人（Arvid Kahl、JK Molina等） |
| **株式比率** | 各0.1%（合計1.2%） |
| **契約内容** | **なし**（自由に使ってもらうだけ） |
| **なぜ可能か** | 完全自己資金、VCなし |
| **効果** | 積極的・継続的に宣伝してもらえる |

### インフルエンサー選定基準

1. **フォロワー数**: 1万人以上
2. **エンゲージメント率**: 5%以上
3. **ターゲット層の一致**: X成長、起業家、インディーハッカー
4. **Build in Public実践者**: 透明性を重視する人

### 成果

- **拡散力**: インフルエンサー経由で月間500万インプレッション
- **信頼性**: 「インフルエンサーが株主」という信頼
- **継続性**: 単発広告ではなく、継続的な宣伝
- **コスト**: 株式1.2%のみ（現金コストゼロ）

### Tiboの名言

> 「インフルエンサーにお金を渡すのではなく、株をもたせることで全員を単なる商売相手ではなく同じ船に乗せる」

## 5. Revenue Impact from Build in Public

### Direct Revenue Attribution

- **Tweet Hunterローンチ時のフォロワー**: 10,000人程度
- **4ヶ月時点のMRR**: $15K（約200万円）/月
- **1年8ヶ月時点のARR**: $1.4M（約2億円）/年
- **コンバージョン率**: 推定1-2%（フォロワー → 有料ユーザー）

### Compounding Effect

- **フォロワー成長** → **インフルエンサー拡散** → **新規フォロワー** → **Tweet Hunter認知** → **購入増加**
- **インフルエンサーエクイティ効果**: 12人の拡散力が複利的に成長

### Build in Public ROI

- **時間投資**: 1日1-2時間（投稿、リプライ、コミュニティ参加）
- **直接コスト**: 株式1.2%のみ（広告費ゼロ）
- **収益貢献**: **80%以上**（フォロワー・インフルエンサー経由がメイン）

## 6. Key Success Factors

### What Worked

1. **インフルエンサーエクイティ**: 革新的マーケティング、契約なし株式配布
2. **完全透明性**: 収益、失敗、すべて公開→信頼獲得
3. **横展開**: Twitter → LinkedIn（Taplio）成功モデル複製
4. **自己資金**: VCなしだから株式配布が可能
5. **コミュニティ参加**: 他のインディーハッカー支援→相互支援

### What Didn't Work

1. **初期の15個のプロジェクト**: ほとんどが失敗（学びに転換）
2. **Pistache**: 50万ユーザーでも売却価格は控えめ（推定）
3. **dreamz.studio**: $500K調達→2年でシャットダウン

## 7. Lessons for Japanese Solopreneurs

### Cultural Adaptation

- **透明性のレベル**: 日本では収益完全公開は抵抗あり→段階的公開が有効
- **失敗の見せ方**: 自虐より「学び」重視で共感獲得
- **インフルエンサーエクイティ**: 日本でも可能、ただし「株式配布」より「アンバサダー契約」名目推奨

### Recommended Timeline（日本版）

1. **0-1Kフォロワー**: 6-12ヶ月（海外より慎重な成長）
2. **1K-10K**: 12-24ヶ月（信頼構築に時間）
3. **10K-50K**: 18-36ヶ月（日本市場では50Kで十分な影響力）

### Platform Mix（日本）

- **X**: 40%（Build in Public）
- **note**: 30%（詳細な失敗談・学び記事）
- **YouTube**: 20%（開発過程、チュートリアル）
- **Zenn**: 10%（技術記事、信頼構築）

## 日本市場適用

### 文化的適応
日本市場で本事例を適用する際の調整ポイント:

1. **言語・ローカライゼーション**
   - UI/UX: 日本語対応必須（機械翻訳ではなくネイティブレビュー）
   - カスタマーサポート: 日本語対応（英語のみは避ける）
   - マーケティング: 日本語コンテンツ作成（X/Twitter、ブログ等）

2. **決済・価格設定**
   - クレジットカード: JCB対応追加
   - 価格表示: 税込表示（消費税10%）
   - 決済手段: 銀行振込、コンビニ決済の検討

3. **マーケティングチャネル**
   - SNS: X/Twitter中心（70%）、LinkedIn補完（20%）、Facebook/Instagram（10%）
   - Build in Public: 日本では慎重（失敗の公開に抵抗あり）
   - コミュニティ: Slack/Discord活用（日本語チャンネル）

4. **タイムライン調整**
   - フォロワー獲得: 米国の1.5倍時間（日本市場は保守的）
   - PMF達成: 米国の1.2-1.5倍時間
   - MRR成長: 米国比80%ペース（市場規模差）
