# Tier 2ケーススタディ: Carrd - 競合調査スキル

## 概要

**AJ**（本名非公開）が開発した**Carrd**は、Wix/Squarespaceの「複雑すぎる」「高い」「遅い」という弱点を突き、**1ページランディングページ特化**という制約を武器に年商$2M（約3億円）を達成。**シンプルさの極致**を追求し、「機能を増やさないことが価値」という逆説的戦略で成功した。

---

## 競合特定方法

### どうやって競合を見つけたか

1. **直接競合（Website Builder）**
   - **Wix**: 汎用Webサイトビルダー最大手、月$10-$30
   - **Squarespace**: デザイン重視ビルダー、月$12-$40
   - **Webflow**: プロ向け高機能ビルダー、月$12-$36

2. **間接競合（Landing Page特化）**
   - **Linktree**: リンクインバイオ特化、月$0-$24
   - **About.me**: 個人ページ特化、無料/有料
   - **Notion Sites**: Notion公開ページ、無料

3. **補完的競合（WordPress系）**
   - **WordPress + Elementor**: オープンソース
   - **Shopify Landing Pages**: EC特化

4. **競合発見の手法**
   - **Product Hunt**: "landing page builder" で上位50製品
   - **Reddit r/webdev**: 「簡単なLP作りたい」スレッド
   - **Google検索**: "Wix alternative simple"
   - **G2/Capterra**: Website Builder カテゴリで評価順

5. **市場セグメンテーション**
   ```
   Webサイトビルダー市場
   ├── 汎用ビルダー（Wix, Squarespace）← 最大セグメント
   ├── プロ向け（Webflow）
   ├── 1ページ特化（Carrd, Linktree）← ここで戦う
   ├── EC特化（Shopify）
   └── ブログ特化（WordPress, Medium）
   ```

## 差別化ポイント（3軸）

### 1. 1ページ特化という制約

| Wix/Squarespace | **Carrd** |
|-----------------|---------|
| **複数ページ対応** | **1ページのみ** |
| 何でもできる | 1ページに特化 |
| 選択肢多すぎ | 選択肢少ない（決断が楽） |
| 学習コスト高 | 学習コスト低 |

**差別化の強さ**: ★★★★★（5/5）
- **制約を武器化**: "できないこと" を明確にすることで、ターゲットを絞る
- **意思決定コスト削減**: 「何ページ作る？」「どの構造？」悩まない
- **ユースケース明確**: ランディングページ、リンクインバイオ、イベント告知等

**AJの哲学**:
> 「1人では全機能ビルダーは作れない。制約を受け入れ、1ページに特化した。」

### 2. シンプルさの極致（美しいテンプレート）

| Wix/Squarespace | **Carrd** |
|-----------------|---------|
| **1,000+テンプレート** | **50+厳選テンプレート** |
| 選択肢多すぎて迷う | 厳選されたデザイン |
| 自由度高いが難しい | 制限あるが簡単 |
| プロデザイナー向け | 非デザイナー向け |

**差別化の強さ**: ★★★★☆（4/5）
- **テンプレート品質**: AJ自身が10年以上HTML5テンプレート作成経験
- **選択肢のパラドックス回避**: 50個から選ぶ vs 1,000個から選ぶ

### 3. 圧倒的な低価格

| プロダクト | 料金 | 年額換算 |
|-----------|------|---------|
| **Wix** | $10-30/月 | $120-360/年 |
| **Squarespace** | $12-40/月 | $144-480/年 |
| **Webflow** | $12-36/月 | $144-432/年 |
| **Linktree** | $0-24/月 | $0-288/年 |
| **Carrd** | **$9-49/年** | **$9-49/年** |

**差別化の強さ**: ★★★★★（5/5）
- **年額$9** = Wix 1ヶ月分以下
- **年額$49** = Wix 4-5ヶ月分
- **心理的障壁**: "年$9なら試してみるか"

## 競合調査から得た教訓

### 1. 制約を武器化せよ

Carrdは「1ページしか作れない」制約を、**明確な差別化**に変えた。

**学び**: "何ができない" を明確にすることで、ターゲットが絞れる。

### 2. シンプルさは機能を減らすこと

Wix/Squarespace: 機能を増やす
Carrd: 機能を減らす

**学び**: 「機能追加」ではなく「機能削減」で差別化できる。

### 3. 価格を1/10にすれば市場が変わる

Wix $120/年 → Carrd $9/年（1/13）

**学び**: 「ちょっと安い」ではなく「桁違いに安い」でブルーオーシャン。

### 4. 既存スキルを活かせ

AJは10年以上のHTML5テンプレート制作経験を活かした。

**学び**: 新しいスキル習得より、既存スキルの転用。

### 5. フリーミアムで試用障壁を下げる

無料版3サイト → 有料版10サイト（$19/年）へ自然アップグレード。

**学び**: Freemiumは試用のハードルを下げ、コンバージョン率を上げる。

   - クレジットカード: JCB対応追加

   - 決済手段: 銀行振込、コンビニ決済の検討

   - SNS: X/Twitter中心（70%）、LinkedIn補完（20%）、Facebook/Instagram（10%）
   - Build in Public: 日本では慎重（失敗の公開に抵抗あり）
   - コミュニティ: Slack/Discord活用（日本語チャンネル）

   - フォロワー獲得: 米国の1.5倍時間（日本市場は保守的）
   - PMF達成: 米国の1.2-1.5倍時間
