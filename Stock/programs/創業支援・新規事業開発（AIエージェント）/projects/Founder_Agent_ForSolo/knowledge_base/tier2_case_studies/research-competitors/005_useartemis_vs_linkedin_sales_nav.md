# Tier 2ケーススタディ: useArtemis - 競合調査スキル

## 概要

**Samuel Rondot**（フランス出身）が開発した**useArtemis**は、「**全く新しいものは作らず、既に売れているツールの代替品（Alternative）を作る**」というコピーキャット戦略で月商$35K（約500万円）を達成。LinkedIn自動化ツール市場で、Sales NavigatorやDux-Soupの弱点を突き、**SEO「〇〇 Alternative」記事**で集客する手法が特徴。

---

## 競合特定方法

### どうやって競合を見つけたか

1. **直接競合（LinkedIn Automation）**
   - **LinkedIn Sales Navigator**: LinkedIn公式、月$79.99
   - **Dux-Soup**: 老舗自動化ツール、月$11.25-$55
   - **Expandi**: クラウドベース、月$99
   - **PhantomBuster**: 汎用自動化、月$56-$440

2. **間接競合（CRM統合型）**
   - **HubSpot Sales Hub**: CRM+LinkedIn、月$45-$1,200
   - **Salesforce + LinkedIn統合**: エンタープライズ
   - **Waalaxy**: フランス発、月€40-€80

3. **競合発見の手法（逆算方式）**
   - **G2検索**: "LinkedIn Automation Tools" カテゴリ
   - **既存ツールのレビュー分析**: ★1-2レビューで不満収集
   - **「〇〇 Alternative」検索**: Google検索で上位記事調査
   - **Product Hunt**: "linkedin" タグで過去2年

4. **Samuelの独自手法: "Reverse Engineering"**
   ```
   1. 人気ツール（Dux-Soup）を発見
   2. G2で低評価レビューを読む
   3. 「高い」「サポート悪い」「機能不足」をリスト化
   4. それを全て解決する代替品を作る
   ```

5. **市場セグメンテーション**
   ```
   LinkedIn自動化市場
   ├── 公式ツール（Sales Navigator）← 最大シェア
   ├── クロームエクステンション（Dux-Soup, Waalaxy）
   ├── クラウドベース（Expandi, useArtemis）← ここで戦う
   └── エンタープライズ（HubSpot, Salesforce）
   ```

## 競合調査から得た教訓

### 1. オリジナリティは不要

Samuelの哲学:
> 「新しいものを考える必要はない。既に売れているものを、少し良くして、安く売れば勝てる。」

**学び**: 独自性より、既存市場の隙間を突く。

### 2. 低評価レビューは宝の山

G2/Capterraの★1-2レビュー = 顧客の不満リスト

**学び**: 競合の弱点は、顧客が教えてくれる。

### 3. SEO「Alternative」記事は最強集客

「Expandi Alternative」で検索する層 = 購買意欲高い

**学び**: 競合名でSEO取れば、低CACで顧客獲得。

### 4. Fast Followerは賢い戦略

First Mover（先行者）がリスク取って市場教育 → Fast Followerが刈り取る

**学び**: 先行者の失敗から学び、改善版で参入。

### 5. コピーキャット戦略の限界

**弱点**: ブランド力では本家に勝てない
**対策**: 価格と特定機能（クラウド実行等）で差別化
