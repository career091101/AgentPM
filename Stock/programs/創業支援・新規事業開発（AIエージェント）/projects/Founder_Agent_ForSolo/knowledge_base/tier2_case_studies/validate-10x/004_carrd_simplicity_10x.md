# Carrd - サイト作成の簡便性10倍向上で年商3億円

## 概要

Carrdは、1ページのランディングページ作成に特化したWebサイトビルダーで、**簡便性10倍向上**を実現。WordPress, Wix等の複雑なビルダーに対し、「1ページだけ」という制約を武器に、1時間でプロフェッショナルなサイトを完成可能。価格も年額$9-49と10倍安く、260万ユーザー・400万サイトを獲得し、年商$2M（約3億円）をほぼソロで達成した事例。

## 教訓・学び

### 1. 「制約を武器にする」

AJは「1人では全機能ビルダーは作れない」という制約を逆手に取り、1ページ特化で差別化。制約こそが明確なポジショニングを生み出し、意思決定コストを90%削減した。

### 2. 「シンプルさを維持する規律」

10年間、機能追加の誘惑に抗い、1ページ特化を貫いた。「Noと言い続けること」がプロダクトの価値であり、多機能化の罠に陥らなかったことが成功の鍵。

### 3. 「既存スキルを活かす」

HTML5 UPで10年以上テンプレート制作経験があったAJは、その経験をそのままCarrdに活かした。新しいスキルを学ぶのではなく、既存の強みを最大化した。

### 4. 「古い技術でも十分」

jQuery + PHPという「古い」技術スタックで年商3億円を達成。最新技術を追うのではなく、自分が熟知した技術で完璧なUXを提供することの価値を示した。
