# Tier 2 Case Study: Desmond - Rise (Instagram Viral型PSF)

**Tier 2専用フォーカス**: validate-psf スキル用ケーススタディ
**PSFスコア**: 87/100点（技術的実現可能性92点、ソリューション適合88点、コア機能88点）

---

## 1. PSF概要（3行まとめ）

- **課題発見**: 習慣化アプリ市場は競合多数だが、「アプリが習慣を定義」するコンセプトなし
- **解決策の検証**: アプリ主導型習慣化 + Instagramリールバイラル → 初月$1K、4ヶ月で$11K
- **PSF達成根拠**: 差別化明確 × Instagramマーケティング成功（年間1億回再生）

---

## 2. PSF評価（6軸詳細分析）

### 2.1 ソリューション適合度（88/100点）

**課題の特定**:
- 既存習慣化アプリ: ユーザーが自分で習慣設定 → 挫折率高い
- Rise の差別化: **アプリが「完璧な習慣」を定義** → ユーザーは従うだけ

**検証データ**:
```
2024年6月: ローンチ → MRR $1K
2024年9月: MRR $11K（4ヶ月で11倍）
2025年10月: MRR $200K（18ヶ月で200倍）
```

**-12点の理由**: 習慣化アプリ市場は競合多数、差別化が必須

### 2.2 MVP実装速度（85/100点）

**開発期間**: 数週間（2024年4-5月、2人チーム）

**Phase 1: コンセプト設計（1週間）**
- 66日間の習慣形成ロードマップ設計
- 「完璧な習慣」の定義（起床、運動、瞑想等）

**Phase 2: モバイルアプリ開発（2-3週間）**
- React Native で iOS/Android対応
- 習慣トラッキング機能
- プッシュ通知機能

**Phase 3: ローンチ（1週間）**
- App Store / Google Play申請
- LP作成

**合計**: 約150-200時間（4-5週間）

**-15点の理由**: モバイルアプリ開発は時間がかかる（Webより遅い）

### 2.3 初期顧客フィードバック（82/100点）

**初月（2024年6月）**:
- ダウンロード: 5,000
- 有料化: 30人（MRR $1K）

**4ヶ月後（2024年9月）**:

**-18点の理由**: 初月MRR $1Kは低い、成長に4ヶ月要した

## 3. 成功要因（PSF視点）

### 3.1 Instagram Viral マーケティング

**ターゲット憑依法**:
```
1. ターゲット（自己啓発層）がフォローする人を分析
2. 毎日10分リールをスクロール
3. バズったリールを見つけて「Riseにどう使えるか」考える
4. アイデアをメモして試す
```

**結果**:
- 年間1億回再生達成
- フォロワー5.8万人
- Instagram → アプリダウンロード転換率: 8%

### 3.2 プロダクト差別化

**「アプリが習慣を定義」**:
- 既存: ユーザーが自分で習慣設定 → 挫折
- Rise: アプリが完璧な習慣を提示 → ユーザーは従うだけ
