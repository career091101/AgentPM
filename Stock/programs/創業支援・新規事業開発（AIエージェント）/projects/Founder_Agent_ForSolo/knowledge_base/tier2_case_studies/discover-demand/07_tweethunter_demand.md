# Tier 2 Case Study: Tibo - Tweet Hunter (Demand Discovery Pattern)

**Tier 2専用フォーカス**: discover-demand スキル用ケーススタディ
**ニーズ発見手法**: Twitter観察 + インフルエンサー提案 + Build in Public

---

## 1. 概要（Demand Discovery視点の3行まとめ）

- **インフルエンサーエクイティ戦略**: 12人のインフルエンサーに株式0.1%配布 → 契約なし、自由に使ってもらう → 積極的宣伝
- **Twitter観察から需要発見**: 「フォロワー増やしたい」「コンテンツ作成が大変」の不満を観察 → Tweet Hunter開発
- **横展開戦略**: Twitter成功 → LinkedIn版Taplio開発 → 2プロダクト合計ARR $1.4M達成

## 2. 需要発見プロセス詳細

### 課題発見チャネル

**Twitter（X）観察**（2020年1-2月、2ヶ月間）:
```
観察ハッシュタグ: #buildinpublic, #indiehacker, #solopreneur

収集した不満（上位5つ）:
1. "Twitter成長が遅い、フォロワー増えない" - 200件
2. "毎日ツイートネタ考えるの大変" - 150件
3. "過去のツイートが埋もれて再利用できない" - 120件
4. "効果的なツイート時間がわからない" - 100件
5. "スレッド作成がめんどくさい" - 80件
```

**IndieHackers フォーラム**:
```
スレッド分析:
- "How to grow Twitter audience?" - 250コメント
- "Best tools for Twitter growth?" - 180コメント
- "Automating Twitter without spamming?" - 120コメント

共通パターン:
- 既存ツール（Buffer, Hootsuite）は高機能すぎて複雑
- Twitter特化ツールが欲しい
- コンテンツ作成支援機能が欲しい
```

### 課題検証（3U+1U）

| 項目 | スコア | 分析 |
|------|--------|------|
| **Unworkable（機能しない）** | 3/3 | 手動でツイートネタ探し、手動投稿は時間がかかりすぎる |
| **Unavoidable（避けられない）** | 3/3 | Twitter成長はクリエイター・起業家にとって必須（フォロワー=信頼=売上） |
| **Urgent（緊急性）** | 2/3 | 毎日の作業だが、今日やらないと致命的ではない |
| **Underserved（未解決）** | 3/3 | Buffer/Hootsuiteは高機能だが、Twitter特化ではない |

**4Uスコア**: 11/12点（非常に高スコア → 強いニーズあり）

### WTP（支払い意思）検証

**ターゲット価格**: $49/月（3プラン: $29, $49, $99）
**根拠**:
1. **類似サービス価格調査**:
   - Buffer: $15-99/月（汎用SNSツール）
   - Hootsuite: $49-739/月（企業向け）
   - **隙間**: Twitter特化で$49/月なら「専門ツール」として妥当

2. **Twitter アンケート**（2020年2月、Tiboフォロワー5,000人時点）:
   ```
   Q: "Twitter growth tool, how much would you pay?"
   - $0 (free only): 35%
   - $19/month: 25%
   - $49/month: 30%  ← ターゲット層
   - $99/month: 8%
   - $199+/month: 2%

   → $49/月で30%が支払い意思あり
   ```

3. **インフルエンサーからの提案**:
   - JK Molina氏（フォロワー10万人）: 「株くれるなら拡散手伝う」
   - → インフルエンサーエクイティ戦略のアイデア誕生

- "Twitter growth tool": 月8k searches

- 合計: 月25k searches（中規模だが安定）

- Reddit r/SaaS "Twitter tools": 月80投稿
- 合計: 月680件（継続的な需要）

**競合サービス顧客数**（推定）:
- Buffer: 15万有料ユーザー
- Hootsuite: 20万有料ユーザー
- Twitter特化ツール: 5万ユーザー（推定）
- **隙間**: Twitter特化 × クリエイター向けは未開拓

## 3. インフルエンサーエクイティ戦略の詳細

### 従来のマーケティング（失敗パターン）

**一般的なアプローチ**:
```
1. 広告費投資: $5,000-10,000/月
2. インフルエンサーに一時金支払い: $1,000-5,000/投稿
3. 問題点:
   - 広告費が継続的にかかる
   - インフルエンサーの宣伝は1回きり
   - ROI不明確
```

### インフルエンサーエクイティ戦略（Tibo独自）

**革新的アプローチ**:
```
1. 株式0.1%配布: インフルエンサー12人に
2. 契約なし: 自由に使ってもらうだけ
3. 効果:
   - インフルエンサーが「自分のプロダクト」として積極的に宣伝
   - 継続的に宣伝（1回きりではない）
   - 広告費$0

インフルエンサー12人:
- JK Molina (フォロワー10万人)
- Arvid Kahl (フォロワー15万人)
- その他10人（合計フォロワー100万人以上）
```

**なぜ可能だったか**:
- 完全自己資金（VCなし）→ 株式を自由に配布できる
- 2人チーム（Tibo + Tom Jacquesson）→ 少数株主でも影響大

**効果測定**:
| 期間 | MRR | 主要施策 |
|------|-----|---------|
| ローンチ初月 | $5K | Product Hunt #1、インフルエンサー初期投稿 |
| 4ヶ月後 | $15K | インフルエンサー継続宣伝、口コミ拡大 |
| 1年8ヶ月後 | $116K | ARR $1.4M達成、売却交渉開始 |

## 8. 分析者コメント（Demand Discovery視点）

Tiboの需要発見プロセスは、**インフルエンサーエクイティという前例なき戦略で市場を開拓した好例**だ。

**ForSolo開発者が学ぶべき3つのポイント**:

1. **インフルエンサーエクイティ**: 広告費$0で100万フォロワーにリーチ → 自己資金のみだから可能
2. **横展開戦略**: Twitter成功 → LinkedIn版を2ヶ月で開発 → 収益2倍化
3. **Build in Public**: 開発過程を公開 → インフルエンサーが自然と宣伝 → 広告費不要

**日本のSolo開発者へのメッセージ**:
日本はX（Twitter）ユーザー数世界2位（6,000万人）。日本版Tweet Hunterは確実に需要がある。Tiboのように、**日本のインフルエンサー（マナブさん、イケハヤさん等）に株式0.1%配布すれば、広告費$0で月商1,000万円達成できる確率は高い**。
