# 企業製品リサーチ: スタディサプリ学習塾向けサービス

## 1. 製品・サービス概要

### 基本情報
- **製品名**: スタディサプリ学習塾向けサービス
- **企業名**: 株式会社リクルート
- **ステータス**: Withdrawn（2026年3月30日サービス終了予定）
- **カテゴリー**: EdTech・学習塾向けICT教材プラットフォーム
- **サービス開始**: 2022年3月
- **終了発表**: 2025年9月1日
- **サービス終了**: 2026年3月30日
- **サービス期間**: 約4年

### 製品概要
「スタディサプリ学習塾向けサービス」は、年間194万人（2020年度）が有料会員として利用するスタディサプリを、学習塾の強みをさらに強化するための機能として圧倒的低価格で提供するBtoBサービスです。小学校1年生から高校3年生までの全講義を視聴可能とし、単元ごとの詳細テストで生徒の学習理解度を測定、弱点に合わせた効率的な学習を実現する設計でした。

リクルートは2022年3月にこのサービスを開始しましたが、事業環境の変化やサービスの利用状況などを総合的に判断し、約4年で提供を終了することを決定しました。

### ビジネスモデル
- **生徒1人あたり月額課金**: 月額料金制（具体的金額は非公開）
- **最小契約単位**: 生徒1人から利用可能
- **初期費用**: なし
- **システム利用料**: なし
- **違約金**: なし（いつでも途中退会可能）
- **収益構造**: 学習塾からの月額利用料

### サービス構成
1. **映像授業**: 小学1年生〜高校3年生までの全講義視聴可能
2. **単元テスト**: 単元ごとに詳細なテストを提供、学習理解度測定
3. **弱点分析**: 小問が対応する講義動画と紐づき、効率的な学習が可能
4. **管理機能**: 塾が生徒の学習進捗を管理・把握する機能
5. **柔軟な契約**: 生徒数に応じた従量課金、途中解約自由

### 対象顧客
- **プライマリー**: 学習塾（個人塾〜大手塾まで）
- **エンドユーザー**: 小学生〜高校生の塾生徒

## 2. 市場・競合環境分析

### ターゲット市場
- **プライマリー**: 日本国内の学習塾（推定約5万教室）
- **セカンダリー**: 学習塾に通う生徒（推定数百万人）
- **市場規模**: 学習塾市場は約1兆円規模、EdTech市場は数千億円規模

### 競合状況

#### AI学習「御三家」
教育業界で「AI御三家」として認識されている主要競合:

1. **atama+（アタマプラス）**
   - 大手塾の30%以上が導入
   - 東大・京大・早慶などの超難関大向けに最適化
   - AIが生徒の理解度・弱点・集中力を分析し個別カリキュラム作成
   - 主要導入先: 駿台予備校等
   - ポジション: 超難関校受験層

2. **AIストライク（ベネッセ）**
   - ベネッセが提供するAI学習サービス
   - Classi（クラッシー）との連携で学校向けにも展開

3. **スタディサプリ**
   - 3,000高校で導入（学校向け）
   - カリスマ講師の映像授業が強み
   - 塾向けは低調（今回撤退）

#### その他主要競合

4. **Monoxer（モノグサ）**
   - 記憶定着に特化したアプリ
   - 2,500教室以上の塾・予備校に導入
   - 創業者は元スタディサプリ中核メンバー（竹内孝太朗氏）
   - スタディサプリとは「補完関係」（理解 vs 記憶定着）

5. **Classi（クラッシー）**
   - 学校・クラス管理プラットフォーム
   - ベネッセの進研模試との連携
   - 会員数145万人（スタディサプリより少ない）
   - 主に学校向け

### 市場トレンド
1. **AI個別最適化学習**: atama+等のAI活用サービスが台頭
2. **学校・自治体へのシフト**: EdTech企業が個人・塾よりも学校向けに注力
3. **複数サービスの併用**: 塾が複数のEdTechサービスを組み合わせて活用
4. **記憶定着系の台頭**: Monoxer等、理解ではなく定着に特化したサービスの成長
5. **ChatGPT等生成AI**: 無料または低価格のAI学習支援が普及

### スタディサプリの市場ポジション
- **学校向け**: 3,000高校導入で圧倒的シェア
- **個人向け**: ベーシックコース（月額2,178円）で高い認知度
- **塾向け**: 低調、継続活用事例がごくわずか

## 3. 技術・製品詳細

### サービスアーキテクチャ
- **映像配信プラットフォーム**: 既存のスタディサプリインフラを活用
- **テスト・評価機能**: 単元テストと講義動画の紐付け
- **進捗管理ダッシュボード**: 塾向けの管理画面
- **マルチデバイス対応**: PC・タブレット・スマートフォン

### 技術的特徴
- **カリスマ講師の映像授業**: プロ講師による高品質なコンテンツ
- **小1〜高3の全範囲**: 幅広い学年・教科をカバー
- **単元テスト連携**: テスト結果から弱点を自動抽出、対応講義を推薦
- **スマホ最適化**: 板書が見やすい設計

### 技術的課題（推定）
- **カスタマイズ性の欠如**: 塾ごとの指導スタイル・カリキュラムに柔軟に対応できない
- **汎用的すぎる設計**: 塾独自の教材・カリキュラムとの統合が困難
- **AI機能の不足**: atama+のような高度なAI個別最適化がない
- **記憶定着機能の欠如**: Monoxerのような反復学習・暗記支援が弱い

## 4. 成長・拡大戦略

### 歴史的展開
1. **2011年**: 受験サプリ（スタディサプリの前身）立ち上げ
2. **2015年頃**: スタディサプリに改称、学校向け展開開始
3. **2022年3月**: 学習塾向けサービス提供開始
4. **2023年頃**: 中核メンバー（山口文洋氏等）が相次いで独立・退職
5. **2025年3月**: 個別指導塾オンライン中学講座終了
6. **2025年9月1日**: 学習塾向けサービス終了発表
7. **2026年3月30日**: サービス完全終了予定

### 想定された成長戦略（実現せず）
- **学校向けの成功を塾向けに展開**: 3,000高校の実績を塾市場に活かす
- **小規模塾からの採用拡大**: 生徒1人から利用可能な低価格設定で浸透
- **大手塾への導入**: ブランド力で大手塾を開拓
- **学年・教科の網羅性**: 小1〜高3の全範囲で塾の補助教材として定着

### 実際の状況
- **継続活用事例はごくわずか**: 塾での定着に失敗
- **カスタマイズ性不足**: 塾ごとのニーズに応えられず
- **中核メンバー離脱**: 推進力を失い、市場存在感が急速に低下
- **4年で撤退**: 成長戦略が機能せず、早期撤退

## 5. 事業パフォーマンス

### 定量データ
- **サービス期間**: 約4年（2022年3月〜2026年3月）
- **導入塾数**: 公開データなし（少数と推定）
- **継続利用塾**: 「ごくわずか」と報道
- **競合比較**:
  - atama+: 大手塾の30%以上導入
  - Monoxer: 2,500教室以上導入
  - スタディサプリ塾向け: 具体的数値なし（大幅に劣ると推定）

### 定性的評価
- **市場浸透度**: 極めて限定的
- **ブランド認知**: 学校向け・個人向けは高いが、塾向けは低い
- **顧客満足度**: 継続活用が少ないことから低いと推定
- **収益性**: 公開データなし（赤字または低収益と推定）

### 推定される事業課題
1. **採用塾の少なさ**: 新規導入が進まなかった
2. **高い解約率**: 導入しても継続利用されない
3. **カスタマイズ要望への対応不能**: 塾の個別ニーズに応えられず
4. **中核メンバー不在**: プロダクト改善・営業推進の牽引力喪失

## 6. 撤退・失敗分析

### 公式発表の撤退理由
リクルートは2025年9月1日、「事業環境の変化やサービスの利用状況などを総合的に判断し、提供を終了する」と発表しました。具体的な失敗要因については明示されていません。

### 詳細な失敗要因分析

#### 1. 製品設計の致命的欠陥

**カスタマイズ性の欠如**
- サービスがもともとカスタマイズ性に乏しい設計
- 塾ごとの指導スタイル・カリキュラムに柔軟に対応できない
- 汎用的な映像授業とテストでは、塾独自の価値を提供できない
- 「学校向け」「個人向け」の設計を塾向けに転用しただけで、塾特有のニーズを理解していなかった

**塾のビジネスモデルとの不整合**
- 塾は「講師の指導力」「独自カリキュラム」で差別化している
- スタディサプリの汎用的な映像授業は、塾の差別化を阻害
- 塾は「スタディサプリを使えば塾に来なくても良いのでは？」という矛盾に直面
- 塾の価値を高めるのではなく、塾の存在意義を脅かす構造

**継続活用の困難さ**
- 実際に塾で継続的に活用できていた事例はごくわずか
- 導入しても、塾のカリキュラムに組み込めず、使われなくなる
- 塾講師が「自分で教えた方が早い」と判断し、活用されない

#### 2. 中核メンバー離脱による推進力喪失

**キーパーソンの連続退職**
数年前に、スタディサプリの開発・営業の中核メンバーが相次いで独立し、教育スタートアップを起業しました。

**山口文洋氏（創業者）の離脱**
- 2011年に受験サプリ（現スタディサプリ）を立ち上げ
- 2018年に株式会社リクルート執行役員に就任
- 2022年4月に株式会社LITALICO副社長執行役員として転職
- 2023年4月にLITALICO代表取締役社長に就任
- スタディサプリの生みの親が不在に

**竹内孝太朗氏の独立**
- 2013年からスタディサプリで高校向け営業組織立ち上げ
- 学習到達度測定テスト、オンラインコーチングサービスを開発
- 退職後、Monoxer（モノグサ）を共同創業
- Monoxerは2,500教室以上の塾に導入され、競合として成長

**影響**
- プロダクト改善の牽引力喪失
- 営業推進力の低下
- 市場における存在感が急速に低下
- 「人がいなくなると伸び悩む」リクルートでも避けられない現象

#### 3. 競合との差別化失敗

**atama+のAI個別最適化に劣る**
- atama+は生徒の理解度・弱点・集中力をAI分析
- 一人ひとりに最適化されたカリキュラムを自動生成
- スタディサプリは汎用的な映像授業を提供するのみ
- AI時代の個別最適化競争で完敗

**Monoxerの記憶定着機能がない**
- Monoxerは記憶定着に特化し、反復学習を支援
- スタディサプリは「理解」に留まり、「定着」が弱い
- 塾は「定着」も重視するため、Monoxerと併用される
- スタディサプリ単体では価値不足

**Classiの学校連携に及ばず**
- Classiはベネッセの進研模試と連携
- 学校の成績管理・クラス運営と統合
- スタディサプリは塾向けの管理機能が弱い

#### 4. BtoB市場の理解不足

**学校向けの成功が塾向けに転用できなかった**
- 学校向け: 教育委員会・校長の一括導入、講師の負担軽減が評価
- 塾向け: 塾長・塾講師が自らの価値を守るため、導入に消極的
- 顧客の購買動機が全く異なる

**BtoB営業の難しさ**
- 学校は「教育の質向上」が目的で、コスト意識が比較的低い
- 塾は「収益性」が最優先で、ROIをシビアに評価
- 月額課金が塾の収益を圧迫する構造
- 塾は「生徒に直接契約させる」方が合理的と判断

**カスタマーサクセスの不在**
- 導入後のフォロー・活用支援が不十分
- 塾が継続的に活用するための伴走支援がなかった
- BtoB事業では、導入後の成功支援が生命線

#### 5. リクルートの戦略転換

**学校・自治体への集中戦略**
- 日本の高校の約半数（3,000校）がスタディサプリを導入
- BtoB市場としては学校・自治体の方が安定・大規模
- 塾向けは撤退し、学校向けにリソース集中

**個人向けは継続**
- ベーシックコース（月額2,178円）は継続
- 個人向けは年間194万人の有料会員で安定収益
- 塾向けのような複雑な対応が不要

**リクルート撤退基準の適用**
- 「3年黒字、5年累損解消では遅すぎる」
- 4年間で事業性が見込めないと判断
- 中核メンバー不在で改善見込みなし
- 優秀な人材を他事業に振り向ける判断

### Lessons Learned（失敗から学ぶ）

#### 1. 製品のカスタマイズ性の重要性（BtoB）

**教訓**: BtoB製品は顧客ごとのカスタマイズが必須
- 汎用的な製品を「学校向け」「塾向け」に転用しても成功しない
- 塾は独自のカリキュラム・指導スタイルで差別化している
- それを尊重し、柔軟にカスタマイズできる設計が必要

**Orchestrate-Phase1への示唆**:
- BtoB製品は「1つの正解」ではなく「顧客ごとの最適解」を提供
- カスタマイズ可能な設計を最初から組み込む
- 顧客のビジネスモデルを深く理解し、それを強化する製品設計
- 「使ってもらう」だけでなく「成功してもらう」設計

#### 2. 顧客のビジネスモデルとの整合性

**教訓**: 顧客のビジネスを脅かす製品は採用されない
- 塾は「講師の指導力」で差別化している
- スタディサプリの高品質な映像授業は、塾の価値を相対的に下げる
- 「スタディサプリがあれば塾不要」という矛盾が発生
- 顧客の存在意義を高める製品でなければ継続利用されない

**Orchestrate-Phase1への示唆**:
- 顧客のビジネスモデルを理解し、それを強化する製品
- 「代替」ではなく「補完」の関係を構築
- 顧客が「自分の価値が高まる」と実感できる設計
- Win-Winの関係を作れない製品は淘汰される

#### 3. キーパーソン依存のリスク

**教訓**: 中核メンバーが抜けると事業が止まる
- 山口文洋氏（創業者）、竹内孝太朗氏（営業・開発リーダー）等が退職
- プロダクト改善・営業推進の牽引力が喪失
- リクルートのような大企業でも、人材流出で事業が停滞
- 「人」がいなければ事業は止まる

**Orchestrate-Phase1への示唆**:
- 特定個人への依存度を下げる組織設計
- ノウハウ・知見の組織的蓄積（属人化回避）
- 後継者育成・チーム体制の構築
- キーパーソン流出のリスクマネジメント
- ただし、優秀な人材の確保・維持が最優先

#### 4. BtoC成功がBtoB成功を保証しない

**教訓**: 学校向け・個人向けの成功が、塾向けに転用できなかった
- BtoC（個人向け）: 年間194万人の有料会員
- BtoG（学校向け）: 3,000高校導入
- BtoB（塾向け）: 継続活用事例がごくわずか
- 顧客セグメントが異なれば、成功パターンも全く異なる

**Orchestrate-Phase1への示唆**:
- BtoC/BtoG/BtoBの顧客特性を理解
- 成功事例の安易な横展開は危険
- 新セグメントには新たな価値提案が必要
- 市場ごとに製品・営業戦略を再設計

#### 5. 競合との差別化と補完関係

**教訓**: atama+（AI個別最適化）、Monoxer（記憶定着）に敗北
- スタディサプリは「理解」に特化
- atama+は「AI個別最適化」で差別化
- Monoxerは「記憶定着」で差別化
- スタディサプリの汎用的な映像授業は差別化不足

**Orchestrate-Phase1への示唆**:
- 競合との明確な差別化ポイントを確立
- 「何でもできる」より「これが圧倒的」を目指す
- 補完関係を築ける競合とは協業も検討
- 差別化なき競争は価格競争に陥る

#### 6. カスタマーサクセスの重要性（BtoB）

**教訓**: 導入後のフォロー・活用支援が不足
- 塾に導入しても、継続活用されない
- 塾講師への研修・活用支援が不十分
- 「売って終わり」では解約される

**Orchestrate-Phase1への示唆**:
- BtoBはカスタマーサクセスが生命線
- 導入後の伴走支援体制を構築
- 顧客が成果を出すまでフォロー
- 継続率（Retention）がKPI
- LTV（顧客生涯価値）を最大化する設計

#### 7. リクルート撤退基準の実践（4年撤退）

**教訓**: 4年間で成長が見込めないと判断し撤退
- 通常、BtoB事業は5〜10年で判断されることが多い
- リクルートは4年で「継続活用がごくわずか」と判断
- 中核メンバー不在で改善見込みなし
- 早期撤退でリソースを学校向けに集中

**Orchestrate-Phase1への示唆**:
- BtoB事業でも、成長が見えなければ撤退
- 「もう少し頑張れば」は沈没コスト
- 人材リソースのROIを常に評価
- 撤退は失敗ではなく、戦略的選択
- 学校向けへの集中は正しい判断
