## 1. 市場分析ルール

- [ ] 市場規模は TAM（総市場）、SAM（対象市場）、SOM（獲得可能市場）の 3 層で定量化せよ

- [ ] 市場成長率（CAGR）を必ず記載せよ
- [ ] メガトレンド（10年スパン）を1つ以上特定せよ

## 3. 課題定義ルール

- [ ] 課題は 3U+1（Unworkable, Unavoidable, Urgent, Underserved）で検証可能な形式で記述せよ

- [ ] 既存の代替案を最低3つ列挙せよ

## 4. ソリューションルール

- [ ] UVP（Unique Value Proposition）を一文で表現せよ

## 5. 競合・差別化ルール
