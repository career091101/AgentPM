## 使い方

1. **テーマ別に記事を探す**: `Pattern_A_Themes/` 配下の各テーマフォルダを参照
2. **記事リストを確認**: 各テーマの `article_list.md` で全記事を閲覧
3. **統計情報を確認**: 各テーマの `README.md` で概要と統計を確認

## 分析手法

- **自動分類**: キーワードマッチングによる記事の自動分類
- **複数テーマ**: 1つの記事が複数のテーマに分類される場合あり
- **時系列追跡**: 年別の記事数推移を可視化
