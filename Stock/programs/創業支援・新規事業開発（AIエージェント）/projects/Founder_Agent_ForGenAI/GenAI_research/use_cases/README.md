# Use Cases - ユースケース別インデックス

このディレクトリには、AI/GenAIの活用ユースケース別に分類されたトランスクリプトへのシンボリックリンクを格納します。

## カテゴリ一覧

### startup_support/ - スタートアップ支援
- アイデア検証
- プロダクト開発支援
- ピッチ資料作成
- 市場調査支援

### automation/ - 業務自動化
- タスク自動化
- ワークフロー最適化
- RPAとの連携
- 業務効率化事例

### customer_support/ - カスタマーサポート
- チャットボット構築
- FAQ自動応答
- 問い合わせ対応支援
- サポート品質向上

## 使用方法

各サブディレクトリには、該当ユースケースを扱っているトランスクリプトへのシンボリックリンクが配置されます。

例：
```bash
cd use_cases/startup_support/
ls -lah  # スタートアップ支援関連の全トランスクリプトを表示
```

## 更新方法

T005-5タスクで自動分類スクリプト（`scripts/t005_5_classify_transcripts.py`）により、メタデータのuse_casesに基づいてシンボリックリンクが自動生成されます。
