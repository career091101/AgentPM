<h2>トランプ大統領の就任式</h2>
<p>先週の月曜日、トランプ大統領の就任式が開かれました。</p>
<p>就任後いきなり、数多くの大統領命令に署名をしました。目立つものとしては、以下のようなものがあります。</p>
<ul>

<li>連邦政府の従業員のリモートワークの停止</li>

<li>TikTokの停止命令を75日間延期</li>

<p>興味深いのはリモートワークの停止です。コロナ禍で、多くの職場がリモートワークを許すようになり、連邦政府もその例外ではありません。</p>
<p>しかし、その結果として、週3日しか働かない人なども増え、全体としての生産性が落ちていることが指摘されています。</p>
<p>連邦政府の従業員は法律で守られているため、一方的な解雇は出来ませんが、まずはリモートワークを禁止することにより、真面目に働かない人は自分から辞めてもらうことを狙っているのです。</p>
<p>関税に関しては、初日からのスタートはしませんでしたが、トランプ氏は、2月1日には、メキシコとカナダ向けの２５％関税をスタートすると宣言しており、その影響が心配です。トランプ氏は、「インフレとの戦い」を重要視していますが、メキシコとカナダからの輸入品の値段が25%も上がれば、物価への影響は避けられません。</p>
<p>また、麻薬との戦いに関しては、メキシコに特殊部隊を派遣して、麻薬密売組織をテロリストとして扱うと宣言しました。現在、米国には中国で製造されたオピオイド系のドラッグが、メキシコ経由で大量に流入し、大きな社会問題を起こしていますが、それを「テロ行為」扱いするのです。</p>

<h2>Stargateプロジェクト</h2>
<p>トランプ氏が合衆国大統領に就任して二日目、Stargateプロジェクトをアナウンスしました。</p>
<p>ソフトバンク、Oracle、OpenAI, MGXの４社が合弁会社Stargateを作り（孫正義さんが会長）、そこがここから4年間で$500billion(約７５兆円)を投じて、AIインフラを作るそうです。</p>
<p>まずは$100billionの投資からスタートするそうですが、$100billionのAIインフラごとに5GWの電力(大型の原発4～5基が必要)が必要です。</p>
<p>ソフトバンクは既に$100billionを米国に投資することをトランプ氏に約束しているし、Oracleは、OpenAI向けのデータセンターを構築することを約束しているので、その手の既存の話を一つにまとめて、より大きな話としてアナウンスしたと見て良いと思います。</p>
<p>ソフトバンクは、主に運営資金の担当で、OpenAIがAIとオペレーション、Oracleがインフラを担当し、そこに、（ソフトバンク傘下の）ARM、(GPUを提供する)NVIDIA、(OpenAIのパートナーである)Microsoftもパートナーとして参加するとのことです。</p>
<p>これを受けて、ARMやOracleの株が大きく上昇しました。</p>
<p>ちなみに、Stargateプロジェクトのアナウンスに招待されたOracleのLarry Ellison氏は、AIが癌の治療に革命をもたらすことを強調した上で、特定の癌を攻撃するmRNAワクチンについて言及していたのが印象的でした。</p>

<h2>DeepSeek-R1 is here!</h2>
<p>先日紹介した、中国のDeepSeekが、今度はOpenAIのo1に匹敵する「考える力」を持った人工知能、DeepSeek-R1を発表しました。</p>
<ul>
<li><a href="https://x.com/deepseek_ai/status/1881318130334814301">DeepSeek-R1 is here!</a></li>
</ul>
<p>各種のベンチマークで、o1と同等かそれ以上のスコアを上げています。</p>
<p><img style="width:380px" src="https://satoshi.blogs.com/mag2/Jan2025/DeepSeekR1a.png" alt=""></p>
<p>にも関わらず、APIのコストは、OpenAIの o1, o1-mini と比べて桁違いに安いのが特徴です。</p>
<p><img style="width:380px" src="https://satoshi.blogs.com/mag2/Jan2025/DeepSeekR1b.jpeg" alt=""></p>
<p>オープンなはずのOpenAIが、パラメータ数も発表せずにクローズドな形でGPT4o、o1などの人工知能を発表しているに対して、中国のベンチャーが全てオープンな形で次々に優秀な人工知能を発表している点が、なんとも皮肉だと指摘する人もいます。</p>
<p>仕組みに関しても、<a href="https://github.com/deepseek-ai/DeepSeek-R1/blob/main/DeepSeek_R1.pdf">DeepSeek-R1: Incentivizing Reasoning Capability in LLMs via Reinforcement Learning</a>という論文で詳しく紹介しており、AI業界全体に対する寄与もとても高く評価できます。</p>
<p>パソコンでも動くように小さくしたモデルも既に公開されており、LM Studio や Ollama を使って簡単に試すことが出来ます。私も、70Bモデルと32Bモデルの両方をOllamaを使って私のMacBook Pro(96GB メモリ)で試してみました。70Bモデルは少し遅いのですが、32Bモデルはとても快適に動きます。有名な「単語、Strawberryの中に&quot;r&quot;はいくつ入っているか」も問題なく解けます。</p>
<p>「考える力」を持ったo1をOpenAIが発表した時は、多くの人が驚くとともに、APIのコストがとても高くなることが懸念でしたが、わずか３ヶ月後に、それと匹敵するレベルのものがオープンソースでリリースされてしまったことは、素晴らしいとしか言いようがありません。</p>
<p>少し前に、このメルマガで「AIの進歩により、知的労働のコストが限りなくゼロに近づく」と言いましたが、まさにその現象が私たちの目の前で起こっているのです。</p>
<p>ちなみに、DeepSeekを作ったのは、High-Flyerという名の中国のヘッジファンドです。ヘッジファンドとは言え、設立したのは浙江大学（Zhejiang University）を卒業したエンジニアたちで、AIの力を使って株の取引をして利鞘を稼ぐことを目指して作られたそうです（設立は2019年）。</p>
<p>DeepSeekは、High-Flyerの子会社として2023年に設立されましたが、外部からは資金は調達しておらず、全ての運営資金はHigh-Flyerから提供されています。</p>
<p>High-Flyerは、2021年に１万台のNvidia A100 GPUから構成されるAIスーパーコンピュータを構築し、その上で動くAIで株の取引を目論んでいましたが、うまく行かなかったようです。経緯から考えると、その資産をDeepSeekで活用していると私は解釈しています。</p>
<p>DeepSeekは、2024年5月にDeepSeek-V2を発表しましたが、その際のAPIの価格設定が桁違いに安かったため、ByteDance、Tencent、Baidu、Alibabaなどのライバル企業は、価格を大幅に下げなければならなくなりました。</p>

<h2>人間と同じ一時記憶を持つ人工知能</h2>
<p>Google Deepmind の研究者が人間の脳の仕組みを参考にして、（与えられたプロンプトのことを覚える）短期記憶、（与えられた一連のプロンプトの中で覚える価値のあるものを記憶する）長期記憶、（学習プロセスで覚えた）持続的記憶の３つのレイヤーからなるニューラルネットを作ることにより、従来よりも長いプロンプトを与えられても、しっかりとした性能を発揮するLLM（大規模言語モデル）を作ることに成功したことを報告する論文を発表して、注目を集めています。</p>
<p>論文は、<a href="https://arxiv.org/abs/2501.00663">Titans: Learning to Memorize at Test Time</a>というタイトルで、Transformerのベースとなった論文、<a href="https://arxiv.org/abs/1706.03762">Attention Is All You Need</a>並みの影響を、今後の人工知能の研究に与える可能性があります。</p>
<p>興味深いのは、「何を記憶するか」を決める段階では「驚き」を重視していることで、これも人間の普段の反応を参考にしているそうです。例えば、通学路で起こる出来事がいつもと同じだと、何も記憶に残りませんが、いつもと違うこと（例えば、「事故」や「地震」）が起こると、それが記憶に残るのと同じです。</p>
<p>また、セッションをまたいだ記憶を保持する長期記憶に関しても、「時間と共に忘れる」仕組みを取り入れることにより、メモリー容量が限られた中で、必要なことだけ覚えておく仕組みとして活用しているそうです。</p>

<h2>賭けゴルフと数字のマジック</h2>
<p>以前にも目撃したことがあって変だと思っていたのですが、今回、全く同じ間違いに遭遇したので、説明したいと思います。実際のお金が関わる、とても重要な問題だし、数字の難しさを表す話でもあります。悪用すれば、詐欺も可能です。</p>
<p>問題が生じたのは、ゴルフにおける「賭け」です。「賭けゴルフ」と聞くと、「違法行為だ！」と条件反射をしてしまう人がいると思いますが、米国では、プレーヤー間の少額の「賭けゴルフ」は合法で、私も含めて、多くのゴルファーが日常的に行っています。</p>
<p>日本では「賭けゴルフ」どころか「賭け麻雀」すら法律上は違法ですが、実際には、多くの人たちが楽しんでおり、たまに野球選手とかが暴力団が絡む「賭け麻雀」をしたことが判明して、メディアに叩かれます（これは、単に警察が目をつけた暴力団を取り締まるためだけに利用されているだけであり、本来の法律のあり方とは違うと思いますが、その話をすると長くなるので、別の時に書きます）。</p>
<p>よく行われるのが、「ジャンク」と呼ばれるゲームで、バーディ（パーよりも一つ少ないスコアで上がること）やサンディ（バンカーに入りながらもパーを取ること）などに１ドルの賞金をつけることです。</p>
<p>４人でプレーしていて、誰かがバーディを取ると、他の３人がバーディを取った人に一ドル支払うことになります。</p>
<p>プレー中に支払うのは面倒なので、それぞれにプレーヤーが「ジャンク（ポイント）」をいくつ取ったかを記録しておき、１８ホールのプレーが終わった後に、バーで精算をします。</p>
<p>例えば、A、B、Cの３人のプレーヤーがいた場合（通常、ゴルフは４人でやりますが、計算しやすいように３人で行います）、それぞれのプレーヤーが、以下のポイントを稼いだとします。</p>
<ul>
<li>A: 40</li>
<li>B: 28</li>
<li>C: 20</li>
</ul>
<p>精算方法はいくつかありますが、一般的に使われるのが、それぞれのプレーヤーがポイントの差だけお金を払う手法です。</p>
<ul>
<li>A - B = 12, BからAに$12の支払い</li>
<li>A - C = 20, BからAに$20の支払い</li>
<li>B - C = 8, CからAに$8の支払い</li>
</ul>
<p>ポイントを１つ稼ぐために、それぞれのプレーヤーから$1もらえる決まりなので、精算の際には、その差を計算すれば良いのです。</p>
<p>ジャンクは、プレーヤーの数が２人、３人、４人のいずれでも可能ですが、いずれの場合においても、それぞれのポイントの差額を支払うことにより精算が可能です。</p>
<p>ゴルフにはジャンク以外にも色々なゲームがありますが、３人でプレーする場合に行われるのが、「９ポイント」と呼ばれるゲームです。</p>
<p>それぞれのホールに９ポイント（９ドル）が割り当てられており、それを３人のプレーヤーで奪い合うのです。</p>
<ul>
<li>三人が同点だった場合、それぞれが３ポイントづつ受け取る。</li>
<li>一人だけ良いスコアだった場合、その人が５点、他の二人は２点づつ受け取る。</li>
<li>二人が良いスコアだった場合、二人が４点づつ、残りの一人が一点を受け取る。</li>
<li>三人バラバラのスコアだった場合、上から、５点、３点、１点を受け取る。</li>
</ul>
<p>プレー中はそれぞれのプレーヤーが受け取った点を記録しておき、最後に精算します。</p>
<p>例えば、それぞれのプレーヤーが、以下のポイントを稼いだとします。</p>
<ul>
<li>A: 70</li>
<li>B: 50</li>
<li>C: 42</li>
</ul>
<p>このゲームの場合、ポイントのトータルは常に162ポイントになります（9 x 18）。</p>
<p>９ポイントゲームの精算は、ジャンクと異なり、それぞれのプレーヤーが、平均の54点に対して、何点稼いだかの差額で、決めます。</p>
<ul>
<li>A: 70 - 54 = 16、$16受け取る</li>
<li>B: 50 - 54 = -4、$4支払う</li>
<li>C: 42 - 54 = -12、$12支払う</li>
</ul>
<p>この精算方法が正しいのは、それぞれが１ホールあたり$3ドル、トータルで$54を参加料として提供し、１ホールあたり$9の賞金を奪い合うゲームだからです。全部で70ポイント(=$70)を稼いだAは、参加料$54を差し引いた$16を受け取るのです。</p>
<p>私から見れば、とても明らかですが、数学が弱い人は、しばしば、9ポイントゲームにおいても、ジャンクと同様に、プレーヤーのポイントの差を計算して、精算しようとします。すると、</p>
<ul>
<li>A - B = 20, BからAに$20の支払い</li>
<li>A - C = 28, BからAに$28の支払い</li>
<li>B - C = 8, CからAに$8の支払い</li>
</ul>
<p>となってしまい、トータルでは、</p>
<ul>
<li>A: 20 + 28 = 48、$48受け取る</li>
<li>B: 20 - 8 = 12, $12支払い</li>
<li>C: 28 + 8 = 36, $36支払い</li>
</ul>
<p>と、本来の３倍の支払い・受け取り、をすることになってしまいます。</p>
<p>よく考えれば明らかな間違いなのですが、ジャンクゲームの方が幅広くされており、９ポイントゲームは稀にしか行われないため、９ポイントゲームにおいても、ジャンクゲームと同様にそれぞれのプレーヤーのポイント数を使って精算をしてしまう間違いが生じるのです。</p>
<p>先日も、この間違った方法で９ポイントゲームの精算をしようとする人がいたので、間違いを指摘しました。その人は、ちゃんと大学（建築科）も出てまっとうな仕事をして来た人ですが、ゲームによって精算の仕方に違いがあることを知らず、これまでずっと、９ポイントゲームにおいても、ジャンクと同じ手法で精算して来たそうです。</p>
<p>ちなみに、この「９ポイントゲームの精算方法が分かりにくいい」という事実を利用して、お金を稼ぐことが可能なので注意が必要です。</p>
<p>９ポイントゲームをし、自分が54ポイントより多く稼いだ時には、ジャンクゲームと同様の間違った精算方法で、本来受け取るべきよりも３倍のお金を受け取り、自分が54ポイント以下しか稼げなかった時には正しい方法で精算すれば、長い目では確実に稼ぐことが出来ます。</p>
<p>この手法は、明らかな詐欺行為ですが、とても分かりにくく（IQ120ぐらいの人までが簡単に引っかかってしまうと思います）、とても厄介です。</p>
<p>普段から親しく遊んでいる友達と少額の賭け事をすることを悪いとは思いませんが、初対面の人と「賭け事」をする場合、こんな「落とし穴」があるので注意が必要です。このケースは、ゲームそのものはフェアに作られているのに、精算方法の部分に「数字のマジック」の落とし穴があります。</p>

<p>ただ、どうしてもタレントマネジメント領域ではタレントに比例する形で従業員を増やす労働集約的なモデルになってしまい、案外AIが入りづらい領域なのかなと思っているのですが何か意見があればお願いします。</p>
<h3>《回答》</h3>
<p>確かに、タレントマネージメントのような領域は自動化しにくいのは分かります。とは言え、タレントのマネージャーがやっていることの中には、無駄だったり、自動化が可能なこともたくさんあると思うので、まずはそこから手をつけて、彼らの生産性を上げるところかは始めるのも悪くないかも知れません。</p>
<p>しかし、本格的にAI-nativeなアプローチを考えるのであれば、インフルエンサーのあり方から根本的に考え直す必要があるようにも思います。極端な話、バーチャル・インフルエンサーをAIを作って複数作り、彼らに人間のインフルエンサーが作るようなコンテンツを作らせ、競争させ、自然淘汰の仕組みで、その中からスターを育てて行くようなビジネスモデルです。</p>
<h3>【質問】</h3>
<p>ちなみに、米国政府がなぜTikTokにこれほど神経質になるかは不明ですが、超党派でこの政策が決まったことからして、議員たちにしか公開されていない、何らかの（TikTokが中国政府によるスパイ行為が行われているという）証拠があるのだと想像できます。</p>
<p>私は最近のTikTokの争いはよくあるアメリカの嫌がらせ、「大量破壊兵器」のようなものと思って報道を見ていました。中国に対してのアメリカの理不尽な圧力は例を挙げたらキリがないと思います。ならず者国家は双方でお互いに思っているかと思います。</p>
<p>中島様にしては珍しく踏み込んだ憶測に思いましたが、何かここまで書かれるに至った思いがあるのでしょうか。</p>
<p>わたしは親中でもないのですが、無闇に中国は悪く書かれすぎていると思っていますし、ロシアに関しては実際にロシアに今も頻繁に行き来している人の話を聞くと、報道とあまりに違う実態に驚いています。当たり前に現在も偏向報道はあることを実感しています。</p>
<p>中島様へのクレームではなく、逆に私が中国を是とする情報に偏っているかもしれないので、質問させていただいております。いかがでしょうか。</p>
<h3>《回答》</h3>
<p>TikTokに関して、特別な情報を得ているわけではありませんが、今回の超党派な措置に強い違和感を感じたので、それを表明したのです。</p>
<p>TikTokが閉鎖させるというニュースを見て、多くのインフルエンサーが別のサービスにもアカウントを作って動画の投稿を始めたそうですが、その運営元も中国の会社だそうです。</p>
<p>少し誤解されているようですが、私は特に中国に対して悪いイメージは持っていないし、悪口を書いたこともありません。米国で暮らし、大半の資産を米ドルベースの株や不動産として持っている私としては、米国の経済が順調に成長することを望んでおり、米中間の経済摩擦はもちろん、台湾有事などとても迷惑だと感じています。</p>
<p>別の言い方をすれば、私が米国よりの発言をするのは、スポーツの試合で地元のチームを応援するのと似ています。相手のチームが憎いわけでも嫌いなわけでもありませんが、地元のチームには勝って欲しくて応援しているだけです。</p>
<h3>【質問】</h3>
<p>中島さんの「なぜ、あなたの仕事は終わらないのか」読ませてもらい、一つだけ気になったことがあります。タクスリストを作成する時に15分程で終わるタクスに分けて書くと書かれていましたが、実際プログラムを書いている際なども細かく「ここのコードを書く所まで終わらせる」等の目標を設定しているのでしょうか？細かくタクスを分けて書く事が大変なのですが具体的にどのように書いてるか教えてほしいです。</p>
<h3>《回答》</h3>
<p>はい。「この関数を実装する」「このバグを取る」「この関数をリファクタリングする」など可能な限り仕事を細かく分割し、その度、テストを実行し、ちゃんと動くことを確認した上で、（Github上のリポジトリに）マージするようにしています。</p>
<p>「その度、テストを実行し」という部分がとても重要で、山を登る際に、一歩一歩、足元の土を踏み固めて行くイメージで開発します。</p>
<p>ちなみに、ある作業をしていて、「思ったように動かない」「想定もしていなかったバグが発生する」などの状況で「煮詰まった」と感じた時には、一度、すべてを捨てて（作業していたブランチを消してしまいます）、フレッシュな気持ちで、ゼロからやり直します。</p>
<p>一塊の作業が大きいと、この「うまく動かないときは、一度捨てて、ゼロからやり直す」ことが苦痛になるため、可能な限り、一塊の作業を小さく分解するのです。</p>
<p>遠回りに感じるかも知れませんが、この手法を採用して以来、私の生産性は爆上げしたし、思うように動かない時のストレスも無くなりました。是非、試してください。</p>
<h3>【質問】</h3>
<p>現在の日本の財政状況につきまして、巷では「財政危機は目前である」という悲観的な意見もあれば、「財政赤字は問題ない」とする楽観的な意見もあり、一体どちらが正しいのか、一般の者には判断が難しい状況です。</p>
<p>つきましては、中島さんのご見解を伺いたく、以下の点についてご教示いただけますでしょうか。</p>
<p>1.現在の日本の財政状況をどのように認識されていますか？
2.「財政危機」という言葉が頻繁に使われていますが、先生は現状を危機的な状況だとお考えでしょうか？
3.そのように判断される理由、根拠となるデータや指標などがあれば、併せてご教示ください。
4.今後、日本が取り組むべき方向性についてお考えをお聞かせください。</p>
<h3>《回答》</h3>
<p>財政危機に関しては、専門家でも言うことが様々なのでとても分かりにくいですが、私なりの理解は以下の通りです。</p>
<ol>
<li>日本のように、円建てで国債を発行している国の場合、借金が返せなくなって破綻することはない。</li>
<li>しかし、今のように大量に赤字国債を発行し続けていると、円の価値は下がり、インフレになる。</li>
<li>しかし、円相場や物価は他の要因でも動くため、必ずしも、リアルタイムで連動しない点が悩ましい。</li>
<li>とは言え、中有長期的には、円安・物価高に動くことは確実。</li>
<li>インフレを抑えるために金利を上げると、国債を発行する際の金利も上昇し、赤字国債が雪だるま式に増えてしまう。</li>
<li>その時に市場がパニックを起こすと、ハイパーインフレが起こり、円の価値が暴落する。</li>
<li>財務省はそれを嫌って、財政赤字を減らすべきだと考えている。</li>
<li>政治家は、緊縮財政は票を減らすので出来ない。</li>
<li>ある意味、ハイパーインフレが起こるまでのチキンゲームをしていると言える。</li>
<li>ハイパーインフレは、大きな痛みを伴うが、結果として、国の過去の借金が、実質、帳消しに出来てしまうから、それで良いと考えている人もいる。</li>
</ol>
