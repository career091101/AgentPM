# プロジェクト憲章: ideal_partner_matching

## 1. プロジェクト概要

### 1.1 プロジェクト名
ideal_partner_matching（理想的パートナー発見プロジェクト）

### 1.2 プロジェクトの目的
30-40代の真剣交際希望者に向け、真剣度スコアリング × AI婚活コーチ × 価値観マッチングを統合したプラットフォームを構築し、6ヶ月以内の真剣交際実現を支援する。

### 1.3 ビジネス価値
- **社会的価値**: 日本の未婚率上昇・出生率低下の一因となっている「良質な出会い不足」を解消
- **経済的価値**: Year 3で ARR 3.3億円、50,000アクティブユーザー達成見込み
- **顧客価値**: 時間効率20倍、使いやすさ30倍の圧倒的な婚活体験を提供

## 2. プロジェクトの背景

### 2.1 市場背景
2025年現在、婚活市場は約1,000億円規模に成長しているが、多くのサービスが「マッチング数の最大化」に注力し、「真剣交際への移行」という本質的な成果を軽視している。

| 指標 | 数値 |
|------|------|
| 日本の婚活市場規模（TAM） | 1,080億円 |
| オンライン婚活市場（SAM） | 300億円 |
| 30-40代独身者数 | 約1,000万人 |
| 既存サービスの真剣交際移行率 | 10-15%（推定） |

### 2.2 解決すべき課題
需要発見リサーチ（スコア18/20点）により、以下3つの主要課題が特定された：

1. **真剣度の低い人との無駄なやりとり**: マッチングアプリは「hookup目的」「wishy-washy」な人が多く、デートが「chore（雑用）」化
2. **条件にこだわりすぎて相手が見つからない**: スペック至上主義・減点法思考で誰も選べない
3. **自分の気持ちを伝えるスキル不足**: Red flagの見極めやコミュニケーションスキルが不足

### 2.3 市場機会
- **マーケットギャップ**: 中価格帯（¥3,980-7,980/月）で真剣度高＋AI支援のサービスが不在
- **CPFスコア**: 75%達成（目標60%以上クリア）
- **PSFスコア**: 3軸で10倍優位性達成（時間20倍、使いやすさ30倍、導入障壁5倍）

- **緊急停止**: 重大なセキュリティインシデント・倫理違反発生時のサービス停止判断

### 11.4 継続的モニタリング

- Phase2以降の実行プロセスにおけるKPI追跡（リテンション率、NPS、真剣交際移行率）
- ユーザーフィードバックの定期収集と対応（週次）
- AI婚活コーチの会話ログレビュー（週次）
- 真剣度スコアリングの精度検証（月次）

## 12. Values（行動指針）

本プロジェクトは、以下の5つのValuesに基づいて運営される：

| Value | 説明 | 行動例 |
|-------|------|--------|
| **Authenticity（本物）** | 偽りのない真剣な出会いを提供する | 真剣交際移行率を正確に開示、プロフィールの虚偽記載を厳格に取り締まり |
| **Empowerment（成長支援）** | AI婚活コーチによる継続的な自己成長をサポート | 失敗（交際終了）も学びの機会と捉え、振り返り支援を提供 |
| **Respect（尊重）** | すべてのユーザーの価値観・人生観を尊重する | ゴーストング、breadcrumbingなど相手を尊重しない行動に警告・ペナルティ |
| **Innovation（革新）** | AI技術を活用した新しい婚活体験を創造する | 月1回の新機能リリース（A/Bテストで効果測定） |
| **Commitment（コミットメント）** | 6ヶ月以内の真剣交際実現に全力でコミットする | 真剣交際モード（1対1集中モード）でコミットメントを後押し |
