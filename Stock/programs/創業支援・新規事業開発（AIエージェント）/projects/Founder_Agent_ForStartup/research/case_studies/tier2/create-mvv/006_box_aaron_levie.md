# ケーススタディ: Box - エンタープライズピボットのMVV

**企業**: Box
**創業者**: Aaron Levie
**創業年**: 2005年
**評価**: $4.3B (時価総額)

---

## Mission/Vision/Values分析

### Mission: エンタープライズコンテンツ管理の民主化

**Mission文**:
「エンタープライズ向けクラウドコンテンツ管理を簡単にする」

**VC評価ポイント**:
- **TAM**: エンタープライズファイル共有市場 $20B以上
- **ピボット成功**: コンシューマー向け → エンタープライズ特化への転換
- **グローバル展開**: ✅ 規制対応（GDPR等）で差別化

### Vision: ボトムアップ採用による成長

**ビジョン進化**:
- **初期**: フリーミアム + 個人利用（大学キャンパス）
- **成長期**: IT管理者を経由しない「シャドウIT」の逆転
- **IPO**: エンタープライズ部門ARR向上

**VC評価ポイント**:
- **ボトムアップ採用**: 個人 → チーム → エンタープライズのアップセル
- **スケーラビリティ**: SaaS+ネットワーク効果
- **カスタマー粘性**: ファイル保存で顧客ロック

### Values: ボトムアップ文化の体現

**Boxの組織値**:
1. **Be Authentic** - 透明な意思決定
2. **Drive Results** - 成果主義
3. **Empower People** - 権限委譲とボトムアップ

## 学習ポイント

### ピボット判断の重要性
- コンシューマー市場の競争激化 → エンタープライズ市場への戦略転換
- Aaron Levie の決断力：初期アイデアにこだわらない柔軟性

### ボトムアップ採用の成功事例
- SharePoint との差別化：使いやすさ × モバイル対応
- 従業員が個人的に導入 → 企業採用へ（VC基準で高評価）

### create-mvvスキル適用時の注意
- Missionはピボット可能性を残す（「エンタープライズ特化」に限定しない柔軟性）
- Visionは段階的成長パス（個人 → チーム → エンタープライズ）を明示
- Valuesは「ボトムアップ」を強調（成長組織の象徴）
