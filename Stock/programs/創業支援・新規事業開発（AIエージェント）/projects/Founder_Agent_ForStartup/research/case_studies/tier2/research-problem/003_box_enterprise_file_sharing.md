# Box: エンタープライズファイル共有の課題発見

**企業**: Box
**創業者**: Aaron Levie
**業界**: クラウドストレージ / エンタープライズコラボレーション
**CPFスコア**: 70%
**実行時期**: 2005-2006年

## 課題発見プロセス

### 1. 定性リサーチ手法
- **手法**: 大学キャンパスでの直接聞き取り + エンタープライズ企業インタビュー
- **対象数**: 80人以上（学生30人、IT部門担当者50人）
- **期間**: 1年間（2005年秋-2006年秋）
- **対象層**: UC Berkeleyの学生 + Salesforce 周辺企業の社員

### 2. 発見した課題

**学生側の課題**:
```
"プロジェクト資料を複数メンバーで共有したい。
メール添付は受信制限（25MB）で送れない、バージョン管理も困る"
→ 欲求: クラウド上で共有・バージョン管理ができる仕組み
```

**企業IT部門の課題**:
```
"SharePoint 導入したが、社内セットアップが3ヶ月かかる。
ユーザー教育も大変。オンプレ保守費も年間$50Kかかる"
→ 欲求: クラウドで即導入可能、保守不要、リモートアクセス可能
```

**営業チームの課題**:
```
"顧客提案資料をオフィス内だけでなく、出張先からもアクセスしたい。
VPN接続は遅い、モバイルでは使えない"
→ 欲求: どこからでもアクセス可能、モバイル対応
```

### 3. 3Uスコアリング（Boxの例）

| 軸 | スコア | 根拠 |
|---|--------|------|
| **Unworkable** | 17/20 | ファイル共有の遅延が営業進捗を阻害。提案資料がタイムリーに顧客に届かないと売上機会喪失 |
| **Unavoidable** | 16/20 | SharePoint は高コスト + 複雑、USB移動は紛失リスク、メール添付は容量制限あり。完全には避けられない |
| **Urgent** | 15/20 | 営業活動は時間的制約あるが、深刻な損失ではない。中期的な課題 |
| **合計** | **48/60** | CPFスコア 80% ✅ |

### 4. 顧客ニーズの定量化

**インタビューから抽出したデータ**:

| 指標 | データ | 根拠 |
|------|--------|------|
| **支払意思（WTP）** | $5-15/ユーザー/月 | 企業IT部門の購買力調査 |
| **課題頻度** | 週3-5回 | ファイル共有操作は日常的 |
| **導入コスト削減効果** | $50K/年 → $10K/年 | SharePoint保守費との比較 |
| **TAM** | エンタープライズ企業 500万社 × $10 = **$50B** |
| **SAM** | USA企業 500万社 × $10 = **$5B** |

### 5. 既存代替案への不満点

**競合分析（インタビュー結果ベース）**:

| 選択肢 | コスト | 導入 | モバイル | 保守 | 不満点 |
|--------|--------|------|----------|------|--------|
| **SharePoint** | $50K+/年 | 3ヶ月 | ❌ | 高い | 複雑、高価、遅い |
| **Google Drive** | $6/月 | 即座 | ✅ | 不要 | 企業向け機能不足 |
| **Dropbox** | $10/月 | 即座 | ✅ | 不要 | 企業向けセキュリティ不足 |
| **FTP/WebDAV** | 無料 | 中程度 | △ | 高い | 複雑、セキュリティ脆弱 |

### 6. 課題の本質理解

**表面的な課題**: 「ファイルを共有したい」
↓
**本質的な課題**: 「エンタープライズは SharePoint のセットアップ/保守に$50K/年費やしているが、エンドユーザーは遅いツールにストレス」

→ **洞察**: **シンプル + クラウド + セキュア** の組み合わせは市場にない

### 7. 市場セグメンテーション

**優先順位**:
1. **大学キャンパス** （初期タッチポイント、口コミ伝播）
2. **成長企業** （IT体質が良い、購買決定が速い）
3. **エンタープライズ** （高額な市場）

## スキル実行時の活用方法

### このスキルで活用されるポイント

1. **複数セグメントへのインタビュー**:
   - 学生 → 成長企業 → エンタープライズ企業
   - 各層で異なるペインポイントを発見できる

2. **既存代替案の詳細分析**:
   - SharePoint の不満（複雑さ + 高価格）から導出できる UVP
   - 市場に「シンプル + 低価格」の選択肢が欠けていることの確認

3. **WTP の具体化**:
   - IT部門: $10-15/月/ユーザーで年間 $50K の削減効果
   - 経営層へのピッチで ROI が明確

4. **TAM/SAM の推定**:
   - エンタープライズ企業数 × 平均支払額で市場規模を推定
   - 初期セグメント（大学）から全体市場への拡張可能性を確認

## 定量データサマリー

- **インタビュー数**: 80+人（学生 30人、企業 50人）
- **3Uスコア合計**: 48/60（CPF 80%）
- **WTP確認**: $5-15/ユーザー/月 + $50K 保守費削減
- **推定市場（SAM）**: $5B（USA エンタープライズ市場）
- **初期セグメント**: 大学キャンパス → 高い口コミ伝播率
