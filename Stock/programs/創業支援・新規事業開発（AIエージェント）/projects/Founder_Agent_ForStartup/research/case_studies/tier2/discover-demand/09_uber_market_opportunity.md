# Uber: 既存市場の破壊的再定義

## 市場機会の本質

**需要の源泉**: タクシー呼び出しの不便さ（つかまらない、高い） + スマートフォンの普及による「いつでも呼べる」可能性

## TAM/SAM/SOM分析

| 指標 | 推定値 | 根拠 |
|------|--------|------|
| **TAM** | $5T+ | グローバルタクシー・輸送市場全体 |
| **SAM** | $300B | 都市部でのオンデマンド輸送市場 |
| **SOM (Year 5)** | $10B | 北米主要都市でのUber利用市場 |

## 市場成長指標

- **市場成長率 (CAGR)**: 50%+（オンデマンド輸送市場の成長は従来タクシー市場を大幅に超過）
- **初期成長**: 月次成長率30-40%（初期都市）
- **市場タイミング**: スマートフォン普及（iPhone 2007年、Android 2008年） + GPS精度向上

## 需要検証ポイント（discover-demand スコア）

| 項目 | スコア | 根拠 |
|------|--------|------|
| **切実度** | 5/5 | 夜間や雨の日、タクシーがつかまらない切実な問題 |
| **頻度** | 4/5 | 都市部ユーザーの定期的なニーズ（毎週〜毎月） |
| **支払い匂い** | 5/5 | 既存タクシー利用実績 = 支払い意思確認済み |
| **未解決度** | 5/5 | 既存タクシー業界は電話呼び出しが主流で改善なし |
| **市場規模** | 5/5 | TAM $5T+、破壊的成長率50%+ |
| **合計** | **24/25** | VC投資推奨レベル |

## スキル活用ポイント

### 1. 既存市場の非効率性の認識
- タクシー業界は100年以上変化なし
- スマートフォンで「今この瞬間」のタクシー配車が可能に
- **活用方法**: 「当たり前の非効率」が大規模市場転換を生む

### 2. ネットワーク両面性（サプライとデマンド）
- ドライバー（サプライ） + ユーザー（デマンド）の両面が必要
- **活用方法**: マーケットプレイス系企業での「鶏と卵」問題の解決策

### 3. ジオグラフィック・エキスパンション戦略
- SF → NYC → 全米 → グローバルの段階的展開
- **活用方法**: 地域限定ビジネスのスケール方法論

## 年次成長率シナリオ

| 期間 | 月次成長率 | 年次成長率 | 達成目標 |
|------|-----------|----------|---------|
| **初期（0-12ヶ月）** | 20-25% | 200-250% | SF市場での基盤確立 |
| **Series A（12-24ヶ月）** | 30-40% | 400%+ | 複数都市への拡張 |
| **Series B（24-36ヶ月）** | 25-35% | 300-400% | 全米展開、国際展開準備 |

## 差別化軸（10倍優位性）

1. **利便性**: タクシー呼び出し（10分待機） → スマートフォン1タップ（10倍）
2. **予測可能性**: 運転手の対応待機 → 配車時間表示（5倍）
3. **価格透明性**: メーター制（不透明） → 事前料金提示（3倍）

## 失敗パターン（回避すべき落とし穴）

- 規制・タクシー業界との対立
- **教訓**: 規制産業への進出は「法的リスク」を最大リスク要因として扱う

## 地域別市場機会の段階化

**市場導入戦略**:
```
1. SF（テック人口多い、既存タクシー態度悪い） → 初期ユーザー獲得
2. NYC（タクシー密集度高い、不満多い） → 市場検証
3. 全米主要都市 → スケール加速
4. グローバル → 各国法制度対応
```

## 日本市場への適用性

- **市場ニーズ**: 日本でもタクシー呼び出しの不便さ（ただし無線タクシーが普及）
- **課題**: 法規制（運送業法）の厳しさ、既存タクシー業界との利益相反
- **機会**: 駅前タクシー不足の時間帯へのオンデマンド対応
