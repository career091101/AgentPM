## 使用方法

### build-pitch-deck スキル実行時

各ケーススタディは以下の構造で記載:

1. **ピッチデッキの特徴**: Problem/Solution/Traction/Market Size/Business Model
2. **投資家説得力スコア**: 各スライド0-15点、総合130点満点
3. **投資家の反応**: 実際の投資判断ポイント、リターン
4. **このスキルでの活用ポイント**: build-pitch-deck実行時の具体的参照方法

### スコア評価基準

| スコア範囲 | 判定 | 説明 |
|-----------|------|------|
| **120-130点** | 最優秀 | Dropboxレベル、即座にSeries A調達可能 |
| **110-119点** | 優秀 | Airbnb/Freshworksレベル、Series A調達成功率高い |
| **90-109点** | 良好 | 改善点あるが投資検討に値する |
| **70-89点** | 要改善 | 3ヶ月改善後に再提出推奨 |
| **70点未満** | 再構築 | CPF/PSF再検証推奨 |
