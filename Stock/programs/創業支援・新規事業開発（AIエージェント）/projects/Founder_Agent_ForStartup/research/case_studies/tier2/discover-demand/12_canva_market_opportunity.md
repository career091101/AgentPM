# Canva: 非デザイナー向け市場の創造

## 市場機会の本質

**需要の源泉**: グラフィックデザインスキルなし + デザインツール高額・複雑 + 「誰でも美しいデザインを作りたい」という潜在需要

## TAM/SAM/SOM分析

| 指標 | 推定値 | 根拠 |
|------|--------|------|
| **TAM** | $100B+ | グローバルグラフィックデザイン・ビジュアルコンテンツ市場 |
| **SAM** | $20B | デジタルマーケティング・ソーシャルメディアコンテンツ市場 |
| **SOM (Year 5)** | $2B | 非プロデザイナー（マーケター、起業家等）の有料ユーザー |

## 市場成長指標

- **市場成長率 (CAGR)**: 15-20%（ビジュアルコンテンツ市場）
- **初期成長**: 月次成長率20-25%（初期段階）
- **市場タイミング**: ソーシャルメディア普及（Instagram, Pinterest） + コンテンツマーケティング需要増加

## 需要検証ポイント（discover-demand スコア）

| 項目 | スコア | 根拠 |
|------|--------|------|
| **切実度** | 4/5 | マーケター・起業家にとって重要だが絶対的緊急性は低い |
| **頻度** | 4/5 | マーケター・コンテンツ作成者の週1回以上の需要 |
| **支払い匂い** | 4/5 | 有料素材・テンプレート購入実績あり |
| **未解決度** | 5/5 | 既存（Photoshop、Illustrator）は高額・学習曲線が急 |
| **市場規模** | 4/5 | TAM $100B+、ただし初期はソーシャルメディア領域 |
| **合計** | **21/25** | VC投資推奨レベル（初期セグメント限定） |

## スキル活用ポイント

### 1. ペルソナ選択による市場機会
- ターゲット: 非プロデザイナー（マーケター、起業家、小規模ビジネス）
- **活用方法**: 「できない人」に売る市場 = プロ向けよりはるかに大きい

### 2. テンプレート・プリセットによる複雑性の排除
- デザイン初心者でも「高品質の結果」を出せる仕組み
- **活用方法**: UXで「複雑性を排除」することが市場機会になる

### 3. ソーシャルメディアとの連携
- Instagramテンプレート、Pinterestテンプレート等
- **活用方法**: 主要プラットフォームとの統合が採用の鍵

| **Series B（24-36ヶ月）** | 15-20% | 200%+ | チーム機能、白レーベル対応 |

## 差別化軸（10倍優位性）

1. **使いやすさ**: Photoshop（専門家向け） → Canva（誰でも使える）（10倍）
2. **テンプレート量**: 数百個 → 数十万個（100倍）
3. **価格**: $19/月（Adobe）→ $10-15/月（3倍）

## 失敗パターン（回避すべき落とし穴）

- プロデザイナー向け機能追加 → 複雑化 → 初心者離脱
- **教訓**: 「シンプルさ」を最高の差別化要因として守ること

## 創業者の背景が生み出した市場洞察

**Melanie Perkinsの視点**:
```
1. 大学生時代：Photoshopは「デザイナーだけのツール」という課題発見
2. 観察：ほとんどのビジネスオーナーはデザイナーを雇えない
3. 仮説：「テンプレート + ドラッグ&ドロップ = 誰でもデザイナーになれる」
→ 2013年Canva創業、2021年ユニコーン化
```

## 日本市場への適用性

- **市場ニーズ**: 日本のマーケター・起業家も同様の課題
- **課題**: 日本語フォント・テンプレート、日本的デザイン文化への適応
- **機会**: SNSマーケティング・インフルエンサー向けコンテンツ作成市場
