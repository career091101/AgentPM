## Key Scores
- Market (7/8): Personal styling $10B+ TAM
- CPF (7/10): 74% - Busy professionals need styling help
- PSF (8/10): Personalization 10x better than generic retail
- Business Model (7/10): LTV/CAC 3.5x (inventory costs)
- Execution (7/10): Hybrid model (stylists + algorithms)
- Uniqueness (8/10): Data flywheel (more orders = better recommendations)

## 10x Advantage
- **Personalization**: 85% accuracy (vs. generic retail 10%)
- **Convenience**: Delivered to home (vs. store shopping)
- **Data flywheel**: Recommendations improve with every order

## Business Model
- Revenue: Keep rate × item price (avg $55/item)
- Keep rate: 50-60% (industry-leading)
- Repeat rate: 80% (strong retention)

## Data Flywheel
1. Customer style quiz → Initial data
2. Stylist + algorithm curation
3. Customer feedback (keep/return)
4. Algorithm improves → Better next box
5. Higher keep rate → More revenue

## Lesson: Human + AI Hybrid
- Not pure algorithm (Stitch Fix uses stylists + data)
- Result: 85% accuracy vs. 50% (pure algorithm)
- Execution complexity: Requires both tech + operations
