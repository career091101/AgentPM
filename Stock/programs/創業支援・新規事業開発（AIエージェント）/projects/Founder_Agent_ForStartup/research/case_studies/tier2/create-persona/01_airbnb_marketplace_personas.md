## ペルソナ1: ホスト側 - Sarah（空きスペース活用中年女性）

### 基本情報
| 項目 | 内容 |
|------|------|
| 名前 | Sarah（仮名） |
| 年齢 | 45歳 |
| 職業 | パートタイム講師 |
| 月額収入 | $2,500 |
| 動機 | 住宅ローン補填、追加収入源 |

### 日常行動
- 09:00-12:00: 教育機関での非常勤講師
- 14:00-16:00: Airbnbリスティング管理（メッセージ対応、清掃スケジュール調整）
- 16:00-18:00: 自由時間（客室清掃、準備）

### ペルソナ設定のためのWTP検証
- **支払い根拠**: 月額$800-1,200の追加収入（年$9,600-14,400）
- **スイッチングコスト**: リスティング削除、新プラットフォームでの信用構築が必要（低～中）
- **実績**: 2008年初期ゲストが$80/泊を実際に支払う → 月4-5組で$320-400の追加収入を確認

### このスキルでの活用ポイント
ホスト側ペルソナは「追加収入」「所有資産の活用」「自動化できない人間関係」の3要素で強いWTPが成立。

### 基本情報
| 項目 | 内容 |
|------|------|
| 名前 | Michael（仮名） |
| 年齢 | 32歳 |
| 職業 | デザインカンファレンス参加者 |
| 月額収入 | $4,500（フリーランス） |
| 課題 | 都市への出張時、ホテル満室・高額費用 |

### 日常行動
- カンファレンス参加日: ホテル満室対応として Airbnb 検索
- チェックイン時間: 柔軟（ホスト対応可能なタイミング）
- 宿泊期間: 3-5日（カンファレンス期間+前後）

### ペルソナ設定のためのWTP検証
- **支払い実績**: $80/泊（ホテル $150-250 vs Airbnb $80） → **47-67%コスト削減**
- **支払い根拠**:
  - ホテルコスト削減: 月$100-200/宿泊
  - ローカル体験価値: Airbnb独特のホスト交流
- **感情**: 緊急性（ホテル満室）× 経済性（コスト削減）で高いWTP

### このスキルでの活用ポイント
初期ゲスト3名の実支払い$80/泊事例は、「実際の支払い実績」の重要性を示す典型例。

## LTV/CAC推定（マーケットプレイス特性）

### ホスト側 LTV
- 平均稼働日数: 60日/年（年$4,800-8,000）
- 稼働期間: 3-5年
- **LTV**: $14,400-40,000

### ゲスト側 LTV
- 平均予約: 3回/年 × $250/予約
- 継続期間: 5-10年
- **LTV**: $3,750-7,500

### ネットワーク効果
- ホスト増加 → ゲスト満足度向上 → ゲスト増加 → ホスト収入向上
- **フライホイール**: ホスト100件でクリティカルマス達成

## ペルソナ生成時の検証ポイント

### Unworkable（切実度）: 85/100
- ホテル満室問題は解決不可（構造的）
- ホスト：住宅ローン補填の必要性は避けられない

### Unavoidable（不可避）:✅
- イベント時のホテル供給不足は業界構造上避けられない
- ホスト：所有資産の活用は経済的必要性

### Urgent（緊急性）: 7/10
- カンファレンス直前のホテル満室 = 即座の対応必要
- ホスト：月額追加収入の即時性

## このケーススタディから抽出すべき ペルソナ設計原則

1. **両面ペルソナの独立性**: ホスト/ゲスト側でペルソナが異なる（動機, WTP, 緊急性）
2. **実際の支払い実績の重要性**: 「支払う意思」ではなく「実際に$80を支払った」が検証根拠
3. **ネットワーク効果の定量化**: クリティカルマス（供給側件数）の明示
4. **初期顧客獲得の具体性**: イベント参加者→満室→Airbnb という具体的カスタマージャーニー
