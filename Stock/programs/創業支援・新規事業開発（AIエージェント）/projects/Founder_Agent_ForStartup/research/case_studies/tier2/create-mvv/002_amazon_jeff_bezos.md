# ケーススタディ: Amazon - 長期思考のMVV

**企業**: Amazon
**創業者**: Jeff Bezos
**創業年**: 1994年
**評価**: $2T+ (時価総額)

---

## Mission/Vision/Values分析

### Mission: 顧客執着（Customer Obsession）

**Mission文**:
「地球上で最も顧客中心的な企業になる」

**VC評価ポイント**:
- **TAM**: E-commerce市場 + クラウドコンピューティング → 無制限
- **10年スケール**: ✅ 顧客中心主義は30年経っても有効
- **グローバル展開**: ✅ 地域制約なし（決済・物流インフラの課題は局所的）

### Vision: 逆算思考による戦略

**ビジョン進化**:
- **初期（1994）**: オンライン書籍販売で$1M売上
- **5年後（1999）**: 多商品展開、評価額$2.2B
- **15年後（2009）**: AWS発表、クラウド市場開拓
- **30年後（2024）**: $2T企業、卸売・SaaS・広告へ拡張

**VC評価ポイント**:
- **年次成長率**: 初期30%以上（VC基準年次3倍超過）
- **ネットワーク効果**:
  - フライホイール: 低価格 → 顧客体験向上 → トラフィック増 → 出品者増 → 品揃え増 → さらなる顧客体験向上
  - データフライホイール: 購買データ → レコメンド精度向上 → 購買増
- **スケーラビリティ**: テクノロジー投資による限界費用削減

### Values: "Day 1" 文化による永遠の変革

**Amazonの組織値**:
1. **Customer Obsession** - 常に顧客中心（社内会議ではEmpty Chairを用意）
2. **Ownership** - オーナーシップの徹底（部門横断的責任）
3. **Bias for Action** - 行動バイアス（実験文化）
4. **Frugality** - 倹約（資源の最適化）
5. **Earn Trust** - 信頼構築

**VC評価ポイント**:
- **採用基準**: 顧客中心性とオーナーシップを組織DNA化
- **スケーラビリティ**: 30年経っても「Day 1思考」を維持 → リーダーシップ原則（Leadership Principles）で全従業員に浸透
- **意思決定**: 6-pagerドキュメント文化により、権限委譲を実現

## 学習ポイント

### VC調達との整合性
- **第一次VC投資**: Nick Rotonowski（天使投資）+ Kleiner Perkins（$12M Series A）
- **ストーリー**: 「Web成長率2,300%/年」という市場データが投資判断を支配
- **後発優位**: 「今すぐ行動しないと後悔する」という心理がVC投資を加速

### create-mvvスキル適用時の注意
- Amazonモデルは「長期利益」より「市場シェア」を優先する戦略
- VC向けには「短期的KPI」ではなく「10年後の市場支配」を語る
- 日本スタートアップが真似しやすい: 「顧客中心主義」は業界問わず有効だが、日本では「品質重視」との整合性を説明する必要

### 失敗パターン回避
- ❌ 短期利益重視 → Amazon的MVVには反対
- ❌ テクノロジー重視 → 顧客体験を二の次にしない
- ✅ 10年戦略 → 初期段階でも「未来市場」を語る
