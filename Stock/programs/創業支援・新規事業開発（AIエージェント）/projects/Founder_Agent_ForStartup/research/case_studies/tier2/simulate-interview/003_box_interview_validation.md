## 戦略的なインタビュー手法: ボトムアップ採用の可視化

### 段階1: 大学キャンパス内インタビュー（初期ユーザー発掘）

**対象者**: 大学生・研究者（ファイル共有が必要な層）

**インタビュー内容**:
1. 「現在どうやってファイルを共有しているか？」（現状把握）
2. 「USBメモリ・メール添付で困ったことは？」（ペイン確認）
3. 「セキュアなクラウドストレージがあれば月$10払う？」（WTP検証）
4. 「友人に勧めたいか？」（バイラリティ検証）

**発見したインサイト**:
- **Unworkable**: 「ファイル紛失のリスク」「バージョン管理が面倒」
- **Unavoidable**: 「全学生が課題グループワークでファイル共有に困っている」（90%）
- **Urgent**: 「来週の提出まで対応したい」
- **Underserved**: SharePoint/Dropbox/Google Driveより「シンプルで信頼できる」

**4Uスコア**: 35/40 ✅

---

### 段階2: フリーミアムでの行動分析インタビュー

**特徴**: 無料版 → 有料版への転換過程をインタビュー

**質問**:
1. 「無料版でどの機能を使った？」
2. 「有料版へのアップグレードを検討した理由は？」
3. 「月$10-15なら企業採用を推奨するか？」

**発見したインサイト**:
- **ボトムアップ採用**: 学生個人 → 研究室 → 大学部門 → エンタープライズ
- **CVR改善**: 無料版ユーザーの15-20%が有料化
- **WTP**: 企業向けに月$15/ユーザーで80%の受容率

**UVP刺さり度**: 36/40 ✅

---

### 段階3: エンタープライズ購買決定者インタビュー

**対象**: IT部門長、CFO

**インタビュー内容**:
1. 「SharePointの問題点は何か？」（現状課題）
2. 「ファイル共有で年間いくら損失しているか？」（定量化）
3. 「Box導入の決裁権は誰か？」（購買プロセス理解）
4. 「トライアル期間はいくらあれば十分か？」（実行可能性）

**発見したインサイト**:
- **ROI**: 「ファイル管理工数月50時間 × $50/時間 = 月$2500削減」
- **Payback期間**: CAC$200 → 月$150削減 → 1.3ヶ月で回収
- **決裁**: CTO/CFO承認必須（3-4ヶ月要）

**LTV/CAC**: 4-6x ✅（Series A基準達成）

## このスキルへの活用方法

### simulate-interview での実装例

```
3段階のペルソナ設定:
1. 個人ユーザー: 学生・研究者（WTP $10/月）
2. 部門管理者: IT部門長（WTP $15/ユーザー）
3. 決裁権者: CFO/CEO（LTV/CAC 5.0基準）

B2B特化のインタビュー設計:
1. 課題認識: ファイル共有工数、セキュリティリスク
2. ROI試算: 年間$100K削減 × 5年 = LTV試算
3. 現状ツール: SharePoint/Dropbox との比較
4. 決裁プロセス: 誰が承認? 期間は?
5. PoC意向: 30日/60日トライアル希望? 成功基準は?
6. 導入決定期間: 即決 vs 3-6ヶ月

期待値:
- WTP確認: 個人$10/月、企業$15/ユーザー/月
- ROI明確性: 年間削減額の定量化
- 4Uスコア: 35点以上（CPF検証達成）
- PoC参加率: 70%以上（高い実装意向）
```

## ベストプラクティス抽出

### ボトムアップ採用戦略の検証
1. **個人層からの開始**: 低CAC（友人紹介、SEO）で早期ユーザー獲得
2. **段階的拡大**: 個人 → チーム → 組織へのオーガニック拡散確認
3. **フリーミアムの活用**: 無料で十分な価値提供 → 有料化への自然な流れ

### B2B インタビュー工夫
- **複数層へのインタビュー**: IT部門とCFO両方に聞く（購買委員会理解）
- **ROI定量化**: 削減できる工数・コストを金額で明示
- **決裁プロセス理解**: 期間・承認者・成功基準を把握
