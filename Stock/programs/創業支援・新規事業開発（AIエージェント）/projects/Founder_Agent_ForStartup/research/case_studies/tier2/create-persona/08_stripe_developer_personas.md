## ペルソナ: スタートアップCTO - Elena（決済システム統合）

### 基本情報
| 項目 | 内容 |
|------|------|
| 名前 | Elena（仮名） |
| 年齢 | 32歳 |
| 職業 | e-commerce スタートアップ CTO |
| 月額支払い | GMVの2.9% + $0.30/取引（従量課金） |
| 課題 | 決済システム構築（複雑 + セキュリティ要件） |

### WTP検証
- **支払い実績**: 月商$10,000 × 2.9% + $0.30 × 500取引 = $440/月
- **支払い根拠**:
  - 従来: 自社決済システム構築 → 開発コスト$50k, 保守$3k/月
  - Stripe: $440/月 + API呼び出し無制限
  - 開発時間削減: 4週間 → 1日（27倍高速化） = $20k-30k相当
  - セキュリティ: PCI DSS コンプライアンス自動提供
  - **LTV**: 月$440 × 60ヶ月（5年） = **LTV $26,400**（初期カスト$20k回収後）

### このスキルでのポイント
**デベロッパーペルソナ**:
- 意思決定: CTO（技術的判断） + CEO（費用判断）の二層構造
- WTP根拠: **開発コスト削減**（定量的ROI）
- リテンション: 統合が深い（スイッチングコスト高）→ 長期LTV確保
