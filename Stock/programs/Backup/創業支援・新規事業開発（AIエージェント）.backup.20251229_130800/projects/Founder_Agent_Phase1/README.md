# Founder Agent：事業計画書自動生成・評価プロジェクト（Phase1）

## 概要
CxO候補／事業会社の新規事業担当が持つ事業アイデアについて、
- 起業の科学をベースに良し悪しを即時評価し
- 思考プロセスを構造化・可視化しながら
- 仮説検証前に使える軽量な事業計画書を自動生成する

思考補助型ファウンダーエージェント（Phase1）を設計・試作すること。

## プロジェクト構造
```
documents/
├── 1_initiating/   # 立ち上げフェーズ（憲章・ステークホルダー分析）
├── 2_discovery/    # 発見フェーズ（ペルソナ・課題定義）
├── 2_research/     # 調査フェーズ（顧客・競合・市場調査）
├── 3_planning/     # 計画フェーズ（PRD・WBS・バックログ）
├── 4_executing/    # 実行フェーズ（スプリント・日次タスク）
├── 5_monitoring/   # 監視・コントロールフェーズ
└── 6_closing/      # 終結フェーズ（教訓・引き継ぎ）
```
