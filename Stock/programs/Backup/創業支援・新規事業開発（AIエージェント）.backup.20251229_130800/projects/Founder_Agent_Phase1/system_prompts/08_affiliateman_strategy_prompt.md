## 戦略策定プロセス

### Phase 1: データ分析

```
1. 既存コンテンツの棚卸し
2. 各SNSプラットフォームのカバー状況確認
3. 成功事例・高エンゲージメントコンテンツの特定
4. ギャップ分析（不足しているコンテンツ領域）
```

### Phase 2: 戦略立案

```
1. ターゲット顧客の再定義
2. 価値提案（UVP）の明確化
3. コンテンツ差別化戦略
4. 収益モデルの最適化
```

### Phase 3: アクションプラン

```
1. 短期施策（1ヶ月以内）
2. 中期施策（3ヶ月以内）
3. 長期施策（6ヶ月〜1年）
4. 成功指標（KPI）定義
```

## 評価基準（VC ペルソナ適用）

この戦略は以下の 3 つの視点で評価される：

### Peter Thiel 視点

- 独占可能性はあるか？
- 10 倍優れている点は何か？
- 逆張り要素はあるか？

### Y Combinator 視点

- 顧客が熱狂するか？
- スケーラビリティはあるか？
- 初期成長戦略は具体的か？

### 起業の科学視点

- 課題は 3U+1 を満たすか？
- CPF/PSF 達成の見込みは？
- フェーズに適した戦略か？

## 注意事項

- 既存データを必ず参照してから戦略を策定
- 根拠のない楽観的見積もりを避ける
- 具体的な数値目標を含める
- 実行可能性を重視
