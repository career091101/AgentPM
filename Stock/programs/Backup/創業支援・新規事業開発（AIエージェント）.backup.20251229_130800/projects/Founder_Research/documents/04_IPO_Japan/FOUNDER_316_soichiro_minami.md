## 2. 創業ストーリー

### 2.1 課題発見（需要発見）

**着想源**:
- モルガン・スタンレー、楽天で優秀な人材採用に携わる
- 「日本の採用市場は非効率すぎる」と痛感
  - ハイクラス人材の転職は人材紹介会社経由が主流
  - 成功報酬が年収の30-35%と高額
  - 企業と人材が直接出会う場がない
- LinkedIn のようなプラットフォームが日本にない
- 「即戦力人材の直接採用プラットフォーム」があれば、企業・人材双方にメリット

**需要検証方法**:
- 企業人事100社にヒアリング
- 「ハイクラス人材採用に困っているか？」→ 90%が「困っている」
- 「人材紹介会社の費用が高いか？」→ 80%が「高い」
- 「ダイレクトリクルーティングに興味があるか？」→ 70%が「興味あり」
- ハイクラス人材500人にアンケート
  - 転職経験: 80%が「ヘッドハンター経由」
  - 直接応募の意向: 60%が「企業と直接やり取りしたい」

### 2.2 CPF検証（Customer Problem Fit）

**インタビュー/顧客検証**:
- 実施数: 100社以上（企業人事） + 500人（ハイクラス人材）
- 手法: 訪問インタビュー、採用プロセス同行
- 発見した課題の共通点:
  - **企業**: 「即戦力人材が見つからない」「人材紹介会社の費用が高い」「採用に時間がかかる」
  - **人材**: 「良い求人情報がない」「ヘッドハンターに依存したくない」「自分で選びたい」

**3U検証**:
- Unworkable（現状では解決不可能）:
  - 人材紹介会社は成功報酬が高額（年収の30-35%）
  - ハイクラス求人は一般の転職サイトにない
  - 企業と人材が直接出会う場がない
- Unavoidable（避けられない）:
  - 企業の成長には即戦力人材の採用が必須
  - ハイクラス人材のキャリアアップには転職が不可欠
  - 優秀な人材は常に市場で引く手あまた
- Urgent（緊急性が高い）:
  - 急成長企業は今すぐ即戦力が必要（中程度の緊急性）
  - 優秀な人材は早く確保しないと競合に取られる

**支払い意思（WTP）**:
- 確認方法: 有料ベータ版を20社に提供
- 結果:
  - 月額100万円で全社が契約
  - 年間契約率85%（継続意思の確認）
  - 「人材紹介会社より安い」との声が多数

### 2.3 PSF検証（Problem Solution Fit）

**10倍優位性**:

| 軸 | 従来の解決策 | 自社ソリューション | 倍率 |
|---|------------|-----------------|------|
| 採用コスト | 人材紹介（年収の30-35%） | ビズリーチ（月額固定+成功報酬15%） | 2-3倍 |
| 採用スピード | 3-6ヶ月 | 1-2ヶ月 | 3倍 |
| 候補者の質 | ヘッドハンター任せ | 自社で直接スカウト | 2倍 |
| 候補者数 | 数名 | 数十〜数百名 | 10倍以上 |
| 透明性 | ブラックボックス | プロセス可視化 | ∞倍 |

**MVP**:
- タイプ: Marketplace Prototype（企業とハイクラス人材のマッチング）
- 初期反応:
  - ベータ版20社、18社が本契約
  - 初回採用成功率60%（業界平均30-40%）
  - NPS（推奨度）: 65（業界平均20-30）
- CVR: 初期10% → 改善後20%（2年後）

**UVP（独自の価値提案）**:
- 「即戦力人材と企業をつなぐ、ハイクラス転職サイト」
- 会員制（審査あり）でハイクラス人材のみ
- 企業が直接スカウト可能
- ヘッドハンターも活用可能（二重構造）

**競合との差別化**:
- 人材紹介会社との差別化:
  - 圧倒的低コスト（成功報酬15% vs 30-35%）
  - 候補者を自社で選べる
  - リアルタイムで進捗確認
- 一般転職サイト（リクナビ、マイナビ等）との差別化:
  - ハイクラス人材特化（年収600万円以上）
  - 会員制（審査あり）で質を担保
  - 企業からのスカウト中心

## 3. ピボット/失敗経験

### 3.1 初期の失敗

- **会員獲得の困難（2009-2010年）**:
  - 当初、ハイクラス人材の会員登録が進まない
  - 「転職サイトに登録=転職活動中」というネガティブイメージ
  - 「キャリア情報の登録=スカウトを受け取る」というポジティブメッセージに変更

- **企業側の理解不足（2010年）**:
  - 「ダイレクトリクルーティング」という概念が未成熟
  - 企業人事への啓蒙活動に注力（セミナー、コンテンツマーケティング）

- **価格設定の試行錯誤（2010-2011年）**:
  - 初期価格: 成功報酬のみ → 収益が不安定
  - 月額固定 + 成功報酬15%に変更 → 安定収益確保

### 3.2 ピボット（該当する場合）

- ピボットは実施していない
- 一貫して「ハイクラス人材のダイレクトリクルーティング」を追求

## 4. 成長戦略

### 4.1 初期トラクション獲得

- **コンテンツマーケティング（2009-2012年）**:
  - 「採用ノウハウ」「業界動向」等のコンテンツ発信
  - SEO/SEM集中投資

- **ヘッドハンター活用（2011年〜）**:
  - ヘッドハンターもプラットフォームを活用可能に
  - ヘッドハンター数千人がビズリーチ経由で活動

- **テレビCM（2013年〜）**:
  - 「ビズリーチ」の認知度向上
  - 会員数が急拡大（年間2倍ペース）

### 4.2 フライホイール

1. **ハイクラス人材の会員獲得**
   → 2. **企業がスカウトを送る**
   → 3. **採用成功事例が増える**
   → 4. **口コミで新規企業・人材が増える**
   → 5. **ヘッドハンターも参加**
   → 6. **マッチング精度向上**
   → 1に戻る

- **ネットワーク効果**:
  - 会員数が増えるほど、企業の採用確率が上がる
  - 企業数が増えるほど、人材の選択肢が増える

### 4.3 スケール戦略

- **周辺サービスへの拡大（2014年〜）**:
  - キャリトレ（20代向け転職サイト）
  - HRMOS（採用管理システム）
  - ビズリーチ・キャンパス（新卒採用）

- **M&A戦略（2016年〜）**:
  - スタンバイ（求人検索エンジン）買収
  - トラボックス（飲食業界特化）買収
  - 複数の人材系スタートアップを買収

- **海外展開（2018年〜）**:
  - 東南アジア（シンガポール、インドネシア）で展開

- **ビジョナルへの統合（2020年）**:
  - 持株会社ビジョナルを設立
  - 複数サービスを統合管理

### 4.4 バリューチェーン

```
ハイクラス人材（会員登録）
  → ビズリーチ（審査・承認）
  → 企業（スカウト送信）
  → 人材（応募・面接）
  → 企業（採用）
  → ビズリーチ（成功報酬受領）
```

## 6. 成功要因分析

### 6.1 主要成功要因

1. **ハイクラス特化の明確なポジショニング**:
   - 年収600万円以上に絞り、質を担保
   - 会員制（審査あり）でブランド構築

2. **二重構造（企業直接 + ヘッドハンター）**:
   - 企業がダイレクトスカウトも可能
   - ヘッドハンターも活用可能
   - 双方のニーズに対応

3. **コンテンツマーケティング**:
  - 「採用ノウハウ」等のコンテンツで啓蒙活動
   - SEO/SEM で継続的な流入

4. **M&A戦略**:
   - 周辺サービスを積極的に買収
   - HR Tech エコシステム構築

5. **ビジョナルへの統合**:
   - 持株会社化で複数サービスを統合管理
   - IPO成功（2022年4月）

### 6.2 タイミング要因

- **2009年**: リーマンショック後、企業の採用コスト削減ニーズ増大
- **2012年**: LinkedIn 日本進出 → ダイレクトリクルーティング認知拡大
- **2015年**: 働き方改革 → 転職市場活性化
- **2020年**: コロナ禍 → オンライン採用の普及

### 6.3 差別化要因

- **ハイクラス特化**: 年収600万円以上に絞る
- **会員制**: 審査ありで質を担保
- **二重構造**: 企業直接 + ヘッドハンター活用
- **M&A戦略**: 周辺サービス買収でエコシステム構築
