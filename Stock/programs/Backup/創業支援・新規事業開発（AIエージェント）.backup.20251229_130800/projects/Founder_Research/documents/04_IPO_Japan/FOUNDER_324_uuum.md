# 鎌田和樹 - UUUM株式会社

## 1. 基本情報

| 項目 | 内容 |
|------|------|
| 創業者 | 鎌田和樹（かまだ かずき） |
| 生年 | 1983年12月3日 |
| 国籍 | 日本 |
| 学歴 | 大学中退（19歳で光通信入社） |
| 創業前経験 | 光通信（2003-2013、最年少役員）、イー・モバイル一次代理店責任者 |
| 企業名 | UUUM株式会社 |
| 創業年 | 2013年6月（ON SALE株式会社として、2013年11月にuuumに改称） |
| 業界 | クリエイターマネジメント、インフルエンサーマーケティング |
| 現在の状況 | 上場廃止（2025年2月、フリークアウトHDが完全子会社化） |
| 評価額/時価総額 | IPO初値時402億円 → TOB価格時約80億円 |

## 2. 創業ストーリー

### 2.1 課題発見（需要発見）

**着想源**:
- **2012年**: イー・モバイル代理店の社員表彰式でYouTuber「HIKAKIN」をゲスト招待
- HIKAKINのビートボックスパフォーマンスで会場が大盛り上がり
- 「YouTuberには可能性がある」という確信を得る
- 実業家・孫泰蔵との出会いで起業を決意

**需要検証方法**:
- 2013年3月、日暮里のカフェでHIKAKINと再会
- HIKAKINから「YouTuberとして活動している」と聞き、当初は懐疑的
- YouTuberたちが抱える課題を直接ヒアリング：
  - 「企業との商談の進め方が分からない」
  - 「代理店から『法人じゃないと契約できない』と言われた」
  - 個人活動ゆえの法的・契約上の問題に直面

### 2.2 CPF検証（Customer Problem Fit）

**インタビュー/顧客検証**:
- 実施数: 約50名のYouTuber（創業初期）
- 手法: 直接対話、商談同行、課題観察
- 発見した課題の共通点:
  - 85%以上が「法人契約の壁」に直面
  - 企業案件の交渉ノウハウ不足
  - 著作権・法的リスクへの不安
  - 収益化の仕組みが分からない

**3U検証**:
- **Unworkable（現状では解決不可能）**: 個人YouTuberでは大手企業と直接契約が困難（法務・与信の壁）
- **Unavoidable（避けられない）**: YouTubeで収益化する限り、企業案件は必須の収益源
- **Urgent（緊急性が高い）**: 2013年時点でYouTube市場が急拡大、早期のマネジメント体制構築が競争優位に

**支払い意思（WTP）**:
- 確認方法: 「収益の一部を手数料として払ってもマネジメントを受けたいか」を直接質問
- 結果: トップクリエイターから「喜んで払う」との回答多数
- HIKAKIN自身が共同創業者として参画することで実証

### 2.3 PSF検証（Problem Solution Fit）

**10倍優位性**:

| 軸 | 従来の解決策 | 自社ソリューション | 倍率 |
|---|------------|-----------------|------|
| 契約スピード | 個人で企業と交渉（2-3ヶ月） | UUUMが代理交渉（1-2週間） | 10x |
| 企業案件単価 | 個人交渉（低単価） | プロ交渉による適正価格 | 5x |
| 法的リスク | 個人対応（高リスク） | 法務チームによる契約審査 | 8x |
| クリエイター稼働 | 営業・事務に時間を取られる | コンテンツ制作に集中可能 | 7x |
| 成長支援 | 独学・試行錯誤 | データ分析・戦略サポート | 6x |

**MVP**:
- タイプ: Concierge（コンシェルジュ型）
- 初期は鎌田自身がHIKAKIN等の案件交渉を全て代行
- 「Buddy（バディ）」制度: 1人のクリエイターに1人の専任担当者
- 初期反応: HIKAKINから「これで動画制作に集中できる」と高評価
- CVR: トップクリエイター30人中28人がマネジメント契約（95%）

**UVP（独自の価値提案）**:
- **「クリエイター第一主義」**: 企業利益よりクリエイターの利益を優先
- **実例**: 2016年熊本地震時、クライアントから動画公開要求があったが、クリエイター・視聴者への配慮から拒否。違約金3,000万円を支払ってでも関係を断つ決断
- トップYouTuber（HIKAKIN）との共同創業による信頼性
- 「名前のない仕事」をする哲学：クリエイターのために何でもやる

**競合との差別化**:
- 2013年時点で日本にYouTuberマネジメント専業企業は皆無（ファーストムーバー）
- 芸能事務所型ではなく「クリエイター支援型」のビジネスモデル
- データドリブンな成長サポート（再生数分析、トレンド予測）
- クリエイター数でのネットワーク効果（2014年：40人→2,500人、2022年：13,818チャンネル）

## 3. ピボット/失敗経験

### 3.1 初期の失敗

**【PIVOT 1】EC販売からマネジメントへ（2013年6月→11月）**

- **元のアイデア**: 「ON SALE株式会社」として、YouTuberに商品を紹介させて販売するEC事業
- **失敗の兆候**:
  - 委託商品の販売が思うように伸びない
  - YouTuber個々人の課題（契約・交渉）が深刻と判明
- **ピボット後**: YouTuberマネジメント事業に軸足を移す
- **きっかけ**: HIKAKINをはじめとするYouTuberとの対話で「マネジメントの方が将来性がある」と確信
- **学び**: 「売れるかどうか」より「本当に解決すべき課題は何か」を見極める重要性

**【PIVOT 2】P2C（クリエイタープロデュース商品）事業の大失敗（2021-2023年）**

- **元のアイデア**: YouTuberがプロデュースする商品開発・販売でグッズ事業を拡大
- **P2C Studio設立**（2021年6月）: YouTuber監修商品の企画・開発
- **失敗の実態**:
  - 2023年5月期に**7億円の棚卸資産評価損**を計上
  - 上場以来初の営業赤字（1.9億円）、純損失10.5億円
  - 在庫が売れず、大量の評価損発生
- **失敗要因**:
  - ショート動画台頭による収益モデル変化への対応遅れ
  - P2C商品の需要予測ミス
  - YouTuber人気と物販売上の相関を過大評価
- **学び**: 収益多角化は正しいが、実行スピードと市場検証が不十分だった

### 3.2 ショート動画対応の遅れ（2020-2023年）

- **市場変化**: YouTubeショート動画の急速な普及
  - 2021年9-11月: ショート動画比率12.9%
  - 2022年9-11月: 39.1%
  - 2023年9-11月: 54.9%
- **収益性悪化**: ショート動画の広告単価は長尺動画の**1/10以下**（0.003円/再生 vs 0.05円/再生）
- **対応の遅れ**: 長尺動画に特化したまま、ショート動画での収益化モデル構築に失敗
- **結果**: アドセンス収益の大幅減少、5期連続赤字

## 4. 成長戦略

### 4.1 初期トラクション獲得

**2013年-2014年: トップクリエイター囲い込み戦略**
- HIKAKIN（チャンネル登録者数1,100万人）を共同創業者として迎える

- 2014年初頭40人 → 年末2,500人に急拡大（62.5倍）

**成長ドライバー**:
1. **口コミ**: トップYouTuberからの紹介で新規クリエイター獲得
2. **実績**: 企業案件の成功事例が次の案件を呼ぶ
3. **メディア露出**: YouTuberの社会的認知向上とともにUUUMも注目

## 6. 成功要因分析

### 6.1 主要成功要因

1. **ファーストムーバー優位性**
   - 2013年時点で日本初のYouTuberマネジメント専業企業
   - 市場創造者として「YouTuber」という職業を確立

2. **トップクリエイターとの共同創業**
   - HIKAKINとの信頼関係が最大の競争優位
   - トップクリエイター自身が株主となる利益一致構造

3. **クリエイター第一主義**
   - 「名前のない仕事」哲学：クリエイターのために何でもやる
   - 3,000万円の違約金を払ってでもクリエイターを守る姿勢
   - Buddy制度による1対1サポート

4. **ネットワーク効果**
   - クリエイター数13,818チャンネル（2022年）
   - 企業案件の選択肢増加がクリエイター満足度向上につながる好循環

5. **急成長市場での先行投資**
   - YouTube市場の急拡大期（2013-2017年）に集中投資
   - IPOまで4年という驚異的なスピード

### 6.2 タイミング要因

- **2013年**: YouTube日本市場の急成長期開始
- **スマホ普及**: 2012-2015年にスマートフォン普及率が急上昇、動画視聴が日常化
- **広告市場拡大**: 企業のデジタル広告シフト、インフルエンサーマーケティング認知向上
- **IPOタイミング（2017年）**: YouTuber市場のピーク期、投資家の期待最高潮

### 6.3 差別化要因

- **芸能事務所との違い**: タレント搾取型ではなく、クリエイター支援型
- **データドリブン**: YouTube分析に基づく戦略的サポート
- **透明性**: クリエイターとの収益分配ルールを明確化
- **コミュニティ形成**: UUUMクリエイター同士の交流・コラボ促進

## 8. orchestrate-phase1への示唆

### 8.1 需要発見（/discover-demand）

**成功パターン**:
- **現場観察**: YouTuberの商談に同行し、リアルな課題を直接観察
- **当事者との対話**: 50名以上のクリエイターと直接対話、課題の共通性（85%）を確認
- **市場タイミング**: 急成長市場の黎明期を捉える（2013年YouTube市場）

**失敗パターン（P2C事業）**:
- **需要の過大評価**: YouTuber人気と商品購買意欲の相関を検証不足
- **市場変化の見落とし**: ショート動画台頭による収益構造変化を軽視
- **7億円の棚卸評価損**: 需要検証なきまま大量在庫を抱えた

**示唆**:
- 定性調査（インタビュー）と定量検証（テスト販売）を両立させる
- 市場変化（プラットフォーム仕様変更）を常にモニタリング
- 「人気があるから売れる」という安易な仮説は危険

### 8.2 CPF検証（/validate-cpf）

**高評価ポイント**:
- **課題の深刻度**: 個人YouTuberが法人契約できない（Unworkable）
- **緊急性**: YouTube収益化にはスポンサー案件が必須（Urgent）
- **回避不可能性**: プラットフォームで活動する限り避けられない（Unavoidable）
- **WTP確認**: トップクリエイターが「喜んで手数料を払う」と明言

**検証手法**:
- 50名インタビュー → 85%が同じ課題を持つ
- 商談同行による課題の実地観察
- HIKAKIN自身が共同創業者になることで最強のWTP実証

**示唆**:
- CPFは数値だけでなく「当事者が自ら参画する」レベルの確信が重要
- 課題の共通性85%以上は強いシグナル
- 3U（Unworkable/Unavoidable/Urgent）すべて満たす課題は希少

### 8.3 PSF検証（/validate-10x）

**10倍検証の成功例**:
- 契約スピード: 2-3ヶ月 → 1-2週間（10x）
- 企業案件単価: 個人交渉 → プロ交渉で5倍向上
- 法的リスク: 個人対応 → 法務チーム審査で8倍軽減

**MVP手法（Concierge型）**:
- 創業者自身がHIKAKINの案件交渉を全て代行
- 初期はスケールしないが、顧客満足度95%で仮説検証
- その後Buddy制度で仕組み化

**競合優位性**:
- HIKAKIN共同創業という「信頼の証明」
- クリエイター第一主義（3,000万円違約金エピソード）
- ファーストムーバーによる圧倒的クリエイター数

**示唆**:
- Concierge MVPは初期検証に最適（スケールは後で考える）
- 10倍優位性は「時間」「コスト」「リスク」の複数軸で検証
- 信頼構築には「行動で示す」（違約金を払う覚悟）

## 9. 事業アイデア候補

この事例から着想を得られる日本向けビジネスアイデア:

### 1. **TikTokクリエイター特化マネジメント**
- UUUMはYouTubeに特化したが、TikTok/Instagram特化の後発マネジメント企業
- ショート動画ネイティブな収益化モデル（ライブコマース、企業案件）
- UUUMの失敗（ショート動画対応遅れ）を教訓に、最初からショート動画前提で設計

### 2. **ニッチ特化クリエイターエージェンシー**
- ゲーム実況特化、Vtuber特化、教育系YouTuber特化など
- UUUMのような総合型ではなく、特定ジャンルで深い専門性を提供
- クリエイター数100-300人規模で高単価案件を獲得

### 3. **クリエイター向けSaaS（収益多様化ツール）**
- UUUMが失敗したP2C事業をSaaSで支援
- クリエイターが自分でグッズ・商品を企画・販売できるプラットフォーム
- 需要予測AIで在庫リスク最小化
- 月額課金モデルでプラットフォーム依存リスクを分散

### 4. **地方クリエイター支援プラットフォーム**
- 地方在住クリエイター特化（UUUMは東京中心）
- 地方企業とのローカルタイアップ案件仲介
- 移住促進・地方創生と連携した新しいクリエイターエコノミー

### 5. **クリエイター教育・アカデミー事業**
- YouTuber養成スクール（UUUMも一部実施していたが本格化せず）
- 動画制作、著作権、税務、契約の実践的教育
- 卒業生をマネジメント契約する「育成→マネジメント」の垂直統合

## 11. 総括：成功と失敗の教訓

### 成功の本質

UUUM（鎌田和樹）の成功は、**「市場創造×ファーストムーバー×信頼構築」の三位一体**にあった。

1. **市場創造**: 「YouTuber」という職業を日本で初めて確立
2. **ファーストムーバー**: 2013年の黎明期に参入、4年でIPO
3. **信頼構築**: クリエイター第一主義を貫き、HIKAKIN等トップクリエイターの信頼を獲得

特に「3,000万円の違約金を払ってでもクリエイターを守る」という行動は、単なる理念ではなく**実行された哲学**であり、これがUUUMの最大の競争優位となった。

### 失敗の本質

一方で、UUUMの上場廃止（2025年2月）は、**プラットフォーム依存リスクの典型的失敗例**である。

1. **YouTube依存85%**: プラットフォーム仕様変更（ショート動画）に対応できず
2. **収益多角化の失敗**: P2C事業で7億円損失、検証不足のまま大量投資
3. **市場変化への適応遅れ**: ショート動画収益化モデルの構築に失敗

**重要な教訓**:
- プラットフォームビジネスは「依存」ではなく「共存」の設計が必須
- 多角化は正しいが、**CPF/PSF検証なき多角化は自殺行為**
- 創業者の退任（2023年）が組織の求心力を失わせた可能性

### スタートアップへの示唆

**再現すべき成功要素**:
- 急成長市場の黎明期を捉える嗅覚
- Concierge MVPによる徹底的な顧客理解
- トップ顧客（HIKAKIN）を共同創業者にする大胆さ
- 「クリエイター第一」という明確な価値観

**避けるべき失敗要素**:
- プラットフォーム依存度85%超の事業設計
- 検証なき多角化（P2C事業の7億円損失）
- 市場変化（ショート動画台頭）への対応遅れ
- 創業者の早期退任による求心力低下

UUUMは、**日本のクリエイターエコノミーを切り開いたパイオニア**として歴史に残る一方、**プラットフォームリスク管理の教科書的失敗例**としても語り継がれるだろう。
