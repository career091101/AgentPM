# Emily Weiss - Glossier

## 1. 基本情報

| 項目 | 内容 |
|------|------|
| 創業者 | Emily Weiss |
| 生年 | 1985年3月22日 |
| 国籍 | アメリカ |
| 学歴 | ニューヨーク大学スタジオアート専攻（2007年卒業） |
| 創業前経験 | Ralph Lauren インターン（高校時代）、Teen Vogue インターン、W Magazine ファッションアシスタント、Vogue スタイリングアシスタント |
| 企業名 | Glossier |
| 創業年 | 2014年（Into The Gloss: 2010年） |
| 業界 | 美容・化粧品（DTC） |
| 現在の状況 | 継続運営中（Emily Weissは2022年5月にCEO退任、現在はExecutive Chairwoman） |
| 評価額/時価総額 | $1.8B（2021年ピーク時） |

## 2. 創業ストーリー

### 2.1 課題発見（需要発見）

**着想源**:
- Vogueで編集アシスタントとして働いていた際、美容ブランドが顧客の声に耳を傾けていないことに気づいた
- 既存の美容業界はトップダウン型のアプローチが主流で、消費者の本当のニーズが反映されていなかった
- 女性が美容について語り、自分のストーリーを共有できるプラットフォームが存在しなかった

**需要検証方法**:
- 2010年、家族とコネチカット州のビーチで休暇中にブログのアイデアを着想
- $750の中古カメラを購入し、intothegloss.comのドメインを取得
- 「Top Shelf」シリーズでファッション業界の著名人のバスルームキャビネットを公開し、彼女たちの美容ルーティンをインタビュー
- 数ヶ月で数百万人がサイトに訪問するほどの反響を獲得

### 2.2 CPF検証（Customer Problem Fit）

**インタビュー/顧客検証**:
- 実施数: Into The Glossで数百件以上の「Top Shelf」インタビューを実施
- 手法: ブログでの対話型コンテンツ、コメント欄でのフィードバック収集
- 発見した課題の共通点:
  - 美容ブランドは消費者の声を聞いていない
  - 製品開発が消費者ニーズと乖離している
  - 自然な美しさを引き出す製品が少ない

**3U検証**:
- Unworkable（現状では解決不可能）: 既存ブランドは大手小売経由で販売し、直接的な顧客フィードバックを得る仕組みがなかった
- Unavoidable（避けられない）: 毎日使用する美容製品は消費者にとって避けられないニーズ
- Urgent（緊急性が高い）: SNS時代の到来により、消費者は自分の声が反映されることを期待するようになった

**支払い意思（WTP）**:
- 確認方法: Into The Glossの読者からの製品リクエストと、発売時の即座の購入行動
- 結果: 製品発売後、60,000人のウェイトリストが形成されるほどの需要を確認

### 2.3 PSF検証（Problem Solution Fit）

**10倍優位性**:

| 軸 | 従来の解決策 | 自社ソリューション | 倍率 |
|---|------------|-----------------|------|
| 顧客の声の反映 | 一方向的なマーケティング | コミュニティ主導のプロダクト開発 | 10x |
| コミュニティ | なし | Instagram中心の双方向コミュニケーション | 10x |
| 購入体験 | 小売店での複雑な購入プロセス | DTC（Direct-to-Consumer）モデル | 5x |
| 価格帯 | 高価格帯のラグジュアリー or 低価格帯のドラッグストア | 手頃な価格（$12-$26） | 3x |
| 美的アプローチ | 完璧なメイクアップ志向 | 「スキンケアファースト、メイクアップセカンド」の自然美志向 | 5x |

**MVP**:
- タイプ: ランディングページ + 4製品でのローンチ
- 初期製品（2014年10月）:
  1. Soothing Face Mist（フェイスミスト）
  2. Priming Moisturizer（プライミングモイスチャライザー）
  3. Balm Dotcom（リップモイスチャライザー）
  4. Perfecting Skin Tint（シアーファンデーション）
- 価格帯: $12-$26
- 初期反応: ローンチ1ヶ月後に$840万のシードラウンドを調達

**CVR**:
- オンライン売上・トラフィックの70%がピアツーピアリファラルから発生
- 顧客の約80%が友人からの紹介で購入

**UVP（独自の価値提案）**:
- 「ボトムアップ型美容ブランド」- 顧客が製品開発に参加できる
- 「スキンケアファースト、メイクアップセカンド」の哲学
- コミュニティの声を直接製品に反映

**競合との差別化**:
- 既存美容ブランドがセレブリティやインフルエンサーを起用するのに対し、一般消費者の声を重視
- DTC（Direct-to-Consumer）モデルによる直接的な顧客関係構築
- Instagramを活用したユーザー生成コンテンツ（UGC）マーケティング

## 3. ピボット/失敗経験

### 3.1 初期の失敗

**資金調達の困難**:
- 最初のピッチで12社中11社から拒否された
- 最終的にForerunner VenturesのKirsten Greenから$200万のシード資金を調達

**Glossier Play の失敗（2019-2020）**:
- 2019年にカラフルなメイクアップラインとして「Glossier Play」をサブブランドとして発売
- 失敗理由:
  - 本体ブランドの自然美志向と矛盾するビビッドなカラー展開
  - 購入者の2/3が既存Glossier顧客で、新規顧客獲得に失敗
  - サステナビリティへの批判（生分解性のないグリッター、過剰包装）
  - 売上不振
- 2020年にサブブランドとしての展開を中止し、一部製品を本体ブランドに統合
- Emily Weissの学び: 「新しいサブブランドを作る必要はなく、単にメイクアップ製品を追加すればよかった」

### 3.2 ピボット（該当する場合）

**ピボット1: ブログから製品へ（2014年）**
- 元のアイデア: Into The Gloss（美容ブログ/メディア）
- ピボット後: Glossier（DTC美容製品ブランド）
- きっかけ: ブログだけでは成長に限界があり、読者が本当に求めていたのは「話題にしている製品そのもの」だと気づいた
- 学び: メディアの信頼とコミュニティを製品開発のインサイトに活用できる

**ピボット2: DTC専売からオムニチャネルへ（2022-2023年）**
- 元のアイデア: DTCオンリーモデル（自社サイト+自社店舗のみ）
- ピボット後: Sephoraとの卸売パートナーシップによるオムニチャネル展開
- きっかけ:
  - 2022年1月に従業員の1/3（80人以上）をレイオフ
  - DTC成長の鈍化
  - 競争激化による市場シェア低下
- 結果:
  - 2023年2月にSephoraで販売開始（600店舗以上）
  - パートナーシップ初年度で$1億以上の売上増加
  - 6ヶ月で$1億近い売上、前年比73%成長

## 4. 成長戦略

### 4.1 初期トラクション獲得

- Into The Glossで4年間培った読者コミュニティをGlossierローンチ時に活用
- Kim Kardashianなどセレブリティのインタビューによる認知拡大
- 60,000人のウェイトリスト形成（9製品に対して）
- ローンチ1ヶ月で$840万のシード資金調達

### 4.2 フライホイール

```
コミュニティフィードバック → 製品開発 → 発売 → UGC(ユーザー生成コンテンツ) → 口コミ拡散 → 新規顧客獲得 → コミュニティ拡大 → さらなるフィードバック
```

**具体的メカニズム**:
1. Into The GlossやInstagramでフィードバック収集
2. 顧客の声を製品開発に反映（例: Milky Jelly Cleanser）
3. 製品発売時にコミュニティに先行告知
4. 顧客がInstagramで製品を投稿
5. GlossierがUGCをリポスト（「Top 5」Instagramストーリーズ）
6. 口コミで新規顧客が流入
7. 新規顧客がコミュニティに参加

### 4.3 スケール戦略

**資金調達履歴**:
- 2014年: シード $200万（Forerunner Ventures）
- 2015年: シリーズA $840万
- 2016年: シリーズB $2,400万
- 2018年: シリーズC $5,200万
- 2019年: シリーズD $1億（評価額$1.2B、ユニコーン達成）
- 2021年: シリーズE $8,000万（評価額$1.8B）
- 累計調達額: $2.66億以上

**成長マイルストーン**:
- 2018年: 売上$1億達成、Boy Browが32秒に1本売れるペース
- 2021年: ピーク評価額$1.8B
- 2023年: Sephora展開で年間売上$3億近くに

## 6. 成功要因分析

### 6.1 主要成功要因

1. **コミュニティ主導のプロダクト開発**
   - Milky Jelly Cleanserは400件のブログコメントから誕生
   - Boy Browも顧客からの「自然なふんわり眉毛」リクエストに応えて開発

2. **メディアからプロダクトへの転換**
   - 4年間のブログ運営で構築した信頼とコミュニティをレバレッジ
   - 顧客のニーズを深く理解した上での製品開発

3. **SNS時代のマーケティング最適化**
   - Instagramフォロワー320万人以上
   - UGCの活用でマーケティングコストを削減
   - 70%の売上が口コミ経由

### 6.2 タイミング要因

- 2010年代のInstagramとDTCブランドの台頭
- 「自然美」「スキンケアファースト」トレンドの到来
- 消費者のブランドに対する直接的な関与への期待の高まり
- ミレニアル世代の購買力向上

### 6.3 差別化要因

- 「ボトムアップ型」ブランド構築（業界初）
- トップダウン型の既存美容業界への明確なアンチテーゼ
- 製品開発への顧客参加を制度化

## 8. orchestrate-phase1への示唆

### 8.1 需要発見（/discover-demand）

- **メディア/コミュニティからの需要発見モデル**
  - ブログやSNSを通じて顧客の「言葉」を直接収集
  - コメント欄での対話から具体的なペインポイントを抽出
  - 「What's Your Dream Face Wash?」のような直接的な質問でニーズを言語化

- **適用可能な手法**:
  - 対象業界のブログ/YouTubeチャンネル/Instagramアカウントでの対話開始
  - コミュニティ内での製品アイデアのクラウドソーシング

### 8.2 CPF検証（/validate-cpf）

- **Into The Gloss式CPF検証**
  - インタビュー形式のコンテンツで顧客の本音を引き出す
  - コメント欄での反応を定量的に分析
  - 400件のコメント = 400件のCPFインタビュー相当

- **Glossierの教訓**:
  - 顧客が「欲しい」と言っているものを作る
  - 既存製品への不満を具体的に収集
  - 支払い意思はウェイトリスト登録数で検証

### 8.3 PSF検証（/validate-10x）

- **10倍優位性の構築**:
  - 顧客の声の反映: 既存ブランドがトップダウンなら、ボトムアップで10倍
  - コミュニティ: 一方通行マーケティングなら、双方向で10倍
  - Milky Jelly Cleanserの例: 40バージョンのプロトタイプを経て最適解を発見

- **MVP戦略**:
  - 4製品に絞ってローンチ
  - 既存コミュニティ（Into The Gloss読者）に先行提供
  - ウェイトリストで需要を可視化

### 8.4 スコアカード（/startup-scorecard）

**Glossierから学ぶ評価基準**:

| 評価項目 | Glossierの実績 | スコアカードへの示唆 |
|---------|--------------|-------------------|
| コミュニティサイズ | 320万人（Instagram）| コミュニティ規模を重要KPIに |
| 口コミ比率 | 70%が口コミ経由 | オーガニック獲得率を測定 |
| ウェイトリスト | 60,000人 | 発売前の需要指標として活用 |
| リピート率 | 80%が友人紹介経由 | NPS/リファラル率を追跡 |
| 製品開発サイクル | 顧客コメント→製品化 | フィードバックループの速度を評価 |

## 9. 事業アイデア候補

この事例から着想を得られる日本向けビジネスアイデア:

1. **日本版「Into The Gloss」→ コスメブランド**
   - 日本の美容ブロガー/インフルエンサーがコミュニティを構築し、そこからプライベートブランドを立ち上げ
   - ターゲット: 日本人の肌質・美意識に最適化された製品開発

2. **メンズ美容コミュニティ → DTC メンズスキンケア**
   - 日本のメンズ美容市場は成長中
   - Glossierモデルを男性向けに適用

3. **ペットケア版Glossier**
   - ペットオーナーコミュニティを構築
   - コミュニティフィードバックから製品開発
   - プレミアムペットケア製品のDTC展開

4. **地方特産品のDTCブランド**
   - 地域の生産者とのコミュニティ構築
   - 消費者との対話から商品開発
   - 「生産者の顔が見える」コミュニティ型EC
