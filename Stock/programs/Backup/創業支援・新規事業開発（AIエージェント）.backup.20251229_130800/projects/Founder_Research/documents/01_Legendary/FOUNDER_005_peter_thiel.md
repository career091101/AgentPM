# Peter Thiel - PayPal/Palantir/Founders Fund

## 1. 基本情報

| 項目 | 内容 |
|------|------|
| 創業者 | Peter Andreas Thiel |
| 生年 | 1967年10月11日 |
| 国籍 | ドイツ系アメリカ人 |
| 学歴 | スタンフォード大学（哲学BA 1989）、スタンフォード法科大学院（JD 1992） |
| 創業前経験 | Sullivan & Cromwell弁護士、Credit Suisseデリバティブトレーダー |
| 主要企業 | PayPal (1998), Palantir (2003), Founders Fund (2005) |
| 現在の状況 | Founders Fund Managing Partner、Palantir取締役 |
| 純資産 | $23.2B（2025年Forbes推計） |

## 2. 創業ストーリー

### 2.1 背景と教育

**幼少期**:
- 1967年、西ドイツ・フランクフルト生まれ
- 1歳で家族とともにアメリカに移住
- その後、南アフリカ・南西アフリカを経て1977年にアメリカ帰国
- カリフォルニア州フォスターシティで育つ
- San Mateo High Schoolで首席卒業、チェスチャンピオン

**スタンフォード大学時代**:
- 20世紀哲学を専攻、1989年にBA取得
- 哲学者ルネ・ジラールの模倣理論（mimetic theory）から強い影響を受ける
- 1987年、保守系新聞「The Stanford Review」を共同創刊
- 「西洋文化」プログラム廃止への批判がきっかけ

**法律家からビジネスへ**:
- スタンフォード法科大学院卒業（1992年）
- 第11巡回区控訴裁判所の法務書記官（1992-1993）
- Sullivan & Cromwell（NY）で証券弁護士として勤務
- 1年未満で退職、金融業界へ転身
- Credit Suisseでデリバティブトレーダーとして勤務

### 2.2 PayPal創業（1998-2002）

**Confinity設立**:
- 1998年12月、Max Levchin、Luke Nosekとともに「Fieldlink」を設立
- Thielは初期資金として$100,000を出資
- 1999年2月、友人・家族から$500,000を調達（Thielの両親から$35,000含む）
- 社名を「Confinity」に変更、Palm Pilot間の暗号化送金ソフトを開発

**PayPalへのピボット**:
- 当初の目標：Palm Pilot向けセキュリティソフトウェア
- 気づき：オンライン決済の隙間を発見
- 1999年、Eメールベースの送金サービス「PayPal」をローンチ
- クレジットカードを受け付けられない加盟店向けのデジタルウォレット

**X.comとの合併とCEO就任**:
- 1999年、Elon MuskのX.comと合併
- 2000年10月、Elon MuskがCEO解任、Peter ThielがCEOに就任
- 2001年6月、社名をPayPalに統一
- 2002年2月、NASDAQ上場（IPO）

**eBayへの売却**:
- 2002年、eBayが$1.5Bで買収
- Thielは3.7%の持分から$55Mを獲得

### 2.3 Facebook投資（2004年）

**歴史的な投資**:
- 2004年8月、19歳のMark Zuckerbergと初対面
- 当時Facebookは20の大学キャンパス、約10万ユーザー
- $500,000のエンジェル投資で10.2%の株式を取得
- 会社評価額：$4.9M
- Facebookへの初の外部投資家となる

**投資条件**:
- 当初は転換社債の形式
- 2004年末までに150万ユーザー達成が条件
- わずかに目標未達だったが、Thielは株式転換を許可
- 取締役にも就任（2022年まで）

**リターン**:
- 2012年5月のIPO時に16.8百万株を$638Mで売却
- 2012年8月、残りの株式の大半を$395.8Mで売却
- 総リターン：$1B以上（投資額の2,000倍）
- 仮に保有し続けていれば約$14.76B（2025年時点）

### 2.4 Palantir創業（2003年）

**9.11からの着想**:
- 2001年9月11日のテロ攻撃がきっかけ
- PayPalで開発した不正検知アルゴリズムをテロ対策に応用できないか
- Thiel曰く：「PayPalが詐欺対策に使ったアプローチを、テロ対策に拡張できると気づいた」

**設立とチーム**:
- 2003年5月、Palantir Technologiesを設立
- 社名はトールキンの「見る石（Seeing Stone）」から
- 共同創業者：Alex Karp（CEO）、Stephen Cohen、Joe Lonsdale、Nathan Gettings
- 2004年、PayPalエンジニアのNathan GettingsとスタンフォードのJoe Lonsdale、Stephen Cohenがプロトタイプを開発

**初期資金調達**:
- Thielが個人で$30Mを出資
- 2005年、CIAのベンチャー部門In-Q-Telが約$2M投資
- In-Q-Telの投資は金額以上に「信頼性の証明」として機能
- 政府・諜報機関との関係構築のきっかけに

**政府契約の獲得**:
- 初期顧客：米国諜報機関・国防総省
- In-Q-Telとの関係が政府契約獲得の土台に
- 2020年9月、NYSE上場

## 3. Zero to One哲学

### 3.1 基本概念

**0から1への進歩**:
- 水平的進歩（1からn）：既存のものをコピー・拡大
- 垂直的進歩（0から1）：まったく新しいものを創造
- 真のイノベーションは「0から1」の創造

**Contrarian Question（逆張りの問い）**:
> 「あなたが信じている真実で、他の多くの人が同意しないものは何か？」

- 良い回答：「多くの人はXを信じているが、真実はXの逆である」
- 起業家は「秘密」を持つべき——ほとんどの人が知らない重要な真実

### 3.2 競争否定論

**「競争は敗者のもの」**:
- 激しい競争は全員の利益を破壊する
- 最良のビジネスは独占企業——他にできないことをするから価値を獲得
- 競争＝縮小するパイの奪い合い、独占＝自分のパイを創造

**幸福な企業の法則**:
- トルストイの逆転：「すべての幸福な会社は異なる。すべての不幸な会社は同じ——競争から逃れられなかった」
- 各独占企業はユニークな問題を解決することで独占を獲得

### 3.3 独占構築の4要素

1. **独自技術（Proprietary Technology）**
   - 競合の10倍以上優れていること
   - 例：Googleの検索アルゴリズム

2. **ネットワーク効果（Network Effects）**
   - ユーザーが増えるほど価値が上昇
   - 例：PayPal、Facebook

3. **規模の経済（Economies of Scale）**
   - 大きくなるほどコスト効率が向上
   - 例：ソフトウェア企業

4. **ブランディング（Branding）**
   - 模倣困難な独自性
   - 例：Apple

## 4. 10倍優位性・独占戦略

### 4.1 10倍ルール

**基本原則**:
- 独自技術は競合の少なくとも10倍優れている必要がある
- 10倍未満は「わずかな改善」と認識され、販売困難
- 10倍改善の最も明確な方法：まったく新しいものを発明すること

**10倍改善の方法**:

| アプローチ | 例 |
|-----------|---|
| 新発明 | Google検索、PayPal送金 |
| 既存の10倍改善 | Amazon（本の品揃え）、iPad（タブレット体験） |
| 10倍安価 | SpaceX（打ち上げコスト） |
| 10倍高速 | Palantir（データ分析） |

### 4.2 市場攻略戦略

**小さく始める**:
- 最初のターゲット市場は非常に小さくあるべき
- 少数の特定の人々が集中し、競合がほぼいない市場
- 支配してから隣接市場へ拡大

**ラストムーバー・アドバンテージ**:
- 「ファーストムーバー」より「ラストムーバー」を目指す
- カテゴリーで最後の、永続的なリーダーになることが重要
- 長期的な市場支配を意識した戦略

### 4.3 Thielの投資基準

| 基準 | 内容 |
|------|------|
| 独占可能性 | 競争から逃れられる構造があるか |
| 10倍優位性 | 既存解の10倍以上優れているか |
| 秘密 | 他者が知らない重要な洞察があるか |
| チーム | 創業者に信頼を置けるか |
| タイミング | 今が適切な時期か |

## 5. 失敗経験と学び

### 5.1 Clarium Capital（2002-2013）

**設立と戦略**:
- 2002年、PayPal売却益$10Mで設立
- グローバルマクロヘッジファンド
- 手数料構造：0%管理費 + 25%成功報酬
- 核心的投資テーゼ：「ピークオイル理論」

**成功期（2002-2008前半）**:
- 2004年：運用資産$260M、設立来+125%
- 2006年：累積リターン+230%
- 2008年半ば：運用資産$7.8B、年初来+58%

**崩壊（2008後半-2013）**:
- 2008年：-4.5%
- 2009年：-25%
- 2010年：-23%
- 2011年：運用資産$350Mに縮小（90%減）
- 2013年：事実上の閉鎖

**失敗の原因**:
- マクロ予測に過度に依存したハイリスク戦略
- 金融危機後の市場回復を予測できず
- 「正しいアイデアを持っていても、実行が伴わなければ意味がない」

**学んだ教訓**:
- 予測困難なマクロ経済より、直接投資（VC）にシフト
- 長期的な価値創造へのフォーカス
- リスク分散とポートフォリオ管理の重要性

### 5.2 その他の失敗・批判

**投資の見落とし**:
- Facebookの早期売却（保有していれば$14.76B相当だった）
- 一部のスタートアップ投資の失敗

**Thiel Fellowshipへの批判**:
- ハーバード大学元学長Larry Summers：「この10年で最も誤った慈善活動」
- Thiel自身も「スケーラブルではない」と認める

## 6. 投資戦略

### 6.1 Founders Fund

**概要**:
- 2005年設立、サンフランシスコ拠点
- 2025年時点で運用資産約$17B
- 創業者主導の企業に投資

**投資哲学**:
- 「困難な問題を解決する賢い人々」に投資
- 逆張り的、長期的なベット
- 政治的に議論を呼ぶ領域でも躊躇しない

**投資先（主要企業）**:
- SpaceX（最初の機関投資家、$19.5B以上のゲイン）
- Palantir（共同インキュベート）
- Airbnb
- Stripe
- Spotify
- Anduril Industries（防衛テック）
- Neuralink
- Nubank

**投資戦略の進化**:
- 初期：インターネット・ソーシャルメディア、ヘルスケア
- 現在：ハードテック、防衛技術にシフト
- 2025年：Andurilに$1B投資（ファンド史上最大）

### 6.3 Thiel Fellowship

**プログラム概要**:
- 2010年設立、2011年開始
- 22歳以下の若者に$200,000を2年間提供
- 大学を中退し、スタートアップや研究に専念することを奨励

**成功事例**:
- Dylan Field（Figma CEO）
- Lucy Guo（Scale AI共同創業者）
- Mercor創業チーム（3名全員がThiel Fellow）
- プログラム卒業生が創出したビジネス価値：$100B以上

## 9. 成功要因分析

### 9.1 主要成功要因

1. **哲学的思考力**: スタンフォード哲学科で培った深い思考、ルネ・ジラールの影響
2. **逆張り思考**: 多数派と異なる視点を持ち、それに賭ける勇気
3. **ネットワーク構築**: PayPal Mafiaという強力な起業家・投資家コミュニティ
4. **長期視点**: 短期の利益より長期的な独占構築を重視

### 9.2 タイミング要因

- インターネット黎明期にオンライン決済市場に参入
- 9.11後のセキュリティ需要を先読みしてPalantir設立
- Facebook初期の可能性を見抜いた投資判断

### 9.3 差別化要因

- 弁護士・哲学者としてのユニークなバックグラウンド
- 「競争を避ける」という一貫した戦略
- 政府・諜報機関との関係構築能力
