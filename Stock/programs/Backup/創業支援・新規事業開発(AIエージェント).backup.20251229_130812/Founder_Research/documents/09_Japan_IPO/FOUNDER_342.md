# FOUNDER_342: Kenji Kasahara - mixi (ミクシィ)

## 1. 創業の経緯・課題認識

### 原体験・課題との出会い

Kenji Kasahara（笠原健治）は1975年12月6日、大阪府箕面市生まれ。東京大学在学中の1997年、シリコンバレーのIT産業の急速な成長とオンラインビジネスの事例研究を通じて、インターネットビジネスの可能性に目覚めた。

その体験から、日本におけるインターネット市場に大きな成長機会があると確信。1997年11月、大学在学中に人材紹介サイト「Find Job!」の運営を開始した。このプロジェクトを通じて、オンラインコミュニティと人間関係構築の重要性を学んだ。

2004年、Friendster等のアメリカのSNSサービスの存在を知り、日本のユーザー特性に合わせたソーシャルネットワーキングプラットフォームの構想を開始。特に、日本文化における「プライバシー」「信頼」「クローズドコミュニティ」への需要の高さに気づき、招待制のSNS「mixi」を立案した。

### 課題の定義

- **国内ソーシャルメディア市場の未成熟**: 2004年当時、日本ではMyspace等の海外SNSが浸透しておらず、日本文化に適応したプラットフォームが存在しなかった
- **プライバシーと信頼の課題**: 日本ユーザーは見知らぬ人からの接触やプロファイル公開に抵抗感が強く、安全性の高いクローズドコミュニティを求めていた
- **グローバル企業での日本向け最適化の不足**: FacebookやMyspaceは西洋文化を前提に設計されており、日本ユーザーのニーズとの乖離があった

### 創業チーム

- **Kenji Kasahara (CEO/創業者)**: 東京大学卒、Find Job! CEO、起業家的思考とエンジニアリング知見を保有
- **Batara Eto (初期メンバー/提案者)**: インターンから参加、ソーシャルメディアの重要性をKasaharaに提案。後のmixi発展に貢献
- **初期メンバー**: エンジニア、デザイナー等の小規模チーム（3-5名）から開始

## 2. ソリューション・事業内容

### プロダクト/サービス概要

**mixi** は、招待制（Invitation Only）のクローズドソーシャルネットワーキングサービス。ユーザーは既存ユーザーからの招待を通じてのみ参加でき、プロフィール、日記、コミュニティ等を通じて友人関係を構築できるプラットフォーム。

**主要機能:**
- **招待制ネットワーク**: 登録ユーザーのみが招待可能。知り合いとのみ繋がるため、安全性と信頼性が高い
- **プロフィール機能**: 自分自身の情報を公開し、友人との共通点を発見
- **日記機能**: 日常の出来事や思いを記述、友人とのコメント交流
- **コミュニティ**: 特定のトピック（趣味、興味、地域など）に基づいたグループ。ユーザーは複数のコミュニティに参加可能
- **マイミク機能**: 互いに認めた友人を「マイミク」として登録
- **メッセージング**: プライベートメッセージ送受信

### 差別化ポイント

1. **招待制による安全性と排他性**: MyspaceやFacebookとは異なり、見知らぬ人からのアクセスがなく、既知の友人のみとのネットワーク形成を実現
2. **日本文化への適応**: プライバシー重視、クローズドコミュニティ志向の日本ユーザーのニーズに合致
3. **コミュニティ中心設計**: 趣味やニーズに基づいたグループ形成により、単なる人間関係ネットワークを超えた価値提供
4. **シンプルで直感的なUI**: 複雑な設定を避け、日本の一般ユーザーでも容易に使用可能な設計

### ビジネスモデル（複数段階での進化）

**フェーズ1 (2004-2010): 広告モデル**
- ユーザー数増加に基づく広告掲載による収益化
- バナー広告、スポンサーシップ

**フェーズ2 (2010-2013): ゲーム内アイテム販売**
- プラットフォーム上でのゲーム提供開始
- Virtual Goods、アイテム販売による内課金モデル

**フェーズ3 (2013-現在): Monster Strike主軸モデル**
- 2013年リリースの「Monster Strike」が爆発的ヒット
- In-app Purchases（特にGacha系）で全収入の70%以上を占有

## 3. 市場環境・競合分析

### 市場規模と成長

- **2004年当時**: 日本SNS市場未成熟。高速インターネット普及率は50%程度
- **2006年IPO時点**: 日本SNS市場が急速に拡大。登録ユーザー数4.7-6.6百万人、日本のSNS市場シェア最大
- **2013年-2015年**: モバイルゲーム市場の爆発的成長（年率30%以上）
- **2024年**: 日本モバイルゲーム市場 約1.8兆円、うちmixi（Monster Strike等）が有力プレイヤー

### 競合状況

**直接競合 (SNS, 2004-2012年)**
- **MySpace**: 当時のグローバルSNS主要プレイヤー。ただし日本ユーザーからは距離感
- **mixi以前の日本SNS**: Friendster等。招待制の要素もあったが、UIやコミュニティ機能でmixi が優れていた
- **Facebook**: 2008年に日本展開開始。全世界ネットワークと強力な機能で、長期的にはmixiの脅威

**間接競合 (ゲーム, 2013年以降)**
- **Puzzle & Dragons (GungHo)**: 2012年リリース。Monster Strikeと同等レベルの高収益ゲーム
- **グランブルーファンタジー、FGO**: 同じアクションRPGカテゴリの競合
- **パズドラ、モンスト以外のGacha系**: 多数のスマートフォンゲーム

### 参入障壁

1. **ネットワーク効果**: SNS、特に招待制ではユーザーベースが重要。先着優位性が高い
2. **ユーザー習慣の変更**: 一度構築された友人ネットワークからの移行は困難
3. **ゲーム開発の高難度性**: Monster Strikeレベルのヒット作品開発には、高度な企画力、UX/UI、エンジニアリング、マーケティングが必要
4. **プラットフォーム依存性**: App Storeのランキング、ストアサーチアルゴリズム、推奨アルゴリズムに大きく左右される
5. **知的財産**: Monster Strikeのキャラクター、ゲームメカニクスの独自性

## 4. 成長プロセス

### 初期段階 (1999-2004)

- **1997年11月**: Find Job!運営開始
- **1999年6月**: E-Mercury Co., Ltd. 設立、法人化
- **2004年2月**: mixi サービス正式ローンチ

### トラクション獲得・初期成長 (2004-2006)

- **2004年**: ローンチから数ヶ月で数千ユーザー。招待制の口コミにより緩やかに成長
- **2005年**: ユーザー数 100万人突破。メディアでの注目開始
- **2006年6月**: ユーザー数 470万人を達成
- **2006年9月**: 東京証券取引所 Mothers市場に上場（IPO）

### 成長加速期 (2006-2012)

- **2006年9月IPO時**: IPO価格 155万円。初日から注文殺到、指値で上場。公開価格の90%高で終値
- **2007年7月**: ユーザー数 1,000万人達成
- **2008年**: ピークユーザー数 約2,200万人に達する。しかし2008年のリーマンショック後、広告収入減少による経営難に直面
- **2010年**: モバイルアプリ化により、スマートフォン時代への対応開始

### ターンアラウンド期 (2012-2015)

- **2013年6月**: **Monster Strike** リリース。初期反応は中程度だったが、夏から秋にかけてバイラル成長
- **2013年Q4**: Monster Strike が App Store ランキング上位を占有。月間売上が数十億円レベルに
- **2014年**: Monster Strike が全ゲーム中で最高売上を記録。Mixi の営業利益が対前年比 10,871% 増（ピーク時）
- **2015年3月**: Monster Strike の累計ダウンロード 3,000万件以上、日次売上 380万ドル（約4.6億円）

### スケール戦略

**SNS期 (2004-2012)**
1. **口コミ・バイラル**: 招待制による自然な拡散。友人から招待される喜びが増幅
2. **メディア露出**: テレビ、雑誌での紹介により、認知度向上
3. **プロダクト改善**: ユーザーフィードバックに基づき、コミュニティ機能等を継続的に拡張
4. **モバイル対応**: iPhoneやAndroidの普及に合わせ、アプリ化

**ゲーム期 (2013-現在)**
1. **Monster Strike**: 独自開発のアクションRPG。AI、イラスト、ゲームメカニクスに高度な工夫
2. **マルチプラットフォーム**: iOS、Android両対応。後にPC、コンソールへも展開
3. **コンテンツ継続配信**: 定期的な新ステージ追加、キャラクター実装、イベント開催によるエンゲージメント維持
4. **IP化**: アニメ化、グッズ販売、カードゲーム等による横展開

## 8. 課題と解決アプローチ

### 直面した主な課題

**課題1: グローバル企業との競争激化 (2008-2012)**
- **背景**: Facebook の日本進出、iPhone 登場による携帯ゲーム市場の急成長で、mixi SNS は相対的な価値低下
- **ユーザー数のピークアウト**: 2008年の2,200万人がピーク。その後は減少傾向
- **広告収入の不安定性**: リーマンショック後、広告費削減により収入急減
- **解決策**: ゲーム事業への大型投資。2013年 Monster Strike リリースで、収益源の多角化と急成長を実現

**課題2: SNS 事業の成熟化と衰退 (2010-2013)**
- **背景**: Facebook、Twitter、LINE 等のグローバル・プラットフォームに機能面で劣る
- **ユーザー離脱**: スマートフォンシフトに伴い、若年ユーザーの流出
- **UI/UX の老朽化**: 2004年設計のコミュニティ機能が、モダンな競合と比較して洗練されていない
- **解決策**: mixi の SNS 事業は維持（コミュニティは根強い利用者が存在）しつつ、ゲーム事業へのリソース配分をシフト

**課題3: Monster Strike の持続性と後続タイトルの開発 (2014-現在)**
- **背景**: Monster Strike は日本国内でのヒット。しかし日本市場は飽和気味で、海外展開が困難
- **後続タイトルの開発困難**: Monster Strike レベルのヒット作を継続的に生み出すことは困難
- **解決策**:
  - Monster Strike への継続的なコンテンツ投資（新キャラ、新ストーリー、イベント等）で、ユーザーロイヤリティを維持
  - 複数の新作ゲーム開発を並行実施（開発成功率を高める戦略）
  - スポーツ事業（Jリーグ等への投資）、ライフスタイル事業（みてね等）への多角化

**課題4: 経営陣の世代交代と組織体制の刷新 (2015-現在)**
- **背景**: Kasahara は創業以来 CEO を務めてきたが、組織の成長に伴い、専門的な経営体制の構築が必要に
- **解決策**: CEO の座を後任に譲り、自身は Chairman に転任。事業ごとのCOO等の配置により、ガバナンス体制を強化

### 現在の課題

1. **ゲームポートフォリオの構成**: Monster Strike 一本足の脱却が不十分。新作ゲーム等の収益化が鍵
2. **海外展開の限定性**: 99% が日本での売上。グローバル市場での競争力確保が課題
3. **人材確保**: ゲーム開発、データサイエンス等の高度な人材を継続的に確保する必要性
4. **多事業化のシナジー**: ゲーム、スポーツ、ライフスタイル各事業間での シナジーを引き出す仕組みづくり

## 9. 学び・洞察

### 成功要因

1. **市場特性への深い理解**: グローバル企業が見落とした、日本文化（プライバシー重視、クローズド性志向）を SNS 設計の中心に据えた
2. **タイミングの妙**: 2004年の招待制SNS は、スパムやプライバシー侵害への懸念が高まっていた時期。この問題を招待制で解決したことが評価
3. **プロダクト・マーケット・フィット**: 当初の SNS 事業で、完全にマーケット・フィットを達成（登録ユーザー数2,000万人規模）
4. **失敗から学んだピボット**: SNS 事業の成熟化に直面した際、同じプラットフォーム（ユーザーベース）を活用して、ゲーム事業への転換を敢行。リスクが高い決定だったが、成功
5. **継続的なコンテンツ投資**: Monster Strike は「一発屋」に終わらず、年単位での新ストーリー、新キャラ、イベント開催により、エンゲージメント維持に成功

### 失敗・ピボット

**SNS 事業のピークアウト (2008-2012)**
- ユーザー数の増加は止まり、Facebook や Twitter への移行が進行
- ただし「失敗」ではなく、成熟化段階。主婦層やシニア層からは根強い利用者がいた

**複数の新作ゲーム開発の失敗 (2010-2015)**
- Monster Strike 以外のゲーム（例: エレメンタルストーリー、戦国IXA 等）は、成功度のばらつきが大きい
- ただし多数試行戦略により、失敗の影響を最小化

### 重要な意思決定

1. **招待制設計**: 当初は制約に見えたが、差別化の源泉となった
2. **モバイル対応 (2010年)**: PC から携帯・スマートフォンへのシフトを早期に判断
3. **ゲーム事業への大型投資 (2010-2013)**: SNS 事業の衰退を見据え、ゲーム開発への組織的・財務的リソース配分をシフト
4. **Monster Strike のグローバル展開の制限判断**: 国内市場での圧倒的な成功に専注し、無理な海外展開は避けた
5. **複数事業ポートフォリオ化 (2015-現在)**: ゲーム、スポーツ、ライフスタイル等への多角化

## 11. 創業者の特徴・思考

### 思考パターン（Kenji Kasahara）

1. **ユーザー中心主義**: 「何がユーザーに必要か」を常に問い直す。Find Job! から mixi への転換も、ユーザーニーズの変化を察知した結果
2. **市場機会の洞察**: シリコンバレーの事例から、日本市場での同種ビジネスの成功可能性を読み取る。また、2008-2012年の SNS 衰退を予測し、ゲーム事業への転換を決断
3. **文化的相対主義**: グローバル企業（Facebook、MySpace）をそのまま模倣するのではなく、日本文化の特殊性（プライバシー重視、信頼重視）を理解し、プロダクト設計に反映
4. **長期忍耐**: SNS 事業の成熟化後も、ゲーム事業が軌道に乗るまでの数年間、赤字や低採算を甘受。短期的な利益より、長期的な企業体質の改善を優先

### 意思決定スタイル

- **データと直感の融合**: ユーザーフィードバック、マーケット分析（データ）と、自身の起業的直感を組み合わせた判断
- **リスク許容度が高い**: ゲーム事業への全社的リソース シフトは、高リスク・高リターンな決定。これを実行できたのは、創業者としての決定権と、ユーザーへの信頼感が土台
- **段階的な撤退ではなく、大胆なピボット**: SNS 事業の完全な放棄ではなく、ゲーム事業との併立。後発のスポーツ、ライフスタイル事業も同時並行

### リーダーシップ特性

- **ビジョンの明確さ**: 「日本文化に合わせたSNS」「ゲーム業界でのトップポジション」など、わかりやすく、かつ達成可能性のあるビジョンを提示
- **チーム編成の工夫**: 初期段階での小規模チーム（3-5名）から、成長に伴い、分野別の有力エンジニア、プロデューサーを採用。特にゲーム開発では、業界経験者を登用
- **失敗への向き合い方**: Monster Strike 以外の新作ゲームの相次ぐ失敗にも、企業体質としての失敗を受け入れ、複数試行戦略を継続

## 12. 追加情報・特記事項

### 問題の共通性（Problem Commonality）

**72% (推定)**

**根拠**:
- 日本におけるプライバシー・信頼・セキュリティへの関心は、グローバル平均（50-60%）より高い
- オンライン上での個人情報提供への抵抗感は、日本で72% に及ぶ（総務省調査等）
- 特に2004年当時、スパムメール、迷惑コンタクト、個人情報流出への懸念が社会的課題
- mixi の招待制は、この問題を直接的に解決する価値提案であった

### インタビュー数（ユーザー検証）

**500+ (推定)**

**根拠**:
- SNS のプロダクト開発には、大規模なユーザーテストが不可欠
- 初期段階での友人ネットワークテスト（50-100人規模）から、数百万ユーザーへのスケール過程で、継続的にユーザーフィードバックを収集
- ゲーム開発（Monster Strike）では、ベータテスト、オープンテスト等で数千～数万人のテストプレイヤーを動員
- ただし「カスタマーインタビュー」というより、プロダクトテスト、アンケート、ユーザーレビュー分析が中心

### ビジネスモデルの進化

**フェーズ1 (2004-2010): 無料プラットフォーム + 広告**
- ユーザーは無料で SNS に参加
- mixi の収益源は、バナー広告、スポンサーシップ

**フェーズ2 (2010-2013): 広告 + アイテム販売**
- mixi モバイル版で、ゲーム内アイテム販売を開始
- Virtual Goods（武器、スキン等）への課金

**フェーズ3 (2013-現在): In-App Purchases（ガチャ中心）**
- Monster Strike で、ガチャシステム（確率的に強力なキャラを獲得するための課金）を実装
- 売上の 70% 以上が In-App Purchases から発生

### 社会的インパクト

- **SNS 文化の形成**: mixi コミュニティを通じて、日本人の「オンラインコミュニティ参加」が一般化
- **ゲーム産業への貢献**: Monster Strike のヒットにより、日本のモバイルゲーム産業が国際的に注目。その後の国産ゲーム（FGO、グランブルーファンタジー等）の成功を促進
- **プライバシー意識**: 招待制 SNS の価値提案を通じて、個人情報保護の重要性を社会に啓発

### 競合優位性の持続性

1. **先行者優位性**: 招待制 SNS の先駆けとして、日本市場での強い地盤を確立。Facebook 等の後発企業も、日本市場では「招待制」の特性は採用しなかった
2. **ユーザーロイヤリティ**: SNS ユーザーの高い継続率。主婦、シニア層での継続利用が強い
3. **Monster Strike のブランド**: キャラクター、ストーリー、ゲームメカニクスが独自。他社での追随は困難
4. **ただし持続性に課題**: Monster Strike のユーザー数は 2022-2024年で 減少傾向。後続タイトルで代替売上を生み出すことに成功していない

### 今後の展望

1. **ゲームポートフォリオの多様化**: Monster Strike への依存を低減し、複数の中規模タイトルでの収益化を目指す
2. **海外市場での新展開**: アメリカ、ヨーロッパでの新規ゲーム等の展開を検討
3. **Web3/ブロックチェーン**: NFT、メタバース関連のゲーム開発への参入の可能性
4. **スポーツ、ライフスタイル事業の成長**: Jリーグの所有権獲得（FC町田ゼルビア）、みてね等のライフスタイルアプリの拡大

### 教訓

1. **市場特性の理解が競争優位の源泉**: グローバル企業の模倣ではなく、ローカル市場の文化・ニーズの深い理解が成功の鍵
2. **成熟化段階での大胆なピボット**: SNS の衰退を見据え、同じプラットフォーム・ユーザーベースを活用して、新事業（ゲーム）へ転換。失敗リスクは高かったが、決断が適切だった
3. **長期的な R&D投資と複数試行**: Monster Strike のような大ヒット作は、多くの失敗を通じてのみ可能。失敗を許容する企業文化が重要
4. **創業者の役割の進化**: CEO から Chairman への転任を通じて、組織のスケーラビリティを高め、後進の経営者に経営権を委譲。創業者として企業の長期的方向性を示しながら、日常経営から距離を置く判断
