# AI企業評価フレームワーク

**作成日**: 2026-01-01
**プロジェクト**: AI_Timeline_Forecast
**目的**: AI関連企業への投資判断を支援する評価フレームワークの構築

---

## 1. 評価フレームワーク概要

### 1-1. 評価の3軸

AI企業の評価は以下の3軸で行う：

```
┌──────────────────────────────────────────┐
│  AI企業評価 = f(技術力, 市場性, タイミング) │
└──────────────────────────────────────────┘
         │           │           │
    ┌────▼────┐ ┌────▼────┐ ┌────▼────┐
    │技術力    │ │市場性    │ │タイミング│
    │(40%)    │ │(40%)    │ │(20%)    │
    └─────────┘ └─────────┘ └─────────┘
```

| 評価軸 | ウェイト | 評価内容 | スコア範囲 |
|--------|---------|---------|-----------|
| **技術力** | 40% | AI技術の先進性、研究開発力、人材 | 0-40点 |
| **市場性** | 40% | ビジネスモデル、収益性、成長性 | 0-40点 |
| **タイミング** | 20% | AI Timelineマイルストーンとの関連性 | 0-20点 |
| **総合** | 100% | 加重平均 | 0-100点 |

---

## 2. 評価軸の詳細定義

### 2-1. 技術力（40点満点）

| 評価項目 | 配点 | 評価基準 |
|---------|------|---------|
| **AIモデル開発力** | 15点 | ・自社独自のAIモデル開発実績<br>・学術論文・特許の質と量<br>・ベンチマークでの性能 |
| **研究開発投資** | 10点 | ・R&D投資額（売上比）<br>・研究者数・質<br>・トップ研究者の在籍 |
| **技術インフラ** | 10点 | ・計算インフラ（GPU/TPU）<br>・データセットの質と量<br>・スケーラビリティ |
| **技術差別化** | 5点 | ・競合との技術的差別化<br>・独自技術の有無<br>・技術的モート |

**スコアリング例**:
- **15点（最高）**: GPT-4レベルのモデル開発、トップカンファレンス採択論文多数
- **10点（高）**: 独自モデル開発、業界標準レベルの性能
- **5点（中）**: 既存モデルのファインチューニング・応用
- **0点（低）**: AI技術の独自開発なし

---

### 2-2. 市場性（40点満点）

| 評価項目 | 配点 | 評価基準 |
|---------|------|---------|
| **ビジネスモデル** | 15点 | ・収益化モデルの明確性<br>・スケーラビリティ<br>・継続課金（SaaS等）の有無 |
| **市場規模・成長性** | 10点 | ・TAM（市場総額）<br>・市場成長率（CAGR）<br>・市場の将来性 |
| **財務健全性** | 10点 | ・売上成長率<br>・利益率またはユニットエコノミクス<br>・資金調達状況 |
| **競争優位性** | 5点 | ・市場シェア<br>・ネットワーク効果<br>・ブランド力 |

**スコアリング例**:
- **15点（最高）**: 明確な収益化、急成長中、高利益率
- **10点（高）**: 収益化済み、成長中、黒字または黒字化見込み
- **5点（中）**: 収益化モデルあり、成長初期段階
- **0点（低）**: 収益化モデル不明確、成長停滞

---

### 2-3. タイミング（20点満点）

**AI Timelineマイルストーンとの関連性を評価**

| 評価項目 | 配点 | 評価基準 |
|---------|------|---------|
| **短期マイルストーン関連性** | 10点 | ・2025-2027年のマイルストーンとの関連<br>・GPT-5、Claude 4等のリリース影響<br>・コーディング自動化の恩恵 |
| **長期マイルストーン関連性** | 10点 | ・2027年AGI到来への対応<br>・2030年代超知能時代への準備<br>・AI規制動向への対応 |

**スコアリング例**:
- **20点（最高）**: AGI到来で事業が飛躍的拡大、短期マイルストーンでも恩恵大
- **15点（高）**: AGI時代の主要プレイヤー候補、短期でも成長加速
- **10点（中）**: AI発展の恩恵あり、ただし間接的
- **5点（低）**: AI発展との関連性薄い
- **0点（リスク）**: AGI到来で事業が脅威にさらされる

## 3. AI企業のカテゴリ分類

### 3-1. レイヤー別分類

```
[AIバリューチェーン]
┌────────────────────────────────────────────┐
│ Layer 4: アプリケーション                    │
│ (ChatGPT, Claude, Gemini等のユーザー向けAI)  │
└────────────────────────────────────────────┘
                  ↑
┌────────────────────────────────────────────┐
│ Layer 3: AIモデル・API                       │
│ (OpenAI API, Anthropic API, Vertex AI等)   │
└────────────────────────────────────────────┘
                  ↑
┌────────────────────────────────────────────┐
│ Layer 2: AI開発ツール・プラットフォーム        │
│ (Hugging Face, LangChain, Vector DB等)     │
└────────────────────────────────────────────┘
                  ↑
┌────────────────────────────────────────────┐
│ Layer 1: インフラ・ハードウェア               │
│ (NVIDIA, AMD, Google TPU, クラウド等)        │
└────────────────────────────────────────────┘
```

| レイヤー | 主要企業例 | 投資魅力度 | リスク |
|---------|----------|----------|--------|
| **Layer 4: アプリ** | OpenAI, Anthropic, Google, Microsoft | 高（直接的な収益化） | 高（競争激化） |
| **Layer 3: モデル/API** | OpenAI, Anthropic, Mistral, Cohere | 高（基盤技術） | 中（非上場多い） |
| **Layer 2: ツール/PF** | Hugging Face, Databricks, Scale AI | 中（エコシステム） | 中（差別化困難） |
| **Layer 1: インフラ** | NVIDIA, AMD, TSMC, AWS, Azure, GCP | 中〜高（安定収益） | 低（確立された市場） |

---

### 3-2. ビジネスモデル別分類

| カテゴリ | 説明 | 主要企業例 | 収益化 |
|---------|------|----------|--------|
| **モデル開発企業** | 独自AIモデルの開発・提供 | OpenAI, Anthropic, Google DeepMind | API課金、ライセンス |
| **垂直統合企業** | モデル開発〜アプリまで一貫提供 | Microsoft (Copilot), Google (Gemini) | サブスク、広告 |
| **開発ツール企業** | AI開発者向けツール・PF | Hugging Face, LangChain, Pinecone | SaaS課金 |
| **インフラ企業** | GPU、クラウド、ストレージ | NVIDIA, AMD, AWS, GCP | ハードウェア、クラウド |
| **応用特化企業** | 特定業界・用途向けAI | Jasper (マーケ), Synthesia (動画), Harvey (法務) | SaaS課金 |
