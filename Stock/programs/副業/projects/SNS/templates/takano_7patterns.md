## パターン選択フローチャート

```
入力のテーマは？
├─ 時事ニュース・大型発表 → パターン1（断定型主張）or パターン3（ニュース深掘り）
├─ スキル・キャリア・経営哲学 → パターン2（問題提起 → 反論 → 正論）
├─ 業界トレンド・複数データ → パターン4（リスト型衝撃ファクト）
├─ イベント・登壇報告 → パターン5（体験レポート型）
├─ サービス・人材紹介 → パターン6（Call-to-Action型）
└─ コミュニティ・サロン訴求 → パターン7（有料サービス型）
```

## パターン1: 断定型主張 → データ展開 → 読者問いかけ

### 構造
```
[強烈な主張・タイトル]

[背景データ・統計]
[具体的事例1]
[具体的事例2]
[自己の解釈・洞察]

[読者への問いかけ]
```

### テンプレート
```
[企業名・話題]が[数値]の[アクション]、[断定的評価]の件。

[人名・企業名]、完全に[状態]。この規模の[アクション]、[業界・歴史]で見たことない。
[数値データ1]。

でもこれ、ただの[表面的解釈]じゃない。注目すべきは[本質的ポイント]。
[詳細データ1]: [数値・事実]
[詳細データ2]: [数値・事実]
[詳細データ3]: [数値・事実]

[権威者名]、[発言の場]で言い切ってた。
「[引用]」って。

あなたはこれ、[成功すると思う？/どう思う？]
```

### 実例
```
SoftBankが3.5兆円ぶち込んだOpenAI投資、マジでヤバい件。

孫正義、完全に本気モード入ってる。この規模の投資、日本企業史上見たことない。
225億ドル追加で出資比率11%。

でもこれ、ただの金の話じゃない。注目すべきは資金調達の方法。
Armの株を担保にしたローンで115億ドル、つなぎ融資で150億ドル。
負債カバー率は10%台で、まだ余裕ありまくり。

孫さん、6月の株主総会で言い切ってた。
「OpenAIは地球上で最も価値ある会社になる」って。

あなたはこれ、成功すると思う？
```

### 推奨用途
- 時事ニュース解説
- 大型投資・M&A発表
- 業界動向分析

### 文字数目安
500-800字

## パターン2: 問題提起 → 反論 → 正論展開

### 構造
```
[タイトル: 〇〇なら、〜するな]

[問題提起]
[よくある誤解・甘い考え]
[その反論]

[正論1]
[正論2]
[正論3]

[最終結論]
```

### テンプレート
```
[職業・立場]になりたいなら、『[スキル・行動]』だけは逃げるな。

[スキル]が必要な理由はシンプル。[理由]。
[失敗パターン]。それだけ。

「[よくある言い訳1]」って? 甘い。[反論1]。
「[よくある言い訳2]」? [反論2]。

[真実1]。
[真実2]。[強調点]だから。
[真実3]。

[覚悟・行動指針]。
この覚悟がないなら[職業]になるな。
```

### 実例
```
経営者になりたいなら、『営業』だけは逃げるな。

起業するなら絶対に必要な資質がある。それが営業力だ。
会社が潰れる理由はシンプル。金がないか、売れないか。

「営業のプロを雇えばいい」って? 甘い。
そのプロがあなたを裏切らない保証はどこにある。

創業社長は自分で営業するしかない。経験の有無は関係ない。
営業力は何歳からでも磨ける。人間力でありコミュニケーション力だから。

自分と家族を食わせる金は自分で稼ぐ。
この覚悟がないなら経営者になるな。
```

### 推奨用途
- スキル論
- キャリア論
- 経営哲学

### 文字数目安
400-600字

## パターン3: ニュース引用 → 深掘り解説 → 示唆

### 構造
```
[タイトル: [組織・業界]に起きている「[評価]」]

[ニュース要約]

答えは[単純な理由]。

[詳細データ1]
[詳細データ2]
[詳細データ3]

でも、ここからが本当の話だ。
[裏側の真実]

[今後の示唆・課題]
```

### テンプレート
```
[組織・業界]に起きている「[衝撃的評価]」

[統計庁・機関名]が発表した[データ名]で、[数値]を記録した。
「[一般的な見方]」と言われてきた[業界・分野]で、なぜこんなことが起きているのか。

答えは単純だ。[本質的理由]。

[詳細1]: [数値・事実]
[詳細2]: [数値・事実]
[詳細3]: [数値・事実]

でも、ここからが本当の話だ。
[表面では見えない問題点・矛盾]

[業界・読者]の未来は、そこにかかっている。
[問いかけ]
```

### 実例
```
日本の労働市場に起きている「ありえない現象」

人口が減ってるのに、働く人が過去最多。
総務省が発表した労働力調査で、2025年11月の労働力人口が7033万人を記録した。

答えは単純だ。今まで働いていなかった人たちが、次々と労働市場に入ってきている。

女性の労働参加が止まらない: 11月時点で3228万人、前年比46万人増。
60代、70代も働き続けている: 65歳以上の労働力人口は961万人。
外国人労働者も急増中: 2024年10月時点で230万人。

でも、ここからが本当の話だ。
働く人は増えたが、働く時間は減っている。

労働力7000万人時代は始まったばかりだ。
これを本物の成長につなげられるかどうか。今が正念場。
```

### 推奨用途
- 統計データ解説
- 業界動向分析
- マクロ経済トレンド

### 文字数目安
700-1200字

## パターン4: リスト型・衝撃ファクト連打

### 構造
```
[タイトル: [年号]、[常識]が完全に壊れる]

[見出し1]: [衝撃的事実]
[詳細]
→ [解釈・結論]

[見出し2]: [衝撃的事実]
[詳細]
→ [解釈・結論]

[見出し3]: [衝撃的事実]
[詳細]
→ [解釈・結論]

今すぐやるべきこと:
[ターゲット1]のあなたへ:
- [アクション1]
- [アクション2]
- [アクション3]

[ターゲット2]のあなたへ:
- [アクション1]
- [アクション2]

[挑発的問いかけ]
```

### テンプレート
```
[年号]、[常識・業界ルール]が完全に壊れる衝撃の事実

[見出し1]:
[データ・統計]
[詳細説明]
→ [だから何が起きるか]

[見出し2]:
[データ・統計]
[詳細説明]
→ [だから何が起きるか]

[見出し3]:
[データ・統計]
[詳細説明]
→ [だから何が起きるか]

今すぐやるべきこと:
[ターゲット1]のあなたへ:
- [具体的アクション1]
- [具体的アクション2]
- [具体的アクション3]

[ターゲット2]のあなたへ:
- [具体的アクション1]
- [具体的アクション2]

あなたの会社、生き残れる？
```

### 実例
```
2026年、採用の常識が完全に壊れる衝撃の事実

ESで差がつかない時代が来た:
就活生の90%がAI、ChatGPTなど使ってエントリーシート書いてる
企業の人事も当然それを知ってる
→ もう誰も紙の文章なんか信じてない

初任給34万円が「普通」になる世界:
三菱商事が2026年卒に提示した額：34万円
ソニーの初任給：31.3万円
これを聞いて「うちは22万だけど、やりがいがあるから」なんて言ってる会社
学生は鼻で笑ってスルーしてる

リモート求人8%に応募が35%殺到する異常事態:
求人全体のたった8%しかないリモート可の仕事なのに
応募者の35%がそこに集中してる
→ 週5出社を求める企業は、残り65%の応募者を92%の求人で取り合ってる

今すぐやるべきこと:
就活生・転職者のあなたへ:
- ChatGPT Plus契約する（月20ドルケチるな）
- 何でもいいから作ってGitHubに上げる
- リモート可の企業だけエントリーする勇気を持つ

採用担当のあなたへ:
- 初任給3万上げる稟議、明日出す
- 求人票の「やりがい」「アットホーム」全部消す

あなたの会社、生き残れる？
```

### 推奨用途
- トレンド分析
- 業界変化の警告
- 複数データの統合提示

### 文字数目安
1000-1500字

## パターン5: イベント・体験レポート型

### 構造
```
[タイトル: [場所・組織]で[活動]させていただきました]

[概要・背景]

[印象的だったポイント1]
[印象的だったポイント2]
[印象的だったポイント3]

[学びや感想]
```

### テンプレート
```
[組織名]で[役割]として[活動]させていただきました。

[組織の特徴・背景を1-2文で]

印象的だったのは[ポイント]。
[詳細・エピソード]

[気づき・学び]

[今後への示唆や感謝]
```

### 実例
```
広島市立基町高校でアントレプレナーシップ推進大使として講演させていただきました。

校訓「自主自律」のもと、自ら考え行動する力を育む進学校。
東大・京大など難関大学への進学実績と、美術を通じて世界に発信する創造学科を併設。

印象的だったのは生徒たちの真剣な眼差し。
起業とは何か、社会課題にどう向き合うかを本気で考えている。

次世代を担う若者たちとの対話は、常に新たな気づきをくれます。
```

### 推奨用途
- 登壇・イベント告知
- ブランディング
- ネットワーキング報告

### 文字数目安
350-500字

## パターン6: サービス・人材紹介型（Call-to-Action）

### 構造
```
【[カテゴリ]】[ターゲット]をお探しの方へ

[背景・課題提起]

そこで、[提供価値]をご紹介します。

[特徴1]
[特徴2]
[特徴3]

[アクション呼びかけ]
```

### テンプレート
```
【ご紹介】[サービス・人材]をお探しの方へ

[よくある相談内容・背景]

そこで、私が信頼できる[提供価値]をご紹介します。

[強み1]: [詳細]
[強み2]: [詳細]
[強み3]: [詳細]

ご興味ある方は[アクション]ください。
```

### 実例
```
【ご紹介】スタートアップに強い税理士をお探しの方へ

起業したばかりの知人から「良い税理士さんいませんか?」と相談されることが増えました。

そこで、私が信頼できる「スタートアップ専門」の税理士をご紹介します。

強み1: 創業融資・資金調達サポートに精通
強み2: クラウド会計freee・マネーフォワード完全対応
強み3: 月額顧問料2万円〜のリーズナブル価格

ご興味ある方はDMください。
```

### 推奨用途
- サービス紹介
- 人材マッチング
- パートナー紹介

### 文字数目安
350-550字

## パターン7: 有料サロン・コミュニティ訴求型

### 構造
```
[タイトル: 【本音だけを話す場所】]

[通常サービスとの価格比較]

[コミュニティの価値1]
[コミュニティの価値2]
[コミュニティの価値3]

[参加者の声・実績]

[限定性・緊急性]
```

### テンプレート
```
【本音だけを話す場所】[価格]で、[提供価値]に触れる

僕の[通常サービス]は[価格]。
それに比べたら、この[サービス名]は破格すぎる。

正直、全く儲かりません。
でも、やる理由がある:
- [価値1]
- [価値2]
- [価値3]

現在[参加者数]が参加中。[実績・成果]

[限定募集・締切情報]
```

### 実例
```
【本音だけを話す場所】月4,980円で、億単位の知見に触れる

僕の顧問料は月20万〜50万。1時間のミーティングだけで、です。
それに比べたら、このサロンは破格すぎる。

正直、全く儲かりません。
でも、やる理由がある:
- スタートアップの成功率を上げたい
- 本気の経営者とつながりたい
- 次世代の起業家を育てたい

現在200名以上が参加中。資金調達、M&A、採用の修羅場を経験した経営者たちが本音で議論しています。

次回募集は11月限定30名。
```

### 推奨用途
- 有料コミュニティ・サロン訴求
- メンバーシップ募集
- 限定サービス告知

### 文字数目安
500-800字

## 共通要素チェックリスト

### 必須要素（全パターン共通）

- [ ] **問いかけ終結**: 必ず読者への問いかけで終わる
- [ ] **断定型表現**: 「〜だ」「〜である」で言い切る
- [ ] **口語体の活用**: 「マジで」「ヤバい」「〜の件」を適度に使用
- [ ] **具体的数値**: 最低3つ以上の数値を含む
- [ ] **企業名・人名**: 最低2つ以上の固有名詞
- [ ] **ハッシュタグ**: 2-3個（#スタートアップ #AI など）

### 展開フレーズ集

| フレーズ | 用途 |
|---------|------|
| 「答えは単純だ。」 | 理由説明の導入 |
| 「でも、ここからが本当の話だ。」 | 深掘りへの転換 |
| 「つまり、」 | 結論・自己解釈 |
| 「ポイントは、」 | 核心の強調 |
| 「注目すべきは、」 | 着目点の提示 |
| 「これ、[解釈]。」 | カジュアルな断定 |

### 問いかけテンプレート

| タイプ | テンプレート |
|--------|------------|
| Yes/No型 | 「あなたはこれ、成功すると思う？」 |
| 戦略型 | 「あなたの会社は、この変化にどう対処する？」 |
| 選択型 | 「あなたならどちらを選ぶ？」 |
| 挑発型 | 「あなたの会社、生き残れる？」 |
| カジュアル型 | 「あなたはこれ、どう思う？」 |

## パターン別推奨頻度

| パターン | 推奨頻度 | 理由 |
|---------|---------|------|
| パターン1（断定型主張） | 週1-2回 | 時事ネタで注目獲得 |
| パターン2（問題提起→正論） | 月2-3回 | 深い洞察で信頼構築 |
| パターン3（ニュース深掘り） | 週1回 | 専門性アピール |
| パターン4（リスト型） | 月1-2回 | 保存・シェア狙い |
| パターン5（体験レポート） | 月1-2回 | 人間味・親近感 |
| パターン6（サービス紹介） | 月1回以下 | 押し売り感回避 |
| パターン7（サロン訴求） | 月1回以下 | 募集時のみ |
