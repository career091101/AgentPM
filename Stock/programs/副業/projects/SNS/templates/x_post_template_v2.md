## 🎯 X投稿テンプレート（130-180字）

### 基本構造（4段階）

```
[1. フック（衝撃的一言）] 40-60字
[固有名詞/権威性] + [数字/断定的評価]

[2. 根拠・データ（1文）] 30-50字
[簡潔な事実・データ]

[3. 洞察・自己解釈（1文）] 30-50字
これ、[自分の解釈・主張]。

[4. 呼びかけ（問いかけ+@メンション）] 30-40字
あなたは[質問]？ @[関連アカウント]
```

### 具体例（170字）

```
OpenAIへの1400億円投資は18兆円の価値になった。

わずか6年で135倍というリターンだ。これ、AI投資は単なる技術投資ではなく、インフラ投資の領域へ。

あなたの会社は、この変化に対処できる？ @OpenAI @Microsoft
```

**分解**:
- **フック**: 「OpenAIへの1400億円投資は18兆円の価値になった」（固有名詞+数字）
- **根拠**: 「わずか6年で135倍というリターンだ」（数字データ）
- **洞察**: 「これ、AI投資は単なる技術投資ではなく、インフラ投資の領域へ」（自己解釈）
- **呼びかけ**: 「あなたの会社は、この変化に対処できる？ @OpenAI @Microsoft」（問いかけ+@メンション）

## 📋 X投稿チェックリスト

投稿前に以下を確認してください：

### 必須要素（85%以上）
- [ ] **固有名詞**が含まれている（企業名・人名・サービス名）
  - 例: OpenAI, Google, Microsoft, Sam Altman, ChatGPT, Gemini
- [ ] **数字**が1つ以上含まれている（実績・金額・率・倍率）
  - 例: 1000億円, 135倍, 2ヶ月, 1億ユーザー
- [ ] **文字数**が130-180字の範囲内

### 推奨要素（30-60%）
- [ ] **@メンション**が1-2個含まれている（関連アカウント）
  - 例: @OpenAI, @Google, @Microsoft
- [ ] **問いかけ終結**で終わっている
  - 例: 「あなたは〜？」「これって〜じゃない？」
- [ ] **バズ構文**が使用されている（フック強化）
  - 例: 「これ本当に〜」「〜曰く」「〜だけど」

### 品質保証
- [ ] **断定的な主張**がある（「〜だ」「〜です」で終わる文）
- [ ] **自己解釈**が含まれている（「これ、〜」「つまり〜」）
- [ ] **視覚的訴求力**がある（改行・句点で区切り）

## 🔄 LinkedIn→X変換ロジック

### ステップ1: 要素抽出

LinkedIn投稿（700-1500字）から以下を抽出:
- **フック**: タイトル・冒頭 → X冒頭1行（50-80字）
- **データ**: 主要数値1-2点 → X根拠1文（40-60字）
- **洞察**: 「つまり」セクション → X自己解釈1文（30-50字）
- **問いかけ**: CTA → X問いかけ+@メンション（30-50字）

### ステップ2: 圧縮

| LinkedIn | X最適化 |
|----------|---------|
| 700-1500字 | **140-170字**（1/4〜1/5） |
| 4-5段落 | 1-2段落 |
| データ5個以上 | 1-2個に絞る |
| 固有名詞3社以上 | 1-2社に絞る |

### ステップ3: X最適化

- **@メンション追加**: 関連アカウント（OpenAI、Google等）
- **問いかけ強化**: CEO向け→広範囲フォロワー向け
- **ハッシュタグ**: 0-2個 → 2-4個（X推奨）

### 変換例

**LinkedIn投稿（950字）**:
```

この変化が示唆するのは、AI投資が単なる技術投資ではなく、
電力・通信と並ぶインフラ投資の領域へと移行しているということです。

（中略...データ・分析が続く...）
```

## 📝 使用方法

### Claude API経由での投稿文生成

```python
import anthropic

client = anthropic.Anthropic(api_key="your_api_key")

prompt = f"""
以下のLinkedIn投稿を、X最適化フォーマット（130-180字）に変換してください。

【変換ルール】
- 固有名詞を1つ以上含める
- 数字を1つ以上含める
- @メンションを1-2個含める
- 問いかけで終わる
- バズ構文100選からフックを選択

【LinkedIn投稿】
{{linkedin_post}}

【X投稿（130-180字）】:
"""

message = client.messages.create(
    model="claude-sonnet-4-5-20250929",
    max_tokens=1024,
    messages=[
        {"role": "user", "content": prompt}
    ]
)

x_post = message.content[0].text
print(x_post)
```
