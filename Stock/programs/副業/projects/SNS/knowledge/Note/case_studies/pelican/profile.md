## 法的コンプライアンス

### 投資助言に該当しない理由

#### ✅ セーフゾーン（法的リスク低い発信方法）

1. **自己の投資実績の開示**
   - 「私が保有している銘柄」の報告（推奨ではない）
   - 配当金受取額の公開（事実報告）

2. **客観的な指標の解説**
   - PER（株価収益率）、PBR（株価純資産倍率）の説明
   - 配当利回り計算方法の教育
   - 「これらの指標で誰でも確認可能」という点を強調

3. **対価を得ない情報発信**
   - ブログ記事は無料公開が基本
   - アフィリエイト（証券口座開設等）は「対価」ではなく「応援金」

4. **「自己責任」の明示**
   - メディア出演でも「投資は自己責任で」というスタンス
   - 損失のリスクを明確に述べている

#### 法的適合性の根拠

```
ペリカンの発信 → 金融商品取引法上の投資助言に該当しない理由：

1. 個別銘柄の「推奨」ではなく「保有銘柄の報告」
   例: 「〇〇株を買え」（助言）×
       「私は〇〇株を保有」（報告）✓

2. 売買タイミングの「指示」がない
   例: 「今が買い時」（助言）×
       「私はこのタイミングで購入」（事例）✓

3. 対価を得て個別アドバイスしない
   → note有料記事で個別相談対応なし
   → ブログとメディア出演は不特定多数向け

4. 免責事項の明示
   → 「投資は自己責任」を常に述べている
```

### 推奨される法的対応

**実装状況の評価**:
- ✅ ペリカンは「投資助言ではなく情報発信」というスタンスを厳密に守っている
- ✅ 免責事項が明示されている
- ✅ 対価を得ての投資相談対応がない
- ✅ 客観的指標の説明に注力している
