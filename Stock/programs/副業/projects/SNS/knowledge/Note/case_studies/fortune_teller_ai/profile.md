# AI占い師（恋愛占い特化）

## 基本情報
- **ジャンル**: 恋愛占い（タロット占い）
- **収益規模**: 月5-32万円（事例により幅広い）
- **販売単価**: 500-3,000円/件
- **フォロワー数**: 不要（ココナラ等で販売）
- **運用期間**: 1年以上（2023年開始）

## プロフィール詳細

| 項目 | 詳細 |
|------|------|
| **属性** | 会社員副業、占い経験なし |
| **開始時期** | 2023年頃（ChatGPT普及後） |
| **開始前知識** | 占い経験なし、ChatGPT利用可能 |
| **販売プラットフォーム** | ココナラ、note、LINE公式 |
| **対応方式** | AI自動生成（鑑定文） + 手動対応 |

## 主な実績

### 収益実績（事例別）

#### 事例1: 30代会社員
- **開始時期**: 2023年
- **初月**: 数件販売（約5,000円）
- **2ヶ月目**: 累計40万円以上の受注成功
- **月間**: 推定20-40万円規模

#### 事例2: note販売（プロンプト集）
- **商品**: 「コピペで使える鑑定プロンプト50選」
- **価格**: 1,980円
- **月間収益**: 約5万円
- **販売数**: 約25部/月

### 全体的な収益レンジ
- **月間**: 5-32万円（事例により大きな幅）
- **成長パターン**: 初月小売り→2-3ヶ月で月10万円超
- **スケーラビリティ**: 高い（対応時間=収益）

## キャリア背景と参入経緯

### 参入のしやすさ
- **占い経験**: なし
- **必要スキル**: ChatGPT操作能力、ターゲット理解
- **初期投資**: ほぼゼロ
- **参入障壁**: 極めて低い

### AI活用のポイント
- **鑑定文生成**: ChatGPTで自動生成
- **効率性**: 1件あたり10-20分で対応可能
- **スケーラビリティ**: 対応件数を増やすだけで売上増

## 提供するサービス

### ココナラ鑑定
- **料金形態**: 500-3,000円/件
- **対象**: 恋愛に悩む20代女性
- **方法**: タロット占い + AI鑑定文生成
- **特徴**: 低価格で評価獲得→値上げ戦略

### note販売（プロンプト集）
- **商品**: 「コピペで使える鑑定プロンプト50選」
- **単価**: 1,980円
- **販売数**: 約25部/月
- **ターゲット**: AI占い副業を始めたい人

### LINE公式アカウント
- **機能**: 自動返信テンプレート
- **戦略**: 返信時間を短縮、スケーラビリティ向上
- **月額制**: 検討可能

## 他プラットフォーム展開

- **ココナラ**: メイン販売チャネル（★★★★★評価獲得が重要）
- **note**: プロンプト集販売（月5万円程度）
- **LINE公式**: 自動返信で省力化
- **X（Twitter）**: note記事の拡散（週数回）
