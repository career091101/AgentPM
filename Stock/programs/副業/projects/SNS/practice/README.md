# SNS_Practice Project

SNS実践プロジェクトのPMBOK管理ディレクトリです。

## 構造

このディレクトリはプロジェクト管理とA/Bテスト記録に特化しています。
SNSナレッジベースは `../knowledge/` を参照してください。

### ディレクトリ構造

- **documents/**: PMBOK準拠のプロジェクト管理ドキュメント
  - 1_initiating/: プロジェクト憲章、ステークホルダー分析
  - 2_discovery/: ペルソナ、ジャーニーマップ
  - 3_planning/: WBS、バックログ
  - 4_executing/: スプリント記録
  - 5_monitoring/: ステータスレポート
  - 6_closing/: レッスンズラーンド

- **experiments/**: A/Bテスト・実験記録
  - 各プラットフォームでの施策実験結果を記録

## SNSナレッジベースとの関係

- **knowledge/**: What/How（アルゴリズム、ベストプラクティス、ケーススタディ）
- **practice/**: When/Where（プロジェクト管理、実験記録）
