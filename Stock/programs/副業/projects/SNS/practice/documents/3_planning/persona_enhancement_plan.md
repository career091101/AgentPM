## 2. Phase 1: JTBD（Jobs to be Done）追加

### 2.1 実施内容
田中ペルソナ（CEO層）が「なぜSNS投稿を見るのか」を3つのJobで定義

### 2.2 JTBDテンプレート

```
【機能的Job（Functional）】
When: [状況]
I want to: [したいこと]
So that: [得られる結果]

【感情的Job（Emotional）】
I want to feel: [感じたい感情]
By: [そのための行動]

【社会的Job（Social）】
I want to be seen as: [周囲からどう見られたいか]
By: [そのための行動]
```

### 2.3 田中ペルソナへの適用（案）

| Job種別 | 内容 |
|---------|------|
| **機能的** | When: 朝の通勤中 / I want to: AI業界の最新動向を5分で把握したい / So that: 役員会議でAIについて発言できる |
| **感情的** | I want to feel: 「自分は最新情報を把握している」という安心感 / By: 信頼できるソースからの要約を読む |
| **社会的** | I want to be seen as: 「AI時代に対応できる経営者」と見られたい / By: 社内外でAI関連の発言をする |

### 2.4 確認ポイント
- [ ] 3つのJobがそれぞれ異なる視点を提供しているか
- [ ] 投稿コンテンツの方向性と一致しているか

## 3. Phase 2: 共感マップ（Empathy Map）作成

### 3.1 実施内容
田中ペルソナの「Says / Does / Thinks / Feels」を可視化

### 3.2 共感マップテンプレート

```
┌─────────────────┬─────────────────┐
│      SAYS       │      DOES       │
│  （言うこと）    │   （行動）      │
├─────────────────┼─────────────────┤
│     THINKS      │      FEELS      │
│ （考えていること）│   （感情）      │
└─────────────────┴─────────────────┘
         ↓
    PAINS（課題）
         ↓
    GAINS（願望）
```

### 3.3 田中ペルソナへの適用（案）

| 象限 | 内容 |
|------|------|
| **Says** | 「AIは重要だ」「でも何から始めればいいかわからない」「うちの業界では本当に使えるのか」 |
| **Does** | LinkedInを朝晩5分チェック / 気になる記事を保存するが読み返さない / 部下にAI調査を指示 |
| **Thinks** | 「競合に遅れを取りたくない」「投資判断を間違えたくない」「社内にAI人材がいない」 |
| **Feels** | 情報過多で焦り / 判断先送りへの罪悪感 / 時間がない苛立ち / 未知への不安 |

### 3.4 確認ポイント
- [ ] Says（表面）とThinks（本音）にギャップがあるか
- [ ] Feelsが投稿コンテンツのトーンに活かせるか
