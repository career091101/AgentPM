# AIエージェント実装ラボ - ランディングページ

## 概要

中級者特化型AIエージェント実装コミュニティのMVP検証用ランディングページ。

## MVP類型

**選択**: Landing Page型
**選定理由**: UVP刺さり度検証が最優先（コスト最小、期間最短）

## ファイル構成

```
mvp/lp/
├── index.html    # 完全なLP（HTML/CSS/JS統合）
└── README.md     # 本ファイル
```

## LP構成（7セクション）

| セクション | 内容 |
|-----------|------|
| **Hero** | UVP + 初月50%OFFバッジ + CTA |
| **Problem** | 3つの悩み（論文実装困難/初心者向けばかり/業界特化なし） |
| **Solution** | Before/After比較（200時間→40時間、完成度50%→95%） |
| **Features** | 3つの特徴（論文実装/中級者専用/業界テンプレート） |
| **How it Works** | 3ステップ（登録→実践→適用） |
| **Pricing** | 月額2,980円 + 競合比較表（vs Udemy/個別コンサル） |
| **CTA** | メール登録フォーム + トラストシグナル |

## UVP

**採用案**:
> 論文実装を40時間で完了。中級者専用の実装コミュニティ

**選定理由**:
- 具体的時間数値（40時間）で信頼性向上
- 「中級者専用」で明確なターゲティング
- 競合との差別化ポイントを凝縮

## 証拠収集計画

### 測定指標

| 指標 | 目標値 | 測定方法 |
|------|--------|----------|
| **CVR**（メール登録率） | 5%以上 | Google Analytics |
| **平均滞在時間** | 60秒以上 | GA / Hotjar |
| **ヒートマップ** | CTA 50%クリック | Hotjar / Crazy Egg |
| **離脱率** | Hero 30%以下 | GA |
| **デバイス比率** | Mobile 60%+ | GA |

### 成功判定（2週間後）

- CVR 5%以上 → ✅ PSF合格候補
- CVR 3-5% → ⚠️ LP改善実施後に再測定
- CVR 3%未満 → ❌ UVP再定義、またはPivot検討
