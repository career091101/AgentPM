# プロジェクト憲章
**プロジェクト名**: SNS

## 1. 背景と目的

### 背景

複数のSNS（Facebook, Instagram, X, LinkedIn, YOUTRUST, Threads）への投稿管理が煩雑化しており、運用コストが増大している。

**現状実績（2025年10月-12月、90日間）**:
- **月間総imp**: 346,766 imp（目標1,000,000の34.7%、-653,234 imp不足）
- **プラットフォーム別貢献度**:
  - Facebook: 172,261 imp/月（49.7%貢献）
  - LinkedIn: 159,264 imp/月（45.9%貢献）
  - X: 15,242 imp/月（4.4%貢献のみ）← 最大の機会損失
- **運用効率**: 47.5分/投稿（目標15分の3.2倍超過）
  - 月間浪費時間: 40時間/月（≒ 1週間分の労働時間）

**市場トレンド（社内ナレッジより）**:
- **Build in Public**: 透明性戦略でコミュニティ売上30%達成事例
- **PLG (Product-Led Growth)**: 製品がマーケティングツール化、SaaS月間MRR $32K成功事例
- **属人性経済**: Instagram事例で、属人性あり（月300-500万円）vs なし（月5-10万円）= **50-100倍の収益差**
- **プラットフォーム特化**: 各SNSアルゴリズム最適化が必須。横展開では各媒体のポテンシャル1/3しか発揮できない

**課題の構造（詳細は別紙レポート参照）**:
1. **コンテンツ品質**: フック不足（Top/Bottom 60倍差）、情報解像度不足、プラットフォーム最適化欠如
2. **戦略・方向性**: トピック偏り（OpenAI 55.8%）、属人性欠如、収益化導線不在
3. **プロセス・体制**: 再現性欠如、KPI管理不備、投稿時間帯集中（深夜0時100%）

### 目的

クロスポストやAI活用によるコンテンツ生成フローを構築し、投稿にかかる時間を最小化（1日15分以内/1投稿）しつつ、全チャネルでのインプレッションを最大化する。

**改善目標（2026年12月31日まで）**:
1. **総インプレッション**: 346,766 → **1,000,000 imp/月**（+188.4%成長）
2. **運用効率**: 47.5分/投稿 → **15分/投稿**（-68%時間削減）
3. **プラットフォーム最適化**:
   - LinkedIn ER: 1.00% → **4.00%**（高野氏ベンチマークレベル）
   - X 貢献度: 4.4% → **25%**（フォロワー基盤構築）
   - Facebook Reels比率: 8.4% → **25%**（動画コンテンツ活用）
4. **再現性向上**: Top/Bottom投稿のimp差 60倍 → **10倍以内**（品質バラツキ抑制）

## 2. スコープ

### 含む

- **imp数が出る投稿案の作成**
- **以下の主要課題への対策（詳細は別紙レポート参照）**:
  1. **LinkedIn品質改善**: 問いかけ終結率8.5% → 80%+、ER 1.00% → 4.00%、高野氏7パターン適用
  2. **X フォロワー基盤構築**: Build in Public開始、Reply Guy活動（毎日10コメント）、スレッド投稿（週1本）
  3. **Facebook Reels活用強化**: Reels比率8.4% → 25%、動画コンテンツの質向上
  4. **運用効率化**: ネタ収集自動化（10-15分 → 3分）、テンプレート適用率0% → 80%
  5. **再現性確保**: トップ/ボトム分析、チェックリスト導入、PDCA構築

### 含まない

- 投稿スケジュールの自動化
- 各SNSへの一括投稿・管理フローの構築
- インプレッション分析ダッシュボードの導入
- 各個別のSNSへの手動返信（AIアシストは検討するが、全自動化は範囲外）
- 有料広告運用の最適化
- 動画コンテンツの制作（テキスト・画像メイン）← ただしReels（15-30秒短尺動画）は含む
- **収益化機能の実装**（本PJは認知・エンゲージメント獲得が主眼、収益化は将来フェーズ）
- **個人ブランド確立の全体戦略**（属人性向上は重要だが、本PJ期間内での完全確立は困難）

## 5. リスクと対策

| # | リスク | 影響 | 発生確率 | 対策 |
|---|--------|------|---------|------|
| 1 | **LinkedIn ER改善が進まない** | imp目標未達（-200K imp/月） | 中 | - 高野氏7パターンテンプレート適用<br>- 問いかけ終結を習慣化（チェックリスト導入）<br>- 週次A/Bテスト実施 |
| 2 | **X フォロワー基盤構築の長期化** | X貢献度4.4%のまま停滞 | 高 | - Reply Guy活動（毎日10コメント）をルーチン化<br>- Build in Public開始（進捗共有）<br>- 3ヶ月で+1,000フォロワー達成を最低目標 |
| 3 | **ネタ収集の効率化失敗** | 作業時間47.5分のまま、浪費継続 | 中 | - RSS自動収集（ロボット、半導体系）<br>- トピック別パフォーマンスDB構築<br>- 高パフォトピック優先選定ルール化 |
| 4 | **Facebook Reels制作の技術的障壁** | Reels比率8.4%のまま | 中 | - 15-30秒短尺動画に限定（長尺動画は対象外）<br>- テンプレート化（ニュース引用型）<br>- 外部ツール活用（CapCut等） |
| 5 | **PDCA未構築によるバラツキ継続** | Top/Bottom 60倍差の改善なし | 高 | - 週次Top 5/Bottom 5分析を習慣化<br>- チェックリスト導入（5桁数字、企業名3社、問いかけ終結）<br>- テンプレート適用率80%達成 |

**重大リスク（発生時の緊急対応）**:
- **データ矛盾の再発**: LinkedIn impが0になる等のデータ異常 → 即座に原因調査（CSVエクスポート、API確認）
- **投稿時間帯最適化の副作用**: 時間帯変更でERが逆に低下 → 即座にロールバック
- **法令違反リスク**: ステマ規制、著作権侵害等 → 該当投稿を即座に削除、再発防止策策定
