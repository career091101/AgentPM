## 6. 勝ちパターン（3プラットフォーム統合）

### パターンA: 衝撃的数字型
```
構成: 冒頭に具体的で衝撃的な数字
例: 「18万台」「1兆7000億円の赤字」「週100時間」
効果: 平均の2〜3倍
対象: 全プラットフォーム共通
```

### パターンB: 権威引用型
```
構成: 著名人の発言を冒頭に配置
例: サム・アルトマン、イーロン・マスク、ジェフ・ベゾス
効果: 安定した高パフォーマンス
対象: Facebook, LinkedIn特に有効
```

### パターンC: 経営者向け示唆型（LinkedIn特化）
```
構成: AI/テクノロジーの経営への影響を明示
例: 「CEOがAIについて知るべき3つのこと」
効果: CEO層4.3%へのリーチ最大化
対象: LinkedIn優先
```
