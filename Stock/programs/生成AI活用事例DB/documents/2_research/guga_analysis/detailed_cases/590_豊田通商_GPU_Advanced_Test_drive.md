# 590_豊田通商_GPU Advanced Test drive（GAT）

## 1. 企業基本情報
- **企業名**: 豊田通商株式会社
- **業種**: 総合商社
- **企業規模**: トヨタグループ
- **従業員数**: 取得不可

## 2. AI導入の概要
- **導入AIツール名**: GPU Advanced Test drive（GAT）
- **提供ベンダー**: ネクスティエレクトロニクス（豊田通商グループ）、NVIDIA
- **導入時期**: 2024年4月1日
- **導入規模**: B2Bサービスとして提供

## 3. 導入目的・背景
豊田通商グループのネクスティエレクトロニクスは、NVIDIA製GPUの試用サービス「GPU Advanced Test drive（GAT）」を提供開始。ソフトウエア定義車両や自動運転などの分野で高まる生成AIの需要に対応し、生成AIの基盤となる大規模言語モデル（LLM）の開発を支援。

## 4. 活用シーン・業務内容
- 大規模言語モデル（LLM）開発
- 画像認識（自動運転向け）
- デジタルツイン
- ソフトウェア定義車両（SDV）開発
- GPU性能検証

## 8. 課題・障壁と解決策
