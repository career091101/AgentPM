# 事例: 損保会社Z（金融・保険）

## 1. 基本情報
| 項目 | 内容 |
|------|------|
| 企業名 | 損保会社Z |
| 業界 | 金融・保険 |
| AIツール | 保険料算定・リスク評価AI |
| 導入時期 | 2024-09 |

## 2. 定量的効果
| 指標 | 数値 |
|------|------|
| 料金計算時間 | 59%削減 |
| 見積精度 | 83% |
| 新規契約数 | 28%増加 |
| 利益率向上 | 3.2% |

## 3. 導入背景・課題
- 保険料計算が複雑で、手作業が多い
- 見積時間が長く、顧客が離脱
- リスク評価が属人的

## 4. ソリューション概要
申込者の情報（運転記録、家族構成、社会データ）をAIで分析し、リスク等級を自動判定。統計的リスク評価から最適な保険料を自動算定。競合保険との価格比較を提示。個別リスク要因を可視化し、リスク低減提案を提供。

## 5. 成功要因
- 統計モデルの精密化
- リスク分類の標準化
- 顧客データの整理

## 6. 他社への示唆
- 保険料設定の自動化で市場対応力強化
- リスク評価の公正性向上
- 顧客獲得と利益率の両立
