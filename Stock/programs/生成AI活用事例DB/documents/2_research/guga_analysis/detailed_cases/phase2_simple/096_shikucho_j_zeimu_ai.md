# 事例: 市区町村J（公共・自治体）

## 1. 基本情報
| 項目 | 内容 |
|------|------|
| 企業名 | 市区町村J |
| 業界 | 公共・自治体 |
| AIツール | 租税債権管理・滞納予測AI |
| 導入時期 | 2024-06 |

## 2. 定量的効果
| 指標 | 数値 |
|------|------|
| 滞納予測精度 | 81% |
| 徴収率向上 | 8.3% |
| 催告効率化 | 44% |
| 年間税収増加 | 3,200万円 |

## 3. 導入背景・課題
- 市税滞納が年々増加している
- 催告対象者の優先順位付けが属人的
- 滞納者の実情把握に時間がかかる

## 4. ソリューション概要
過去の納税履歴、雇用状況、家族情報をAIで分析し、滞納リスクを予測。催告対象者の優先順位を自動提案。納税相談に応じやすい時期・方法を提案し、滞納解消率を向上。

## 5. 成功要因
- 税務職員の納税相談スキル向上
- 個人情報保護への十分な配慮
- 定期的なモデル精度検証と改善

## 6. 他社への示唆
- 公共財源の確保にAI活用は有効
- 機械的な催告から対話型サポートへの転換
- データ駆動の施策で実績を可視化できる
