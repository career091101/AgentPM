# 事例: クレジット会社R（金融・保険）

## 1. 基本情報
| 項目 | 内容 |
|------|------|
| 企業名 | クレジット会社R |
| 業界 | 金融・保険 |
| AIツール | 延滞・デフォルト予測AI |
| 導入時期 | 2024-08 |

## 2. 定量的効果
| 指標 | 数値 |
|------|------|
| 延滞予測精度 | 77% |
| 延滞件数削減 | 22% |
| 回収率向上 | 16% |
| 不良債権削減 | 31% |

## 3. 導入背景・課題
- 延滞者の事前把握ができていない
- 回収活動が後手後手になる
- 不良債権が増加している

## 4. ソリューション概要
利用パターン、支払い遅延履歴、失業リスク、家族状況をAIで分析し、延滞リスク者を早期予測。支払い困難になる前に相談窓口を案内。個別の返済相談を提案。回収優先度を自動設定。通知タイミングを最適化。

## 5. 成功要因
- 利用者データの詳細な分析
- コールセンターとの連携
- 返済支援制度の充実

## 6. 他社への示唆
- 予防的な回収戦略で資産質向上
- 利用者との良好関係維持
- 顧客生涯価値の向上
