# 事例: 消防本部Q（公共・自治体）

## 1. 基本情報
| 項目 | 内容 |
|------|------|
| 企業名 | 消防本部Q |
| 業界 | 公共・自治体 |
| AIツール | 119通報分析・出動支援AI |
| 導入時期 | 2024-07 |

## 2. 定量的効果
| 指標 | 数値 |
|------|------|
| 通報内容理解精度 | 84% |
| 出動判断時間短縮 | 38% |
| 現場到着時間 | 12%短縮 |
| 誤報対応削減 | 22% |

## 3. 導入背景・課題
- 119通報が増加し、通報内容の判断に時間がかかる
- 誤報や不適切な出動が発生
- 消防士の対応負担が増加

## 4. ソリューション概要
119通報の音声・テキストをAIがリアルタイム分析し、緊急度と最適な出動体制を自動判定。背景情報（過去の同所出動、建物構造等）を自動表示。通報者の説明不足箇所をAIが質問提案。出動予定表を自動生成。

## 5. 成功要因
- 消防士との密接な連携で現場ニーズを把握
- 通報音声データの学習
- 定期的なシステム改善

## 6. 他社への示唆
- 緊急対応領域でのAI活用は生命に関わる
- 確実な判定が求められる領域での実装
- 人間の判断を支援する役割が重要
