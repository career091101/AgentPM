# 事例: クレジット会社Q（金融・保険）

## 1. 基本情報
| 項目 | 内容 |
|------|------|
| 企業名 | クレジット会社Q |
| 業界 | 金融・保険 |
| AIツール | 与信審査・限度額最適化AI |
| 導入時期 | 2024-07 |

## 2. 定量的効果
| 指標 | 数値 |
|------|------|
| 審査時間短縮 | 73% |
| 承認率向上 | 12% |
| デフォルト率低減 | 8.3% |
| 年間利益増加 | $18M |

## 3. 導入背景・課題
- 与信判定に時間がかかり、顧客満足度が低い
- デフォルト予測が不十分
- 与信額の設定が非効率

## 4. ソリューション概要
申込者の収入、雇用形態、信用情報、過去の利用パターンをAIで分析し、与信可否と限度額を自動判定。申告内容と利用行動の矛盾を検出。今後のデフォルトリスクを予測。リスク調整金利を自動設定。

## 5. 成功要因
- 信用情報機関との連携
- デフォルト予測モデルの精密化
- リスク管理との連携

## 6. 他社への示唆
- スコアリング精度の向上でリスク最適化
- 与信効率化で事業規模拡大
- 顧客満足度と安全性の両立
