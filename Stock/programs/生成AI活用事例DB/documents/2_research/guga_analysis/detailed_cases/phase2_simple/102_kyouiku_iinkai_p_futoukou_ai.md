# 事例: 教育委員会P（公共・自治体）

## 1. 基本情報
| 項目 | 内容 |
|------|------|
| 企業名 | 教育委員会P |
| 業界 | 公共・自治体 |
| AIツール | 不登校予測・支援AI |
| 導入時期 | 2024-03 |

## 2. 定量的効果
| 指標 | 数値 |
|------|------|
| 不登校予測精度 | 72% |
| 早期発見率 | 64%向上 |
| 復学支援成功率 | 58% |
| スクールカウンセラー業務効率化 | 26% |

## 3. 導入背景・課題
- 不登校生が年々増加
- 初期段階での発見が難しい
- カウンセラーの負担が大きい

## 4. ソリューション概要
出欠状況、成績、学習態度、行動パターンをAIで分析し、不登校リスク生徒を早期発見。個別の支援方針を提案。保護者・教員への働きかけ時期と内容を提案。カウンセリング記録から効果的な支援プランを自動生成。

## 5. 成功要因
- スクールカウンセラーとの連携
- プライバシー保護への配慮
- 学校全体の支援体制整備

## 6. 他社への示唆
- 社会課題解決にAIが貢献
- 早期予防が効果的な領域に活用可能
- 人間的サポートとのハイブリッドが重要
