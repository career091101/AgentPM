# 事例: リース会社S（金融・保険）

## 1. 基本情報
| 項目 | 内容 |
|------|------|
| 企業名 | リース会社S |
| 業界 | 金融・保険 |
| AIツール | 資産価値評価AI |
| 導入時期 | 2024-09 |

## 2. 定量的効果
| 指標 | 数値 |
|------|------|
| 価値評価精度 | 84% |
| リース料金最適化 | 12%向上 |
| 残存価値予測精度 | 79% |
| 年間利益増加 | $22M |

## 3. 導入背景・課題
- 車両や設備の価値評価が非効率
- リース料金の算定が属人的
- 残存価値の予測が不正確

## 4. ソリューション概要
中古市場の価格、走行距離、年式、保守履歴をAIで分析し、現在価値を自動評価。残存価値を予測してリース料金を最適化。保守費用予測で総コストを算定。リース終了時の処分計画を自動生成。

## 5. 成功要因
- 中古市場データの整備
- 保守業者との連携
- 統計モデルの精密化

## 6. 他社への示唆
- 資産評価の自動化でリスク低減
- 最適価格設定で競争力強化
- 利用者との信頼構築
