# 事例: 自治体F（公共・自治体）

## 1. 基本情報
| 項目 | 内容 |
|------|------|
| 企業名 | 自治体F |
| 業界 | 公共・自治体 |
| AIツール | ゴミ分別判定AI |
| 導入時期 | 2024-03 |

## 2. 定量的効果
| 指標 | 数値 |
|------|------|
| 誤分別率削減 | 42% |
| 市民相談減少 | 38% |
| アプリダウンロード | 12,000件 |
| ゴミ処理費削減 | 15% |

## 3. 導入背景・課題
- ゴミ分別の複雑化で市民からの問い合わせが急増
- リサイクル率が全国平均以下
- 清掃職員の負担増加

## 4. ソリューション概要
スマートフォンカメラでゴミを撮影すると、AIが自動判定して分別方法を提案。多言語対応で外国籍住民もサポート。リアルタイムでAIモデルを更新し、最新のゴミ分別ルールに対応。

## 5. 成功要因
- ゴミ分別ルール変更時の迅速な学習データ更新
- 市民向けプロモーション活動
- 清掃職員との協働で精度向上

## 6. 他社への示唆
- 市民向けAIサービスの利用促進が重要
- 行政サービスの複雑性の解決にAI活用は有効
- ローカル言語対応がユーザー増につながる
