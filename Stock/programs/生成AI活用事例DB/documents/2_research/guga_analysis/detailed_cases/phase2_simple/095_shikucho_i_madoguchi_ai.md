# 事例: 市区町村I（公共・自治体）

## 1. 基本情報
| 項目 | 内容 |
|------|------|
| 企業名 | 市区町村I |
| 業界 | 公共・自治体 |
| AIツール | 住民相談チャットボットAI |
| 導入時期 | 2024-07 |

## 2. 定量的効果
| 指標 | 数値 |
|------|------|
| 問い合わせ対応率 | 67% |
| 対応時間短縮 | 51% |
| 職員負担削減 | 25時間/週 |
| 利用者満足度 | 82% |

## 3. 導入背景・課題
- 住民からの問い合わせが増加・多様化
- 窓口職員が情報検索に時間を取られている
- 時間外の相談対応ができない

## 4. ソリューション概要
住民からの質問をAIチャットボットが24時間対応。手続き方法、必要書類、申請期限等を自動回答。複雑な案件は職員に自動エスカレーション。多言語対応で外国籍住民にも対応。

## 5. 成功要因
- 全庁的な手続き情報の整理・標準化
- 利用者向けのプロモーション活動
- チャットボットの継続的なチューニング

## 6. 他社への示唆
- 住民サービスの質向上とコスト削減を同時実現
- チャットボット導入で職員の付加価値業務を増やせる
- 多言語対応は市民サービス向上のカギ
