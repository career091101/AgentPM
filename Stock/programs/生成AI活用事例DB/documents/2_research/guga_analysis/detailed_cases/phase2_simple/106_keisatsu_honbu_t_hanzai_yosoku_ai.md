# 事例: 警察本部T（公共・自治体）

## 1. 基本情報
| 項目 | 内容 |
|------|------|
| 企業名 | 警察本部T |
| 業界 | 公共・自治体 |
| AIツール | 犯罪発生予測・警察配置AI |
| 導入時期 | 2024-06 |

## 2. 定量的効果
| 指標 | 数値 |
|------|------|
| 犯罪予測精度 | 68% |
| 犯罪件数削減 | 19% |
| 検挙率向上 | 12% |
| パトロール効率化 | 34% |

## 3. 導入背景・課題
- 犯罪発生場所の予測が困難
- パトロール体制の最適化ができていない
- 被害者支援の優先順位付けが属人的

## 4. ソリューション概要
過去の犯罪統計、時間帯、天候、イベント情報をAIで分析し、犯罪リスク地区を予測。警察官の巡回エリアを最適配置。重要な道路や駅周辺の警戒強化を提案。被害者支援の優先度を自動判定。

## 5. 成功要因
- 警察官からのニーズヒアリング
- 犯罪統計データの整理
- 継続的なモデル改善と検証

## 6. 他社への示唆
- 治安維持へのAI活用で社会貢献
- データドリブン警察活動への転換
- 市民の安心感向上に有効
