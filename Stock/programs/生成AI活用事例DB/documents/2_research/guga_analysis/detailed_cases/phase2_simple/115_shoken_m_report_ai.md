# 事例: 証券会社M（金融・保険）

## 1. 基本情報
| 項目 | 内容 |
|------|------|
| 企業名 | 証券会社M |
| 業界 | 金融・保険 |
| AIツール | 投資レポート自動生成AI |
| 導入時期 | 2024-09 |

## 2. 定量的効果
| 指標 | 数値 |
|------|------|
| レポート作成時間 | 78%削減 |
| 月間レポート数 | 5.2倍増加 |
| 顧客満足度 | 87% |
| 営業効率向上 | 38% |

## 3. 導入背景・課題
- 投資レポート作成に膨大な時間がかかる
- 全顧客への対応が不可能
- 分析精度にばらつきがある

## 4. ソリューション概要
企業決算、業界動向、株価チャートをAIが自動分析し、投資レポートを生成。顧客ポートフォリオに基づいて個別最適レポートを作成。アナリスト提案を自動作成。営業担当者のプレゼン資料を自動生成。

## 5. 成功要因
- アナリストと連携し、AI学習データ整備
- レポートテンプレートの標準化
- ユーザーフィードバック活用

## 6. 他社への示唆
- 知識型業務の自動化で生産性向上
- 全顧客対応が可能に
- アナリストは高度な分析に専念
