# 事例: 保険会社P（金融・保険）

## 1. 基本情報
| 項目 | 内容 |
|------|------|
| 企業名 | 保険会社P |
| 業界 | 金融・保険 |
| AIツール | リスク評価・保険料算定AI |
| 導入時期 | 2024-09 |

## 2. 定量的効果
| 指標 | 数値 |
|------|------|
| リスク評価精度 | 82% |
| 保険料最適化 | 18%削減 |
| 不正申告検知 | 71%精度 |
| 新規契約数 | 25%増加 |

## 3. 導入背景・課題
- リスク評価が属人的
- 保険料設定が非効率
- 不正申告の検出が困難

## 4. ソリューション概要
申込者の年齢、健康履歴、職業、ライフスタイル、医療検査結果をAIで分析し、リスク等級を自動判定。過去データから統計的に最適な保険料を算定。申告内容と医療記録の矛盾を自動検出。個別リスク評価で差別化商品を設計。

## 5. 成功要因
- 医療機関との連携データ整備
- 統計モデルの高度化
- 規制対応の適切な実施

## 6. 他社への示唆
- リスク評価の標準化で公正性向上
- 最適価格設定で競争力強化
- 不正検知で不健全な契約防止
