## ファイル形式・内容構成

各ファイルは以下の標準構成で作成されています：

### セクション構成
1. **ヘッダー情報**: 業界名、作成日、データソース
2. **概要表**: 総事例数、Phase別内訳
3. **事例一覧表**: 企業名、公表日、ツール、用途、効果、カテゴリ、Phase
4. **カテゴリ別集計**: 活用カテゴリの件数・比率
5. **主要AIツール**: ツール別導入数
6. **用途別分類**: セクター別/タイプ別の詳細分類
7. **定量効果サマリー**: 代表的な数値化効果
8. **特記事項**: 業界固有の特性・課題・展望

### 表形式
- Markdownテーブル形式で統一
- No、企業名、公表日、ツール、用途、効果、カテゴリ、Phaseの8列

### データ出典
- ソース: `/Flow/202601/2026-01-07/research_guga/05_data_all.md`
- 詳細事例: `/04_detailed_cases/` 配下の個別ファイル

## 使用方法

### 業界別データの確認
```
1. 対象業界のファイルを開く
2. 事例一覧表で企業名・用途を検索
3. 定量効果で具体的な数値を確認
4. 活用カテゴリ別集計で傾向を分析
```

### クロス業界比較
```
00_INDEX.md (本ファイル) の統計表を参照
- 活用カテゴリ分布の比較
- AIツール選択の傾向分析
- 業界特性の相違点確認
```

### 詳細事例の参照
```
各業界ファイルの事例一覧から企業名を確認
→ /04_detailed_cases/ で該当企業の詳細ドキュメントを検索
```
