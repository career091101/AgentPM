# 642_日本生命_N-Chat / Microsoft Copilot

## 1. 企業概要
- **企業名**: 日本生命保険相互会社
- **業種**: 生命保険
- **従業員数**: 約70,000人
- **内勤職**: 約20,000人

## 2. AI導入の背景・課題
2029年度までに内勤職の既存業務量最大30%削減目標、業務効率化、英語対応能力向上

## 3. 導入したAIツール
### 主要ツール
1. **N-Chat** (2024年全内務職員2万人展開)
   - 約20種類のテンプレート
   - 2023年1月から約1,000名で実証実験

2. **Microsoft Copilot** (2024年6月試験導入)
   - 1日平均18分の業務時間削減
   - 94%が継続利用希望
   - 数千人規模で展開予定（2024年度内）

3. **保険事務支援AIツール** (2025年1月試作)
   - 回答精度約9割
   - 保険契約管理事務の質問対応

4. **レゾリューションライフとのハッカソン** (2025年2月)
   - 業務サポートツール開発
   - 2025年7月実装予定

## 4. 導入プロセス
- **2023年1月**: N-Chat実証実験（約1,000名）
- **2024年**: N-Chat全内務職員2万人展開
- **2024年6月**: Copilot試験導入
- **2024年度内**: 10案件の実証実験実施
- **2025年1月**: 保険事務支援AIツール試作
- **2025年2月**: ハッカソン開催

## 5. 主な機能・活用シーン
- N-Chat: 資料要約、翻訳、企画業務支援
- Copilot: 業務時間削減（1日平均18分）
- 保険事務AI: 契約管理事務の質問対応（回答精度約9割）

## 6. 導入効果・成果
| 指標 | 成果 |
|------|------|
| Copilot業務時間削減 | 1日平均18分 |
| Copilot継続利用希望率 | 94% |
| 保険事務AI回答精度 | 約9割 |
| 目標 | 2029年度までに業務量最大30%削減 |

## 7. 今後の展開予定
- 2029年度までに内勤職の既存業務量最大30%削減
- Copilot展開拡大（数千人規模）
- レゾリューションライフ共同開発ツール実装（2025年7月）

## 8. 他社への示唆
**成功要因**: 段階的実証実験（1,000名→2万人）、明確な数値目標（2029年度30%削減）、複数ツールの並行展開
