# 655_小田急電鉄_AcroChatAI

## 1. 企業名・業界
- **企業名**: 小田急電鉄株式会社
- **業界**: 運輸業（鉄道）
- **企業規模**: 大企業（従業員数: 約3,500人）

## 2. 導入AIツール名・提供企業
- **ツール名**:
  - AcroChatAI（議事録作成・社内検索）
  - 画像解析AI安全確認システム
  - サポートチャットボット
  - FAQシステム
- **提供企業**:
  - アクロクエストテクノロジー（AcroChatAI）
  - ユーザーローカル（チャットボット・FAQシステム）
- **AI分類**: 生成AI、画像解析AI、チャットボット

## 3. 導入目的・解決すべき課題
- 議事録作成の効率化
- 社内ドキュメント検索の効率化
- 列車出発時の安全確認自動化
- 駅構内の異常検知
- メール対応の自動化
- 2035年度の運行体制確保（係員30%減対応）

## 4. 導入プロセス
### AcroChatAI
1. **2024年**: アクロクエストテクノロジーと導入プロジェクト開始
2. **2024年**: 議事録作成・社内検索機能の展開

### 画像解析AI安全確認システム
1. **2024年8月21日**: 実証実験開始（新百合ヶ丘駅5番ホーム）
2. **実験期間**: 2024年8月～2025年2月頃

### 駅構内AI警備システム
1. **2024年10月10日**: 実証実験開始（柿生駅・玉川学園前駅）
2. **実験期間**: 2024年10月～2025年1月9日
3. **対象カメラ**: 既設カメラ全20台

### チャットボット・FAQシステム
1. **2024年2月**: ユーザーローカル製サポートチャットボット導入発表

## 5. 具体的な活用内容
### AcroChatAI
- **議事録作成**: 会議音声の自動テキスト化・要約
- **社内ドキュメント検索**: Azure AI SearchとAzure OpenAI Serviceを組み合わせたナレッジ検索
- **AIアシスタント**: 社内情報資産を活用した質問応答

### 画像解析AI安全確認システム
- **検知機能1**: ホーム端の黄色い点字ブロックと列車の間に人がいる場合の検知
- **検知機能2**: 閉扉後に傘などが挟まっている場合の検知
- **アルゴリズム**: 独自開発の画像解析AI

### 駅構内AI警備システム
- **設置場所**: 柿生駅・玉川学園前駅
- **対象**: 既設カメラ全20台の映像
- **機能**: 高度認識AIによる異常検知（転倒、不審者等）

### チャットボット・FAQシステム
- **公式サイト**: チャットボット・お問い合わせページに配置
- **機能**: メール受付の自動化、よくある質問への自動応答

## 6. 技術的詳細
- **AcroChatAI**: Azure OpenAI Service + Azure AI Search
- **画像解析AI**: 独自開発の画像認識アルゴリズム
- **駅構内AI**: アジラ社の高度認識AI技術
- **チャットボット**: ユーザーローカルのAIチャットボット技術
