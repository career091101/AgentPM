### 効果の詳細

- **予測精度90%以上**: 大規模イベント開催時に発生しうる通信品質の変動を事前に高精度で予測
- **国産AIモデル**: 「Sarashina」と組み合わせることで、日本語に最適化された通信業界特化AI基盤を実現
- **リアルタイム対応**: 予測結果に基づく事前のネットワーク調整が可能

---

## 4. 導入背景・課題

### 導入前の課題

1. **大規模イベント時の通信品質低下**
   - コンサート、スポーツイベント等で一時的にユーザーが集中
   - 従来は過去データに基づく経験則での対応が中心

2. **予測精度の限界**
   - 従来の統計モデルでは複雑な要因の組み合わせを捉えきれない
   - イベントの種類、規模、天候等の多変量分析が困難

3. **海外LLMへの依存リスク**
   - セキュリティ面での懸念
   - 日本語対応の精度課題
   - 通信業界特有の専門用語・知識の不足

---

## 5. ソリューション詳細

| 項目 | 内容 |
|------|------|
| **プロダクト名** | Large Telecom Model（LTM） |
| **ベースモデル** | Sarashina（国産LLM） |
| **主要機能** | 通信品質予測、ネットワーク最適化支援 |
| **技術スタック** | 大規模言語モデル、時系列予測、通信ドメイン特化学習 |
| **カスタマイズ** | 通信業界特化型生成AI基盤モデル |

### 技術的特徴

1. **Large Telecom Model（LTM）**
   - 通信業界向けに設計された生成AI基盤モデル
   - 通信トラフィック、ネットワーク構成、イベント情報を学習

2. **Sarashina連携**
   - SB Intuitions社が開発する国産LLM
   - 日本語処理に最適化
   - LTMと組み合わせて国産AIモデルに発展

3. **予測アルゴリズム**
   - 過去のイベントデータ
   - リアルタイムトラフィックデータ
   - 気象情報、SNS動向等の外部データ

## 7. 成功要因分析

| 要因 | 詳細 |
|------|------|
| **自社開発へのコミット** | 海外LLM依存ではなく、国産LLM開発に投資 |
| **業界特化モデル** | 汎用LLMではなく、通信業界に特化したモデル設計 |
| **データ資産活用** | 長年蓄積した通信トラフィックデータを学習に活用 |
| **Sarashina連携** | SB Intuitions社との連携で日本語処理を強化 |
| **実運用での検証** | 実際のイベントでPoC実施し、90%+の精度を実証 |

3. **業界標準化への貢献**

   - 他通信事業者への技術提供

---

## 10. 他社への示唆

1. **業界特化型AI開発の有効性**: 汎用LLMをそのまま使うのではなく、業界固有データで追加学習することで精度向上が見込める

2. **国産LLM活用のメリット**: セキュリティ、日本語対応、国内規制対応の観点から国産LLMの価値が高い

3. **データ資産の活用**: 長年蓄積したデータが生成AI時代の競争力の源泉となる

4. **90%精度の実用性**: 予測精度90%は実運用に十分なレベルであり、完璧な100%を待つ必要はない
