# 657_JAL_JAL-AI

## 1. 企業名・業界
- **企業名**: 日本航空株式会社（JAL）
- **業界**: 航空運輸業
- **企業規模**: 大企業（グループ従業員数: 約35,000人）

## 2. 導入AIツール名・提供企業
- **ツール名**:
  - JAL-AI（全社員向け生成AIツール）
  - JAL-AI Report（客室乗務員向けレポート作成AI）
  - LITRON Multi Agent Simulation（マーケティング）
  - 機内持ち込み手荷物解析AI
- **提供企業**:
  - アバナード（JAL-AI開発支援）
  - 富士通、ヘッドウォータース（JAL-AI Report）
  - マイクロソフト（Phi-4小規模言語モデル）
  - NTTデータ（LITRON Multi Agent Simulation）
  - NEC（手荷物解析AI）
- **AI分類**: 生成AI（RAG、オンデバイスAI）、画像認識AI

## 3. 導入目的・解決すべき課題
- 全社員の業務効率化
- 議事録作成の効率化
- 社内ファイル横断検索の効率化
- 客室乗務員のレポート作成業務効率化
- オフライン環境でのAI活用
- マーケティング施策の高度化
- 機内持ち込み手荷物の積載量推定

## 4. 導入プロセス
1. **2023年4月**: 生成AIリスク評価・活用検討ワーキンググループ立ち上げ
2. **2024年1月**: アバナードを新パートナーとしてプロジェクト再スタート
3. **2024年度**: JAL-AIの間接部門への展開（利用率100%達成）
4. **2024年4月～9月**: NEC手荷物解析AI実証実験
5. **2025年3月**: 富士通・ヘッドウォータースとJAL-AI Report実証実験発表
6. **2025年1月**: NTTデータとLITRON実証実験開始

## 5. 具体的な活用内容
### JAL-AI（全社員向け）
- **機能**:
  - RAG技術を使った社内文書検索（マニュアル、FAQ、業務ガイドライン）
  - 議事録自動作成
  - ドライブ内ファイル横断検索
- **利用状況**: 間接部門社員の実質100%が利用

### JAL-AI Report（客室乗務員向け）
- **動作環境**: オフライン（通信環境不安定な機内でも利用可能）
- **基盤モデル**: マイクロソフトPhi-4（小規模言語モデル）
- **機能**: キーワード・フレーズ入力とチェックボックス選択でレポート自動生成
- **デバイス**: タブレット端末

### LITRON Multi Agent Simulation（マーケティング）
- **提供**: NTTデータ
- **機能**: AIバーチャル顧客同士の会話からマーケティング施策導出
- **成果**: 従来ターゲットより購買率3.0%向上

### 機内持ち込み手荷物解析AI
- **共同開発**: NEC
- **期間**: 2024年4月～9月実証実験
- **機能**: 搭乗口で手荷物の個数・種類を機械学習で自動解析
- **特徴**: 世界初のソリューション

## 6. 技術的詳細
- **RAG（Retrieval-Augmented Generation）**: 社内文書検索と回答生成
- **マイクロソフトPhi-4**: 小規模言語モデル、オンデバイス動作
- **Azure OpenAI Service**: JAL-AIの基盤（推定）
- **LITRON Multi Agent Simulation**: 生成AI活用のマルチエージェントシミュレーション
- **機械学習**: 画像認識による手荷物解析
