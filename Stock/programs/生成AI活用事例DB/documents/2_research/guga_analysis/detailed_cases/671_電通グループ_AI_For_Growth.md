# 671_電通グループ_AI For Growth

| 項目 | 内容 |
|------|------|

| **業種** | 広告・マーケティング |
| **従業員数** | 約1,114人がG検定資格取得（2024年11月時点） |
| **導入時期** | 2024年8月～（AI For Growth発表） |
| **AI戦略名** | AI For Growth / AI For Growth 2.0 |

## 2. 導入したAIツール・技術

| AIツール/技術名 | 提供元 | 主な機能 |
|----------------|--------|----------|
| **AICO2** | 電通（自社開発） | AI広告コピー生成（音声広告対応） |
| **People Model** | 電通（自社開発） | 1億人規模のAIペルソナ再現 |
| **Creative Thinking Model** | 電通×東京大学AIセンター | 高度なビジュアルアイデア生成 |

| **minnect AIアシスト** | 電通総研 | 自治体向けセキュアなSaaS型生成AI |
| **Amazon Bedrock** | AWS | 生成AIサービス基盤 |
| **Amazon SageMaker** | AWS | 機械学習プラットフォーム |

## 3. 導入目的・背景

### 戦略的背景

2024年8月に発表した「AI For Growth」は、「人間の知（Intelligence）」と「AIの知」を掛け合わせて顧客企業や社会の成長に貢献する方針を打ち出した。2025年5月には「AI For Growth 2.0」として刷新し、マーケティング領域の「AIネイティブ化」を目指す。

### 具体的な目的

- マーケティングプロセスの高度化
- クリエイティブ業務の効率化と創造性向上
- 社会課題解決への貢献
- 従業員のAIリテラシー向上

## 4. 具体的な活用方法・事例

### AICO2（AIコピーライター）

電通のコピーライターの思考プロセスを学習したAIが広告コピーを生成。音声広告にも対応している。

### People Model

電通独自の大規模調査データをLLMでファインチューニングし、1億人規模のAIペルソナを仮想的に再現。マーケティングターゲット分析に活用。

### People IMC Prototyping（2025年5月本格提供）

マーケティングプロセスの4ステップを生成AIで高度化：
1. ターゲット規定
2. コミュニケーション戦略設計
3. 戦術設計
4. 評価

### 藤沢市への自治体向け生成AI導入

電通総研が「minnect AIアシスト」をセキュアな生成AI環境として藤沢市に導入。

## 5. 導入効果・成果

### 定量的成果

- グループ39社で1,114人がG検定資格取得（2024年11月時点）
- マーケティングプロセスの高度化による業務効率向上（具体的数値は取得不可）

### 定性的成果

- 独自AIモデル開発による競争優位性の確立
- 社内業務の効率化および提供サービスの高度化
- クライアント企業のマーケティング支援強化
