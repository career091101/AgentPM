## 2. AI導入サマリー

JFEスチールは製鉄DXを3つの柱で推進している:(1)独自生成AI「Chat JFE」による業務効率化、(2)高炉デジタルツイン化(CPS)による操業最適化、(3)AI画像解析による労働災害削減。2024年度までに年間25万時間以上の業務時間創出を実現。

**主な導入事例:**
- **Chat JFE**: セキュアな社内生成AIサービス(2,000名以上利用)
- **高炉CPS**: 12時間先の高炉状態を予測するデジタルツイン
- **AI不良検知**: NEC RAPID機械学習で緩衝材トラブル検知率80%
- **労働災害削減AI**: Lightblueと協業で動作検知システム導入
- **RPA/ローコード**: 1,400種類以上の業務に導入、年間25万時間創出

## 4. 導入背景・課題

**経営課題:**
- カーボンニュートラル実現に向けた製鉄プロセスの革新
- 熟練作業者の高齢化と技能伝承
- 労働災害ゼロの実現
- 国際競争力の強化

**具体的な課題:**
1. **生成AI活用**: セキュリティを確保した社内利用環境の整備
2. **高炉操業**: 複雑な化学反応プロセスの最適化と予測精度向上
3. **品質管理**: 製品歩留まり改善と稼働率向上
4. **安全管理**: 労働災害削減と基本行動遵守の徹底
5. **データ活用**: 現場社員によるセルフサービス分析環境の不足

## 5. ソリューション詳細

### 1. Chat JFE(独自生成AIサービス)
**概要:**
- 独自のセキュリティ対策とガイドラインを整備した社内専用生成AI
- 2,000名以上の従業員が文章作成、翻訳、情報検索に活用

**技術スタック:**
- Azure OpenAI Service利用(CPS開発環境から利用可能)
- GitHub Copilot統合(コーディングアシスト)

### 2. インテリジェント製鉄所構想 - CPS化
**J-DNexus(Digital Nexus):**
- 2024年9月運用開始のデジタルプラットフォーム
- 主要ラインのCPS(サイバー・フィジカル・システム)化を加速
- Microsoft Azureを全面採用

**高炉デジタルツイン:**
- センサーで温度・圧力データをリアルタイム収集
- 高炉内部状態をCPSで可視化
- 12時間先の高炉状態を予測

### 3. AI不良検知システム(日立製作所と協業)
**システム概要:**
- 圧延機による鋼板の形状補正を自動制御
- JFEスチールのノウハウとAIを組み合わせたソリューション

**導入効果(2021年導入):**
- 製品歩留まり改善
- 稼働率向上
- オペレーター作業負荷軽減

### 4. AIデータ分析プラットフォーム「dotData」
**特徴:**
- データサイエンス専門知識不要で現場社員が自ら分析
- 300人以上が活用
- 30件以上のプロジェクトが実用段階

### 5. 労働災害削減AI(Lightblueと協業)
**システム概要:**
- 画像解析AIによる動作検知システム
- 基本行動遵守を検知・評価

### 6. RPA・ローコード開発
**実績:**
- 1,400種類以上の業務に導入
- 年間25万時間以上創出
- 市民開発者630名(2024年度末)

## 7. 成功要因分析

### 組織的要因
1. **全社DX推進体制**: J-DNexusを基盤とした統合デジタル戦略
2. **市民開発者育成**: 630名の市民開発者による現場起点のDX
3. **セキュアな環境整備**: Chat JFEで独自ガイドライン策定

### 技術的要因
1. **クラウドファースト**: Microsoft Azure全面採用
2. **ノーコード/ローコード**: dotDataで専門知識不要の分析環境
3. **デジタルツイン**: 高炉CPSで12時間先予測実現

### マネジメント要因
1. **外部連携**: Microsoft、日立、Lightblue、dotDataと戦略的提携
2. **実証から実用へ**: 2021年自社導入→2024年外販の段階的展開
