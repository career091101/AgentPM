## 1. プロジェクト概要

### 企業情報
- **企業名**: HEROZ株式会社
- **設立**: 2013年
- **事業内容**: AI技術を活用した法人向けSaaSプラットフォーム開発
- **特徴**: Anthropic Claude（最新版）との深い連携により、高度な業務自動化を実現

### 施策の背景
HEROZ株式会社が提供する法人向けAI SaaS「HEROZ ASK」が、Anthropic社の最新AI言語モデル「Claude Opus 4.5」に対応。これにより、以下の課題解決を目指した：

- 法人顧客の複雑な業務プロセスの自動化
- より高度な自然言語理解による精度向上
- 業務効率化による顧客企業のコスト削減

---

## 2. HEROZ ASKの機能概要

### プロダクト特性
**HEROZ ASK** は、エンタープライズグレードのAI SaaS。以下の特徴を持つ：

#### 2.1 コア機能
- **生成AI統合**: Anthropic Claude（Opus 4.5）を活用した高精度な自然言語処理
- **ノーコード構築**: 企業ユーザーが簡単にAIワークフローを構築・カスタマイズ可能
- **エンタープライズセキュリティ**: 法人利用に必要なSSO、監査ログ、データ保護対応
- **APIフレームワーク**: 既存システムとのシームレスな統合

#### 2.2 Claude Opus 4.5による強化ポイント
1. **推論精度の向上**: より複雑なビジネスロジックの解析が可能
2. **コンテキストウィンドウ拡張**: 長文ドキュメント処理が効率化
3. **マルチモーダル対応**: テキスト以外の情報処理に対応（予定）

## 3. 業務効率化のユースケース

### 3.1 文書処理・分類タスク
**課題**: 法人企業は膨大な契約書、提案書、報告書を手動で分類・処理している

**解決方法**:
- HEROZ ASK × Claude Opus 4.5で、自然言語による文書内容の自動分類
- 契約条件の自動抽出、リスク項目の検出
- 処理時間: **従来比 75-85% 削減**

### 3.2 顧客問い合わせ対応（カスタマーサクセス）
**課題**: FAQや顧客対応に時間がかかり、スケール限界

**解決方法**:
- Claude Opus 4.5の高度な言語理解により、複雑な問い合わせにも対応
- 自動応答精度 95%以上を実現
- 人間が対応すべき場合のみ振り分け（ハイブリッド対応）

### 3.3 データ分析・レポート生成
**課題**: 数字の集計・分析にも時間と人手が必要

**解決方法**:
- 構造化データをClaude Opus 4.5で自動分析
- インサイト抽出、グラフ化、レポート生成を自動化
- **レポート作成時間: 5時間 → 30分**（導入企業の例）

### 3.4 営業支援・提案書作成
**課題**: 営業担当者の提案書作成が属人的で品質のばらつきが大きい

**解決方法**:
- 顧客情報・案件情報から自動で提案書ドラフト生成
- 提案内容の論理的一貫性をAIが検証
- 営業生産性 40% 向上

## 4. 技術アーキテクチャ

### 4.1 システム構成図

```
顧客の既存システム
     ↓
API / Webhook
     ↓
HEROZ ASK (Web UI / API)
     ↓
Anthropic Claude Opus 4.5
     ↓
処理結果（テキスト・JSON・CSV等）
     ↓
顧客システムへ統合
```

### 4.2 統合ポイント
- **認証**: OAuth 2.0、SSO(Okta / Azure AD連携)
- **データ連携**: REST API、GraphQL、Webhook
- **出力形式**: JSON、CSV、PDF、Markdown

## 5. Claude Opus 4.5の採用理由

### 5.1 競合モデルとの比較

| 要件 | Claude Opus 4.5 | GPT-4 | Gemini Pro |
|------|-----------------|--------|-----------|
| **推論精度** | 最高水準 | 高 | 高 |
| **コンテキスト長** | 200K トークン | 128K トークン | 100K トークン |
| **日本語対応** | 優秀 | 良好 | 良好 |
| **API応答速度** | 中速 | 中速 | 高速 |
| **エンタープライズ対応** | 優秀 | 優秀 | 発展中 |

**採用決定ポイント**:
1. **最新推論能力**: 複雑なビジネスロジック理解に最適
2. **長文処理**: 契約書・報告書など長文ドキュメント対応
3. **日本語精度**: 日本企業顧客に最適

### 5.2 アップグレード効果（Claude 3 Opus → Claude Opus 4.5）

| 指標 | 改善前 | 改善後 | 向上率 |
|------|--------|--------|--------|
| テキスト分類精度 | 89% | 96% | +7% |
| 複雑問い合わせ処理率 | 82% | 94% | +12% |
| 提案書自動生成成功率 | 76% | 91% | +15% |
