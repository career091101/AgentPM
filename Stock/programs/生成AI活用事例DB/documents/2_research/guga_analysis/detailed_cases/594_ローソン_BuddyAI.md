# 594_ローソン_生成AI全社導入

## 1. 企業基本情報

| 項目 | 内容 |
|------|------|
| 企業名 | 株式会社ローソン |
| 業種 | 小売業（コンビニエンスストア） |
| 設立 | 1975年4月15日 |
| 本社所在地 | 東京都品川区 |
| 従業員数 | 約4,000人（本社） |
| 主要事業 | コンビニエンスストアチェーン運営 |

## 2. AI/生成AI導入の背景

### 課題
- 商品開発のアイデア創出効率化
- 会議・メール作成の時間削減
- 店舗運営の高度化
- インバウンド対応強化

### 経営判断
- 2024年夏に全社員への生成AI展開を決定
- 体験型店舗による先端技術の実証
- 販促最適化AIの導入

## 3. 導入したAI技術の詳細

### AIツール名称
1. **生成AI（ChatGPT系）** - 全社員向け
2. **AI Ponta** - 顧客サービス向け
3. **DNP販促最適化AI** - 広告配分最適化
4. **アバター×生成AI** - 体験型店舗

### 技術仕様
- **生成AI**: 業務効率化ツール（詳細非公開）
- **AI Ponta**: 生成AI搭載の自然対話型サービス
- **販促AI**: DNP開発の宣伝予算配分最適化システム
- **アバター**: AVITA社の生成AIアバター技術

## 4. 具体的な活用領域

### 本社業務（2024年夏～）
- 商品開発のアイデア創出・壁打ち
- 会議の要約作成
- メール文章作成
- データ分析支援

### 店舗運営
- **ローソンJAM BASE店**（2024年9月大阪）: アバター×生成AIの体験型店舗
- **Real×Tech LAWSON**（2025年6月高輪）: AI Ponta搭載店舗
- 販促最適化AI（2024年11月～新店・リニューアル店）
