| 項目 | 内容 | ソース |

| 企業名 | 全日本空輸株式会社（ANAホールディングス） | [公式サイト](https://www.ana.co.jp/) |

| 本社所在地 | 東京都港区 |  |
| 協業先 | neoAI、ソフトバンク、avatarin |  |
| 公表日 | 2024年〜（継続発表） |  |

## 2. AI導入サマリー

| 項目 | 内容 |
|------|------|
| AIツール名 | ANA Blue LLM、neoAI Chat、avatarin |
| ベンダー | neoAI、ソフトバンク |
| 導入時期 | 2024年8月〜 |
| 対象範囲 | グループ約4万人、航空機オペレーション部門 |
| 主要技術 | 生成AI、LLM、ロボティクス |

## 3. 定量的効果

| 指標 | 数値 | 備考 |
|------|------|---------:|
| 情報検索時間 | 約90%削減 | 見込み |
| 文書作成時間 | 約75%削減 | 見込み |
| 対象従業員 | 約4万人 | グループ全体 |
| 現業職対象 | 約2,000人 | 先行導入 |
| 資金調達 | 77億円累計 | avatarin |

## 4. 導入背景・課題

- 航空機オペレーションの効率化
- 膨大な規程・マニュアルの検索負荷
- 従業員の生産性向上
- 顧客体験価値の向上

## 5. ソリューション詳細

### ANA Blue LLM
- **社内クローズド環境**
- 独自生成AI基盤

**特徴**:
- ChatGPT等LLMを社内で構築
- セキュアな情報取り扱い
- グループ4万人対象

### 航空機オペレーション向け生成AI
- **2024年8月試験導入**
- neoAI Chat採用

**機能**:
- 対話形式での指示に応答
- 規程・過去議事録から情報提示
- 根拠文書も同時に表示

**効果見込み**:
- 情報検索時間: 約90%削減
- 文書作成時間: 約75%削減

**対象業務**:
- 報告書作成
- 規定作成
- 教育資料作成

### 本格導入（2025年10月）
- **運航現場向け**
- 約2,000人から順次拡大

**対象職種**:
- 空港係員
- 整備士
- 客室乗務員（CA）

### avatarin（ロボティクスAI）
- **ソフトバンクとの協業**
- 2024年7月発表

**内容**:
- AI・5G・ロボティクス活用
- 人手不足業界向け接客ソリューション

**資金調達**:
- ソフトバンク含む6社から37億円
- 累計77億円達成

### AI活用の目的（ANA方針）
- **2つの目的**
- デジタル変革室主導

**目的**:
1. 従業員の生産性向上
2. 顧客サービス体験価値の向上

| 段階的展開 | 試験→検証→本格導入 |

## 8. 課題・対策

| 課題 | 対策 |
|------|------|

## 9. 今後の展開

- 航空機オペレーションAIの全社展開
