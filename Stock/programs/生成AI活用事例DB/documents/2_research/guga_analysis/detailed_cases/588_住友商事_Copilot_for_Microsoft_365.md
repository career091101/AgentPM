# 588_住友商事_Copilot for Microsoft 365

## 1. 企業基本情報
- **企業名**: 住友商事株式会社
- **業種**: 総合商社
- **企業規模**: 五大商社
- **従業員数**: グループ約9,000人（対象範囲）

## 2. AI導入の概要
- **導入AIツール名**: Microsoft 365 Copilot
- **提供ベンダー**: Microsoft
- **導入時期**: 2024年4月
- **導入規模**: 海外グループ会社を含む約9,000人（日本企業初のグローバル全社導入）

## 3. 導入目的・背景
住友商事は、日本企業初となるMicrosoft 365 Copilotのグローバル全社導入を実施。2023年4月に生成AI活用ワーキンググループを組成し、2023年5月にはグループにおける生成AIの実装を支援するCoE組織「SC-Ai Hub」を設立し、全社的なAI活用を推進。

## 4. 活用シーン・業務内容
- メールの下書き作成
- 資料作成
- Web会議の要約
- 汎用的な業務での生成AIフル活用
- 投融資委員会議事録分析
- リテイルデータ活用の高度化

## 5. 具体的な成果・効果
### 定量的成果
- **月間アクティブユーザー**: 約90%（2025年10月時点）
- **利用者数**: 約9,000人（2024年12月時点）
- **コスト削減効果**: 年間約12億円

### 定性的成果
- 日本企業初のグローバル全社導入
- 経営層肝煎りの導入により全社的な活用文化を醸成

## 6. 技術スタック
- **AIモデル**: Microsoft 365 Copilot
- **クラウド環境**: Microsoft Azure
- **CoE組織**: SC-Ai Hub

## 7. 導入プロセス・期間
- 2023年4月: 生成AI活用ワーキンググループ組成
- 2023年5月: CoE組織「SC-Ai Hub」設立
- 2024年4月: 約9,000人へのCopilot一斉利用開始
- 2025年10月: 月間アクティブユーザー約90%達成

## 8. 課題・障壁と解決策
### 直面した課題
取得不可

### 解決策
- CoE組織「SC-Ai Hub」の設立により全社実装を支援
- 経営層肝煎りの導入により組織的な推進体制を構築

## 9. 今後の展開・計画
- 投融資委員会議事録分析システムを2025年1月にリリース予定
- 東京大学越塚研究室と共同で、生成AIを用いたリテイルデータ活用の高度化に向けた研究開発を推進（2025年度内にプロトタイプ完成予定）

## 10. ROI・投資対効果
年間約12億円のコスト削減効果を達成。

## 11. 他の導入事例（同社）
