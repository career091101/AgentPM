## 🔻 リトレースメント（調整目標）

### 第2波の調整
```
第2波は第1波の 50% 〜 61.8% まで調整することが多い
```

| 調整レベル | 解釈 |
|------------|------|
| 38.2% | 浅い調整（強いトレンド） |
| **50%** | **標準的な調整** |
| **61.8%** | **典型的な調整** |
| 78.6% | 深い調整（注意） |
| 100%超 | ルール違反 → カウント誤り |

### 第4波の調整
```
第4波は第3波の 38.2% 〜 50% まで調整することが多い
```

| 調整レベル | 解釈 |
|------------|------|
| **38.2%** | **よくある調整** |
| 50% | やや深い調整 |
| 61.8% | 深い調整（ダイアゴナル可能性） |

### 修正波B波の調整
```
B波はA波の 38.2% 〜 78.6% まで戻すことが多い
```

| パターン | B波の戻し |
|----------|-----------|
| ジグザグ | 50% 〜 61.8% |
| フラット | 78.6% 〜 100% |
| 拡大フラット | 100%超 |


## 📏 計算方法

### リトレースメント計算
```
調整目標 = 高値 - (高値 - 安値) × フィボナッチ比率
```

**例**: 第1波が100から110に上昇、第2波の61.8%調整
```
調整目標 = 110 - (110 - 100) × 0.618 = 103.82
```

### エクステンション計算
```
拡大目標 = 起点 + (基準波の値幅) × フィボナッチ比率
```

**例**: 第1波が10ポイント、第3波の161.8%目標
```
第3波目標 = 第2波終点 + 10 × 1.618 = 第2波終点 + 16.18
```

### 値幅の目安（日足ベース）
| 波動 | 典型的な値幅 |
|------|-------------|
| 1日の動き | 200-500円 |
| 1波 | 1,000-3,000円 |
| 3波 | 1,600-8,000円 |
| 5波 | 1,000-3,000円 |

### 計算例
```
現在値: 38,000円
第1波: 37,000 → 39,000（2,000円上昇）
第2波: 39,000 → 37,800（61.8%調整）
第3波目標（161.8%）: 37,800 + (2,000 × 1.618) = 41,036円
```
