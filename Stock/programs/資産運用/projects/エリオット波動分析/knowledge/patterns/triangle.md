# トライアングル（Triangle）

> **出典**: セミナー第8回、エリオット波動研究.pdf

---

## 基本定義

**修正波**の一種。3-3-3-3-3の5波動構成（A-B-C-D-E）。

---

## 波動構成

```
3 - 3 - 3 - 3 - 3
│   │   │   │   └─ E波: 修正波
│   │   │   └───── D波: 修正波
│   │   └───────── C波: 修正波
│   └───────────── B波: 修正波（何でもOK）
└───────────────── A波: 修正波（トライアングル出ない）
```

---

## 4つのバリエーション

| 型 | 特徴 |
|----|------|
| **収縮型** | 幅がだんだん狭くなる（最も一般的） |
| **ランニング** | B波がA波始点を超える（収縮型の40%） |
| **バリア型** | BDラインが水平（株価はBD方向に抜ける） |
| **拡大型** | 幅がだんだん広がる（性質が異なる） |

---

## 絶対ルール

1. **各複事波は必ず修正波**
2. **A波にトライアングルは出ない**
3. **C波はA波終点を超えて動かない**（収縮型）
4. **E波はC波終点を超えて動かない**
5. **D波はB波終点を超えて動かない**

---

## 出現位置

| 位置 | 備考 |
|------|------|
| インパルスの4波 | 最も一般的 |
| ジグザグのB波 | よく出る |
| ダブルジグザグのX波 | よく出る |
| フラットのB波 | 稀 |
| トライアングルのB/C/D/E波 | A波には出ない |
| W3の最後のアクション波 | W波には出ない |
| トリプル3の2つ目のX波のみ | 1つ目のXには出ない |

---

## スラスト

4波にトライアングル出現 → 5波は**スラスト**（短時間で終了）

- 5波の大きさ ≒ トライアングルの最大範囲
- スラスト範囲を超えたら **5波延長の前兆**

---

## カウントミス例

1. **ダブルジグザグのX波だった**
2. **C波/D波がダブルジグザグだった**
3. **実はダイアゴナルの一部だった**
