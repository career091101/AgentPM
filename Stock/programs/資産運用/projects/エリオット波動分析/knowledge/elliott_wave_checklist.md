## 🔍 分析前チェック

### 時間軸の確認
- [ ] 分析対象の時間軸を決定した（日足/4時間足/1時間足）
- [ ] 上位時間軸のトレンドを確認した
- [ ] 下位時間軸でサブ波を確認した
- [ ] ログスケール表示で分析している（セミナー2,8）

### データ準備
- [ ] 十分な過去データがある
- [ ] 直近の高値・安値をマークした

---

## ✅ 3つの絶対ルール検証（セミナー4）

### ルール1: 第2波の位置
```
□ 第2波の安値 > 第1波の起点
```
- [ ] 第2波が第1波の起点を下回っていないか
- [ ] **違反 → 第1波の認識が誤り**

### ルール2: 第3波の長さ
```
□ 第3波 > 第1波 または 第3波 > 第5波
```
- [ ] 第3波が最短にになっていないか
- [ ] **違反 → カウントをやり直す**

### ルール3: 第4波と第1波の重複
```
□ 第4波の安値 > 第1波の高値
```
- [ ] 第4波が第1波の価格帯に入っていないか
- [ ] **違反 → ダイアゴナルか別パターン検討**
