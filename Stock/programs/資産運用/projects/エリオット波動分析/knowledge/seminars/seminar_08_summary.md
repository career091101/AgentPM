## キーポイント

### 1. トライアングルの波動構成（33333）

| 波 | 構成 | 補足 |
|---|---|---|
| **A波** | 修正波 | トライアングル **出ない** |
| **B波** | 修正波 | トライアングル出ることあり |
| **C波** | 修正波 | トライアングル出ることあり |
| **D波** | 修正波 | トライアングル出ることあり |
| **E波** | 修正波 | トライアングル出ることあり |

> **重要**: A波にはトライアングルは絶対に出ない（フラットでも同様）

### 2. トライアングルの4つのバリエーション

| 型 | 特徴 |
|---|---|
| **収縮型** | だんだん幅が狭くなる（最も一般的） |
| **ランニング** | B波がA波始点を超える（収縮型の40%） |
| **バリア型** | BDラインが水平（株価はBD方向に抜ける） |
| **拡大型** | だんだん幅が広がる（性質が異なる） |

### 3. トライアングルの絶対ルール

1. **各複事波は必ず修正波**
2. **A波にトライアングルは出ない**
3. **C波はA波終点を超えて動かない**（収縮型）
4. **E波はC波終点を超えて動かない**
5. **D波はB波終点を超えて動かない**

### 4. トライアングルの出現位置

| 位置 | 備考 |
|---|---|
| インパルスの4波 | 最も一般的 |
| ジグザグのB波 | よく出る |
| ダブルジグザグのX波 | よく出る |
| フラットのB波 | 稀に出る |
| トライアングルのB/C/D/E波 | A波には出ない |
| W3の最後のアクション波 | W波には出ない |
| トリプル3の最後のX波のみ | 1つ目のXには出ない |

### 5. ログスケール上の波動の大きさ計算

**手順**:
1. 変化率（%）を計算
2. 変化率に1を足す（例: 30.02% → 1.3002）
3. log₁₀（1+変化率）を計算
4. この値が波動の大きさ

> **注意**: 変化率同士を直接比較するのは間違い

### 6. スラスト（4波トライアングル後の5波）

- 4波にトライアングル出現 → 5波はスラスト（短時間で終了）
- 5波の大きさ ≒ トライアングルの最大範囲（A波始点からACライン延長まで）
- スラスト範囲を超えたら **5波延長の前兆**

## トライアングルのカウントミス例

### パターン1: ダブルジグザグのX波だった

トライアングル完成と思ったら、実はW波の一部で、さらにY波が続く

### パターン2: C波/D波がダブルジグザグだった

リトレースの深いダブルジグザグがC波やD波に出現

### パターン3: 実はダイアゴナルの一部だった

4波トライアングルと思ったら、5波ダイアゴナルの途中だった

## 重要な用語・概念

| 用語 | 定義 |
|---|---|
| **ランニングトライアングル** | B波がA波始点を超える（40%出現） |
| **バリア型** | BDラインが水平、株価はBD方向に抜ける |
| **スラスト** | 4波トライアングル後の5波が短時間で終わる現象 |
| **ACライン/BDライン** | アクション波/リアクション波の終点を結んだライン |

---

## 実践ポイント

1. **A波にトライアングルは出ない**: これを覚えれば間違いが減る
2. **C波/D波は複雑化しやすい**: 早合点せず待つ
3. **E波は極端に小さく終わることがある**: 「足が1本」のこともある
4. **BDラインの方向に株価は抜ける**: 緩やかな方へ抜ける
5. **ACE同士、BD同士の大きさがフィボナッチ関係になりやすい**
