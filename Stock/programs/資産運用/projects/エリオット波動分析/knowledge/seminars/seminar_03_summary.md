## キーポイント

### 1. ディグリー（波動の階層）

エリオット波動では波の規模を「ディグリー」で分類

| ディグリー | 日本語訳 | 記号例 |
|---|---|---|
| Grand Supercycle | グランドスーパーサイクル | (I) (II) |
| Supercycle | スーパーサイクル | (I) (II) |
| Cycle | サイクル | I II |
| Primary | プライマリー | ① ② |
| Intermediate | インターミディエット | (1) (2) |
| Minor | マイナー | 1 2 |
| Minute | ミニュット | i ii |
| Minuette | ミニュエット | (i) (ii) |
| Subminuette | サブミニュエット | i ii |

### 2. 推進波と修正波の違い

| 項目 | 推進波（Motive Wave） | 修正波（Corrective Wave） |
|---|---|---|
| **別名** | アクション波 | リアクション波 |
| **方向** | 一回り大きな波と同じ方向 | 一回り大きな波と逆方向 |
| **構成** | インパルス、ダイアゴナル | ジグザグ、フラット、トライアングル |
| **波動数** | 5波動 | 3波動 |

### 3. インパルスの基本

| 波 | 性質 |
|---|---|
| **1波** | アクション波 |
| **2波** | リアクション波 |
| **3波** | アクション波（最も強い） |
| **4波** | リアクション波 |
| **5波** | アクション波 |

### 4. FX市場への適用

- エリオット波動原理は**株式市場**を前提に開発された
- **FX市場**への適用は可能だが注意が必要
- 通貨は相対的な価値なので、片方の通貨の波動を追う必要がある

## 重要な用語・概念

| 用語 | 定義 |
|---|---|
| **ディグリー** | 波動の規模・階層を示す概念 |
| **推進波（Motive）** | 一回り大きな波と同じ方向に動く波 |
| **修正波（Corrective）** | 一回り大きな波と逆方向に動く波 |
| **インパルス** | 5波構成の推進波 |

## 実践ポイント

1. **ディグリーを意識**: 今見ている波は全体のどの規模か
2. **表記法を統一**: チーム内で記号を統一する
3. **アクション/リアクションの区別**: 方向と形を混同しない
