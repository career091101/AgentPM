## キーポイント

### 1. インパルスの波動構成

**5-3-5-3-5**: 推進波-修正波-推進波-修正波-推進波

| 波 | 波動構成 | 性質 |
|---|---|---|
| **1波** | 5波動 | 推進波 |
| **2波** | 3波動 | 修正波 |
| **3波** | 5波動 | 推進波（必ずインパルス） |
| **4波** | 3波動 | 修正波 |
| **5波** | 5波動 | 推進波 |

### 2. インパルスの3つの絶対ルール

| # | ルール | 説明 |
|---|---|---|
| 1 | **2波は1波の始点を割らない** | 2波のリトレースは1波の100%未満 |
| 2 | **3波は最短にならない** | 1, 3, 5波のうち3波が最も短くなることはない |
| 3 | **1波と4波は重ならない** | 4波は1波の終点を超えない |

> **重要**: これらは「ルール」であり「ガイドライン」ではない。1つでも破ればインパルスではない。

### 3. インパルスのガイドライン

| ガイドライン | 説明 |
|---|---|
| **チャネリング** | 1波と3波の終点を結んだ線と平行線でチャネルを形成 |
| **1波と5波の等距離** | 延長がなければ1波と5波は同程度の大きさ |
| **2波の深いリトレース** | 2波は1波を深くリトレース（50%〜78.6%） |
| **4波は3波を浅くリトレース** | 4波のリトレースは38.2%程度が多い |
| **オルタネーション** | 2波と4波は性質が異なる（時間、深さ、複雑さ） |

### 4. 延長波（エクステンション）

1, 3, 5波のいずれか1つが「延長」することがある

| 延長波 | 特徴 |
|---|---|
| **1波延長** | 稀、見分けにくい |
| **3波延長** | 最も一般的 |
| **5波延長** | 次に多い、完成後に2波終点まで急反転 |

## 重要な用語・概念

| 用語 | 定義 |
|---|---|
| **インパルス** | 5波構成の推進波、3つの絶対ルールを満たす |
| **オルタネーション** | 2波と4波が異なる性質を持つこと |
| **延長（エクステンション）** | 1, 3, 5波のいずれかが著しく大きくなること |
| **チャネリング** | 平行線で価格の範囲を想定する技法 |

---

## 3つの絶対ルール詳細

### ルール1: 2波は1波の始点を割らない

- 2波のリトレースがどんなに深くても、1波の始点（0%地点）を超えない
- 超えた時点でインパルスではない

### ルール2: 3波は最短にならない

- 1波、3波、5波を比較して、3波が最も短いことはない
- 3波 > 1波 または 3波 > 5波 が必ず成立

### ルール3: 1波と4波は重ならない

- 4波の終点が1波の終点を超えない（価格帯が重ならない）
- **例外**: レバレッジ商品やダイアゴナルでは重なることがある

---

## 実践ポイント

1. **3つのルールを最初に確認**: カウント前にルール違反がないか
2. **延長を探す**: どの波が延長しているか判断
3. **オルタネーションを活用**: 2波の性質から4波を予測
4. **チャネルを引く**: 1-3, 2-4ラインで5波の目標を設定
