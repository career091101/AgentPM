## 1. 理論の起源と背景

- **提唱者**: ラルフ・ネルソン・エリオット（1938年）
- **原理**: 相場の値動きは群衆心理の反映であり、一定のパターンで繰り返される
- **適用市場**: 株式、FX、先物、暗号通貨など（本プロジェクトはJP225）

## 2. 波動の基本構造

### 2.1 完全サイクル（8波構成）
```
上昇5波 + 下降3波 = 1サイクル

      5
     /\
    /  \
   /    4
  3      \
 /\       \
2  \       A
    1       \
             B
              \
               C
```

### 2.2 波動の階層

| 階層 | 時間軸 | 例 |
|------|--------|-----|
| Grand Supercycle | 数十年〜数百年 | 歴史的な超長期トレンド |
| Supercycle | 数年〜数十年 | 長期的な景気サイクル |
| Cycle | 1年〜数年 | 年単位のトレンド |
| Primary | 数ヶ月〜1年 | 主要トレンド |
| Intermediate | 数週間〜数ヶ月 | 中期トレンド |
| Minor | 数日〜数週間 | 短期トレンド |
| Minute | 数時間〜数日 | デイトレレベル |
| Minuette | 数分〜数時間 | スキャルピングレベル |

## 3. 推進波の詳細

### 3.1 各波の特徴

#### 第1波
- トレンド転換の始まり
- 多くの参加者はまだ前のトレンドを信じている
- 出来高は比較的少ない

#### 第2波
- 第1波の調整
- **絶対ルール**: 第1波の起点を割らない
- リトレースメント: 通常50%〜61.8%

#### 第3波
- **最も重要な波**
- 最も長く、力強い動き
- 出来高が最大になる
- **絶対ルール**: 最短にならない
- 典型的な拡大率: 161.8%〜261.8%

#### 第4波
- 第3波の調整
- **絶対ルール**: 第1波の価格帯と重ならない
- 複雑な形状になりやすい（横ばい調整）

#### 第5波
- トレンドの最終局面
- 第3波より弱いことが多い
- ダイバージェンスが発生しやすい
- フェイリャー（未達）の可能性

### 3.2 推進波のパターン

| パターン | 説明 |
|----------|------|
| インパルス | 標準的な5波構成 |
| ダイアゴナル（エンディング） | 第5波でくさび形に収束 |
| ダイアゴナル（リーディング） | 第1波でくさび形 |
| エクステンション | 第3波が延長（最も一般的） |

## 4. 修正波の詳細

### 4.1 修正波の種類

#### ジグザグ（5-3-5）
```
    B
   /\
  A  \
      C
```
- 急なA波とC波
- B波は浅い戻り

#### フラット（3-3-5）
```
  A   B
 ___/\___
        C
```
- A波とB波がほぼ同じ高さ
- C波で下落

#### トライアングル（3-3-3-3-3）
```
  A   C   E
   \ / \ /
    B   D
```
- 5つのサブ波で構成
- 通常、第4波や修正波Bで出現

### 4.2 複合修正
- ダブルスリー（W-X-Y）
- トリプルスリー（W-X-Y-X-Z）

## 5. フラクタル構造

エリオット波動は**入れ子構造**を持つ：
- 大きな波は小さな波で構成される
- 小さな波も同じ5-3パターンを持つ

```
1波（大）= 1-2-3-4-5（小）
2波（大）= a-b-c（小）
...以下同様
```

## 7. 波動カウントの実践

### 7.1 カウントの手順

1. **大きな時間軸から始める**
2. **明確な5波を見つける**
3. **3つのルールをチェック**
4. **サブ波を確認**
5. **複数シナリオを用意**

### 7.2 カウントの検証

- [ ] 第3波は最短でないか？
- [ ] 第2波は第1波の起点を割っていないか？
- [ ] 第4波は第1波と重なっていないか？
- [ ] フィボナッチ比率と整合性があるか？

## 8. よくあるパターン

### 8.1 第3波のエクステンション
最も一般的。第3波が第1波の1.618倍以上に延長。

### 8.2 トランケーション（フェイリャー）
第5波が第3波の高値を超えられない現象。トレンド転換のサイン。

### 8.3 オルタネーション
第2波と第4波は異なる形状になりやすい：
- 第2波がシャープ → 第4波はサイドウェイ
- 第2波がサイドウェイ → 第4波はシャープ
