# 波動カウント分析プロンプト

> **用途**: 特定のチャート範囲の波動カウントを依頼する  
> **対象**: JP225（日経平均先物）

---

## 目的

指定されたチャート範囲について、エリオット波動のカウントを行い、各波の位置と特徴を特定する。

---

## 入力テンプレート

```
【分析対象】
- 銘柄: JP225
- 時間軸: [日足/4時間足/1時間足]
- 期間: YYYY/MM/DD 〜 YYYY/MM/DD

【価格データ】
- 期間始値: ¥XX,XXX
- 期間終値: ¥XX,XXX
- 期間高値: ¥XX,XXX（MM/DD）
- 期間安値: ¥XX,XXX（MM/DD）

【主要な価格ポイント】
1. ¥XX,XXX（MM/DD）- [高値/安値]
2. ¥XX,XXX（MM/DD）- [高値/安値]
3. ¥XX,XXX（MM/DD）- [高値/安値]
...

【質問】
この期間の波動カウントを行い、現在の位置を特定してください。
```

---

## 期待される出力

### 波動カウント結果

```
【大局観】
- 全体トレンド: [上昇/下降/レンジ]
- 上位波動での位置: 第X波

【詳細カウント】

         ¥40,000 ────────────── 5波頂点
              /\
             /  \
            /    \
¥38,500 ──/──────\───── 3波頂点
          /  4波   \
         /          \
¥37,000 /────────────\──── 1波頂点
       1波     2波
      
起点: ¥36,000（MM/DD）
```

### 各波の詳細

| 波 | 開始 | 終了 | 値幅 | 特徴 |
|----|------|------|------|------|
| 1波 | ¥36,000 | ¥37,000 | +1,000 | トレンド転換開始 |
| 2波 | ¥37,000 | ¥36,400 | -600 | 60%調整 |
| 3波 | ¥36,400 | ¥38,500 | +2,100 | 最大波（210%拡大）|
| 4波 | ¥38,500 | ¥37,800 | -700 | 33%調整 |
| 5波 | ¥37,800 | ¥40,000 | +2,200 | 進行中 |

### ルール検証

- [x] 第3波は最短でない（2,100円 > 1,000円）
- [x] 第2波は第1波起点を割っていない（36,400 > 36,000）
- [x] 第4波は第1波と重なっていない（37,800 > 37,000）

### フィボナッチ整合性

| チェック項目 | 理論値 | 実際値 | 判定 |
|-------------|--------|--------|------|
| 2波調整（61.8%） | ¥36,382 | ¥36,400 | ✅ 一致 |
| 3波拡大（161.8%） | ¥38,018 | ¥38,500 | ✅ 達成 |
| 4波調整（38.2%） | ¥37,698 | ¥37,800 | ✅ 近似 |

## 追加分析オプション

必要に応じて以下も依頼可能：

1. **サブ波分析** - 各波の内部構造（5波/3波）
2. **時間分析** - 波動間の時間比率
3. **オルタネーション確認** - 2波と4波の形状比較
4. **上位波との整合性** - より大きな時間軸での位置
