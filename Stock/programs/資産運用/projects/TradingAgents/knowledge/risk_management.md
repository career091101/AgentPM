# リスク管理指針

## 概要

トレード戦略の実行におけるリスク管理の原則とガイドラインを定義します。

---

## リスク管理の3原則

### 1. 資金管理（Position Sizing）

**2%ルール**: 1トレードのリスクは資金の2%以内

**計算例**:
```
資金: 100万円
最大リスク: 100万円 × 2% = 2万円

エントリー価格: 33,000円
ストップロス: 32,000円
リスク幅: 1,000円

ポジションサイズ = 最大リスク / リスク幅
                = 20,000円 / 1,000円
                = 20単位（ミニ先物なら20枚）
```

**重要**: 勝率に関わらず、常に2%ルールを遵守

### 2. 損切りの徹底

**ストップロスは絶対**:
- 設定したストップロスに到達したら、例外なく即座に損切り
- 「もう少し待てば戻るかも」という希望的観測は禁物

**損切り価格の設定方法**:
1. **ATRベース**: エントリー価格 - 1×ATR
2. **直近安値**: 直近のサポートレベル
3. **固定%**: エントリー価格の-2%

**例**:
```
エントリー: 33,000円
ATR: 700円
ストップロス: 33,000 - 700 = 32,300円（-2.1%）
```

### 3. 利益確定の段階化

**段階的利確**:
- **第1目標**: 50%のポジションを決済
- **第2目標**: 残り50%を決済

**例**:
```
エントリー: 33,000円（20単位）
第1目標: 34,000円 → 10単位決済（利益確定）
第2目標: 34,500円 → 残り10単位決済
```

**トレーリングストップ**（オプション）:
第1目標達成後、ストップロスを建値（エントリー価格）に移動

## リスク・リワード比率

### 最小R:R比率: 1:1.5

**定義**:
```
R:R比率 = (目標価格 - エントリー価格) / (エントリー価格 - ストップロス)
```

**判定**:
- **R:R ≥ 2.0**: 優秀、積極的にトレード
- **1.5 ≤ R:R < 2.0**: 良好、条件が揃えばトレード
- **1.0 ≤ R:R < 1.5**: 慎重に判断
- **R:R < 1.0**: トレード非推奨

**例**:
```
エントリー: 33,000円
目標価格: 34,500円（+1,500円）
ストップロス: 32,300円（-700円）
R:R = 1,500 / 700 = 2.14 ✅ 優秀
```

## 期待値計算

### 期待値の重要性

**期待値が正であることが必須**

**計算式**:
```
期待値 = (勝率 × 平均利益) - (負け率 × 平均損失)
```

**例**:
```
勝率: 60%
平均利益: +2.0%
負け率: 40%
平均損失: -1.0%

期待値 = (0.60 × 2.0) - (0.40 × 1.0)
       = 1.2 - 0.4
       = +0.8% ✅ 正の期待値
```

**最小基準**: 期待値 > +0.5%

## ポジションサイズの調整

### ボラティリティに応じた調整

**ATR%別のポジションサイズ**:

| ATR% | ボラティリティ | ポジションサイズ |
|------|--------------|----------------|
| < 2% | 低 | 通常（100%） |
| 2-3% | 中 | 80% |
| > 3% | 高 | 60% |

**理由**: 高ボラティリティ時はストップロスに引っかかりやすい

### 連敗時の調整

**連続損失後のポジションサイズ**:

| 連敗数 | ポジションサイズ |
|-------|----------------|
| 0-1回 | 100% |
| 2回 | 75% |
| 3回以上 | 50%、戦略見直し |

**理由**: 連敗時は市場環境が戦略に合っていない可能性

## 最大ドローダウン管理

### 許容最大DD: 資金の20%

**ドローダウンが15%に達したら**:
1. 新規ポジション停止
2. 戦略の見直し
3. バックテストの再実行

**ドローダウンが20%に達したら**:
1. すべてのポジション決済
2. トレード一時停止
3. 戦略の全面見直し

**計算例**:
```
初期資金: 100万円
許容最大DD: 20万円

現在資金: 85万円
DD: 15万円（15%） → 警戒レベル

現在資金: 80万円
DD: 20万円（20%） → 停止レベル
```

## トレード頻度の管理

### 過度なトレード（Over-Trading）の防止

**週間最大トレード数: 5回**

**理由**:
- スイングトレードは中期戦略
- 頻繁なトレードは手数料増加
- 衝動的なトレードの防止

**1日の最大トレード数: 1回**

**理由**:
- 冷静な判断を保つ
- 複数ポジションのリスク累積防止

## 時間帯別のリスク管理

### 日経225先物の取引時間

| 時間帯 | セッション | 流動性 | リスク |
|-------|----------|-------|--------|
| 9:00-15:15 | 日中取引 | 高 | 低 |
| 16:30-翌5:30 | ナイトセッション | 中 | 中 |
| 5:30-9:00 | 取引停止 | - | - |

**推奨**:
- **エントリー**: 日中取引時間（9:00-15:15）
- **エグジット**: 両セッションで可
- **ストップロス**: 24時間有効

## 市場環境別のリスク管理

### Bull Market（上昇相場）
- **戦略**: 買いポジション中心
- **リスク**: 標準（2%）
- **注意**: 天井圏での過熱

### Bear Market（下降相場）
- **戦略**: 売りポジション or ポジション縮小
- **リスク**: 保守的（1.5%）
- **注意**: 急反発リスク

### Sideways（レンジ相場）
- **戦略**: レンジの上限/下限での逆張り
- **リスク**: 保守的（1.5%）
- **注意**: ブレイクアウトリスク

### High Volatility（高ボラ）
- **戦略**: ポジションサイズ縮小
- **リスク**: 保守的（1.0-1.5%）
- **注意**: スリッページ、ギャップ

---

## 心理的リスク管理

### 感情的トレードの防止

**禁止事項**:
1. **リベンジトレード**: 損失を取り戻すための無計画なトレード
2. **過信トレード**: 連勝後の過度なリスクテイク
3. **FOMO（乗り遅れ恐怖）**: 既に大きく動いた後のエントリー

**対策**:
- **トレード日誌**: 全トレードの記録と振り返り
- **ルールの明文化**: 戦略を文書化し、遵守
- **休息**: 連敗時は一時休止

## リスクチェックリスト

### トレード前の確認事項

- [ ] ポジションサイズは資金の2%以内か？
- [ ] ストップロスを設定したか？
- [ ] R:R比率は1.5以上か？
- [ ] 期待値は正か？
- [ ] 今日のトレード回数は1回以内か？
- [ ] 連敗中ではないか？（連敗中ならポジション縮小）
- [ ] ドローダウンは15%未満か？
- [ ] 感情的になっていないか？

**すべて✅の場合のみトレード実行**

---

## 緊急時の対応

### 想定外の事態

**急激な暴落/暴騰**:
1. すべてのポジション即座に決済
2. 市場が落ち着くまで新規エントリー停止

**システム障害**:
1. 電話での決済手段を確保
2. ブローカーの緊急連絡先を控える

**重大ニュース**:
1. ポジション保有中なら、ニュースの影響を即座に判断
2. 不確実性が高い場合は決済

## リスク管理の評価指標

### 定期的な見直し（月次）

| 指標 | 目標値 | 評価方法 |
|-----|--------|---------|
| 最大ドローダウン | < 20% | 累積損益曲線から計算 |
| 月間勝率 | > 50% | 勝ちトレード数 / 総トレード数 |
| 平均R:R | > 1.5 | 平均利益 / 平均損失 |
| 月間リターン | > 2% | (月末資金 - 月初資金) / 月初資金 |

**目標未達の場合**: 戦略の見直し

---

## まとめ

リスク管理の本質は、**長期的に生き残ること**です。

**黄金律**:
1. 1トレードのリスクは資金の2%以内
2. ストップロスは絶対に守る
3. R:R比率 ≥ 1.5
4. 期待値が正の戦略のみ実行
5. 感情に支配されない

これらを遵守すれば、長期的に利益を積み上げることができます。
