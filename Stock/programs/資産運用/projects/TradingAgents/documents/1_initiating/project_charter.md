## プロジェクト目的

24 時間取引可能な日経平均先物（日本 225 株価指数）市場において、今後 1 週間（5 営業日）で確実な利益を狙える実用的なスイングトレード戦略を立案するシステムを構築する。

### ビジネス上の必要性

1. 日経平均先物市場は 24 時間取引可能であり、スイングトレードに適している
2. テクニカル分析に基づく体系的な戦略立案が必要
3. リスク管理を含めた包括的なトレード判断の自動化が求められる

---

## スコープ

### 含まれるもの（In Scope）

- テクニカル分析モジュール（MA, RSI, MACD, ボリンジャーバンド）
- エントリー/イグジットシグナル生成
- リスク管理（ストップロス/テイクプロフィット設定）
- 週次戦略レポート生成
- バックテスト機能

### 含まれないもの（Out of Scope）

- 自動発注システム（実取引との連携）
- リアルタイム価格アラート
- 複数市場対応（日経平均以外）

## リスクと制約事項

### 主要リスク

1. **市場リスク**: 急激な市場変動による予測精度低下
2. **データリスク**: 外部 API の停止・制限
3. **技術リスク**: 過剰最適化（オーバーフィッティング）

### 制約事項

- 金融商品取引法の遵守が必要
- 投資助言ではなく、最終判断は人間が行う
- API 制限を考慮したデータ取得

---

## 前提条件

- Python 3.9 以上の環境が利用可能
- yfinance 等のデータ取得手段が利用可能
- 過去 3 年分以上の日経平均先物データが取得可能
