# 2026年 金融占星術 重要イベントカレンダー

このドキュメントは、Merriman Financial Astrology Analysis 2026年次レポートに基づき、2026年の主要なジオコスミックイベント、逆行期間、および各市場の重要反転日（CRD: Critical Reversal Dates）を集約したものです。

## 凡例
*   **Stock**: 株式市場（DJIA, Nikkei 225等）
*   **T-Notes**: 米国債
*   **FX**: 為替市場（主にUSD, EUR, CHF）
*   **Gold/Silver**: 金・銀
*   **BTC**: ビットコイン
*   **Oil**: 原油
*   **Grains**: 穀物（コーン、小麦、大豆）
*   **CRD**: Critical Reversal Date（重要反転日）。前後3営業日（場合によりそれ以上）のオーブを考慮すること。
