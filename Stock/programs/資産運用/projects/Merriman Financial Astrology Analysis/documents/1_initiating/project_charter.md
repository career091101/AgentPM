# プロジェクト憲章: Merriman Financial Astrology Analysis

## 1. プロジェクト概要

### 1.1 プロジェクト名
Merriman Financial Astrology Analysis（メリマン金融占星術分析）

### 1.2 プロジェクトの目的
Merriman Market Analyst（MMA）の金融占星術レポートを活用し、ジオコスミック（天体配置）情報を基に金融市場の転換点とサイクルタイミングを分析する。エリオット波動分析との統合により、「何が起きるか（波動構造・価格目標）」と「いつ起きるか（タイミング・サイクル）」を組み合わせた精緻な相場分析システムを構築する。

### 1.3 ビジネス価値
- **タイミング精度の向上**: 相場転換点の予測精度を向上させることで、ポジション取り・利益確定のタイミングを最適化
- **リスク管理の高度化**: 長期サイクルの把握により、大幅下落（37-63%）前の利益確定とキャッシュポジション構築を実現
- **投資機会の最大化**: サイクルボトム（19年サイクルなど）での長期投資開始により、世代を超えた資産形成機会を獲得
- **知識資産の蓄積**: 50年の歴史を持つMMAレポートの体系的整理により、再利用可能な知識ベースを構築

## 2. プロジェクトの背景

### 2.1 背景
金融市場の相場分析において、テクニカル分析（エリオット波動、フィボナッチ等）は価格目標と波動構造の予測には有効であるが、「いつ転換が起きるか」のタイミング特定には限界がある。一方、Raymond Merrimanが1976年から50年間発行し続けているMMAレポートは、天体配置（ジオコスミック）と市場サイクルの相関関係を実証的に研究しており、特に長期サイクル（19年、9.5年、6.33年等）の転換点予測において高い精度を示している。

**2026年は特に重要な年であり、以下の歴史的天体配置が集中する**：
- **土星-海王星コンジャンクション**（36年周期、2026年2月20日）：過去2回（1953年、1989年）でバブルと長期天井を形成
- **Aries Vortex**（2025-2027年）：天王星-冥王星トライン、土星-海王星合、複数の外惑星が牡羊座0度に集中する史上初の配置
- **日経225の19年サイクル終盤**：2008年10月から17年経過、2027-2030年に長期ボトム予想

**2025年12月31日時点の状況**：
- 土星-海王星コンジャンクションまで**あと約2ヶ月**（カウントダウン期）
- 知識ベース構築および2026年年次分析が完了済み
- 最重要イベントに向けた準備フェーズが最終段階

### 2.2 ビジネスニーズ
1. **定期レポートの体系的管理**: 年次レポート（全般・JP225特化）、週次レポートを知識ベースとして整理
2. **トレードカレンダーの構築**: 重要転換日（CRD: Critical Reversal Date）をカレンダー化し、トレード判断の参考とする
3. **エリオット波動分析との連携**: 「統合分析プロジェクト」へのインプット提供により、波動カウントとタイミングの両面から相場を分析
4. **長期投資戦略の策定**: 19年サイクルボトム（2027-2030年）に向けた資金配分計画の立案

## 付録C: 用語集

- **CRD（Critical Reversal Date）**: ジオコスミック重要反転日。重要な天体配置が形成される日付で、市場の転換点と高い相関を示す。
- **コンジャンクション**: 2つの惑星が同じ度数に位置する配置（0度）。新しいサイクルの開始を示す。
- **オポジション**: 2つの惑星が180度の位置関係。サイクルの頂点・緊張の最高潮を示す。
- **トライン**: 2つの惑星が120度の位置関係。調和的なエネルギーを示す。
- **レトログレード（逆行）**: 惑星が見かけ上逆方向に動く期間。市場の不安定性や再検討と関連。
- **Aries Vortex**: 2025-2027年に牡羊座0度を中心に複数の外惑星が集中する史上初の配置。
- **プライマリーサイクル**: 主要サイクル。本プロジェクトでは23ヶ月サイクル以上を指す。
- **ハーフプライマリーサイクル**: プライマリーサイクルの半分の期間。
- **57ヶ月移動平均線**: 9.5年サイクルの半分に相当。長期サイクル転換の重要指標。
