# 惑星サイクルと相場

## サイクル概要

惑星の公転周期に基づく市場サイクル。長期惑星ほど大きなサイクルに対応。

## 主要惑星サイクル

| 惑星 | 公転周期 | 市場への影響 |
|------|---------|-------------|
| 水星 | 約88日 | 短期的なボラティリティ |
| 金星 | 約225日 | 価値・通貨に関連 |
| 火星 | 約687日 | エネルギー・衝動的動き |
| 木星 | 約12年 | 拡大・楽観相場 |
| 土星 | 約29年 | 収縮・調整相場 |
| 天王星 | 約84年 | 突発的変化・イノベーション |
| 海王星 | 約165年 | 幻想・バブル |
| 冥王星 | 約248年 | 構造的変革 |

## 重要なサイクル組み合わせ

### 木星-土星サイクル（約20年）
経済の大きなトレンド転換に関連。
コンジャンクション（合）は新時代の始まりを示唆。

### 土星-天王星サイクル（約45年）
既存秩序と変革の緊張関係。
スクエア・オポジションで市場の大変動。

### 土星-冥王星サイクル（約33-38年）
権力構造の再編、危機と再生のテーマ。

## MMAレポートでの活用

年次レポートにて、該当年の重要サイクル位相を確認し、大局的な相場観を形成する。
