# Merriman金融占星術分析

## 概要

Merriman Financial Astrology Analysis（MMA）のレポートを活用した金融占星術による相場分析プロジェクト。

ジオコスミック（天体配置）情報を基に、相場の転換点・サイクルのタイミングを分析する。

## レポート種別

### 年次レポート
- **全般レポート**: 翌年1年間の主要市場のジオコスミック展望
- **JP225特化レポート**: 日経225に特化した年間展望

### 週次レポート
- 毎週配信される短期的なジオコスミック分析
- 直近の重要天体配置とその影響

## 活用方法

1. **年次レポート**
   - 年初に重要なジオコスミック日付を抽出
   - トレードカレンダーとして反転日候補をマーキング

2. **週次レポート**
   - 週ごとの重要天体配置をトラッキング
   - 短期的な反転・加速ポイントの確認

## エリオット波動分析との連携

| 分析手法 | 役割 |
|---------|------|
| エリオット波動 | **何が**起きるか（波動構造・価格目標） |
| 金融占星術 | **いつ**起きるか（タイミング・サイクル） |

両プロジェクトは独立して運用しつつ、相互参照により分析精度を高める。
