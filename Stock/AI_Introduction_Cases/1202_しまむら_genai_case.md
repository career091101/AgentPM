# しまむら - 生成AI導入事例

## 1. 企業概要

- **企業名**: 株式会社しまむら
- **業種**: 小売業（衣料品専門店）
- **事業内容**: 郊外型衣料品チェーンストアの運営、ファッション衣料・生活雑貨の販売
- **従業員数**: 約8,000名（グループ全体）
- **店舗数**: 約1,400店舗（ファッションセンターしまむら）
- **売上高**: 約5,800億円（2023年度）
- **証券コード**: 8227（東証プライム市場）
- **本社**: 埼玉県さいたま市北区宮原町
- **特徴**: 日本最大級の衣料品小売チェーン、郊外型低価格業態

## 2. 生成AI導入の背景・目的

### 背景
- **若年層顧客獲得の課題**: 10代〜20代の若い女性顧客層の取り込みが課題
- **トレンド対応のスピード**: 移り変わりの早いファッショントレンドへの迅速な対応が必要
- **撮影コスト・時間**: 従来の人間モデルを使った撮影は、スケジュール調整や撮影に時間とコストがかかる
- **プロモーションの柔軟性**: 頻繁なビジュアル更新が求められるSNS時代への対応

### 目的
1. **若年層顧客の獲得**: 10代〜20代女性をターゲットとした新しい顧客層の開拓
2. **プロモーションのスピード向上**: リードタイム短縮による迅速な販促活動
3. **コスト削減**: モデル撮影に関わる「さがけ業務」（撮影・採寸・原稿作成）のコスト削減
4. **SNS活用強化**: Instagram等のSNSを中心とした情報発信の活性化
5. **トレンド対応力向上**: 高速で変化するファッショントレンドへの即応性向上

## 3. 導入した生成AIソリューション

### 主要ツール
1. **AI model株式会社のAIモデル生成サービス**
   - AIモデル「瑠菜（るな / Luna）」の生成・運用
   - 画像生成AI技術を活用したバーチャルモデルの作成
   - SNS投稿用コンテンツの自動生成

### プロジェクト体制
- **3社共同プロジェクト**:
  1. **株式会社しまむら**: 販促活用・店舗展開
  2. **タキヒヨー株式会社**: 繊維商社として企画・コーディネート
  3. **AI model株式会社**: AIモデル生成技術の提供

### 技術的特徴
- **画像生成AI**: 実在しない人物（AIモデル）の高精度生成
- **コスチューム合成**: しまむらの商品をAIモデルに着用させる画像生成
- **ポーズ・表情バリエーション**: 多様なビジュアル表現の自動生成
- **リアルタイム更新**: トレンドに応じた迅速なビジュアル変更

## 4. 主な活用領域

### 1. SNSプロモーション（2024年5月〜）
- **Instagram**: 公式アカウントでAIモデル「瑠菜」を活用した投稿
- **名前募集キャンペーン**: 2024年5月にSNSでAIモデルの名前を公募
- **キャラクター設定**: 身長158cm、20歳、服飾専門学校生、ファッションモデル志望という人格設定

### 2. 店舗プロモーション（2024年6月〜）
- **下高井戸店（東京都杉並区）**: ポスター・チラシでAIモデル「瑠菜」を起用
- **店頭ビジュアル**: 今後、全国店舗への展開を計画

### 3. 広告展開
- **デジタル広告**: Web広告でのAIモデル活用
- **ECサイト**: オンラインショップでの商品紹介
- **カタログ**: 印刷物でのAIモデル採用

### 4. 商品企画・トレンド対応
- **迅速なビジュアル制作**: トレンドアイテムの即座のビジュアル化
- **多様なスタイリング提案**: 1つの商品に対する複数のコーディネート提案

## 5. 導入効果・成果

### 定量的成果
**現時点では公開情報なし**（プロジェクト開始が2024年5月のため、具体的な数値成果は今後公開予定）

### 定性的成果
1. **高い完成度**: 「言われないとわからない」レベルのAIモデルの品質
2. **SNS反響**: 「これAIなの？」「かわいい」といったポジティブな反応
3. **スキャンダルリスク回避**: 人間モデルと異なり、スキャンダルによるブランドイメージ毀損リスクがゼロ
4. **柔軟性向上**: スケジュール調整不要で、いつでも撮影・ビジュアル制作が可能
5. **若年層への訴求**: 10代〜20代女性をターゲットとしたSNS中心の情報発信を実現

### 期待される効果
- **リードタイム短縮**: 従来数週間かかっていた撮影・制作が数時間〜数日に短縮
- **コスト削減**: モデル撮影に関わる「さがけ業務」（撮影・採寸・原稿作成）のコスト削減
- **ファン獲得**: 10代〜20代女性顧客のファン化促進
- **トレンド対応スピード向上**: 高速で変化するファッショントレンドへの即応性向上

### 具体的な反響
- SNSコメント欄では「どこにでもいそうな可愛い子」「リアルで驚いた」「違和感がない」といった好意的な反応
- AIモデル活用に対する関心の高まり（報道多数）

## 8. 課題と対応策

### 課題1: AIモデルの違和感・不自然さ
**対応策**:
- 高精度な画像生成技術による「言われないとわからない」レベルの品質確保
- 服飾専門学校生という親しみやすいキャラクター設定
- SNSでの名前募集による顧客参加型のブランディング

### 課題2: 倫理的懸念・透明性
**対応策**:
- AIモデルであることを公開し、透明性を確保
- 実在人物の肖像権侵害リスクを回避
- 3社協業による権利関係の明確化

### 課題3: 若年層への訴求力
**対応策**:
- 10代〜20代女性をターゲットとしたキャラクター設定（20歳、服飾専門学校生）
- Instagram中心のSNS戦略
- トレンドに即したビジュアル展開

### 課題4: コスト・リードタイム
**対応策**:
- AIモデル活用による撮影コスト削減
- スケジュール調整不要で迅速なビジュアル制作
- 「さがけ業務」（撮影・採寸・原稿作成）の効率化

### 課題5: ブランドイメージ
**対応策**:
- 郊外型低価格業態のイメージから、若年層向けファッションブランドへの転換
- SNSを通じたコミュニケーション強化
- トレンド感のあるビジュアル展開
