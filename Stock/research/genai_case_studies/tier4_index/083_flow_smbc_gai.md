## 2. AI導入サマリー

| 項目 | 内容 |
|------|------|
| AIツール名 | SMBC-GAI、AI-CEO |
| ベンダー | Microsoft Azure OpenAI Service、OpenAI（GPT-4o）、富士通 |
| 導入時期 | 2023年7月〜（継続展開） |
| 対象範囲 | グループ全社員 |
| 主要技術 | Azure OpenAI、RAG、AIアバター |

## 3. 定量的効果

| 指標 | 数値 | 備考 |
|------|------|---------|
| 生成AI投資枠 | 500億円 | 次期中計期間まで累計 |
| デジタル投資枠 | 8,000億円 | 2025年度まで中期経営計画 |
| 非互換情報抽出時間削減 | 約65% | 富士通共同実証 |
| 開発期間 | 4ヶ月 | ルールづくりから実用化まで |

## 4. 導入背景・課題

- 業務効率化とDX推進
- 従業員の生産性向上
- セキュアなAI活用環境の構築
- システム開発の効率化

## 5. ソリューション詳細

### SMBC-GAI（グループ専用生成AIアシスタント）
- 2023年7月に従業員専用の利用を開始
- Azure OpenAI Serviceを活用
- 開発からルールづくりまで4ヶ月で実用化
- グループ専用環境上で動作
- 情報が社外に流出しないセキュアな設計

### AI-CEO（2025年8月発表）
- グループCEO中島達を模したAIモデル
- OpenAI社のGPT-4oを活用して開発
- システムプロンプトとRAG技術を使用
- AIチャットボットおよびAIアバターのUI設計
- 2025年度内に三井住友銀行で試行開始予定

### システム開発への生成AI活用（富士通共同実証）
- 日本総合研究所と富士通が共同実証
- Red Hat Enterprise Linux（RHEL）のバージョンアップに活用
- 2024年11月5日〜2025年1月15日に検証
- 非互換情報の抽出時間を約65%削減

### AIソリューション事業会社の設立（2025年7月発表）
- シンガポールにAIソリューション事業会社を設立予定
- AIエージェント関連ソリューションの開発・導入支援
- SMBCグループをファーストカスタマー
- 将来的には法人顧客へのソリューション提供を展望
