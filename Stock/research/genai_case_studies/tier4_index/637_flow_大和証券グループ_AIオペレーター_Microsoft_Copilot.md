# 637_大和証券グループ_AIオペレーター / Microsoft Copilot

## 1. 企業概要
- **企業名**: 大和証券グループ本社
- **業種**: 金融・証券
- **従業員数**: 約15,000人（グループ全体）
- **事業内容**: 証券、投資銀行、アセットマネジメント等

## 2. AI導入の背景・課題
顧客からの問い合わせ対応効率化、営業職員の面談記録作業削減、全社員の生産性向上

## 3. 導入したAIツール
### 主要ツール
1. **AIオペレーターサービス** (2024年10月7日リリース)
   - 株価・市況ニュース等のマーケット情報対応
   - ログイン手続き、NISA関連の一般的な問い合わせ対応
   - ピーク時1日1.5万件超の問い合わせ対応（2025年6-7月）

2. **Microsoft Copilot 365** (2025年6月全社導入)
   - 利用社員のうち83%が業務活用達成（2025年9月時点）

3. **ChatGPT** (2023年4月全社導入)
   - 全社員9,000人が利用可能

4. **顧客面談内容自動記録システム** (2025年1月開始)
   - コンサルティング後の記録時間45%削減

5. **生成AIチャットサービス** (2025年6月23日開始)
   - ホームページ「よくあるご質問」に応対

## 4. 導入プロセス
- **2023年4月**: ChatGPT全社導入（9,000人）
- **2024年5月**: AIオペレータープロジェクト始動
- **2024年10月**: AIオペレーターサービスリリース
- **2025年1月**: 面談内容自動記録システム開始
- **2025年6月**: Microsoft Copilot 365全社導入、生成AIチャット開始
- **2025年10月**: Sakana AIと戦略的技術パートナーシップ契約

## 5. 主な機能・活用シーン
- AIオペレーター: 1日1.5万件超の問い合わせに自動対応
- 面談記録: 記録時間45%削減
- Microsoft Copilot: 83%の社員が業務活用成功

## 6. 導入効果・成果
| 指標 | 成果 |
|------|------|
| AIオペレーター対応件数 | ピーク時1.5万件/日 |
| 面談記録作成時間削減 | 45% |
| Copilot業務活用率 | 83% |
| ChatGPT利用規模 | 9,000人 |

## 7. 今後の展開予定
- Sakana AIとの長期的パートナーシップによる新サービス開発
- AIオペレーターの機能拡張
- 全社的なAI活用のさらなる深化

## 8. 他社への示唆
**成功要因**: 早期のChatGPT全社導入（2023年4月）、段階的な機能追加、複数パートナーとの連携（Microsoft、Sakana AI）
