# 703_日立コンサルティング_生成AI

## 1. 基本情報

| 項目 | 内容 |
|------|------|
| 企業名 | 株式会社日立コンサルティング / 日立製作所 |
| 業種 | コンサルティング・ITサービス |
| 従業員数 | 取得不可 |
| 調査日 | 2026-01-07 |

## 2. AI/生成AIツール名称

- **MA-ATRIX**（生成AI活用成熟度モデル、2025年10月無償公開）
- **生成AI活用プロフェッショナルサービス powered by Lumada**（2024年7月開始）
- **OT専用AIエージェント**（2025年3月発表）
- **AI駅係員サービス**（京王電鉄、2024年12月実証実験）

## 3. 導入時期

- **2024年7月**: 生成AI活用プロフェッショナルサービス提供開始
- **2024年12月〜2025年3月**: 京王電鉄AI駅係員実証実験
- **2025年3月**: OT専用AIエージェント発表
- **2025年10月**: MA-ATRIX無償公開
- **継続中**: SI工程での生成AI活用拡大

## 4. 導入目的・背景

- **業務変革加速**: 生成AI活用による段階的な業務変革支援
- **システム開発効率化**: SI工程での生産性30%向上目標
- **保守業務高度化**: インフラ設備メンテナンスの効率化
- **顧客体験向上**: AI駅係員など公共サービスの高度化

## 5. 主な機能・活用方法

### MA-ATRIX（成熟度モデル）
- 生成AI活用の成熟度を客観的に評価
- 段階的な業務変革を支援
- 日立とGen-AXが共同開発
- 無償公開により業界全体の底上げ

### 京王電鉄AI駅係員（2024年12月〜2025年3月）
- お客さまが求める情報をタイムリーに提供
- 快適な駅利用の実現

### SI工程での生成AI活用
- **2024年度効果**: 50億円
- **2027年度目標**: 適用効果1000億円規模、生産性30%向上
- Java 8→Java 17バージョンアップなどで実績

### OT専用AIエージェント（2025年3月）
- 日立ビルシステム、日立パワーソリューションズで活用
- インフラ設備の施工・メンテナンス業務支援
- 問い合わせ対応、初動判断の効率化
- **業務効率約30%向上見込み**

## 6. 導入効果・成果

### SI工程
- **2024年度**: 50億円の適用効果
- **2027年度目標**: 1000億円規模（20倍）、生産性30%向上

### OT専用AIエージェント
- **業務効率約30%向上見込み**: 保守受付担当の初動判断

### 京王電鉄実証実験
- 2024年12月〜2025年3月実施
- お客さま快適性向上
