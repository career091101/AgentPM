## 3. 導入目的・背景

### 戦略的背景

2024年にAI推進チームを発足し、新たな表現手法、業務効率化、課題解決のためのAI活用を推進。Google Cloud「AI Agent Summit '25 Spring - 第3回生成AIイノベーションアワード」でファイナリストに選出。

### 具体的な目的

- 社内情報アクセスの効率化
- 映像資産の最大活用
- 報道品質の向上（ファクトチェック）
- 放送事故防止

## 4. 具体的な活用方法・事例

### Go-chat（社内情報検索AI）

約7,000ページの社内イントラネット情報（総務、人事、経理、技術部門など）を横断検索できるシステム。RAG（Retrieval-Augmented Generation）とGoogleのGemini AIを組み合わせた生成AI。

Google Cloud「AI Agent Summit '25 Spring - 第3回生成AIイノベーションアワード」でファイナリスト選出。

### カオメタ®（顔認識AI映像管理）

東芝の顔認識AI「カオメタ®」をテレビ朝日アーカイブセンターに導入し、300万件超の映像資産を管理。人物特定・メタデータ付与業務を劇的に効率化。

### ファクトチェックLLM開発（2025年8月～2026年2月）

NABLAS（東京大学発ベンチャー）がNEDO採択事業でテレビ朝日と連携し、実際の報道現場での検証とソーシャル実装を目指すファクトチェックLLMとAIエージェントを開発。

### ハザード事象抽出AI（PoC実施）

株式会社Emuni（京都大学・松尾研発ベンチャー）がテレビ朝日と生成AIを用いたヒヤリハット事象の抽出とチェックリストの自動作成のPoCを実施。放送現場のトラブル防止を目指す。

### 日本民間放送連盟賞・技術部門優秀賞W受賞

テレビ朝日はAI応用事績で日本民間放送連盟賞・技術部門の優秀賞をダブル受賞。
