# 634_りそなホールディングス_Microsoft 365 / Data Ignition

## 1. 企業概要
- **企業名**: 株式会社りそなホールディングス
- **業種**: 金融・銀行
- **従業員数**: 約23,000人（グループ全体）
- **事業内容**: 商業銀行、信託銀行等の金融サービス

## 2. AI導入の背景・課題
### ビジネス課題
- 全社員の業務変革とワークスタイル改革
- マネロン・テロ資金供与対策の高度化
- 地域金融機関向けAIツール開発とビジネス展開
- AIガバナンス体制の構築

### 技術的課題
- セキュアな生成AI環境の構築
- 金融犯罪対策へのAIスコアリング適用
- 地域金融機関との協業体制

## 3. 導入したAIツール
### 主要ツール
1. **Microsoft 365 E5 & 生成AI**
   - 導入時期: 2024年3月
   - 戦略的提携: 2025年5月（日本マイクロソフト）
   - 対象: 全社員
   - 特徴: 経営層80名以上が「生成AIを活用した業務改革ハンズオン」を受講

2. **Data Ignition（銀行業務支援ツール）**
   - 開発: りそなHD、りそな銀行、ブレインパッド共同開発
   - 提供開始: 2024年12月30日
   - 対象: 地域金融機関
   - 第1弾導入: 静岡銀行

3. **SAS Financial Crimes Analytics**
   - 用途: マネロン・テロ資金供与対策
   - 運用開始: 2024年1月
   - 特徴: AIスコアリング、アラート確率算定

4. **AIガバナンス構築（NTTデータ共創）**
   - ガイドライン策定
   - リスクチェックプロセス構築
   - 社員向け教育コンテンツ整備

## 4. 導入プロセス
### 導入スケジュール
- **2024年1月**: マネロン対策AI運用開始
- **2024年3月**: Microsoft 365 E5導入
- **2024年10月**: ワークスタイル変革室立ち上げ
- **2024年12月**: Data Ignition提供開始
- **2025年5月**: 日本マイクロソフトと戦略的枠組契約締結
- **2025年6月**: 経営層向け生成AIハンズオン開催（4回、80名以上参加）

### 実装方法
1. **トップダウン戦略**
   - 南昌宏グループCEO以下経営層が率先してAI活用研修参加
   - 全4回の「生成AIを活用した業務改革ハンズオン」

2. **外部パートナー連携**
   - マイクロソフト: 人財育成から業務変革まで包括的パートナーシップ
   - ブレインパッド: AIツール共同開発
   - NTTデータ: AIガバナンス構築

## 5. 主な機能・活用シーン
### 生成AI活用
- 融資稟議書・意見書の自動作成
- 顧客面談内容の音声入力→要約自動化
- 各種資料作成・翻訳・企画業務支援

### Data Ignition
- 地域金融機関向け銀行業務支援
- AI活用による業務効率化
- 静岡銀行が第1弾として導入決定

### マネロン対策AI
- AIを使ったアラート届出確率スコア算定
- 理由の付与
- モデル管理機能による継続的AI運用
- 単一プラットフォームでの一元管理

## 6. 導入効果・成果
### 定量的効果
- マネロン対策AI: 2024年3月末までに本番運用開始（金融庁ガイドライン期限に対応）
- 経営層研修: 4回開催、80名以上参加
- Data Ignition: 静岡銀行導入決定

### 定性的効果
- 経営層のAIリテラシー向上
- 全社的なワークスタイル変革推進
- 地域金融機関への新規ビジネス展開
- マネロン・テロ資金供与対策の高度化

## 7. 技術スタック
### アーキテクチャ
- **Microsoft 365 E5**: 全社基盤
- **Azure OpenAI Service**: 生成AI基盤
- **SAS Financial Crimes Analytics**: マネロン対策専用プラットフォーム
- **Data Ignition**: ブレインパッドとの共同開発AI
