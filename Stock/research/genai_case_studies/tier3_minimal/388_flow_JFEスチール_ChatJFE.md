- デジタル人材の育成

## 5. ソリューション詳細

### Chat JFE（生成AIサービス）
- **2024年構築**
- 独自セキュリティ対策・利用ガイドライン整備

**利用状況**:
- 2,000名を超える従業員が利用
- 文章作成、翻訳、情報検索に活用

**特徴**:
- 安全かつ効率的に利用できる環境
- 全従業員対象

### J-DNexus（CPS基盤）
- **2024年9月運用開始**
- 多工程一貫品質データ解析

**機能**:
- プロセスCPSを全社展開
- 高炉の状態をサイバー空間に再現
- センサーデータ・映像をリアルタイム収集
- 高炉内部状態の可視化

### AIデータ分析プラットフォーム（dotData）
- **現場向けデータ分析ツール**
- データサイエンス専門知識不要

**利用状況**:
- 300人以上の社員が活用
- 30件以上のプロジェクトが実用段階

**特徴**:
- 現場社員自らが課題解決に取り組み可能

### 日立との冷間圧延自動形状制御
- **2024年3月提供開始**
- JFEスチールは2021年に自社導入

**効果**:
- 製品歩留まりの改善
- 稼働率の向上
- オペレーター作業負荷の軽減

**展開**:
- 国内外の鉄鋼業向けにソリューション提供

### 画像解析AIによる安全行動サポート
- **国内業界初（2018年）**
- Lightblueとの協力

**機能**:
- 労働災害削減を目指した動作検知システム
- 基本行動遵守の検知・評価
- 現場従業員の安全行動サポート

### RPA・ローコード開発
- **2024年度までの実績**
- 1,400種類以上の業務に導入

**効果**:
- 年間25万時間以上を創出
- 市民開発者630名（2024年度末）

### GitHub Copilot導入
- **CPS開発環境で活用**
- Azure OpenAI Serviceと連携

**効果**:
- 開発効率の向上
- データ解析・研究開発業務の効率化

### 技能伝承・MR活用
- **複合現実（MR）研修システム**
- 製鉄所の技能伝承に活用

**内容**:
- 溶けた鉄を鋳型に流し込む作業の疑似体験
