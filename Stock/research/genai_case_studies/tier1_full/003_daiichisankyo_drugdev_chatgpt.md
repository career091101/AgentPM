# Tier 1 生成AI導入事例

**事例ID**: 003
**企業名**: 第一三共（Daiichi Sankyo）
**業界**: 医薬品・創薬
**導入AI**: ChatGPT (GPT-4)、Azure OpenAI Service、独自生成AIシステム
**導入時期**: 2023年9月～2024年
**プロジェクト規模**: エンタープライズ全社展開
**導入対象部門**: R&D、臨床開発、医療従事者対応、全社業務

## ビジネス背景

### 課題定義
- **研究効率化**: 膨大な論文・文献データの高速処理が必要
- **開発加速**: AI創薬による開発期間短縮が急務
- **全社デジタル化**: 事務作業の効率化により研究時間確保が重要

### 戦略背景
- **AI戦略**: 「膨大なデータや専門知識がなくても誰でもAIを使える世界」実現
- **プラットフォーム戦略**: 独自生成AIシステムの段階的進化（2023年9月→2024年2月）
- **業界協業**: Tokyo-1プロジェクト（アステラス、小野薬品との連携）

## 導入内容

### 主要プロジェクト1: 独自生成AIシステム「第一三共版ChatGPT」

**進化プロセス**:

| 時期 | 実装内容 | 利用効果 |
|------|--------|--------|
| 2023年9月 | シンプルなAI対話インターフェース | AI基本機能の検証 |
| 2023年11月 | GPT-4実装 | 応答品質向上 |
| 2023年12月 | 社内ドキュメント解析機能 | 文献検索自動化 |
| 2024年1月 | 画像生成機能追加（マルチモーダル） | クリエイティブ業務支援 |
| 2024年2月 | Code Interpreter + データセット解析 | デジタル分析機能強化 |

**技術構成**:
- ベース: Azure OpenAI Service経由GPT-4
- インターフェース: 社内統一ポータル
- セキュリティ: 機密データマスキング機能搭載
- 監視: 利用ログ・効果測定システム

**導入範囲**:
- 全社員が利用可能
- 部門別カスタマイズテンプレート搭載
- API連携で既存システムと統合

### 主要プロジェクト2: AI創薬プロジェクト（エクサウィザーズ連携）

**プロジェクト概要**:
- パートナー: 株式会社エクサウィザーズ
- 開始時期: 2019年から共同開発
- 成果実装: 2023年からヒット化合物創出プロジェクト

**技術内容**:
- ターゲット: 難易度の高い標的タンパク質
- 手法: 超大規模バーチャルスクリーニング（約60億種類の化合物から検索）
- AI活用: 構造予測、親和性評価、候補絞り込み

**成果**:
- 約60億化合物から約2カ月で約400の有望候補を抽出
- 従来プロセスでは6～12カ月要する工程を2カ月に短縮
- 候補化合物の品質も従来法より向上

### 主要プロジェクト3: DIチャットボット「いつでもDI 24」

**概要**: 医療従事者向けAIサービス

**機能**:
- 第一三共製品に関する24時間365日の質問応対
- AI質問理解 + 最適Q&A自動マッチング
- 3年間の継続学習により精度向上

**成果**:
- 医療従事者問い合わせの即座対応率向上
- 製品情報センター業務の30%以上を自動化
- 医療現場の満足度向上

## ChatGPT/Azure OpenAI活用シーン

### シーン1: 論文・文献の要約・検索
- 新規治療法や化学構造の概要把握
- 関連研究論文の自動抽出
- 時間短縮率: 70%以上

### シーン2: 化合物設計の補助
- AI設計値と研究者の知見の融合
- 新規化学構造の提案と検証
- イノベーション加速

### シーン3: 臨床開発ドキュメント作成
- 治験計画書、統計解析計画の自動生成補助
- 多言語翻訳による国際対応
- ドキュメント作成時間30%削減

### シーン4: 医学情報DB検索
- 医療従事者からの製品関連質問への自動応答
- ガイドライン、使用方法の迅速提供

## 実装プロセス

### フェーズ1: 評価・準備（2023年1月～8月）
- ChatGPT/GPT-4のエンタープライズ評価
- セキュリティ・データガバナンス要件定義
- 全社展開に向けた基準・ガイドライン作成

### フェーズ2: MVP～段階展開（2023年9月～2024年1月）
- 初版ポータルの構築（9月）
- 各部門での試験的導入
- フィードバック収集による改善

### フェーズ3: 本格展開～拡張（2024年1月～）
- マルチモーダル機能搭載
- Code Interpreter実装
- 各部門への定着支援・教育

## 投資規模と経済効果

### 投資額
- **初期構築**: 推定10～20億円
- **年間運用**: 3～5億円
- **回収期間**: 2～3年

### 経済効果（実績＆期待値）
- **研究効率化**: 文献検索時間70%削減 → 年間2～3億円相当
- **AI創薬成果**: バーチャルスクリーニング短縮 → 開発期間3～6年短縮
- **全社業務効率**: オフィス業務30%削減 → 年間10～15億円相当
- **トータルROI**: 初期投資3年で回収見込み

## リスク管理

### セキュリティ・ガバナンスリスク
- **対策**: 6つのリスク領域を事前特定
  1. 知財・著作権侵害
  2. 偏ったアウトプット
  3. 個人情報・機密データ漏洩
  4. 目的外利用
  5. 信ぴょう性の欠如
  6. シャドーIT

- **実装対策**:
  - 機密データマスキング
  - 利用ガイドライン・教育の徹底
  - 監視・監査ログの完備
  - 定期的なリスク評価

### 技術的リスク
- **AI精度**: AI創薬の候補化合物品質をバリデーション
- **システム統合**: レガシーシステムとの互換性課題
- **スケーラビリティ**: 全社展開時の負荷対策

## 成功要因

### 戦略的要因
1. **早期評価**: 2023年初からChatGPT/GPT-4を積極評価
2. **セキュリティ先行**: ガバナンスを明確化したうえで展開
3. **全社展開**: 特定部門ではなく全社利用を目指す

### 技術的要因
1. **段階的機能拡張**: 9月シンプル版→2月フル機能版への進化
2. **複数AI併用**: ChatGPT + 独自AI + エクサウィザーズ連携
3. **カスタマイズ**: 部門別テンプレートによる高度な活用

### 組織的要因
1. **教育・啓発**: 全社員向けの利用教育を継続的に実施
2. **ベストプラクティス共有**: 部門間での活用事例の横展開
3. **継続投資**: 新機能開発と既存機能改善の並行

## 教訓・ベストプラクティス

### 成功要因
1. **セキュリティ・ガバナンスを最優先**: エンタープライズAI導入の成否の鍵
2. **段階的展開**: 小さく始めて継続的に改善
3. **複数AI活用**: 単一AIではなく複数技術の組み合わせ

### 留意事項
1. **AI生成物の責任**: 医薬品開発では最終責任は人間が保有すること
2. **競争環境**: 他大手製薬企業も同様に推進中（中外、武田等）
3. **知財管理**: AI生成物の権利帰属を契約で明確化
