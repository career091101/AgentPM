## Tier 1 構造（60フィールド）

各ファイルは以下の8セクションで構成：

1. **基本情報**: 企業名、所在地、業界、従業員数、設立年、AI導入年
2. **導入背景**: 導入前課題、目的、AI選定理由、予算規模
3. **技術詳細**: 使用AI/LLM、統合方法、データソース、インフラ、セキュリティ
4. **実装プロセス**: フェーズ、期間、チーム構成、パートナー
5. **成果・効果**: 定量効果、定性効果、ROI、競争優位性
6. **課題・対策**: 技術的課題、組織的課題、人材育成、解決策
7. **将来展望**: 拡張計画、推奨事項、長期戦略
8. **参考情報**: データソース、更新日

## 使用方法

### 検索
```bash
# 企業名で検索
grep -l "トヨタ" *.md

# 業界で検索
grep -l "金融" *.md

# AIツールで検索
grep -l "Claude" *.md
```

### 分析
```bash
# 総ファイル数
ls -1 *.md | grep -E '^[0-9]{3,4}_' | wc -l

# 日本企業の抽出
grep -l "country: 日本" *.md
```
