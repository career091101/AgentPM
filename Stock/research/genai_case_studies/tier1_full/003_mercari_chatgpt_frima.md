# Tier 1 生成AI導入事例：メルカリ - ChatGPT フリマAI

## 基本情報

### 企業プロフィール
- **企業名**: 株式会社メルカリ
- **業種**: プラットフォーム・フリマアプリ
- **事業規模**: GMV（流通額）1,000億円以上（2024年）
- **本社**: 東京都渋谷区
- **主力サービス**: メルカリ（日本）、Mercari（米国）

### AI導入概要
- **AI ツール**: ChatGPT、GPT-4o、GPT-4o mini
- **導入時期**: 2023年～継続中
- **対象部門**: 商品レコメンド、出品支援、カスタマーサービス
- **導入規模**: 日本版・US版の両プラットフォーム

## 導入背景と課題

### ビジネス課題
1. **出品効率化**: 出品者が商品説明を手入力するコスト
2. **レコメンド精度**: パーソナライズ精度の向上
3. **カスタマーサポート**: 顧客問い合わせ対応の効率化
4. **市場拡大**: US版での利用体験向上

### 市場環境
- フリマアプリ競争の激化
- 出品者数増加に伴うサポートコスト上昇
- ユーザー体験の差別化需要

## 導入内容と実装

### 1. メルカリAIアシスト（日本版）

**概要**:
出品時と商品検索時にAIが支援する機能。

#### a) 出品支援機能
- **機能**: 商品情報入力時にAIが改善提案
- **動作**: 「商品がより売れやすくなるよう改善提案」
- **内容**: タイトル・説明文・カテゴリの最適化提案
- **成果**: 出品者の時間削減、売上向上

#### b) 商品検索支援機能
- **方法**: AIアシストからの質問に回答
- **質問項目**: 予算、サイズ、色、商品状態など
- **結果**: 回答をもとにパーソナライズされた検索結果を提示
- **効果**: ユーザーの検索効率向上

### 2. ChatGPTプラグイン統合（2023年7月）

**概要**:
ChatGPT有料版（ChatGPT Plus）でメルカリ商品を検索可能に。

**利用方法**:
- ChatGPTにメルカリプラグインをインストール
- 自然言語で商品を検索（「予算5,000円以内の冬用コート」など）
- メルカリからおすすめ商品を自動提示

**特徴**:
- ChatGPT内で検索から購入まで完結可能
- 複雑なニーズを自然言語で表現可能
- 従来の検索UI不要

### 3. Merchat AI（US版メルカリ）

**概要**:
2023年8月、US版メルカリでChatGPT搭載型のAIアシスタント「Merchat AI」をベータ版で提供開始。

**特徴**:
- **チャットボット型**: 店員のように接客するAI
- **インタラクティブ**: 予算・サイズ・色などを対話で聞き取り
- **レコメンド**: 十分な情報を得たら最適商品を提示
- **多言語対応**: グローバル対応

**ユーザーメリット**:
- 自然な会話で買い物できる
- 検索UI不要で簡単
- パーソナライズされた提案

### 4. 生成AI専門チーム設立（2023年5月）

**概要**:
メルカリが生成AI・LLM専門チームを社内に設立。

**チームの役割**:
1. **商品説明文の自動生成**: 出品者が簡単に説明文を作成
2. **商品価格推定**: AIが出品商品の適切な価格を自動提案
3. **レコメンドエンジン開発**: パーソナライゼーション精度向上

### 5. OpenAI GPT-4o / GPT-4o miniの統合

**統合時期**: 2024年～

**統合内容**:
- AI Listing Support: 商品リスティングの自動最適化
- Mercari AI Assistant: アシスタント機能の高度化
- Price Optimization: 動的価格推定

**成果**:
- 出品から販売までの時間短縮
- 売上20～30%向上（オプティン者）
- カスタマーサポートコスト削減

## 技術スタック

### 使用技術
| 項目 | 技術 | 用途 |
|------|------|------|
| LLM | ChatGPT, GPT-4o, GPT-4o mini | メイン生成AI |
| Embedding | OpenAI Embeddings | ベクトル化・検索 |
| ベクトルDB | Pinecone, Weaviate | セマンティック検索 |
| APIプラットフォーム | OpenAI API | 統合基盤 |
| プラグイン | ChatGPT Plugins | 検索機能統合 |
| インフラ | GCP, AWS | クラウド運用 |

### レコメンドパイプライン
```
ユーザー行動データ → 埋め込み化
商品データ → チャンキング+埋め込み化
    ↓
ベクトル相似度計算
    ↓
LLMによるランキング・説明生成
    ↓
パーソナライズレコメンド表示
```
