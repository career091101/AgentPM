# Tier 1 生成AI導入事例：スクウェア・エニックス - ChatGPT ゲーム開発AI

## 基本情報

### 企業プロフィール
- **企業名**: 株式会社スクウェア・エニックス
- **業種**: ゲーム・デジタルエンタテインメント
- **事業規模**: 売上高2,000億円以上（2024年）
- **本社**: 東京都新宿区
- **代表作**: FF（ファイナルファンタジー）、DQ（ドラゴンクエスト）シリーズ

### AI導入概要
- **AI ツール**: ChatGPT（GPT-4）、ChatGPT-4o、Azure OpenAI Service
- **導入時期**: 2023年～継続中
- **対象部門**: ゲーム開発全般、AI & エンジン開発ディビジョン
- **導入規模**: 全社規模（100人以上が利用）

## 導入内容と実装

### 1. 独自社内チャットボット「ひすいちゃん」の開発

**概要**:
スクウェア・エニックス独自の社内チャットボット「ひすいちゃん」を開発。

**特徴**:
- **LLMベース**: ChatGPT（GPT-4o）をベースに開発
- **内製エンジン知識保有**: 社内エンジンの仕様・API・best practiceに精通
- **マルチユーザー対応**: 質問者以外も回答と質問を閲覧可能

**利用方法**:
- Slack/社内チャットツール統合
- 自然言語での質問に即座に回答
- コード例や仕様書への参照を自動提示

**利用状況**:
- **ユーザー数**: 100人以上
- **質問頻度**: 月数十～数百件
- **運用コスト**: Azure OpenAI Service 月額5万円程度
  - 周辺サービス含めても月額10万円以下

### 2. ChatGPT / Azure OpenAI Service統合開発環境

**CEDEC2024 発表内容**:

#### a) ChatGPTによるシェーダー・シーケンス自動生成
- **内容**: 内製エンジンのシェーダーやゲームシーケンスをChatGPTで生成
- **方法**: エンジン仕様書をプロンプトに含めてAPI呼び出し
- **効果**: 手作業で数日かかる処理を数時間に短縮

#### b) エンジン知識の学習・回答体制
- **方法**: Azure OpenAI Serviceにエンジン仕様書を学習させる
- **ツール**: Semantic Kernel等でAPI連携
- **効果**: 社員の質問に対して正確なエンジン解説を自動提供

#### c) グラフ構造データの自動生成
- **対象**: ゲーム内イベントシーケンス、スキルツリー等のグラフ構造
- **方法**: ChatGPT-4で構造定義をJSONで出力
- **精度**: 90%以上の成功率で実装可能なコードを生成

### 3. 社内向け多機能チャットボット統合

**利用シーン**:
1. **ゲームエンジン質問**: 「このAPIの使い方は？」→ ひすいちゃんが即座に説明
2. **仕様書参照**: 「イベントシステムの仕様は？」→ 関連ドキュメント自動提示
3. **コード生成支援**: 「このロジックのコード例を」→ ChatGPTが提案
4. **バグ分析**: エラーメッセージをコピペして質問→ 原因と対応策を提示

**セキュリティ対策**:
- **社内データ保護**: Azure OpenAI Service利用で企業データ保護
- **アクセス管理**: 適切なロール設定とAudit log管理
- **データ学習設定**: ChatGPTのモデル学習に社内データを使用させない

### 4. FF/DQシリーズへの適用

**FF XIV（オンラインゲーム）での活用**:
- イベントシナリオの自動生成支援
- NPCダイアログの複数パターン生成
- ゲームロジックのテスト項目書作成

**DQ（ドラゴンクエスト）シリーズ**:
- ダンジョン生成ロジックの最適化
- バトルシステムの複雑なルール検証
- テスト自動化プロンプトの生成

### 5. 2024年7月「スクウェア・エニックスのAI」書籍出版

**概要**:
同社AI開発陣による解説書『スクウェア・エニックスのAI』を出版。

**内容**:
- ゲーム内AI（キャラクターAI）の基礎
- エンジンAI（開発支援AI）の実装例
- 生成AIの活用事例

**出版社**: ボーンデジタル（2024年7月30日）

**影響度**:
- 業界内での知見共有
- 採用候補者への技術力アピール
- 学生向けの学習教材
