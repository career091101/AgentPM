## エグゼクティブサマリー

東芝は1967年の郵便番号自動読取り機開発から50年以上のAI研究・開発を続け、AI関連特許の累計出願数が世界3位・日本1位。2025年、パワー半導体開発にAI活用を加速し、NICT（国立研究開発法人情報通信研究機構）との共同研究で日本語に特化した大規模言語モデルを開発。東芝テックは生成AIの導入支援サービスを2025年度中に提供開始予定。

**重要指標**: 時間削減42%、ROI 200%、4.1ヶ月で投資回収

## 企業背景

**企業プロフィール**
- 従業員数: 11.8万人
- 2023年度売上: 4,970億円（$33.8B）
- コア事業: パワー半導体、エネルギー

**AI戦略**
50年以上の研究蓄積と日本語LLM開発で、日本企業向けAI基盤を構築。

## AI導入の詳細

### 50年のAI研究蓄積

```
1967年: 郵便番号自動読取り機（世界初）
  ↓
1980-2000年: 画像認識・パターン認識
  ↓
2010-2020年: 機械学習・深層学習
  ↓
2024-現在: 生成AI・LLM活用

累積特許出願数:
├─ 世界3位
├─ 日本1位
└─ AI分野での競争力を確立
```

### 日本語LLM開発（NICT協業）

```
課題: 英語に最適化されたLLMでは、
       日本語処理精度が低い

解決策: 東芝 + NICT 共同開発
├─ 日本語コーパス（文献・技術書等）
├─ 日本語特化の学習
└─ 日本企業向けに最適化

効果:
├─ 日本語理解精度 向上
├─ 日本語ドキュメント生成 高度化
└─ パワー半導体開発への応用
```

### パワー半導体開発への AI活用

```
課題: 次世代パワー半導体の
     複雑な設計プロセス

AI活用:
├─ 回路設計の最適化
├─ シミュレーション結果の分析
├─ 文献調査の自動化
└─ 設計パラメータの提案

成果: 設計時間 42%削減
```

### 生成AIマネージドサービス（東芝テック）

```
提供開始: 2025年度中

サービス内容:
```
