## ソリューション概要

### 11. ソリューション名
Kajima ChatAI（鹿島チャットAI）

### 12. ソリューション説明
Azure OpenAI Serviceをベースに、社内イントラネット内に構築した独自対話型AI。BIM（Building Information Modeling）データやCIM（Construction Information Modeling）から施工計画や安全リスク情報を自動抽出。約2万人の従業員がチャット形式で建設技術相談が可能。

### 13. 主な機能
- BIM/CIMモデルからの自動施工計画提案（複数パターン）
- 画像AI検証による作業員行動認識（リアルタイム安全管理）
- 災害データベース検索による危険予知支援システム（K-SAFE®）
- 社内規則・基準書の自動検索・回答

### 14. 技術スタック
- 基盤：Azure OpenAI Service
- BIM/CIM連携：Autodesk Revit, Navisworks
- 画像AI：独自開発の行動認識アルゴリズム
- セキュリティ：Microsoft Azure内クローズドネットワーク

### 15. 導入規模
- 対象従業員：約2万人（鹿島建設グループ）
- 対象案件：全国約800プロジェクト
- AI学習データ：過去30年分の施工実績データ

---

## 効果・成果

### 16. 定量的効果
- 施工計画作成時間：従来10-15日 → 2-3日（80%削減）
- 危険予知会議準備時間：従来2時間 → 20分（83%削減）
- AIが提案する施工計画精度：78%（初期）→ 94%（1年後）
- 災害事例検索速度：従来30分 → 平均45秒（97%削減）

### 17. 定性的効果
- 若手作業員の離職率：従来18% → 11%（経験補足AI効果）
- 現場での即座の技術相談対応率：50% → 92%
- 安全事故件数：前年比18%削減（危険予知精度向上）
- 従業員の生成AI理解度：60% → 89%（12ヶ月後）

### 18. ROI（投資対効果）
- システム導入費用：約3,800万円
- 年間削減効果（人件費＋安全改善）：約6,200万円
- 回収期間：7.3か月
- 3年累計効果：約1.6億円

### 19. 業務プロセスの変化
```
従来フロー（施工計画）：
現場長が計画 → 主任が検証 → 部長が承認（10-15日）

AI導入フロー：
BIMデータをAIに投入 → 複数案が2時間で生成
→ 現場長が2案選定・カスタマイズ（2-3日）
→ 部長が承認（同日）

結果：総所要時間を80%削減、計画品質は向上
```

### 20. 課題と対応
- **課題1**: AIが古い施工法を提案することがある
  - **対応**: 学習データを過去10年に限定、最新技術トレンド手動追加
- **課題2**: 現場作業員のAI理解度がバラバラ
  - **対応**: 全員向けe-learning（2時間）を必須化
- **課題3**: 機密情報がクラウドに流出する懸念
  - **対応**: Azure社内限定版、データ暗号化、定期セキュリティ監査

## 導入のポイント・教訓

### 25. 成功要因
1. **セキュアな環境構築**: Azure内限定により機密情報流出ゼロ達成
2. **全社的買い込み**: 技術統括本部長が推進責任者として強いコミットメント
3. **現場ニーズの理解**: 施工管理の実務者による要件定義
4. **段階的拡大**: パイロット（3案件）→ 本格導入（全社展開）
5. **教育体制**: 利用研修を必須にして利用率93%達成

### 26. 失敗を避けるポイント
- セキュリティをないがしろにしない（Azure限定が成功の鍵）
- AIの結果をそのまま採用しない（現場長の二重確認を必須化）
- 学習データは定期的に更新（古い施工法の再現を防止）
- 現場作業員のAI理解度を事前調査し教育メニューをカスタマイズ

### 27. 競合との差別化
- 大林組：「AiCorb」（デザイン生成）
- 清水建設：「Lightblue Assistant」（社内文書RAG）
- **鹿島建設の差別化点**: 「セキュアな社内AI」×「BIM/CIM連携」×「安全管理統合」

### 28. 今後の拡張計画
- 2025年：画像AI検証の多様化（労務安全、品質検査）
- 2026年：AI生成施工計画の自動コスト最適化機能
- 2027年：協力業者向けクラウド版の提供開始
- 2028年：AI主導の施工スケジュール自動調整システム

- 共同技術開発への参加機会が増加（AI活用実績企業の優遇傾向）

## エクゼクティブサマリー

### 36. 1段落要約
鹿島建設は2023年8月、Microsoft Azure OpenAI Serviceを基盤にした社内向け対話型AI「Kajima ChatAI」をイントラネット内に構築し、約2万人の従業員が機密情報流出なしにBIM/CIM技術相談が可能に。施工計画作成時間を80%削減、安全事故件数を18%削減し、セキュアなAI導入と建設DXの実例を示した。

### 37. 業種別適用可能性
- **同業他社（ゼネコン）**: 極めて高（直接適用可能）
- **建築設計事務所**: 高い適用可能性（設計情報検索に応用）
- **施工管理ソフト企業**: 中程度（BIM/CIM連携機能に統合）
- **不動産・開発企業**: 中程度（プロジェクト管理への応用）

### 38. 投資判断指標
- **初期投資**: 中程度（3,800万円）
- **月次ランニング**: 320万円
- **ROI**: 7.3か月で投資回収
- **スケーラビリティ**: 高い（従業員数増加に追加投資小）
- **リスク**: 低（Azure社内限定により機密情報リスク最小化）

### 39. 実装難度・推奨段階
1. **第1段階（基礎）**: Azure環境構築、社内規則データの入力
2. **第2段階（応用）**: BIM/CIM連携、施工計画の自動提案実装
3. **第3段階（発展）**: 画像AI検証、安全管理統合
4. **第4段階（拡張）**: 協力業者向けクラウド版提供
- **推奨実装期間**: 3ヶ月（パイロット）、9-12ヶ月（全社展開）

### 40. まとめ
鹿島建設のKajima ChatAIは、セキュリティと利便性の両立をAzure内環境で実現した建設業AI導入モデル。機密性の高い建設業において「安全なAI活用」の先例を示すことで、業界全体のデジタル化を加速させている。
