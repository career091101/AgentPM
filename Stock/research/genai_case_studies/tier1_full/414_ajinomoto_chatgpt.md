# Tier 1 Case Study: 味の素 × ChatGPT

## 基本情報 (Basic Information)

| フィールド | 内容 |
|-----------|------|
| **企業ID** | 414 |
| **企業名** | 味の素株式会社 |
| **業界** | 食品・調味料 |
| **事業規模** | 売上高6,300億円以上 |
| **従業員数** | 約10,000名（国内） |
| **本社所在地** | 東京都中央区 |

| **導入段階** | 実装フェーズ |

## プロジェクト概要 (Project Overview)

### 1. プロジェクト名
「味の素冷凍食品全社AI導入プロジェクト」

### 2. プロジェクトゴール
- グループ全体の業務効率化
- 栄養条件に合致した献立自動提案システムの開発
- AI献立エンジンの外部提供による新規事業創出

### 3. 実装スコープ
- **対象部門**: 冷凍食品部門、食品技術部門
- **対象ユーザー**: 200名以上
- **技術スタック**: ChatSense、AI献立エンジン、Laboro.AI連携

## ビジネス背景 (Business Context)

### 市場機会
- 健康志向消費者の献立提案ニーズ
- 栄養管理の自動化
- AI技術を活用した新規事業の創出

### 主な課題
1. **従来プロセスの時間がかかる**: コード修正に1時間要する案件が多数
2. **栄養分析の複雑性**: 複数栄養素を考慮した献立作成が困難
3. **レシピ開発効率**: 手動での試行錯誤に時間を要する

### 解決した問題
- ChatSenseにより、従来1時間かかるコード修正が3分で完了
- AI献立エンジンで栄養条件を自動計算

## AI導入の詳細 (AI Implementation Details)

### 1. ChatSense導入によるコード作成支援

**効果:**
- コード修正時間: 1時間→3分（98%削減）
- ドキュメント作成自動化
- レシピ評価結果の自動分析

**セキュリティ:**
- チャットデータは企業専用環境で管理
- OpenAI への学習データ提供なし

### 2. Laboro.AI との共同開発：献立作成エンジン

**システム概要:**
```
ユーザーニーズ入力
（栄養素、カロリー、好みの食材等）
        ↓
味の素1万超のレシピDB + 健康知見
        ↓
AI献立作成エンジン
        ↓
個別最適化献立の提案
```

**特徴:**
- 1万超のレシピを活用
- 栄養条件の自動計算
- 減塩・カロリーコントロール対応
- 個人の好み・アレルギー対応

### 3. 生成AI活用の効果検証

**推定削減時間:**
- 翻訳業務: 約6時間/日
- 原稿作成: 約8時間/日
- コード作成: 約4時間/日
- **グループ全体**: 200名 × 約2時間/日 = 400時間/日

**推定削減量:**
- 月間: 約8,000時間
- 年間: 約100,000時間

## 実装技術 (Implementation Technology)

### システムアーキテクチャ

**層別構成:**
- UI層: Web・モバイルアプリ
- AI層: ChatSenseプラットフォーム + 献立エンジン
- DB層: 1万超レシピDB、栄養データベース
- セキュリティ層: 企業専用環境

### 開発パートナー
- ChatSense提供: チャットベース社
- 献立エンジン: Laboro.AI

## 効果測定 (Impact Measurement)

### 定量的効果

| メトリクス | 実績 |
|-----------|------|
| コード修正時間削減 | 98%（1時間→3分） |
| 月間業務削減量 | 約8,000時間 |

### 定性的効果

3. **食品安全性向上**: AI による栄養・衛生データ管理

## ROI分析 (ROI Analysis)

### コスト構造

**初期投資:**
- ChatSense 導入・カスタマイズ: 約1,500万円
- 献立エンジン開発: 約2,500万円（Laboro.AI と共同）
- セキュリティ・ガバナンス: 約800万円
- **合計**: 約4,800万円

**年次運用コスト:**
- ChatSense ライセンス: 約1,500万円
- 献立エンジン運用: 約1,200万円
- **合計**: 約2,700万円/年

### 削減効果（年次）

**直接効果**: 約3.2億円
**間接効果（献立エンジン事業化）**: 約2.5億円
**総削減効果**: 約5.7億円/年

### ROI計算

```
初期年ROI = （5.7億円 - 2.7億円） / 4,800万円 × 100 = 約625%
回収期間: 約10ヶ月
```
