# Infosys - Topazプラットフォームで2,000超AI プロジェクト実行と AWS統合

## 1. エグゼクティブサマリー

Infosys は、2023年に「**Infosys Topaz**」を発表し、**12,000以上のAI資産と150以上の事前学習モデル**を備えたエンタープライズGenAI統合プラットフォームを構築。AWS（Amazon Q）、Google Cloud、Anthropic Claude と複数LLMを統合し、**2,000以上のAIプロジェクトを顧客向けに実行**、顧客満足度 92% を達成している。

特に AWS との戦略的パートナーシップにより、Infosys Topaz + Amazon Q Developer を統合し、ソフトウェア開発ライフサイクル（SDLC）全体での AI活用が加速。コード生成、デバッグ、レガシーコード現代化など幅広い用途で活用され、顧客の開発効率が 25-35% 向上しているという実績を記録している。

## 3. AI導入スキーム

### 3.1 Infosys Topaz プラットフォーム

**主要特徴**

| 機能 | 説明 |
|------|------|
| **AI資産ライブラリ** | 12,000+ のAI資産、150+ 事前学習モデル |
| **マルチLLM統合** | Claude、GPT-4、Gemini、オープンソース統合 |
| **業界別テンプレート** | 金融・製造・医療等の業界特化ソリューション |
| **Responsible AI** | 説明可能性、バイアス検知、監査ログ |

### 3.2 AWS 統合（Amazon Q）

Infosys Topaz を Amazon Q Developer と統合することで、以下の機能を提供：

- **コード生成**: 自然言語 → コード自動生成（精度 85%+）
- **デバッグアシスタント**: エラー箇所の自動特定・修正提案
- **レガシーコード現代化**: COBOL → Java/Python への自動変換（60%の精度）
- **ドキュメント自動生成**: コード → API ドキュメント自動作成
