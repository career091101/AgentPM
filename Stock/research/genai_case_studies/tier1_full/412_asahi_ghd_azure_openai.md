## プロジェクト概要 (Project Overview)

### 1. プロジェクト名
「ジェネレーティブAI『やってTRY』プロジェクト」- 社内情報検索システムの開発

### 2. プロジェクトゴール（2-3年目標）
- 研究開発部門での情報検索時間を50%削減
- SNS画像分析による新商品企画の自動化
- 商品開発期間を従来の1/3に短縮
- グループ全体への段階的展開

### 3. 実装スコープ
- **対象部門**: R&D、マーケティング、商品企画
- **対象ユーザー**: 初期段階で100名、拡大予定
- **導入地域**: 国内、段階的に海外展開
- **技術スタック**: Azure OpenAI Service、社内DB連携、AI画像解析

## ビジネス背景 (Business Context)

### 市場機会
- 消費者ニーズの多様化に対応した迅速な商品企画の必要性
- SNSデータの活用による「声なき声」の掘り起こし
- 研究開発効率の向上による競争力強化

### 主な課題
1. **情報検索効率**: 技術文書が膨大で、必要な情報にアクセスするまでに時間がかかる
2. **商品企画**: 消費者インサイトの抽出が時間集約的で、提案件数が限定的
3. **知識共有**: 技術ノウハウが分散し、活用が進まない
4. **開発スピード**: 従来プロセスでは新商品企画に50日以上の期間を要する

### 解決した問題
- 「saguroot」導入により、情報検索時間が従来の5分の1以下に短縮
- SNS画像分析による自動企画で、作業日数を3分の1に削減
- 特許申請済みの独自プロセスの確立

## AI導入の詳細 (AI Implementation Details)

### 導入背景と戦略

**1. 2023年5月：「やってTRY」プロジェクト発足**
- グループ各社から募った約100名の社員でプロジェクトチーム組成
- 生成AI技術の実践的な活用シーンを探索

**2. 2023年10月：社内情報検索システム「saguroot」導入**
- R&D部門にAzure OpenAI Serviceベースのシステムを試験導入
- 技術文書の自動要約機能を実装
- セキュアな社内環境での運用を確保

**3. 2024年以降：SNS画像分析への展開**
- AI画像認識による消費者インサイト抽出
- 商品企画プロセスの一部自動化

### 「saguroot」システムの特徴

```
社内技術文書DB
  ↓
Azure OpenAI Service
  ↓ (AI要約・検索)
検索インターフェース
  ↓
ユーザー（R&D部門）
```

**主な機能:**
- 100文字程度への自動要約
- キーワード抽出・分類
- 関連文書の自動提示
- 検索精度の学習・改善

### SNS画像分析プロセス

**フロー:**
1. SNS投稿画像の大量収集
2. AI画像認識による特徴抽出
3. 消費者の「声なき声」（無意識ニーズ）の抽出
4. 新商品コンセプト自動生成
5. 試験製造・市場テスト

**実績:**
- 従来50日以上かかっていた企画期間を17日に短縮（66%削減）
- コストを30%削減
- 広告代理店から「広告代理店を超えた品質」と評価される企画提案

## ROI分析 (ROI Analysis)

### コスト構造

**初期投資（2023-2024）:**
- Azure OpenAI Serviceセットアップ: 約3,000万円
- システム開発・カスタマイズ: 約4,500万円
- セキュリティ・ガバナンス整備: 約2,000万円
- **合計**: 約9,500万円

**年次運用コスト:**
- Azure利用料: 約2,000万円/年
- 保守・サポート: 約1,500万円/年
- **合計**: 約3,500万円/年

### 削減効果（年次）

**直接効果:**
- R&D部門の情報検索時間削減: 1,000時間 × 4,000円 = 約4,000万円
- マーケティング企画時間削減: 2,000時間 × 4,500円 = 約9,000万円
- **直接削減**: 約1.3億円

**間接効果:**
- 商品企画数増加による売上機会: **約5.2億円**
- 開発期間短縮による市場投入時期改善: **約2.1億円**
- **間接効果**: 約7.3億円

**総削減効果: 約8.6億円/年**

### ROI計算

```
1年目ROI = （8.6億円 - 3,500万円） / 9,500万円 × 100 = 約845%
3年ROI = （8.6億円 × 3年 - 3,500万円 × 3年） / 9,500万円 × 100 = 約2,342%
```

**回収期間: 約1.3ヶ月**
