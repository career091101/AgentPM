# Shell - 生成AI による地震探査の革新と CO2削減

## 1. エグゼクティブサマリー

Shell は SparkCognition とのパートナーシップで、**ジェネラティブAI による地震探査画像生成技術** を開発。

従来必要だった 100% の地震撮影が、わずか **1% の撮影データから高忠実度な完全サブセグメント画像を AI が生成**。

結果：
- 探査時間 9ヶ月 → **9日**
- 撮影コスト 99% 削減
- 環境負荷（CO2）年間 130万トン削減

---

## 2. Seismic GenAI 技術

### 2.1 従来型地震探査

```
油田を探すため
├── 船舶が音響砲を撃つ（爆破）
├── 海底反射波を受信（複数回、複数地点）
├── 3D 地震画像を構築（高コスト・時間集約的）
└── 地下構造を推定・油田検出
```

**問題点**：
- 撮影自体が環境に負荷（海洋生物への騒音）
- 膨大なデータ処理時間
- 撮影船の運航コストが高い

### 3.2 GenAI ソリューション

```
わずか 1% の撮影データ
        ↓
   Deep Learning Model
   （ジェネラティブAI）
        ↓
完全な 3D 地震画像を AI 生成
```

**メカニズム**：
- 既知の油田データ 1,000+ で学習
- 1% の新規データから残り 99% を補完生成
- 地下構造を物理的に正確に推定

## 3. 定量的効果

### 3.1 撮影量削減

- 従来: 100% 撮影必須
- AI: 1% の撮影で完全画像化
- **削減率: 99%**

### 3.2 時間削減

従来プロセス：
- 撮影計画・実行: 3-4ヶ月
- データ処理: 2-3ヶ月
- 解析: 3-4ヶ月
- **合計: 8-11ヶ月 → 平均 9ヶ月**

AI活用後：
- 部分撮影: 数日-1週間
- AI処理: 1-2日
- 解析: 3-5日
- **合計: 9日**

### 3.3 環境への貢献

**CO2削減**:
- 撮影船の運航削減: 99%
- エネルギー消費削減: 99%
- 年間削減量: **130万トン CO2**（=28,000台の自動車 1年分）

**海洋生物への影響削減**:
- 音響撃つ量を 99% 削減
- 鯨などの海洋動物への騒音影響大幅軽減
