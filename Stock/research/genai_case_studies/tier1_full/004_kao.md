# 花王 - SAPPHIRE生成AI導入による経費精算業務の自動化

## 1. エグゼクティブサマリー

花王株式会社は、AI経費精算ソリューション「SAPPHIRE」（Miletos株式会社開発）を導入することで、経費精算業務における**年5.5万時間の業務削減、約1.5億円のコスト削減**を実現した。初期段階の花王ビューティブランズカウンセリング（約6,000名の美容部員）での成功を経て、2023年6月より国内全グループ企業（3万人超）への展開を完了し、大規模生成AIプロジェクトの実装モデルとして注目されている。

## 2. 企業・プロジェクト背景

### 2.1 企業概要

| 項目 | 内容 |
|------|------|
| 企業名 | 花王株式会社（Kao Corporation） |
| 業種 | 化学・化粧品・生活用品製造販売 |
| 創業年 | 1887年 |
| 主要事業 | ビューティケア、ヘルスケア、ファブリックケア |
| 導入時従業員数 | 国内グループ全体で3万人超 |

### 2.2 プロジェクト背景と課題

**初期段階（花王ビューティブランズカウンセリング、2020年）**

- **主な課題**:
  - 花王ビューティブランズカウンセリングの美容部員約6,000名は、百貨店やスーパーなどの化粧品売り場に分散勤務
  - 通勤費・交通費などの経費精算が複雑で、手作業による確認・承認業務が膨大
  - 月末月初の経理部門への業務負荷が集中
  - 精算エラーや不正の可能性を最小化する必要性

- **定量的な課題**:
  - 年間約5.5万時間を経費精算関連業務に費やしていた
  - 経理チェックプロセスが現地支払いと本部確認で二重化
  - 通勤ルート確認、交通費計算の属人化

**全社展開フェーズ（2022-2023年）**

- グループ企業全体への拡大により、さらに大規模な業務効率化が必要に

## 3. 導入スキーム

### 3.1 SAPPHIREの概要

**SAPPHIRE**は、Miletos株式会社が開発した経費領域専門のAI生成ソリューション。従来、人間が行っていた経費精算・チェック業務をAIに置き換える。

**主要機能**:

1. **AIによる経費チェック機能**
   - 申請内容の自動検証
   - ルール違反の自動検出
   - 不正申請の事前防止

2. **AIによる通勤費・交通費自動精算機能**
   - 住所・勤務地からの最短ルート自動計算
   - 交通費自動算出
   - 複数経路の比較・最適化

3. **チャットボット機能**
   - 申請者からの問い合わせ対応自動化
   - FAQ自動応答

### 3.2 導入フェーズ

**フェーズ1: 小規模パイロット実装（2020年10月-2021年3月）**

- 対象: 花王ビューティブランズカウンセリング（約6,000名）
- 開発期間: 6ヶ月
- アプローチ: ビジネス課題（通勤費精算）に特化したAI実装

**フェーズ2: 本番運用と効果検証（2021年4月-2023年5月）**

- 運用ユーザー数: 約6,000名
- 運用期間: 約2年間の本番実績を蓄積
- 効果: 確定的な削減数値を実証

**フェーズ3: 全社グループ展開（2023年6月開始）**

- 対象: 国内全グループ企業
- ユーザー数: 3万人超
- アプローチ: パイロット成功モデルの水平展開

## 4. 実装技術とAI活用

### 4.1 AI技術スタック

| 技術要素 | 説明 |
|---------|------|
| **機械学習モデル** | 交通費算出、ルール判定向けの分類・回帰モデル |
| **自然言語処理（NLP）** | チャットボット実装、ユーザー質問の意図解析 |
| **ルール・エンジン** | 会社規定に基づいた自動チェックロジック |
| **OCR/画像認識** | 領収書・交通系ICカード画像の読み込み（オプション） |

### 4.2 AI導入のポイント

- **領域特化型アプローチ**: 経費処理という限定的で規則性の高い領域に特化することで、精度を高める
- **既存システム連携**: 花王の既存の経費管理システム・給与システムとの連携
- **段階的な導入**: パイロット→検証→全社展開という段階的なアプローチにより、リスク最小化
- **人間とAIの役割分担**: AI自動判定 + 人間による例外処理という二段構え

## 結論

花王のSAPPHIRE導入事例は、**生成AI・機械学習を用いた業務自動化における国内有数の成功事例**として位置づけられる。年5.5万時間、約1.5億円の削減効果を達成し、パイロット段階での成功を全社グループ展開（3万人超）へと拡大したことは、以下の点で示唆的である:

1. **適切な領域選定**: 規則性・反復性の高い経費処理領域への特化が成功の前提条件
2. **段階的導入の有効性**: 小規模検証→本番運用→全社展開というフェーズ分けが組織的リスクを最小化
3. **定量的な効果の可視化**: 時間・金銭の削減を具体的に示すことで、経営層・従業員の支援を獲得
4. **長期的なビジネス価値**: 初期投資を短期間でペイバックし、継続的な改善を実現

今後、グローバル展開や生成AIの高度化により、さらなる効果拡大が期待される。
