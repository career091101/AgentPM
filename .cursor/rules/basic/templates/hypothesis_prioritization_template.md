## 優先順位付けマトリクス

```
          高影響度
              ↑
   ┌──────────┼──────────┐
   │  検証必須  │  最優先   │
   │ (要注意)  │ (今すぐ)  │
   ├──────────┼──────────┤
   │  低優先度  │  確認のみ  │
   │ (保留)   │ (簡易検証) │
   └──────────┼──────────┘
              ↓
          低影響度
   
   低確信度 ←──────→ 高確信度
```

### 優先度判定基準

| 優先度 | 条件 | アクション |
|-------|-----|----------|
| **最優先** | 高影響度 × 低確信度 | 今すぐ検証を開始 |
| **検証必須** | 高影響度 × 中確信度 | 計画的に検証 |
| **確認のみ** | 高影響度 × 高確信度 | 簡易な確認で十分 |
| **低優先度** | 低影響度 | 後回しまたは保留 |
